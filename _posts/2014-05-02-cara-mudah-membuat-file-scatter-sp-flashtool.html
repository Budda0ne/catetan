Contoh File Scatter : Scatter MTK 6572

File scatter adalah file berupa file teks (.txt)yang menyimpan data block partisi pada cpu chipset medidiatek. Selain berfungsi sebagai acuan SP Flashtool saat memproses file yang akan diproses agar sesuai dengan letak/lokasi yang tepat, file ini juga biasa digunakan untuk melakukan read back ROM dari SP Flashtool utuk medapatkan stock firmware mentah yang belum tersentuh (perawan :D).

Anda bisa membuat file scatter sendiri menggunakan MTK Droid Tools.

Membuat File Scatter dengan MTK Droid Tools


Sebelum membuat file scatter untuk SP Flashtool, anda harus sudah:

1. Meng-instal driver ADB Debugging.
2. Meng-install MTK Droid Tools (usakan versi yang baru).
3. Kabel data yang masih bekerja dengan baik.

Setelah anda siap, anda bisa mengikuti langkah-langkah berikut:

1. Buka Software MTK Droid Tools
2. Aktifkan terlebih dahulu USB Debugging di perangkat Android. Buka menu Pengaturan, pilih Opsi Pengembang dan cek/centang pada USB Debugging.
3. Tacapkan perangkat ke komputer dengan kabel data. Lebih disarankan lewat panel belakang, agar aliran data stabil.
4. Tunggu MTK Droid Tools mendeteksi perangkat android anda.
5. Setelah terkoneksi, pilih menu Blocks maps
6. Pada jendela yang muncul, klik create scatter file dan tentukan tempat untuk menyimpan file scatter emmc yang sudah dibuat.

7. Tutup dengan klik return kemudian close program.

Sekarang, anda sudah mempunyai file scatter khusus untuk perangkat android anda sendiri. Dari file scatter yang sudah anda buat, anda bisa melakukan readback menggunakan SP Flashtool untuk men-dump-ing stock ROM yang masih perawan ting-ting.
