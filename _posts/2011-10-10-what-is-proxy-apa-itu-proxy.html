Dalam jaringan komputer, server proxy adalah server (sistem komputer atau program aplikasi) yang bertindak sebagai perantara untuk permintaan dari klien mencari sumber daya dari server lain. Klien terhubung ke server proxy, meminta beberapa layanan, seperti file, koneksi, halaman web, atau sumber daya lainnya, tersedia dari server yang berbeda. Server proxy mengevaluasi permintaan tersebut sesuai dengan aturan penyaringan. Sebagai contoh, mungkin filter lalu lintas berdasarkan alamat IP atau protokol. Jika permintaan divalidasi oleh filter, proxy menyediakan sumber daya dengan menghubungkan ke server yang relevan dan meminta layanan atas nama klien. Sebuah server proxy opsional dapat mengubah permintaan klien atau respon server, dan kadang-kadang dapat melayani permintaan tanpa menghubungi server tertentu. Dalam hal ini, tanggapan 'cache' dari server jauh, dan mengembalikan permintaan berikutnya untuk konten yang sama secara langsung.

Dengan kata lain, server proxy adalah jenis buffer antara komputer Anda dan sumber daya Internet Anda mengakses. Mereka mengumpulkan dan menyimpan file yang paling sering diminta oleh ribuan pengguna internet di database khusus, yang disebut "cache". Oleh karena itu, proxy server dapat meningkatkan kecepatan koneksi Anda ke Internet. Cache proxy server mungkin telah berisi informasi yang Anda butuhkan pada saat permintaan Anda, sehingga memungkinkan proxy untuk mengirimkannya segera. Peningkatan keseluruhan dalam kinerja mungkin akan sangat tinggi. Juga, proxy server dapat membantu dalam kasus-kasus ketika beberapa pemilik sumber daya Internet menerapkan beberapa pembatasan pada pengguna dari negara tertentu atau wilayah geografis. Selain itu, antara server proxy ada yang disebut anonymous proxy server yang menyembunyikan alamat IP Anda dengan demikian menyelamatkan Anda dari kerentanan yang bersangkutan dengan itu.

Ada 4 macam proxy:

  • Transparent Proxy
    • - Jenis server proxy mengidentifikasi diri sebagai proxy server dan juga membuat alamat IP asli tersedia melalui http. Biasanya ini digunakan untuk kemampuan mereka untuk cache situs Web dan tidak efektif memberikan anonimitas orang-orang yang menggunakannya. Namun, penggunaan transparent proxy akan membuat Anda sekitar larangan IP sederhana. Mereka transparan dalam hal alamat IP yang terkena, tidak transparan dalam hal yang tidak Anda tahu bahwa Anda menggunakannya (sistem anda tidak dikonfigurasi secara khusus untuk menggunakannya).
  • Anonymous Proxy
    • - Jenis server proxy indentifies dirinya sebagai server proxy, tetapi tidak membuat alamat IP asli tersedia. Jenis server proxy detectable, tetapi memberikan anonimitas wajar bagi sebagian besar pengguna.
  • Distorting Proxy
    • - Jenis server proxy mengidentifikasi dirinya sebagai server proxy, tetapi membuat alamat IP yang tidak benar asli tersedia melalui http.
  • Anonimitas tinggi Proxy
    • - Jenis server proxy tidak mengidentifikasi dirinya sebagai server proxy dan tidak menyediakan alamat IP asli.
