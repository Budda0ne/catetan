Nubia OS 2.6.0 for Lenovo A6000
Custom ROM Nubia versi 2.6.0 ini sebenarnya untuk Lenovo K30, namun ternyata cocok juga dipakai untuk ponsel Lenovo A6000. Kesan pertama saat mencoba ROM yang dishare developer bbs.vibeui.com ini adalah serasa memakai ROM Color OS dengan menu-menu yang mirip dengan ROM MIUI. Sedangkan kalau dibandingkan dengan Stock ROM Lenovo A6000, tidak begitu banyak perbedaan.

Hampir sama dengan OS buatan developer china yang lain (Color OS, MIUI, atau FIUI dan lainnya) Nubia 2.6.0 juga tidak mendukung layanan google. Artinya, anda tidak akan bisa menggunakan aplikasi-aplikasi google kecuali anda meng-install GAPPS. Tentu saja instalasinya harus menggunakan custom recovery.

Nubia OS v.2.6.0 berjalan sempurna kecuali bug dibagian Video. Sementara fitur yang lain masih bisa berfungsi normal. Dari sisi penerimaan jaringan juga lancar, tidak ada kendala berarti. Bahkan ketika diuji dengan kartu BOLT, bisa langsung menangkap sinyal 4G.

Jika anda ingin mencoba ROM yang tidak dipakai oleh mainstream, Nubia OS ini recomended sekali untuk anda install di ponsel Lenovo A6000 anda. Selain beberapa hal diatas yang sudah saya bahas, dari segi tampilan user interfacenya juga patut dipertimbangkan. Lihat screenshoot berikut:

Instalasi Custom ROM Nubia OS Lenovo A6000 (Kraft)

Sebelum melaukan instalasi, pastikan bahawa anda sudah mempunyai/memasang custom recovery di Lenovo A6000 anda. Jangan menggunakan recovery bawaan untuk meng-install Custom ROM. Jangan lupa juga lakukan Full Back-up menggunakan custom recovery, dan pindahkan file back-up ke tempat lain (PC/Laptop atau media penyimpanan lain) untuk mencegah hal-halyang tidak di-inginkan.
Peringatan: Dengan mengikuti tutorial install Nubia OS Lenovo A6000, berarti anda sudah setuju dengan disclaimer KnoAcc Blog dan Jika ingin menyebarluaskan ulang artikel ini silahkan baca dan patuhi artikel DMCA.
Langkah-langkah instalasi:
  1. Letakkan file ROM di sdcard (external/microsd)
  2. Letakkan juga file zip GAPPS dan zip SuperSU di sdcard
  3. Mematikan ponsel terlebih dahulu kemudian tekan tombol Volume Up bersamaan dengan tombol Power untuk masuk recovery mode
  4. Lakukan Back-up jika belum melakukan back-up
  5. Lakukan Wipe System, Data, dan Cache (atau opsi Wipe to Install new ROM)
  6. Pilih menu Install from SDcard dan arahkan ke lokasi file ROM Nubia dan pilih install, tunggu hingga proses komplit.
  7. Kembali ke menu utama dan pilih reboot recovery
  8. Pilih lagi install from sdcard dan install supersu zip
  9. Kembali ke menu utama dan pilih Reboot system.
  10. Selesai.
Biasanya, Untuk booting awal setelah instalasi akan memerlukan waktu yang agak lama. Ini karena system memebuat dalvik cache dari aplikasi bawaan dan system juga menyelesaikan konfigurasi awal system android.

Setelah Lenovo A6000 ter install Nubia OS v.2.6.0 sempurna dan berjalan normal, anda bisa melakukan instalasi GAPPS karena saat ini agar bisa meng-install aplikasi dari playstore dan menikmati aplikasi-aplikasi dari google. Caranya hampir sama dengan langkah di atas. Masuk ke recovery mode kemudian install zip from sdcard dan pilih file zip GAPPS tanp wipe/format.
