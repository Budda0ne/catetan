Mi Flash Tool di atas saat ini merupakan versi terbaru 20160830. Bagi yang sudah pernah menginstall Mi Flash Tool versi apapun silahkan anda tidak perlu download lagi Mi Flash Tool di atas karena Mi Flash versi lama juga bisa digunakan untuk flashing. Bedanya Mi Flash yang terbaru di atas bisa dipakai untuk Windows 64bit maupun Windows 32bit, sedangkan Mi Flash yang keluaran 2015 atau lebih lama hanya bisa dipakai untuk pengguna Windows 64bit. Tapi rekomendasi saya adalah gunakan Windows 64 bit karena banyak yang gagal pakai Windows 32bit.

Buka Mi Flash Tool, kemudian pilih lokasi ROM dengan mengklik tombol Select, kemudian pilih folder ROM armani_images_xxx, jangan pilih folder images, kemudian klik OK

Pada bagian kanan bawah Mi Flash Tool klik pada menu dropdown, di situ ada beberapa pilihan tipe flash. Flash_all jika ingin clean install menghapus semua data juga internal memory, flash_all_except_data_storage jika ingin install tapi tidak menghapus data di internal memory, dan flash_all_lock jika ingin clean install kemudian lock bootloader lagi. Silahkan pilih terserah anda, yang penting itu tadi jangan lupa backup data dan aplikasi.

Setelah Mi Flash Tool siap kini beralih ke ponsel Redmi 1S . Matikan ponsel kemudian masuk ke fastboot mode dengan cara, dalam keadaan mati tekan dan tahan tombol volume bawah + tombol power, sampai masuk ke fastboot mode baru lepaskan kedua tombol.

Dalam mode fastboot ini hubungkan Redmi 1S ke PC/ komputer

Pada Mi Flash Tool klik tombol Refresh untuk mendeteksi ponsel Redmi 1S yang telah terhubung

Tunggu sampai proses flashing selesai, biasanya tidak sampai 5 menit. Jika proses flashing ROM selesai maka ponsel Xiaomi Redmi 1S anda akan melakukan restart sendiri

Tunggu sampai ponsel restart dan masuk ke sistem, biasanya proses booting setelah flashing ROM ini agak lama daripada proses booting/ restart biasa
