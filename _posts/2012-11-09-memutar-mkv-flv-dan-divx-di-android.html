Untuk memutar file MKV, FLV dan DivX di smartphone Android, tetap membutuhkan codec untuk memainkan file video yang tidak di support oleh player default semacam MKV atau FLV. Bahkan untuk memutar file MKV dan FLV di windows sendiri pun membutuhkan codec tertentu. Lalu bagaimana jika ingin memutar file multimedia seperti MKV atau file lain yang seharusnya tidak bisa diputar di android, bisakah? bisa dengan cara menambahkan codec yang mendukung file MKV, FLV dan DivX.

Diantaranya, codec yang mendukung file MKV, FLV dan DivX adalah MX Player Codec. Yang biasanya digunakan bersamaan dengan MX Video Player. Versi terbaru dari MX Player Codec saat saya menulis artikel ini adalah MX Player Codec 1.7.7.
