Namun beberapa file video seperti MKV, FLV (video YouTube), 3GP (dari handphone) tidak bisa diputar tanpa codec tambahan.

Atau kamu mendapati file MP4 (H.264) yang ketika dimainkan yang keluar hanya suaranya saja sementara gambarnya tidak muncul (serta extensi file seperti rmvb, xvid, ogg, webm atau file audio FLAC, ogg, mod, alac yg tidak bisa diputar --hadeewh). Cara memutar file video tersebut di Windows agar dapat ditonton -salah satunya- adalah dengan menginstall codec video bernama Shark007.

Ada juga alternatif lain seperti K-Lite Codec Pack (lebih sering saya pakai untuk klien-klien saya), tapi saya pribadi lebih suka menggunakan applikasi Shark007 karena kemudahan dalam proses installasinya, karena tetap menggunakan media player bawaan windows sebagai video player default.

Cara memutar file FLV – MKV – RMVB – 3GP dan format video / audio lain:

Pertama : Download Shark007 dari link di bawah yang saya dapat dari [MajorGeeks] dan pilih sesuai OS:

  • XP / Vista (23.61MB)
  • Windows 7 (29.91MB)
  • Windows 8. (17.26MB)

Kedua : Install applikasi yang baru saja kamu download. kemudian buka Setting aplikasi dari Start Menu -> Shark007 Codecs > Setting Application

Ketiga : Klik / centang kotak dengan tulisan “Shark007 SUGGESTED Settings” supaya applikasi men-setting otomatis codec video yang digunakan (direkomendasikan).

Cara memutar file mkv flv 3gp video

Keempat : Kamu juga bisa merubah setting codec yang digunakan secara manual untuk masing-masing tipe file multimedia. Jangan lupa untuk mengklik tab Help > Windows Filetype Associations bila setting dilakukan secara manual.

Setelah semua proses tadi dilakukan dengan benar, maka seharusnya Windows Media Player sudah bisa memutar/memainkan file video seperti MKV, FLV, RMVB, 3GP, MP4 serta extensi file multimedia lain dengan baik. Fitur tambahan lainnya adalah menampilkan subtitle video (.SRT) dan juga header: teaser video di Windows Explorer.
