Dibekali dengan OS Android Kitkat versi 4.4, SPC S12 Razor merupakan jajaran ponsel murah dengan android kitkat, di situs lazada, ponsel ini ditawarkan seharga 699.000,-.. Smartphone dari SPC Mobile ini ditenagai prosesor dual core berkecepatan 1 GHz dan mempunyai RAM sebesar 512 MB. Dengan layar berteknologi IPS luasan 4 inchi dan resolusi sebesar 480 x 800 piksel.

Ponsel Kitkat murah ini termasuk 'lumayan' dari segi performa. Jika anda merasa kurang puas dengan performanya, dan ingin melakukan rooting agar bisa melakukan modifikasi terhadap OS Android, ada cara untuk root kitkat langsung tanpa PC yabisa anda coba. Hanya saja, meskipuyn mudah, terkadang proses tersebut bisa gagal karena beberapa faktor.

Ingat, melakukan rooting menghilangkan garansi. Lanjutkan jika anda setuju dengan disclaimer dan dmca blog ini
Untuk menjamin keberhasilan root, anda bisa menggunakan tutorial root SPC S12 Razor dengan PC yang akan saya berikan. Secara umum langkahnya adalah dengan menginstall CWM Recovery ke SPC S12 Razor terlebih dahulu, kemudian melakukan root via CWM recovery yang sudah terpasang.

Sebelum memulai, silahkan persiapkan terlebih dahulu bahan-bahan rooting yang diperlukan dalam proses root ini.

  1. Minimal ADB Fastboot Tool.
  2. CWM Recovery image untuk SPC S12 Razor, dan rename menjadi REC.img
  3. File Update-Supersu 2.46 atau versi stabil lainnya, rename menjadi superuser.zip dan letakkan di external sdcard ponsel anda.
Jika sudah siap, lanjutkan langkah-langkah berikut:

Memasang CWM Recovery pada SPC S12 Razor

1. Install ADB Fastboot Tool di PC/Laptop. lokasi istall di C:\Minimal ADB and Fastboot:

2. Copy file REC.img ke folder C:\Minimal ADB and Fastboot

3. Aktifkan USB debugging. dengan cara: buka menu setting, pilih about phone dan ketuk build number hingga 7x sampai ada pernyataan anda sudah menjadi pengembang. kemudian buka Developer Mode, aktifkan USB debugging dengan mencentang-nya

4. Tancapkan SPC Razor anda ke PC dengan kabel data USB

5. PC akan secara otomatis menginstall driver Android USB Debuging.

6. Jalankan Command Promt dan ketik:

cd C:\Minimal ADB and Fastbootlanjutkan dengan mengetik:

adb reboot bootloader7. Setelah ponsel masuk fastboot mode, instal CWM recovery dengan mengetik perintah:

fastboot flash recovery REC.img

Jika proses sudah selesai, berati SPC S12 anda sudah mempunyai CWM recovery. Anda bisa melakukan rooting dengan CWM recovery dengan mengikuti langkah selanjutnya.

Root SPC S12 Razor dari CWM Recovery

Setelah SPC S12 anda terpasang CWM Recovery, Andabisa melakukan backup system terlebih dahulu menggunakan CWM Rcovery sebelum melakukan rooting. Untuk melakukan rooting, berikut caranya

1. Pastikan file supersu.zip sudah ada di external sdcard

2. Tancapkan ke PC/Laptop dengan mode usb debug aktif.

3. Masuk mode recovery dengan mengetik: adb reboot recovery

4. Lihat ponsel anda yang sudah dalam recovery mode

5. Lakukan flashing dengan cara memilih menu update from sdcard dan pilih file supersu.zip.

6. jika proses flashing sudah selesai, pilih menu reobot to system

Selesai, kini ponsel SPC S12 Razor kitkat anda sudah root dan selain itu anda mempunyai CWM Recovery ter-install di ponsel anda. Jika sebelum root anda sempat mem-back-up system, anda bisa menggunakannnya untuk mengembalikan ponsel anda jika mengalami bootloop (reboot berulang atau stick di logo)
