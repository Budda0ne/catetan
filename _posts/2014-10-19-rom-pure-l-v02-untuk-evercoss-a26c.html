Cara Install Custom ROM PURE L di Evercoss A26C

Ingat, semua resiko anda sendiri yang tanggung. Lanjutkan jika anda setuju dengan disclaimer dan dmca blog ini
Silahkan gunakan tuorial di bawah dan dapatkan ROM beserta patch Dolby di Download Section. Sedangkan untuk Mito A77 akan saya posting lain waktu setelah mendapat izin dari yang porting. :)

  1. Pastikan anda sudah pernah melakukan downgrade ke Firmware lama.
  2. Pastikan sudah meng-install CWM Recovery atau Carliv Touch Recovery untuk A26C
  3. Copy file ROM Pure L ke sdcard dan Patch Dolby Sorround --optional
  4. Masuk mode recovery dengan cara matikan HH dan tekan tombol Power bersamaan dengan tombol Volume Up.
  5. Sesudah masuk mode Recovery, pilih menu wipe data, wipe cache juga pilih advanced menu pilih wipe dalvik-cache.
  6. Kalau proses wipe selesai, pilih install zip from sdcard dan pilih file ROM Pure L zip
  7. Installer menggunakan Aroma Installer, klik next saja sampai proses install.
  8. Tunggu hingga selesai dan klik finish kemudian reboot HH secara normal dengan memilih menu reboot to system.

Sampai disini install ROM sudah selesai dan bisa berfungsi normal. Tatapi jika anda ingin menggunakan Dolby sebagai Pengatur equalizer audio anda harus meng-install Patch Dolby yang sudah anda letakkan di sdcard.


Patch Dolby Sorround

Jika Evercoss A26C anda sudah boot normal (bisa masuk system), dan ingin melakukan patch dolby reboot ke recovery mode lagi
Install seperti langkah instalasi ROM diatas. Hanya saja file yang di-flash adalah file patch dolby.

Setelah proses flash patch selesai, pilih menu reboot to system dan lihat di drwaer aplikasi. Kini sudah ada Dolby sebagai pengatur efek audio anda.
Menurut saya pribadi, saya lebih suka menggunakan Viper4Android daripada Dolby karena volume level masih bisa diatur sesuai kemampuan "gendang telinga" saya :D. Jika menggunakan Dolby mau tidak mau anda mesti melakukan setelan volume maksimal secara manual.
