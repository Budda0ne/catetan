Panduan instalasi YAKI v1.2.2 ROM via CWM Recovery:

Dengan Mengikuti Tutorial ini anda setuju dengan Disclaimer

Agar bisa berfungsi dengan baik, anda perlu mengganti file boot.img dalam file flashable .zip recovery YAKI v1.2.2 ROM@webz yang anda download dengan file boot.img dari ROM Cross A28 asli hasil back-up CWM Recovery.

Jika tidak punya boot.img, download di sini: via Mediafire.

Setelah boot.img diganti, maka anda bisa melanjutkan sesuai panduan instalasi berikutnya:
  1. Download ROM (pilih salah satu, Balance MOD atau Gaming Mode, link ada di bawah) dan tempatkan di SDcard (jangan di ekstrak).
  2. Matikan Handheld ke mode CWM Recovery (jika belum punya, silahkan install CWM Recovery terlebih dulu).
  3. Pilih Wipe Data, Cache dan Advanced -> Dalvik-Cache. Jika ada menu lagi pilih yes.
  4. Pilih Install zip from SD Card.
  5. Cari file .zip di SDCrad dan pilih. Jika ada menu lagi pilih yes.
  6. Tunggu hingga selesai.
  7. Reboot.
Perlu di ketahui bahwa untuk meng-install YAKI v1.2.2 ROM anda juga bisa melewati langkah no. 3. Ini berguna ketika anda ingin berganti Mode dari Balance MOD ke Gaming Mode maupun sebaliknya tanpa harus kehilangan data. (sejauh yang saya tahu sih begitu)

Fixing Bug untuk Cross A28

  • Buka menu Setting pilih Root Cleaner
  • Cari Full Cleaning
  • Centang pilihan Battery Calibration
  • Pilih Full Cleaning.
