Apa kabar pilot kaleng? Jehehe, Kalau anda termasuk pemain game Walking War Robots (WWR), pasti tahu bahwa Pixonic selaku pengemvang game perang robot ini membuka pendaftaran untuk bergabung sebagai pemain test server. Apa itu test server om? Bagaimana cara gabung test server? Ada syarat khusus gak untuk ikutan main di Test Server WWR? Selow, nanti saya jelaskan satu-satu.

Sebelum itu, pertimbangkan pernyatan dari pixonic hal berikut ini sebelum bergabung: ﻿
Support service is not provided for the inquiries related to the test server.

The content on the test server is on the early stage of development and its final version is subject to change and may not be added to the game.
Kurang lebih begini penjelasnnya: "Dalam test server tidak akan ada dukungan apapun dari pixonic mengingat semua konten yang ada dalam test server adalah fitur ujicoba dan masih dalam tahap awal pengembangan. Bisa jadi akan dimasukkan dalam Game WWR untuk umum, tapi batal masuk karena dianggap tidak layak.

Kalau menurut saya sih lumayan juga bisa mencicipi ide-ide aneh dari developer pixo. Bahkan pernah ada robot yang aneh muncul dalam test server.

Jadi, apa sih test server itu? Test Server merupakan sebuah server terpisah yang disediakan untuk memperluhatkan konten game baru Walking War Robots. Baik itu fitur, aturan gameplay, termasuk kemungkinan senjata dan robot jenis baru yang baru dalam tahap ujicoba. Dan konten ini tidak bisa didapatkan di game WWR biasa.

Apakah boleh pemain Game WWR biasa bergabung? Atau ada syarat khusus? Sejauh yang saya tahu, siapapun boleh bergabung menjadi pemain tester Walking War Robots dan tidak ada persyaratan khusus harus begini-begitu. Tapi harus daftar dulu. Untuk pemain WWR dengan ponsel Android bisa daftar dari sini, sedangkan untuk pemain WWR dengan iOS (iPhone/iPad) daftar dari formulir ini. Isi sesuai yang diminta dalam formulir. Ada dua halaman formulir singkat untuk masing-masing platform (Android/iOS).

Nanti setelah mendaftar anda akan menerima e-mail undangan untuk bermain di test server sekaligus panduan untuk instalasi game khusus test server WWR dan link download aplikasi. Undangan bermain biasanya akan diterima jika ada fitur baru yang ingin di-ujicoba oleh pixo. Jadwal ujicobanya di rilis dari fanspage faceboom mereka. Jadi kalau saat kamu daftar tidak menerima e-mail berarti belum ada jadwal main. Soal jadwal main di test server biasanya akan diposting di wall sehari sebelum dimulai.

Oh iya, mungkin kamu bertanya-tanya, kok daftar test server pixonic malah masuk situs HockeyApp? Jangan jangan ini situs phising/tipuan? Wohoho... Gak cuma anda yang berpikir begitu, beberapa kawan kita yang baru ikut bermain kemudian bertanya di group WWR Indonesia juga bilang begitu. Sekali lagi link yang saya berikan diatas dan kawan-kawan di group facebook itu asli dari pengembang game WWR.

Mengenai HockeyApp yang jadi partner Walking War Robots sendiri adalah layanan untuk mengkoordini test aplikasi untuk publik. Jadi yang memanfaatkan jasa HockeyApp bukan hanya Pixonic saja.

Yang penting untuk anda ketahui: Aplikasi Game WWR Test Server hanya untuk sekali pakai, begitu juga akun yang anda gunakan bermain di test server. Setelah masa ujicoba berakhir atau ada aplikasi test yang baru proges akun yang lama juga akan hilang. Begitu juga aplikasinya menjadi tidak bisa dipakai lagi. Dan setiap kali ada aplikasi WWR test server baru, anda harus mencopot game WWR test versi sebelumnya sebelum memasang yang versi barum

Karena tujuan dibuatnya Test Server WWR adalah untuk men-ujicoba fitur baru, maka pemain akan dimintai pendapatnya setelah bermain di test server. Setiap feedback akan ada reward-nya dan di bagikan sehari setelah event ujicoba berakhir. Jadi masukkan ID pilot game WWR reguler anda dengan benar agar hadiah-nya tidak nyasar.

Bagaimana para pilot kaleng? Tertarik menjadi beta tester game perang robot yang paling banyak dimainkan ini? Gabung saja menjadi pilot Walking War Robots test server, gak rugi.
