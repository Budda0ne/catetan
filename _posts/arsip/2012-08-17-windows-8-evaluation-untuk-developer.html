Windows 8 Enterprise evaluasi 90-hari tersedia bagi pengembang untuk membangun dan menguji aplikasi pada Windows 8 versi final dari Windows 8. Setelah menginstal Windows 8, Anda(Developer) bisa ke halaman download untuk developer di Windows Dev Center untuk menginstal Visual Studio 2012 dan tool terkait untuk membangun/mengembangkan aplikasi di Windows 8.

Apa maksud dari Windows 8 Enterprise Evaluasi ini?


Evaluasi ini memberikan masa trial selama 90-hari untuk Windows 8 edisi Enterprise. Versi 32-bit dan 64-bit yang tersedia dalam bentuk ISO images dalam bahasa berikut: Cina (Simplified), Cina (Tradisional), Inggris (Universal), Inggris (Britain), Perancis, Jerman, Italia, Jepang, Korea, Portugis (Brasil) , dan Spanyol. Untuk menggunakan versi evaluasi ini, Anda harus registrasi terlebih dahulu dan produk harus diaktifkan secara online melalui host layanan aktivasi dan validasi Microsoft. Namun Microsoft tidak menyediakan dukungan teknis untuk perangkat lunak ini.

Untuk Siapa?


Windows 8 Enterprise Evaluasi ditujukan untuk pengembang yang membangun aplikasi Windows 8 dan profesional TI yang tertarik untuk mencoba Windows 8 Enterprise atas nama organisasi mereka. Jika Anda memenuhi syarat di bawah salah satu program berikut (tercantum di bawah ini), Anda mungkin sudah memiliki akses dan tidak perlu men-download evaluasi 90-hari.
Syarat tersebu antara lain:
  1. Berlangganan MSDN
  2. Microsoft Mitra Jaringan
  3. TechNet Profesional Berlangganan
  4. Microsoft Volume Licensing / Software Assurance

Hal-hal yang perlu diketahui sebelumnya antara lain:
Edisi evaluasi akan berakhir dan tidak dapat ditingkatkan (upgrade).
Untuk meng-upgrade, versi evaluasi harus dihapus dan versi non-evaluasi Windows 8 harus diinstal ulang dari media instalasi yang asli.
Lebih disarankan untuk menjalankan edisi evaluasi dalam lingkungan virtual atau instalasi pada hard drive atau partisi yang terpisah. Ini akan memudahkan Anda untuk meng-upgrade instalasi Windows 8 original jika nantinya anda berminat untuk menggunakan Windows 8.
Selama pendaftaran  Anda harus login dengan account Microsoft dan memberikan nama, alamat e-mail dan Negara Anda.
Anda diminta untuk mengaktifkan produk secara online dalam waktu 10 hari setelah menginstal.
Setelah evaluasi terinstal, Anda tidak bisa meng-upgrade. Untuk kembali ke versi sebelumnya dari Windows, Anda harus melakukan instalasi murni (terj. clean install, pen) dari media instalasi asli Anda.

Petunjuk Penggunaan dan Aktivasi


Back up file dan pengaturan sebelum menginstal versi evaluasi ini dan sebelum berakhirnya 90-hari masa trial/evaluasi.
Setelah instalasi, Anda akan memiliki 10 hari untuk mengaktifkan versi evaluasi ini secara online. Windows akan meminta Anda untuk mengaktifkannya. Product Key tidak diperlukan untukmengaktifkannya.
Anda harus melengkapi aktivasi sebelum 15 Agustus 2013, untuk menggunakan evaluasi ini.
Setelah Anda mengaktifkan, Anda memiliki hingga 90 hari untuk menggunakan perangkat lunak. Anda dapat melacak jumlah waktu yang tersisa dengan melihat/mengecek pada "watermark" di sudut kanan bawah desktop Windows.
Jika Anda gagal/tidak mengaktifkan evaluasi ini dalam tenggang waktu 10-hari, atau jika masa evaluasi Anda berakhir, latar belakang desktop akan berubah menjadi hitam, Anda akan melihat pemberitahuan desktop secara terus-menerus yang menunjukkan bahwa sistem ini tidak asli alias bajakan, dan PC akan mati setiap jam. (resikonya: kehilangan pekerjaan yang belum disimpan).
Setelah masa evaluasi, Anda akan perlu untuk mengganti sistem operasi pada komputer pengujian dan menginstal ulang semua program dan data. Tidak akan ada opsi untuk meng-upgrade dari versi Windows 8 Evaluation ke versi Windows 8 yang original. Jadi tetap harus menginstall ulang dengan Clean Install.

Persyaratan Sistem


Windows 8 bisa berjalan pada hardware yang sama denagn spesifikasi untuk Windows 7:

  • Prosesor: 1 gigahertz (GHz) atau lebih cepat
  • RAM: 1 gigabyte (GB) (32-bit) atau 2 GB (64-bit)
  • Hard disk space: 20 GB
  • Kartu grafis: Microsoft Perangkat grafis DirectX 9 dengan driver WDDM

Persyaratan tambahan untuk menggunakan fitur tertentu:

  • Untuk menggunakan fitur Touchscreen, Anda memerlukan pc-tablet atau monitor yang mendukung multitouch.
  • Untuk mengakses Windows Store dan untuk men-download dan menjalankan aplikasi, Anda perlu koneksi Internet aktif dan resolusi layar minimal 1024 x 768.
  • Untuk menjepret (screenshoot) aplikasi, Anda memerlukan resolusi layar minimal 1366 x 768.
  • Akses internet.
