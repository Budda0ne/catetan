Jika iya, apakah anda pernah merasa kesulitan ketika ingin mengembalikan font Android seperti aslinya? Dan jika iya (lagi), silahkan anda teruskan membaca tutorial untuk me-restore stock font ini..

Stock font yang digunakan dalam ponsel android adalah Roboto fonts, Jika anda meng-install custom font dengan flashable zip melalui CWM Recovery, anda sebenarnya me-replace file font roboto dengan file font lainnya yang sudah diberi nama roboto ke dalam system. Untuk mengembalikan font seperti semula, maka anda harus me-replace ulang file font custom dengan file font roboto asli dengan metode yang sama (via CWM Recovery)

Untuk mengembalikan font bawaan android dengan cara ini setidakya anda harus:
- Android sudah ter-install CWM Recovery
- Anda sudah mempunyai stock font flashable zip bisa di download di bawah

Tutorial Restore Roboto Stock Font:

- Siapkan file Font_Roboto442r2.zip di sdcard anda.
- Reboot ke recovery mode (CWM Mode)
- Setelah dalam mode recovery, silahkan pilih menu wipe cache dan wipe dalvik cache (optional)
- Kemudian, pilih menu install zip from sdcard, dan pilih file Font_Roboto442r2.zip, yes.
- Setelah sukses terinstall, pilih menu reboot system, yes.

Sekarang, Anda bisa melihat font Android anda sudah kembali seperti aslinya.
