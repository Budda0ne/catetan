Update Link 19 Desember 2015: ROM Baru CM 12.1 by StuepD, yang lama masih ada.

Setelah mencoba men-downgrade Lenovo A6000 ke Kitkat (demi tutorial, lama-lama bosok ini hape), 3 hari lalu, saya mencoba meng-install Custom ROM CM 12.1 32bit yang di porting oleh agan Stuep Dec (dan akhirnya artikel ini publish juga, setelah sekitar 2 minggu membusuk di draft postingan). Berbeda dengan yang 64 bit, versi 32bit ini lebih nyaman dipakai di Lenovo A6000 tetapi tetap bisa di pakai untuk Lenovo A6000 plus.

Secara umum, fitur-fitur sudah berjalan dengan baik. Berbeda dengan CM 12 yang ber-basis Lollipop 5.0.2 yang sedikit-sedikit mencret, inilah alasan kenapa tidak memajang ROM CM 12 5.0.2di blog ini.
Sedangkan Custom ROM CM 12.1 yang ber-basis Lollipop 5.1 32bit ini tidak ada lagi notifikasi "unfortunately ...." dan "... stopped unexpectedly" secara random lagi. Hanya saja saat saya membuka menu Security > Settings tiba tiba mencret. Awalnya agak kecewa karena masih ada penyakit yang tersisa, dan restore lagi ke Stock ROM Kitkat dari backup-an, dan langsung upgrade ke Stock ROM S058 (Lollipop).

Tetapi, karena masih penasaran saya coba install lagi Custom ROM CyanogenMod 12.1 32bit. Kali ini saya coba lagi untuk membuka pengaturan keamanan (Security > Settings) ternyata sudah tidak mencret :D.

Lippo Securities, link ini gak ada hubungannya dengan Security di android, gak usah dibuka.

Lanjut dengan memaksa si Lenovo A6000 memakan aplikasi-aplikasi sosmed kelas berat macam BBM, Facebook dkk dan membiarkannya tetap berjalan di background sambil memainkan Game COC dan Need for Speed No Limit, hasilnya? Lag setelah 15 menit bermain. Masih bisa dibilang lumayan, artinya: jangan biarkan aplikasi sosmed berjalan saat main game. Selebihnya smooth dan mulus seperti p4h4 artis korea.

Cukup review singkatnya. Dan jika anda ingin ikut-ikutan memakai Custom ROM CM 12.1 untuk Lenovo A6000 ini ikuti tutorial instal-nya, setelah mendownload dan mempersiapkan hal-hal berikut ini:

1. Sebaiknya upgrade ke OS Lollipop terlebih dahulu, meskipun bisa dari OS KitKat.
2. Sudah ter-install TWRP Recovery untuk Lollipop, bisa agan baca tutorialnya
3. Sudah mengunduh file ROM CM 12.1 32bit yang ada di bawah
4. Siapkan juga GAPPS untuk Lollipop 5.1.1 32 bit saya rekomendasikan memakai OpenGAPPS pilih dan download dari halaman berikut.

Tutorial Instalasi Custom ROM CM 12.1 32bit Untuk Lenovo A6000/plus

Dengan mengikuti panduan yang saya tulis, berarti anda sudah setuju dengan disclaimer dan harus memahami copyright notice di KnoAcc Blog jika ingin membagikan tulisan ini. Jika tidak setuju silahkan minggir.

  1. Copy File ROM CM 12.1 32bit dan GAPPS untuk Lollipop 5.1.1 32 bit yang sudah di download ke sdcard, sebaiknya diluar folder.
  2. Matikan ponsel dan boot ke recovery dengan menekan tombol volume min dan plus bersamaan dengan tombol power
    3. .
  3. Lakukan Wipe, system, cache, dalvik-cahe dan harus wipe data jika install dari kitkat. Proses wipe lumayan lama, jadi sabar saja.
  4. Kembali ke menu utama dengan menekam menu back dan pilih install zip dan cari ROM CM 12.1 di sdcard anda.
    Geser indikator (swipe) dibawah untuk melakukan instalasi.
  5. Tunggu hingga selesai dan pilih menu Reboot dari menu utama
  6. Jika pertanyaan untuk root mauapun disable install recovery abaikan saja, jangan di-iyakan. Karena root permission bisa di enable di pengaturan
    7. .

Terakhir, untuk mengaktifkan akses root (native root, tidak perlu melakukan rooting lagi), buka Settings > Developer options > Root access, dan aktifkan untuk aplikasi, adb atau keduanya.

Mungkin saya akan memakai ROM ini selama 2 minggu ke depan sebelum balik memakai Stock ROM Lollipop.
