Cantik kan? ColorOS merupakan ROM yang digunakan di Handheld Oppo. ColorOS ini juga sama dengan MIUI OS yang juga dikembangkan dari OS Android oleh pabrikan Xiaomi. ROM ColorOS ini berbasis Android 4.2.2 (Jelly Bean)

Kali ini ada developer kita yang berhasil mem-porting Color OS untuk Evercoss A7S (lihat bagian kredit). Karena kini anda juga bisa merasakan ColorOS di Handheld kesayangan anda: Evercoss A7S.

Baca juga: Custom ROM Oppo R831T (ColorOS) untuk Evercoss A26C

Note: Work fine di Handheld milik kawan saya.

Tidak ada custom ROM yang sempurna, apalagi ROM portingan. Bugs yang paling terasa :
  • Audio
  • Kamera
  • GPS tidak berfungsi
Selain bugs diatas, init.d support ROM Color OS ini belum aktif.

Bugs di atas sudah fix semua. Lihat bagian fixing
Fitur di Update Fix All:
  • Boot.img (kernel Des for Oppo)
  • Boot.bin (logo Boot Mod)
  • Camera ( fix 8mp dan record video 1080 HD)
  • Tweak (Tweakforall, swap cache 300mb, juwe, Unlock Fps 60)
  • Fix Compas
  • Fix Sensor
  • Fix GPS

Istimewanya adalah, ROM COLOR OS V2 ini sangat cantik. Sama cantiknya dengan BAIDU OS

Untuk menunjang kinerja Evercoss A7S anda, anda bisa menambahkan Turbo Boost Swap karena init.d support sudah aktif.

Cara Pasang/Instalasi Custom ROM Oppo COLOR OS V2

Sebelum meneruskan tutorial Instalasi ini, anda berarti sudah setuju dengan artikel Disclaimer dan mengerti serta patuh dengan artikel DMCA Copyright di blog ini. Dan anda juga paham bahwa dengan menginstall Custom ROM di handheld anda bisa menyebabkan garansi anda gugur resiko kehilangan data, backup data penting anda.

Syarat: harus sudah ter-install CWM Recovery, jika belum, baca di tutorial install CWM Recovery.

Untuk memasang ROM ini, anda harus melakukan Backup IMEI melalui aplikasi Mobile Uncle atau aplikasi backup IMEI lainnya, sebagai antisipasi jika terjadi INVALID IMEI. Buat juga Nandroid Backup melalui CWM Recovery sebelum melakukan flashing.

Langkah-langkah instalasi:

1. Download ColorOS_v2_A7S.zip dan letakkan di sdcard tapi jangan di dalam folder. Letakkan di /sdcard/--disini- - (menjadi /sdcard/ColorOS_v2_A7S.zip)
2. Matikan Evercoss A7s anda, dan hidupkan lagi dalam mode CWM Recovery dengan menekan tombol [Power] dan [Volume Up]
3. Ketika sudah masuk mode CWM Recovery, gunakan tombol [Volume Up] dan [Volume Down] untuk navigasi dan tombol [Power] untuk memilih menu:
  • pilih menu Wipe Data/Factory Reset pilih yes
  • pilih lagi menu Wipe Cache Partition pilih yes lagi
  • pilih menu Advaced, pilih Wipe Dalvik Cache
  • Pilih Install Zip From SdCard kemudian pilih file ColorOS v2 A7S.zip
5. CWM Recovery akan melakukan proses flashing, tunggu hingga selesai.
6. Setalah Selesai, pilih GO BACK (menu paling bawah) dan pilih Reboot System Now

Jika Proses Instalasi berjalan mulus, anda akan mendapati Evercross A7S anda sudah terpasang/terinstall ColorOS V2. Tetapi masih ada beberapa bug yang mengganggu, seperti di bagian audio dan kamera depan yang tidak berfungsi normal.

Melakukan Fix Bug Audio dan Kamera ColorOS v2 A7S

1. Masuk Recovery mode, langkahnya sama.
2. Langsung pilih menu Install from sdcard kemudian pilih file Update_Oppo_Fixed_V2 Full.zip
3. CWM Recovery akan memproses update, tunggu sampai selesai.
4. Jika sudah pilih GO BACK dan pilih Reboot System Now
