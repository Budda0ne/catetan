Samsung Galaxy J1 Ace mempunyai beberapa varian, diantaranya: SM-J110FN, SM-J110F, SM-J110H, SM-J110G/DS. Spesifikasi ketiga varian ini beda-beda tipis.< p>

Untuk melakukan root Galaxy J1 Ace, anda harus mengetahui ponsel anda termasuk variant yang mana. Karena anda harus memasang Custom Recovery pada Galaxy J1 Ace lebih dahulu. Custom Recovery yang di rgunakan adalah TWRP. Setelah TWRP terpasang, baru anda bisa melakukan rooting dengan melakukan flashing file SuperSU.zip untk melakukan rooting.

Spesifikasi lebih detail bisa dilihat di GSM Arena melalui link berikut ini

Sebelum melakukan root Galaxy J1 Ace, silahkan download file yang dibutuhkan di bawah:
TWRP Recovery Samsung Galaxy Ace LTE terbaru saat artikel ini dibuat:

TWRP 3.0.2 [Official]:
TWRP J1 Ace mirror link
SuperSU.zip v2.67

Berikutnya simpan file twrp-3.0.2-0-j1acelte.img.tar dan UPDATE-SuperSU-v2.76-20160630161323.zip yang sudah anda download di sdcard/internal memori ponsel anda.

Peringatan!
Pastikan anda mengaktifkan pilihan Enable OEM Unlock di pengaturan ponsel jika anda menggunakan TWRP. Karena ada kasus dimana saat saya mencoba melakukan factory reset, Samsung Galaxy J1 Ace tidak bisa boot ke system dan tidak bisa di flash dengan Odin dengan status binary blocked.

Jika anda sudah yakin ingin melakukan root, siap kehilangan garansi, kehilangan data dan resiko lain yang lebih parah silahkan proses dengan mengikuti langkah berikut dengan semua konsekuensi anda tanggung sendiri.
Selanjutnya anda tinggal mengikuti cara root Samsung Galaxy J1 Ace langkah demi langkah berikut:

Install TWRP Recovery Galaxy J1 Ace

1. Install samsung drivers, jika ada masalah lihat bagian troubleshoot
2. Buka Odin3_v3.11.1.exe dan klik AP dan pilih file ***.tar.md5.
3. Untuk J120H saja, klik options dan hilangkan centang auto restart.
4. Matikan ponsel dan nyalakan dengan menekan tombol [Vol down] + [Home] + [Power] untuk masuk download mode
5. Jika muncul peringatan, lanjut dengan menekan tombol [Vol up] untuk melanjutkan.
6. Pasang kabel data pada PC dan sambungkan dengan Samsung Galaxy J1 2016 anda.
7. Lihat status koneksi dengan melihat kotak biru di bagian atas Odin. Jika ID:COM muncul dan berwarna biru muda berarti sudah terdeteksi.
8. Klik Start untuk melakukan install TWRP.
9. Jika di kota ID:COM berubah warna menjadi hijau dengan teks PASS berarti sudah terpasang.

Ingat! setelah flashing selesai jangan reboot ke system terlebih dahulu. Jika anda langsung menyalakan ponsel, recovery akan kembali menjadi stock recovery dan yang anda lakukan sebelumnya menjadi sia-sia. Jadi, cabut kabel data dan matikan dengan mencabut baterai. dan lanjutkan langkah root dengan TWRP berikut.

Langkah Root dengan TWRP dengan flashing SuperSU.zip

  1. Nyalakan Galaxy J1 2016 anda langsung ke recovery mode. Caranya dengan menekan tombol [vol up] + [home] + [power]
  2. Jika anda gagal masuk TWRP dan boot ke system, anda harus mengulang proses instal TWRP lagi.
  3. Dalam mode TWRP recovery, lakukan flashing SuperSU.zip dimana sebelumnya sudah anda copy di sdcard/internal memori anda.
  4. Pilih meno reboot dan tunggu. proses boot pertama akan memakan waktu lumayan lama.

Setelah menyala, cek apakah aplikasi SuperSU sudah muncul di menu aplikasi. Jika sudah muncul, artinya anda sudah berhasil melakukan root pada Samsung Galaxy J1 2016 anda. Selamat!!

Untuk memastikan bahwa Samsung Galaxy J1 Ace sudah 100% root, anda bisa menggunakan aplikasi root checker.

Troubleshoot:
1. Ada masalah dengan Odin? Cari artikel di google dengan keyword "stuck at pit file mapping".
2. Posel tidak terdeteksi? Download dan install smartswitch from samsung.com jalankan dan jika sudah terdeteksi silahkan buka kembali Odin.
