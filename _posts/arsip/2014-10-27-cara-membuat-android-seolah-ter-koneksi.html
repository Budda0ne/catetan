Anda mungkin pernah menjumpai dan meng-install beberapa aplikasi yang megharuskan Anda untuk menggunakan wifi atau aplikasi tidak akan bekerja atau beberapa fitur tidak akan tersedia untuk Anda. Hal ini kadang sangat mengganggu.

Tetapi, dengan patch dari Xposed Framework dapat mengatasi masalah ini dengan membuat aplikasi percaya menipu ponsel Anda sedang terhubung ke Wifi, meskipun sebenarnya Anda terhubung dengan data seluler saja (3G).

Wi-Fi Connection required

Xposed Module ini memberi pilihan, untuk mengaktifkan atau menonaktifkan fitur ini untuk aplikasi tertentu sesuai keinginan Anda. Modifikasi akan langsung di-aplikasikan. Beberapa fitur yg sebelumya tidak muncul pada daftar, setelah itu bisa dibuka. Anda juga bisa memilih untuk menjalankan modul ini saat aplikasi sedang berjalan.

Cara menginstal Modul Xposed Android Fake Wifi Connect


Sebelum kita mulai, saya sangat menyarankan Anda membuat back-up perangkat Android Anda. Pelajari cara melakukannya di sini
  1. Agar modul ini berfungsi, Xposed Framework harus sudah ter-install. Anda dapat membaca tentang Xposed Framework dan cara menginstalnya di tutorial berikut ini
  2. Download file FakeWifiConnection.apk di bawah.
  3. Buka Xposed Installer
  4. Arahkan ke Module dan
    centang "FakeWifiConnection"
  5. Reboot Ponsel Anda.

Dan kini anda bisa menikmati fitur aplikasi (umumnya game) yang sebelumnya tidak bisa diakses tanpa koneksi wifi.
