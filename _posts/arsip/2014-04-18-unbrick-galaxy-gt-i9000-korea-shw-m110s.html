Tutorial ini bertujuan untuk mengembalikan SHW M110S yang brick dan masalah masalah yang terjadi gara-gara label/stiker dibalik baterai yang menyatakan kalau hanphone ini adalah I900. Unbrick Galaxy GT-I9000 Korea SHW M110S (Galaxy S)

Langkah Pertama: Membuat SHW M110S Agar Berfungsi Normal

1. download repair kit and unzip file repair.zip, di dalamnya ada 6 file.
2. matikan dengan cara mencabut baterai dan nyalakan telepon dalam download mode.
3. jalankan odin apilih file m110s.pit di kolom PIT (pilihan Re-partion otomatis tercentang)
4. pilih file M110S-PDA-PHONE-MOVI-REV03-DG23-1409-USER.tar.md5 di kolom PDA
5. colok-kan handphone via kabel usb dan tekan tombol start (prosesnya 10 menitan, dan SHW M110S akan reboot otomatis)

Sampai sini, Handphone anda akan normal kembali tetapi dengan sistem operasi Froyo. Lanjutkan ke langkah kedua.

Langkah Ke-dua: Mengisi Region SHW M110S

1. matikan dengan cara mencabut baterai dan nyalakan telepon dalam download mode.
2. jalankan odin apilih file M110S-NPS-IMAGE-REV03-DG23-1409-USER.tar.md5 di kolom PDA.
3. pilih file M110S-PHONE-REV00-DG24.0151_Develop.tar.md5 di kolom PHONE.
4. colok-kan handphone via kabel usb dan tekan tombol start (prosesnya sekitar 5-10 menit, dan SHW M110S akan reboot otomatis)

Langkah Ke-tiga : Meng-update PDA dan flashing PHONE

1) matikan lagi dengan cara mencabut baterai dan nyalakan telepon dalam download mode.
2) pilih file M110S-PDA-PHONE-MOVI-REV03-DG23-1409-USER.tar.md5 di kolom PDA
5) colok-kan handphone via kabel usb dan tekan tombol start (prosesnya sekitar 5-10 menit, dan SHW M110S akan reboot otomatis)

Langkah Ke-empat : Update Firmware terbaru.

Di tahap ini, ada bisa menggunakan Stock Firmware SHW M110S yang paling baru (2012) di bagian download di bawah. Atau menggunakan Custom ROM khusus untuk SHW M110S. Ingat jangan melakukan flashing dengan firmware I9000 international maupun Firmware I9000 yang lain, karena handphone anda akan mengalami brick kembali.
1. matikan SHW M110S dengan cara mencabut baterai dan nyalakan telepon dalam download mode.
3. pilih file M110S-PDA-PHONE-MOVI-REV03-DG23-1409-USER.tar.md5 di kolom PDA
4. colok-kan handphone via kabel usb dan tekan tombol start (prosesnya sekitar 5-10 menit, dan SHW M110S akan reboot otomatis)

Untuk melakukan flashing via odin, ikuti urutan prosedur/langkah-langkah diatas. Dan ingat, setelah melakukan flasing firmware, anda harus masuk ke recovery mode dan melakukan wipe cache dan wipe data dari recovery bawaan (3e recovery).
