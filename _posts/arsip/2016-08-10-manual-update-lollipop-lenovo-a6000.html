Cara Singkat Upgrade Firmware Manual

Cara update secara manual cukup sederhana:

Langkah pertama, Pastikan anda menggunakan stock recovery dan kondisi Lenovo A6000 masih belum root. Kalau sudah root install official firmware S032 Lenovo A6000 di halaman ini. Atau jika menggunakan TWRP Recovery anda bisa lanjut ke langkah kedua.

Langkah kedua, Download file Update Lollipop Lenovo A6000 di atas dan letakkan di internal sdcard dan rename menjadi update.zip jika menggunakan stock recovery dan bebas untuk yang menggunakan TWRP recovery.

Langkah ketiga, Reboot ke recovery mode dan lakukan full backup jika menggunakan TWRP dengan mencentang semua opsi dan pilih eksternal sdcard untuk lokasi backup. Jika menggunakan stock recovery, abaikan langkah ini, cukup reboot biasa.

Langkah ke empat, untuk Stock Recovery, pilih menu install update di pembaruan versi. Jika masih tidak bisa update, rename file menjadi update.zip dan letakkan di sdcard kemudian install update zip from sdcard di stock recovery.

Sedangkan bagi yang menggunakan TWRP Recovery pilih install zip dan pilih file Update Lollipop yang tadi anda download.

Tara... Lenovo A6000 anda kini sudah menggunakan Firmware Lollipop Kraft-A6000 S058 150825 untuk A6000 polos dan Kraft-A6000-s S028 150827 untuk A6000 plus.

Troubleshoot failed update


1. Ekstrak file zip, kemudian ambil System.zip dan Indonesia.ota.zip (atau sesuai dengan negara anda) dan hapus file lainnya dan lakukan flash melalui recovery.
2. Jika masih gagal, ekstrak file boot.img dalam System.zip dan flash image > boot, dengan twrp recovery. lakukan flashing lagi.
3. Jika langkah kedua masih gagal, gunakan boot.img dari versi yang lebih rendah.
