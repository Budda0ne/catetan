Untuk melakukan rooting, cukup sekali pencet menggunakan aplikasi rooting populer KingRoot.

Note: Tutorial ini juga saya buat untuk memeragakan cara rooting dengan Kingroot mulai versi 4.5.0 hingga yang terbaru saat tutorial ini di tulis (Kingroot v 4.6.2).

Untuk melakukan rooting, sebaiknya anda meng-aktifkan mode USB debug dari Developer Options.

Cara Instan Root Infinix Zero 2 Tanpa PC


Baca juga: Mengatasi Bootloop setelah melakukan Rooting, sebagai bahan tambahan untuk referensi anda.

Dengan mengikuti tutorial berikut, anda sudah setuju dengan pernyataan disclaimer dan hak cipta KnoAcc Blog

Sebelum melakukan rooting download aplikasi KingRoot terbaru di halaman ini.
Cek juga apakah pilihan Sumber tak dikenal (Unknown source) di Pengaturan Keamanan > Aplikasi sudah aktif.

Jika sudah, lanjutkan dengan meng-install-nya di Infinix Zero 2 Anda.

Jalankan aplikasi KingRoot setelah ter-install.

Ketika terbuka pertama kali, aplikasi KingRoot akan mengecek status root Infinix Zero 2 Anda.

Di tahap ini ponsel Anda HARUS terhubung ke Internet, agar KingRoot dapat memeriksa ketersediaan cara root.

Setelah proses pengecekan selesai, klik tombol Try to Root.

Tunggu hingga proses rooting selesai. mungkin akan memakan waktu lama, sekitar 5 menit, bisa juga lebih. Tergantung kecepatan koneksi internet.

Nantinya, kalau aplikasi ini sudah mendapatkan metode root yang cocok dan sudah mendowload data exploit-nya, ponsel Anda akan reboot/restart sendiri, untuk melakukan penanaman exploit untuk rooting.

Proses rooting akan dijalankan setelah ponsel Anda reboot/restart.

Jika muncul kotak peringatan, pilih Install anyway (ini untuk menanam kinguser di System).

Kemudian proses rooting akan kembali di lanjutkan, tunggu hingga selesai.

Selesai, proses rooting pada Infinix Zero 2 X509 Anda telah berhasil.

Rekomendasi: Melakukan upgrade (manual) Infinix Hot 2 ke Android 5.1.1 Lollipop.

Untuk memastikan proses rooting telah berhasil, silakan buka kembali aplikasi KingRoot, atau jika masih kurang yakin, gunakan aplikasi root checker lainnya dari PlayStore.
