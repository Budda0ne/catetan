Tutorial Flashing Stock ROM Axioo Picopad 5 GEA

Anda membutuhkan SP Multiport Flash untuk mengembalikan Phablet Picopad 5 anda. Download software untuk flashing firmware di sini ATAU yang ini terlebih dulu. Jangan lupa download juga Driver USB MTK6575nya di sini. Siapkan juga camilan dan kopi, anda bisa mendownload di toko terdekat :D.
Lihat Gambar Proses Flashing Picopad 5 GEA diatas dan ikuti langkah langkah berikut :
  1. Ekstrak file NXGStockROMICS4.0.3Xfab.zip atau Stock ROM yang lain.
  2. Matikan Phablet Picopad anda dan sambungkan ke komputer menggunakan kabel USB.
  3. Install driver USB MTK6575
  4. Cabut kabel data dari PC, dan biarkan Picopad tetap dalam kondisi off alias mati
  5. Jalankan SP Multiport Flash dan lanjutkan dengan langkah berikut:
  6. Buka menu File > Open Download Agent dan pilih file ber-ekstensi .BIN (MTK_Allinone DA.bin).
  7. Buka Factory Database -> cari file dengan extensi .MDB (MTK_DATABASE.mdb)
  8. Klik Scatter file dan cari file scatter dengan extensi TXT (MT6575_Android_scatter_emmc.txt)
  9. Pilih opsi High speed dan No battery atau Battery sesuai keadaan Phablet Picopad anda
  10. Pada opsi USB Port pilih BootRom DA (No Battery) atau Preloader DA (Battery)
  11. Pada kotak Sensor1 isikan dengan ov5647_raw, Isi kotak Sensor2 hi704_yuv, Sensor3 ini adalah Display driver, isikan saja T050UWV014NS atau jika gagal isi dengan LM050AMYBO1E
  12. Hubungkan kabel data USB dengan Komputer/Laptop terlebih dahulu.
  13. Klik tombol start allSekarang tancapkan Kabel data USB ke Phablet Picopad 5 GEA
  14. Tunggu hingga proses selesai sambil minum kopi dan makan snack, rileks saja.
  15. Setelah proses flashing selesai, anda bisa mencabut kabel data dan cabut baterai.
  16. Pasang lagi batera, kemudian nyalakan kembali Axioo Picopad 5 GEA anda.
Jika proses flashing firmware ini berjalan lancar, maka Phablet Axioo Picopad 5 GEA anda sudah kembali ke Original Stock ROM.
