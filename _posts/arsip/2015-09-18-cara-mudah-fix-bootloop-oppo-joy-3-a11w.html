Oppo Joy 3 sendiri menggunakan chipset Mediatek MT6582 dengan CPU quad core 1.3 Ghz dan RAM 1 GB dan GPU Mali 400MP2. Artinya seharusnya flashing bisa dilakukan dengan SP Flashtool.



Tetapi, kebetulan saya ada cara yang mudah dan tidak memerlukan software (SPFlashtool) untuk melakukan flashing. Metodenya sangat sederhana yakni dengan menggunakan file OTA update resmi dari Oppo mobile sendiri. Jadi untuk mengikuti tutorial yang saya berikan, ponsel anda harus mempunyai stock recovery (native oppo recovery). Jika anda sudah menggunakan custom recovery sebaiknya anda melakukan flashing recovery melalui fastboot mode.

Flashing Firmware OPPO Joy 3 A11W via Stock Recovery

Saya anggap anda masih menggunakan stock recovery di ponsel anda. Jika anda sudah menggunakan custom recovery (entah itu CWM maupun TWRP) lihat bagian mengembalikan stock recovery di bawah.
1. Download Firmware Oppo Joy 3 dibagian download.
2. Letakkan file zip yang sudah di download di external sdcard dan masukkan ke dalam ponsel.
3. Masuk ke recovery mode dengan menekan tommbol power dan volume down (atau volume up, saya lupa) secara bersamaan hingga masuk Oppo Recovery.
4. Dalam Oppo Recovery mode, pilih bahasa English jika ada pilihan bahasa.
5. Masih dalam Oppo Recovery, pilih menu wipe data and cache. Pilih Yes jika ada pertanyaan. Tunggu hinga selesai, dan pilih ok.
6. Lanjutkan dengan memilih menu Install from sd lalu pilih From sd dan arahkan ke lokasi file Stock ROM Oppo Joy 3 A11W yang sudah anda letakkan di sdcard sebelumnya.
7. Pilih Yes saat ditanyakan apakah ingin meng-install update ini. Dan tunggu hingga proses selsesai.
8. Setelah proses flashinf selesai, pilih Reboot agar ponsel menyelesaikan proses flashing dan masuk ke syatem.

Saat boot pertamakali setelah flashing biasanya memakan waktu agak lama (sekitar 5 menit) karena android sedang mempersiapkan file file yang dibutuhkan seperti menyusun dalvik cache dan menyelesaikan pembersihan script update. Dan jreng... ponsel Oppo Joy 3 anda kini seperti baru lagi.

Perlu anda ketahui, file stockrom Oppo Joy yang digunakan dalam tutorial ini adalah firmware terbaru yang dirilis tanggal 22 Juli 2015 dan di rilis untuk publik tanggal 3 Agustus 2015. Jadi, jika andabelum sempat meng-update Firmware ponsel anda, dengan mengikuti tutorial ini anda sekaligus melakukan upgrade firmware ke versi terbaru.

Mengembalikan Oppo Stock Recovery dari Custom Recovery


Untuk mengembalikan dari custom recovery ke recovery oppo, ikuti langkah berikut jika menggunakan CWM recovery:
1. Anda harus sudah mempunyai Minimal ADB Fastboot Tool
2. Download file oppo recovery dan rename menjadi recovery.img dan letakkan dalam folder adb fastboot.
3. Matikan ponsel dan nyalakan dalam recovery mode
4. Setelah masuk custom recovery (CWM) pilih menu reboot dan pilih reboot bootloader agar masuk mode fastboot.
5. Buka folder ADB Fastboot dan tekan dan tahan tombol [ctrl] + [shift] bersamaan dengan klik kanan di ruan kosong dalam folder dan pilih opend command promt here
6. Tancapkan ponsel ke komputer, tunggu hingga terdeteksi dan driver fastboot ter-install
7. Dalam Command Promt ketik perintah fastboot flash recovery recovery.img
8. setelah partisi recovery sudah di flash dengan stock recovery, ketik lagi perintah fastboot reboot
Seadangkan jika anda menggunakan TWRP recovery atau Philz Recovery anda bisa melakukan install image dan meilih partisi recovery sebagaitujuan flashing dan mengarahkannya ke file recovery.img kemudian melkukan flashing partisi recovery.
