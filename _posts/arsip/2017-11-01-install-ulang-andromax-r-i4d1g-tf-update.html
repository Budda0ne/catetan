Andromax R atau Haier I4D1G termasuk generasi awal jajaran ponsel bundling Smartfren 4G di Indonesia. Hingga saat ini, penggunanya masih cukup banyak dan basis komunitas pecinta ponsel ini termasuk besar. Bahkan dukungan Official group untuk berdiskusi juga ada di Facebook loh.. Anda bisa ikut bertanya tanya di group tersebut jika ada masalah terkait jaringan atau official support untuk firmware. Namun, jangan sekali-kali bertanya soal bagaimana caranya unlock jaringan 4G agar bisa dipakai untuk kartu provider lain ya? Kamu bisa membaca artikel di sini kalau soal itu.

Nah, ada kemungkin juga saat melakukan unlock tersebut ada kendala mulai dari masalah koneksi yang jadi tidak stabil, atau bahkan mentok di logo smartfren a.k.a bootloop. Untuk itu ada bebeerapa cara untuk memperbaiki dan menjadikan Andromax R and kembali seperti semula. Salah satunya dengan install ulang (flashing) Andromax R pakai file TF Update ini.

Untuk melakukan flash kamu tidak membutuhkan bantuan PC/Laptop. Cukup sediakan sdcard kosong saja untuk menampung file TF Update versi 5.0 untuk Andromax R. Oh iya, jika kamu masih awam, mungkin Anda bertanya apa sih TF update? TF sendiri (setahu saya) singkatan darib Trans Flash yakni istilah lain dari MMC/sdcard. Jadi TF Update kurang lebih maksudnya melakukan update OS via memori eksternal. Nah, ternyata cara ini tak hanya untuk melakukan update, tetapi juga ampuh untuk melakukan install ulang OS Android untuk memperbaiki masalah bootloop, crash dan lainnya.

Namun jika masalah sudah parah, kadang cara ini tidak berfungsi/gagal. Untuk itu ada cara lain, silagkan baca: Cara Flash Andromax R I4D1G terdeteksi 9008 di PC Kita abaikan dulu, untuk saat ini anda hanya perlu fokus dengan langkah-langkah flash Andromax R I4D1G versi 5.0 dengan TF Update berikut ini:

Flash Andromax R I4D1G dengan TF Update

Sebelum melakukan flashing, siapkan dulu sdcard kosong. Hapus semua file didalamnya atau format saja dulu. Dan jangan lupa untuk mengisi baterai setidaknya hingga 60% agar tidak mati tiba-tiba dan malah gagal flashing.

Selanjutnya, tentu saja anda perlu menyiapkan file TF Update-nya terlebih dulu. Silahkan unduh di sini. Ekstrak file-nya dan salin ke sdcard langsung tanpa dimasukkan folder.

Sekarang matikan Andromax R anda dan sidipkan sdcard yang sudah berisi file TF Update tadi.

Selanjutnya tinggal anda tekan dan tahan tombol Volume Up + Volume Down + Power bersamaan. Jika kebetulan anda lupa tidak mengisi baterai ponsel, anda bisa menekan dan tahan tombol volume up + down bersamaan kemudian tancapkan ke charger.

Tunggu sebentar hingga ponsel menyala dan lepaskan jari Anda jika proses sudah mulai berjalan. Seluruh proses update berjalan secara otomatis.
