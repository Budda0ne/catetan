Setelah beberapa kali mencoba beberapa Custom ROM dari hasil porting dan sudah hampir 3 bulan memakai Stock ROM. Akhirnya tertarik juga dengan Custom ROM MIUI V.6 hasil porting dari agan Myname Ariefï»¿. Perlu di ketahui bahwa ini bukanlah MIUI6 tetapi MIUI5 versi 6. Jadi, nanti jangan protes sambil maki-maki ke saya.

Tampilan MIUI yang khas inilah yang membuat banyak pengguna ponsel dari vendor lain ingin mencicipinya. Tak hanya Evercoss A26C saja, tetapi banyak pengguna ponsel sekelas Samsung pun ikut mencicipi ROM MIUI.

Khusus ROM port MIUI v.6 untuk A26C mungkin anda akan sedikit terganggu dengan dialpad yang agak "menceng". Tetapi rasanya tidak ada masalah karena tampilan dan UI yang lain masih normal.

Performa ROM ini termasuk lancar untuk penggunaan harian. Jika ingin meningkatkan performa, anda bisa mencoba NiceTweak v.1 untuk meningkatkan kinerja ponsel dan PureGraphic untuk meningkatkan kualitas grafis.

Selain sudah mendukung dual 3G (switch), ada fitur-fitur yang ditambahkan dalam ROM ini diantaranya sudah dalam kondisi root dan init.d aktif. ROM porting ini berupa ROM deodex. Untuk anda yang lebih suka ROM odex seperti saya ada cara mudah untuk melakukan odex ROM. Jadi jangan kuatir.

Baca artikel ini untuk referensi mengenai ROM ODEX dan ROM DEODEX.

Dengan mengikuti tutorial instalasi ini, anda berarti sudah setuju dengan Disclaimer dan mengerti serta patuh dengan DMCA Copyright di blog ini. Dan anda juga paham bahwa dengan meng-install Custom ROM di handheld anda bisa menyebabkan garansi anda gugur resiko kehilangan data, backup data penting anda.

Syarat: harus sudah ter-install CWM Recovery, jika belum, baca di tutorial install CWM Recovery.

Untuk memasang ROM ini, anda harus melakukan Backup IMEI melalui aplikasi Mobile Uncle atau aplikasi backup IMEI lainnya, jika nanti terjadi invalid IMEI. Buat juga Nandroid Backup melalui CWM Recovery sebelum melakukan flashing.

Langkah-langkah instalasi:

Download ROM di bawah dan letakkan di sdcard (/sdcard/MIUI V.6 BETA JUICE BY Myname are EVE.zip) agar lebih mudah saat instalasi.
Reboot ke mode CWM Recovery (tekan tombol [Power] dan [Volume Up] saat menyalakan ponsel)
Dalam mode CWM Recovery, gunakan tombol [Volume Up] dan [Volume Down] untuk navigasi dan tombol [Power] untuk memilih menu:
pilih menu Wipe Data/Factory Reset pilih yes.
pilih lagi menu Wipe Cache Partition pilih yes lagi
pilih menu Advanced, pilih Wipe Dalvik Cache
Pilih Install Zip From SdCard kemudian pilih file MIUI V.6 BETA JUICE BY Myname are EVE.zip
Tunggu hingga CWM Recovery selesai melakukan proses flashing.
Setelah Selesai, pilih GO BACK (menu paling bawah) dan pilih Reboot System

Setelah ROM ter-install dan bisa booting dengan normal, anda bisa melanjutkan memasang update untuk fixing music player yang force close (FC) saat membuka playlist dengan cara:

Download fix update di bawah
dan letakkan di sdcard (/sdcard/MIUI V.6 BETA JUICE BY Myname are EVE.zip) agar lebih mudah saat update.
Lakukan seperti saat anda melakukan install ROM sebelumnya tanpa melakukan wipe data, cache dan dalvik-cache.
Setelah selesai, reboot ponsel anda

Jika Music Player sudah tidak FC saat membuka playlist, anda bisa melanjutkan memasang Dolby Juice untuk meningkatkan performa audionya atau anda bisa melewatkannya jika anda tidak suka (saya lebih suka Viper4Android FX). Caranya sama dengan saat melkukan update fix playlist, tetapi andaharus menghapus (atau memindah) file /system/etc/audio_effect.conf
