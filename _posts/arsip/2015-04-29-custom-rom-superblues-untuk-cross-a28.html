Setelah sekian lama tidak memegang ponsel Cross A28, hari ini saya dititipi kawan saya untuk meng-install Custom ROM untuk Cross A28 lagi. Dia meminta Custom ROM yang ringan dan terbaik untuk A28 miliknya.

Pikir saya, masa iya saya harus mencicipi satu persatu dari sekian banyak Custom ROM untuk A28? Apalagi saat ini sudah sangat banyak sekali Custom ROM baru untuk A28. Daripada pusing saya comot saja salah satu Custom ROM Superblues bikinan gan Izzy Bluessy. Yang agak mengecewakan, ROM ini terasa sangat berat ketika Facebook dan BBM berjalan bersama. Aplikasi sering berhenti tiba-tiba.

ROM Superblues ini mempunyai 2 file update. Update 1 untuk memperbaiki System UI dan Framework sedangkan Update 2 (final) untuk memperbaiki Login/Account Google. Yang kemudian saya repack (sudah ada izin) untuk menggabungkan 2 update tersebut kedalam satu installer saja agar lebih mudah dan singkat saat melakukan instalasi.

Selain menggabungkan 2 file update ke dalam ROM, saya tambahkan script dukungan init.d yang ramah SuperSU di bagian system agar tidak bentrok dengan fungsi OTA survival SuperSU yang baru agar membantu kinerja RAM. Untuk versi repacked rencananya akan saya jadikan 2 versi, odex dan deodex.

Baca: Update SuperSU v2.46 [stable] via CWM

Sedikit informasi tambahan, ROM ini base-nya adalah ICS. Jadi jangan terkecoh dengan versi android 4.4.2 yang ada di menu Phone info. Jika anda sudah terbiasa mengoprek android, anda pasti tahu kalau hal ini bisa dilakukan dengan merubah nilai di build.prop.

Install Custom ROM Superblues [repacked]


Sebelum melakukan instalasi, anda harus sudah meng-install CWM recovery.

Untuk melakukan instalasi, cara dan langkah-langkahnya sama saja dengan rom adul yang lain. lebih jelasnya, ikuti panduan berikut:
  1. Download dulu ROM Superblues [repacked] dari Download section dan letkakkan di /sdcard/disini
  2. Matikan ponsel anda dan nyalakan kembali (reboot) dengan menekan tombol volume down bersamaan dengan tombol power sekitar 5 detik agar masuk mode recovery.
  3. Dalam mode recovery, lakukan double wipe dengan cara pilih menu Wipe Data dan Wipe cache
  4. Masih dalam mode recovery, lakukan format partisi system agar bisa clean install dengan cara pilih menu mount and storage pilih format system. Langkah ini bisa dilewati.
  5. Kembali ke menu awal, dan pilih menu Install from sdcard dan cari file Superblues zip yang anda download sebelumnya.
  6. Proses instalasi akan berjalan menggunakan Aroma installer, anda tinggal mengikuti instruksi yang ada hingga selesai.
Jika proses sudah selesai, anda bisa reboot Cross A28 anda dengan memilih menu reboot to system. Boot pertama setelah install ROM akan sedikit lama, sekitar 5 - 10 menit, jadi yang sabar saja.

Berikan tanggapan anda di kolom komentar, positif atau negatif asal sopan. Jika ada masalah, sertakan juga code error yang muncul, saat anda menggunakan aplikasi apa dan seterusnya karena saya bukan dukun.
