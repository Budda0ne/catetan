Sudah tahu tentang Videoder? Yap! Ini adalah aplikasi untuk download video YouTube yang sedang naik daun. Hasil Download bisa berupa file mp3, jadi sangat berguna saat ingin mengunduh lagu-lagu yang ada di YouTube tanpa harus repot melakukan convert dari video menjadi mp3 sendiri. Bandingkan dengan Tubemate yang membutuhkan aplikasi tambahan agar file video YouTube menjadi file mp3.

Nah, saat saya baru-baru ini mencari aplikasi Videoder di Playstore banyak sekali aplikasi dengan nama Videoder. Aplikasi-aplikasi yang muncul dalam daftar rata rata berisi iklan. Padahal seingat saya, dulu Videoder itu aplikasi tanpa iklan. Dan sepertinya aplikasi buatan Rahul Verma tidak bakal ada lagi di Playstore.

Perhatian: mengunduh konten melalui aplikasi pihak ketiga seperti ini sebenarnya menentang ToS Youtube. Karena itulah aplikasi seperti ini kebanyakan hanya berumur singkat di Playstore. Mau tidak mau Anda mesti mencari file apk-nya.

Akhirnya saya putuskan untuk mencari saja aplikasi Videoder versi lama. Setelah mencoba beberapa versi lama Videoder, akhirnya saya memutuskan untuk mengunduh Videoder versi paling lama dan stabil. Yakni Videoder versi 11. Jika Anda juga sedang mencari Videoder versi lawas yang tanpa iklan, silahkan unduh apk di bawah ini dan pasang langsung di ponsel Android Anda.

Unduh saja via Mega atau via PCloud

Meskipun sedikit terganggu dengan pemberitahuannya update versi terbaru, saya lebih menyukai versi lama Videoder ini karena di versi 11 tidak ada iklan yang ditampilkan. Jadi lebih nyaman saat memakai Videoder untuk mengunduh lagu-lagu YouTube. Jika Anda lebih menyukai versi terbaru Videoder, Anda bisa langsung menuju situs resmi Videoder di sini

Meskipun sebelumnya saya bilang OG YouTube menjadi aplikasi terbaik untuk download video YouTube pengganti Tubemate, rasanya untuk saat ini sudah tidak lagi. Karena dengan adanya Videoder, Tubemate dan OG YouTube sepertinya layak untuk ditinggalkan.
