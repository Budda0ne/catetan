EFS Professional Tool

Back-up data partisi EFS menggunakan software EFS Profesional Tool:


Alat ini mudah digunakan dan memelihara, dikembangkan oleh anggota XDA, LiquidPerfection. Hal ini dikembangkan sedemikian rupa bahwa setiap orang dapat menggunakannya tanpa kesulitan, dan ini membuat cara termudah untuk backup data EFS Anda. Pertama-tama pastikan perangkat Anda berakar, jika tidak berakar maka akar perangkat Anda.

Disclaimer: Kami tidak bertanggung jawab atas kerusakan yang terjadi pada perangkat Anda. Anda sendiri yang tanggung risiko!!!

Langkah-langkah menggunakan EFS profesional untuk Backup EFS Data:

  • Download EFS alat profesional dari XDA (mirror) ke PC Anda.
  • Aktifkan USB Debugging di perangkat Anda dari Developer Options.
  • Instal driver USB yang sesuai untuk perangkat Anda di PC Anda.
  • Hubungkan perangkat Anda ke PC menggunakan kabel data USB asli.
  • Jalankan EFS Professional.exe sebagai Administrator.
  • Klik pada EFS profesional pada jendela yang muncul.
  • Muncul jendela lain, setelah perangkat terdeteksi, Anda akan di beritahukan model perangkat Anda dan semua info lain yang dideteksi di jendela tersebut (pastikan sama).
  • Klik tab Backup dan kemudian klik pada Device Filter dan pilih model ponsel Anda. Jika Anda tidak melihat apa pun di tab Filter coba untuk klik di tombol Refresh.
  • EFS profesional akan meminta anda untuk menunjukkan letak Partisi Sistem yang akan di back-up, klik Pilih Semua, lalu klik Backup dan data EFS-nya tersimpan di ponsel dan PC Anda.

Di PC, hasil back-up disimpan dalam folder EFSProBackup yang ada di folder EFS profesional dengan nama seperti ini: "GT-xxxxxxx-xxxxx-xxxxxx.tar.gz"

Cara mengembalikan (restore) data EFS menggunakan EFS Professional Tool:

  • Hubungkan perangkat Anda ke PC menggunakan kabel data bawaan.
  • Jalankan EFSProfessional.exe sebagai Administrator.
  • Sekarang klik pada Restore Tab dan pilih file backup yang Anda ingin mengembalikan.
  • Anda dapat memformat data EFS rusak saat dengan mudah.
    • Klik tombol Restore.
