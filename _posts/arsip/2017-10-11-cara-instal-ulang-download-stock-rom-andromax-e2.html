Cara Install Stock ROM Andromax E2+

Sebenarnya tidak bisa dikatakan install rom. Tetapi lebih tepat dikatakan melakukan restore rom Andromax E2+ karena menggunakan rom hasil back-up seperti sudah dijelaskan. Karena file nandroid backup dihasilkan dari TWRP recovery, maka anda pun harus menggunakan custom recovery yang sama untuk melakukan restore. Jika anda belum mempunyai TWRP recovery, silahkan buka tutorial pasang TWRP recovery Smartfren Andromax E2+ dan ikuti panduan di halaman tersebut. Sudah punya? Okay, silahkan lompat ke tutorial cara pasang StockROM Andromax E2+ di bawah saja.

Sebelum itu, silahkan backup aplikasi dan data penting menggunakan Titanium Back-up atau tool back-up lain kesukaan anda. Karena proses ini meskipun sudah tested namun tetap beresiko. Belum punya Titanium Backup? Silahkan ambil versi pro di sini

Ekstrak dulu file stock ROM (.zip) yang sudah anda unduh sebelumnya. Baca: Aplikasi Ekstrak File Zip/Rar

Silahkan matikan Andromax dan nyalakan kembali dalam mode recovery. Caranya, tekan ketiga tombol (power, volume -, volume +) secara bersamaan hingga muncul logo TWRP dan lepaskan semuanya.

Anda juga sebaiknya melakukan backup EFS / Modem terlebih dahulu. Demi keamanan IMEI/ESN ponsel. Pilih menu backup, centrang beberapa partisi yang dirasa perlu termasuk boot dan recovery. Hasil back-up akan ada di dalam folder /sdcard/TWRP/BACKUP/[deviceid]/[tanggal-backup]/

Jangan melakukan restore terlebih dahulu. Pilih menu reboot dan pilih system untuk menyalakan ponsel secara normal.

Buka folder /sdcard/TWRP/BACKUP/[deviceid]/[tanggal-backup]/ dan pindahkan isi folder ke folder lain, misalnya /sdcard/backupEFS/. Ini agar anda bisa memindahkan file stock rom yang sudah di ekstrak sebelumnya (langkah 1) keÂ dalam folder /sdcard/TWRP/BACKUP/[deviceid]/[tanggal-backup]/

Jika sudah dipindahkan, silahkan matikan Andromax E2+ dan masuk ke TWRP lagi (seperti langkah 2)

Di sini, anda boleh langsung melakukan restore. Namun ada baiknya melakukan wipe data (factory reset) agar tidak muncul masalah (tidak wajib)

Untuk melakukan restore, pilih menu Restore dan arahkan ke /sdcard/TWRP/BACKUP/[deviceid]/[tanggal-backup] proses akan berjalan jika anda sudah menggeser sluder kearah kanan. Tunggu hingga proses sukses atau done!

Silahkan pilih menu home dan reboot system untuk menyalakan kembali Andromax E2 plus anda. Anda juga boleh menggunakan opsi wipe dalvick and reboot to system. Sama saja.

Dan ingat, proses reboot pertama setelah restore biasanya memakan waktu lumayan lama. Namun stetelah itu, boot akan menjadi normal.
