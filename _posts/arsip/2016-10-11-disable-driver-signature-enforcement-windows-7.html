Dan awalnya saya tidak menyangka bahwa Windows 7 versi 64 bit juga sudah dipaksa menggunakan Digital Signature, sampai beberapa waktu lalu saya menggunakan kembali Windows 7 64bit (sebelumnya upgrade otomatis ke Windows 10 dan malah Crash/Bluescreen) mengingat sudah hampir 2 tahun menggunakan Windows 8/8.1 tak ada masalah.

Masalah Driver Signature Enforcement di Windows 7 ini, saya sempat heran juga. Sebab dulu (mungkin karena memakai 32bit?) tidak pernah mengalami hal ini. Tetapi ternyata memang ada informasi resmi dari Microsoft bahwa Windows 7 keatas versi 64 bit mulai menggunakan Digital Signature. Dan saya baru tahu beberapa hari yang lalu, kudet.

Disable / Enable Digital Driver Signature Enforcement Windows 7 64bit

Bagi anda yang kebetulan baru tahu bahwa Windows 7 juga ada fitur Driver Enforcement, saya hampir yakin anda bingung bagaimana cara disable (me-non-aktifkan) secara permanen. Kalau untuk Disable di Windows 8/8.1 dan Windows 10 mudah untuk di akses, bahkan bisa dari Start Menu/Control Panel.

Sedangkan di Windows 7, bisa anda temukan dari boot menu [F8] saat booting dan memilih opsi paling bawah. Sayangnya hanya sementara, sampai anda mereboot PC/Laptop anda.

Oke, untuk disable/non-aktif-kan Digital Signature Enforcement secara permanen, Klik kanan pada Command Promt dan pilih Run as administrator.

Saat jendela CMD terbuka, ketik perintah berikut dalam satu baris perintah:

bcdedit.exe -set loadoptions DDISABLE_INTEGRITY_CHECKS
Tekan enter, lanjutkan lagi dengan mengetik:
bcdedit.exe -set TESTSIGNING ON
Tekan enter, silahkan install driver yang sebelumnya kesandung masalah driver signature. Signing Enforcement non-aktif/disable setelah PC di Restart. Tandanya, tulisan Test Mode akan muncul di pojok kanan bawah.

Sedangkan untuk mengaktifkan lagi (enable) fitur ini, hampir sama bedanya di perintah akhir memakai perintah OFF. Secara urut:
bcdedit.exe -set loadoptions DDISABLE_INTEGRITY_CHECKS
Tekan enter, dan ketik lagi:
bcdedit.exe -set TESTSIGNING OFF
Akhiri dengan enter. Reboot PC/Laptop.

Ternyata tak serumit yang dibayangkan.
