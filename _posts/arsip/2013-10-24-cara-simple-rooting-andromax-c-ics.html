Karena harganya yang cukup murah, Andromax C langsung laris manis, meskipun performanya tidak begitu meyakinkan (menurut saya).

Nah, jika anda ingin meningkatkan performa dengan mengutak-utik system android, tentu Andromax C-nya harus di root terlebih dahulu. Langkah ini sekaligus meng-install CWM recovery di Andromax C, CWM Recovery berfungsi untuk memback-up data-data pada Andromax C yang bisa digunakan jika dikemudian hari terjadi sesuatu ketika mengedit file system ponsel Andromax C tersebut.

Karena itu, kali ini akan jelaskan dulu cara Rooting Andromax C sekaligus dengan Instalasi CWM Recovery. Lakukan Back Up ROM terlebih dahulu untuk mengantisipasi resiko terburuk.

Update 17 November 2013
Tutorial ini tidak untuk Smartfren Andromax C yang telah Upgrade ke Versi Jelly Bean.
Untuk Versi Jelly Bean bisa anda dapatkan di Cara rooting Andromax C Jellybean.

Seperti biasa, semua resiko yang terjadi akibat rooting Smartfren Andromax C ini sepenuhnya tanggung jawab Anda. Anda perlu mengikuti langkah-langkah ini penting untuk anda lakukan dengan teliti dan berhati-hati, agar tidak terjai hal-halyang tidak di-inginkan.

Lanjutkan dengan mempersiapkan bahan yang dibuthkan untuk rooting dan instal CWM di Andromax C berikut:
  1. CWM-MaxC-Temp.Zip ( Temporary ) - [ DopBox ]
  2. CWM-MaxC.Zip ( Permanen ) - [ DataFileHost ]
  3. Root-Max-C-via-Recovery.Zip ( Rooting File ) - [ DopBox ]

Setelah ketiga file di atas telah anda download, pindahkan dulu file tersebut ke folder /SDcard/ pada smartphone Andromax C anda. Setelah siap, anda bisa mengikuti langkah-langkah berikut:
  1. Matikan Smartfren Andromax C ( Hisense HS EG-902 ) anda. setelah shutdown nyalakan dengan masuk ke recovery mode dengan menekan tombol POWER dan VOLUME DOWN ( - )
  2. Pilih "Instal From SDcard" lalu pilih CWM_Temp.Zip ( CWM recovery temporer ) yang telah anda simpan pada folder /SDcard. ingat, setelah proses instalasi selesai jangan di reboot dulu.
  3. Lanjutkan dengan melakukan instal ROOT_Max-C-via-Recovery.Zip ( Rooting ) cara instalasi sama dengan langkah 3.
    setelah instalasi selesai lakukan "reboot recovery" dari menu Advance >> Reboot Recovery.
  4. Setelah kembali ke Recovery mode install CWM_MAX.Zip ( Permanen ) dengan cara yang sama seperti langkah nomor 3.
    jika instalasi berjalan dengan lancar lakukan "reboot system"
Cara simpel diatas bukan hanya rooting, tetapi sekaligus memasang CWM Recovery kedalam system Android Smartfren Andromax C anda. Untuk mengecek, anda bisa menggunakan aplikasi terminal emulator dan ketikkan "su" tanpa tanda petik. Jika muncul notifikasi permintaan akses root berarti rooting pada Andromax C sudah berhasil 100%. Jangan lupa untuk membackup Andromax C anda melalui CWM Recovery untuk jaga-jaga.
