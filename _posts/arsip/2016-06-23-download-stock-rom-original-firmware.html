Cara Unbrick Lenovo A6010-nya bagaimana? Nanti akan di jelaskan, mumpung ada tikus percobaan yang mampir di meja saya. Proses Unbrick ini juga bisa anda pakai untuk Lenovo A6000 dan device Lenovo lainnya yang bercipset Qualcomm.

Sebelum itu, perlu saya jelaskan sedikit mengenai 2 file firmware diatas. File dengan nama A6010_S036_151105_16G_ROW_WC5C.zip merupakan file stock rom yang tidak bisa langsung di flash melalui recovery, melainkan dengan QFIL QPST. Dalam beberapa kasus softbrick (bootloop) bisa memakai file system.zip yang bisa anda ekstrak dari dalam file tersebut.

Namun dalam kasus yang lebih parah, seperti tidak bisa masuk recovery bisa menggunakan file A6010_S033_150928_16G_ROW.rar yang di dalamnya sudah termasuk tool dan driver yang bisa digunakan untuk melakukan Unbrick Lenpovo A6010 antara lain:
  • Folder COUNTRY_CODE ( archives region )
  • Folder SW (firmware itself)
  • SW_tool folder (flasher)
  • Qualcomm_USB_Driver_Setup.exe

Oke, lanjut dengan step flashing Lenovo A6010. Silahkan Ekstrak Firware terlebih dahulu.

Untuk OS sebaiknya menggunakan Windows 7 saja agar lebih mudah saat pemasangan driver. namun jika anda menggunakan Windows 8 keatas, sebaiknya disable driver signature enforcement terlebih dulu sebelum melakukan instalasi driver. Jika belum tahu bisa lihat caranya di postingan ini.

Untuk langkahnya sebenarnya cukup mudah. Paling penting adalah anda HARUS meng-instal Driver USB Qualcomm, agar nantinya ponsel anda terdeteksi sebagai Qualcom HS-USB QDLoader 9008 di COM Port Device Manager.

Nyalakan Lenovo A6010 anda ke mode download dengan menekan tombol [POWER] + [Volume UP] dan pilih menu Download dan sambungkan ke PC/Laptop dengan kabel data USB. Bagaimanapun caranya harus terdeteksi. Jika langkah ini saja belum berhasil bagaimana mau melakukan flashing?

Nah, selanjutnya anda tinggal meng-instal Software QPST dengan benar. Install saja seperti biasa hingga selesai. Buka dari Menu > QPST > QFIL

Kini setelah anda membuka Tool QFIL (Qualcomm File Image Loader) dari Software QPST, Pada bagian Programmer patch pilih tombol Browse dan cari file MBN dengan nama prog_emmc_firehose_xxxx.mbn di folder SW (Firmware) dan pilih Open. Abaikan kolom Select Build di bawahnya.

Lanjutkan dengan memilih tombol Load XML dan pilih file rawprogram0.xml pilih open, Load XML lagi dan pilih file patch0.xml satu persatu.

Nah disinilah penting langkah awal tadi. Jika instalasi driver HS-USB QDLoader sukses, QFIL akan mendeteksi Lenovo A6010 anda sebagai Qualcom HS-USB QDLoader 9008 secara otomatis, jika tidak silahkan klik tombol select port dan pilih port yang bernama Qualcom HS-USB QDLoader 9008 sendiri.

Lanjutkan dengan menekan tombol Download untuk memulai proses Flashing.
