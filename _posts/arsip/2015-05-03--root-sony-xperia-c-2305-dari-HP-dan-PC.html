Cara Root Sony Xperia C 2305 dari PC/Laptop


Sebelum melakukan proses rooting, persiapkan terlebih dahulu hal-hal berikut:
  1. Aktifkan USB debugging di Xperia C2305 anada, caranya: masuk menu setting, pilih menu about phone, tap 7 kali di build number, kembali dan pilih developer option, centang opsi usb debugging, dan pilih yes
  2. Download & Install Sony Xperia C USB Drivers (Sony PC companion)(link ada di download section), tancapkan ponsel anda hingga PC/Laptop anda mendeteksi ponsel anda dan Driver ADB terpasang.
  3. Pastikan baterai ponsel cukup untuk melakukan proses rooting, jangan sampai kurang dari 50%.
  4. Sebaiknya back-up data penting anda, semisal kontak (sinkronkan dengan akun google anda), sms dan lainnya.
  5. Untuk proses root, download juga file Root via PC di bagian download.

Setelah hal-hal di atas sudah terpenuhi, lanjutkan dengan prosedur berikut:
  1. Matikan software Sony PC Companion dan Antivirus, karena akan menghalangi proses root.
  2. Tancapkan Sony Xperia C2305 anda dengan USB debugging aktif dengan kabel usb ke PC/Laptop. Tunggu hingga terdeteksi.
  3. Ekstrak file Root via PC yang anda download, buka dan jalankan Sony Xperia C Root.exe
  4. Proses root akan berjalan secara otomatis. Tunggu hingga selesai.

Jika instruksi di atas anda lakukan dengan benar, root Xperia C2305 sukses dengan mulus.

Cara Root Sony Xperia C 2305 dari HH/Ponsel Langsung


Jika anda tidak mempunyai PC/Laptop, anda juga masih bisa melakukan rooting langsung dari ponsel anda. Proses rooting ini tidak akan menghapus data. Meskipun demikian, sebaiknya anda juga melakukan langkah jaga-jaga dengan mem-back-up data penting anda. Untuk proses rooting dari HH/Ponsel, anda membutuhkan aplikasi framaroot, download yang versi 1.93. Lanjutkan dengan langkah-langkah berikut:
  1. Masuk menu Settings, pilih Security dan centang opsi Unknown Sources.
  2. Install aplikasi framaroot versi 1.93 yang sudah anda download.
  3. Setelah terinstal, buka framaroot, dan pada dua menu dropdown pilih opsi "Boromir" dan "Install SuperSU", pilih OK untuk memulai proses root.
  4. Jika sudah muncul pemberitahuan "Super SU Binary Installed" artinya proses rooting sudah sukses.
Reboot ponsel anda. Lihat di drawer aplikasi apakah sudah ada aplikasi SuperSU, jika belum anda bisa mendownload dan meng-install SuperSU secara manual dan Root Checker untuk mengecek status rooting
