Ganti Template Blog
Sekitar Sehari yang lalu saya di tanyai salah seorang teman fesbuk saya, "Masbro.. Gimana caranya biar blog saya keliatan cantik?" saya jawab "Ganti template" Dia tanya lagi "Caranya?". Karena saya memang belum pernah posting artikel cara ganti template blog saya jawab saja "Belum bikin". Karena ada waktu luang di malam Minggu ini, saya coba buat langsung tutorialnya dengan full gambar screenshoot dan sedikit keterangan.

Persiapan Mengganti Template Blog

Sebelumnya, persiapkan dulu template blog yang akan kamu gunakan dengan mendownload di beberapa situs/blog yang menyediakan template semacam zoomtemplate atau jika tidak mau repot silahkan download template yang saya download untuk membuat tutorial ini di sini. Simpan, kemudian ekstrak filenya di komputer Kamu

Sekarang pengerjaannya...

Langkah pertama, buka akun Blogger kamu, Login. Setelah kamu masuk akan disuguhi penampilan Dashboard Blogger seperti Tampilan 1 berikut dengan beberapa list blog kamu (jika kamu sudah punya beberapa blog)
Tampilan 1

Langkah kedua, lihat Tampilan 2 berikut :

Tampilan 2
- Tentukan Blog yang ingin kamu ganti, misalnya blog yang bernama "Cara Ganti Template Blog" (saya beri kotak ungu)
- Klik tanda segitiga kebawah (saya beri kotak hijau) maka akan muncul menu dropdown
- Pilih Template (saya tandai dengan kotak oranye)
Maka akan muncul seperti Tampilan 3 seperti ini :

Tampilan 3
Langkah ketiga :
- Jika kamu sudah mengerti dan sangat faham tentang CSS dan HTML tentu sudah tidak asing dengan peg-editan secara manual. Kamu bisa memilih menu Edit HTML (saya beri kotak hijau). tapi saya tidak menganjurkan untuk mengedit tampilan blog secara keseluruhan dengan cara online seperti ini, karena sangat beresiko (tampilan malah rusak). lebih baik mengedit secara offline dan menyimpannya dalam bentuk file XML.
- Nah, cara amannya, kamu klik saja menu yang ada di pojok kanan atas, Back Up/Restore (saya tandai dengan kotak warna ungu)
Berikutnya Blogger akan menampilkan halaman seperti Tampilan 4 di bawah :

Tampilan 4
Langkah keempat :
- Jika nantinya blog kmau sudah terpasang template kamu bisa mem-back-up template dulu dengan Klik tombol Download full template (saya tandai dengan kotak hijau muda) untuk mencegah hal-hal yang tidak diinginkan (tampilan rusak), untuk yang ini tidak saya bahas sekarang.
- Karena ini adalah pertama kalinya kamu mengganti template, langsung klik pada tombol Chose File (kotak biru muda) untuk menentukan letak file template yang sudah kamu download terlebih dahulu.
Kemudian akan muncul jendela Windows seperti Tampilan 5 berikut :

Tampilan 5
Langkah keempat :
- Tentukan letak file xml yang sudah kamu download tadi (kotak ungu)
- Pilih file yang berextensi (berakhiran) .xml (seperti dalam kotak oranye)
- Klik Open yang saya tandai kotak hijau.

Langkah kelima :
- Klik saja tombol Upload (saya beri kotak hijau) seperti pada Tampilan 6 berikut :
Tampilan 6

- Tunggu prosesnya selesai
Dan akan muncul seperti Tampilan 7 berikut :

Tampilan 7
Langkah keenam :
- Kamu sudah berhasil mengganti Template Blog kamu dan bisa mengecek dengan menekan tombol View Blog yang berada di tengah atas (saya beri kotak warna hijau) seperti yang terlihat di tampilan 7

Yang segera saja browser kamu akan memperliahtakan Antar-muka Blog kamu sudah berubah
Cotohnya seperti Tampilan 8 berikut :

Tampilan 8 : Preview Blog

Selamat!! Blog Kamu sudah jadi Cantik!! :p
