Cara Unlock Bootloader Sony Xperia Dengan Cara LEGAL

Berikut adalah instruksi yang saya terjemahkan secara bebas dari situs resmi unlock bootloader Sony mobile dan beberapa instruksi tambahan atau alternatif.
Langkah awal, Anda harus menge-cek bahwa perangkat anda mendukung service unlock boot loader dengan membuka menu Service, caranya: pada perangkat Anda, buka tombol dialer (Phone), lalu masukkan kode *#*#7378423#*#*. Jika Anda perangkat anda adalah tablet, klik di sini untuk melihat cara mengakses menu Service.
  1. Pada perangkat Anda, tekan kontak Layanan > Konfigurasi > Rooting Status. Jika Unlock Bootloader allowed mengatakan Ya, maka Anda dapat melanjutkan dengan langkah berikutnya. Jika tertulis Tidak, atau jika Status Hilang, perangkat Anda tidak dapat di-Unlock.
  2. Pada komputer Anda, klik Continue di bagian bawah menu  ini untuk memulai proses unlock boot loader.
  3. Pada informasi pop up yang muncul klik Ya, saya yakin, jika Anda masih itetap ingin meng-unlock boot loader.
  4. Klik Saya Setuju, di bagian informasi legal untuk mengkonfirmasi
  5. Masukkan nama Anda, alamat email yang valid dan 14 digit IMEI atau MEID atau nomor IDID (hapus digit terakhir jika tidak cukup) perangkat Anda Klik Submit. Anda dapat melihat IMEI atau nomor MEID dengan mengetik kode *#06# pada menu Dialer atau Phone.
  6. Anda akan diberi Key untuk Unlock Boot Loader melalui email Anda misal: 11234ABCD. Simpan untuk digunakan nanti.
  7. Download dan install SDK Android terbaru dari developer.android.com. atau gunakan AIO toolkit ini dan lewati langkah 8 dan langkah 9
  8. Download dan unzip file ini. Ini adalah file standar android_winusb.inf, dengan beberapa baris kode yang ditambahkan agar memungkinkan Fastboot untuk mendukung perangkat Sony Ericsson. Pastikan Anda menjalankan paket driver USB Google revisi 4 Android SDK atau lebih tinggi. Anda dapat melihat revisi saat Google USB Driver package di SDK Manager.
  9. Tempatkan file unzip dalam folder usb_driver, terletak di SDK Android > ekstra > folder google pada komputer Anda. Terima untuk menggantikan yang lama android_winusb.inf-file dengan file baru. Jika Anda tidak dapat menemukan folder usb_driver, silahkan install Packager Driver Google USB menggunakan manajer SDK Android.
  10. Matikan perangkat Anda.
  11. Pada perangkat Anda, tekan dan tahan tombol Fastboot pada saat yang sama ketika Anda menghubungkan perangkat ke komputer dengan kabel USB. Tombol Fastboot mungkin berbeda antara perangkat yang satu dengan yang lain. Lihat halaman tombol Fastboot Untuk informasi lebih lanjut tentang tombol yang digunakan pada perangkat Anda.
  12. Pada komputer Anda, Anda akan diminta untuk meng-install driver jika tidak menggunakan AIO ADB Fastboot toolkit . Arahkan ke folder usb_driver dan pilih file android_winusb.inf, lalu pilih file Android Bootloader Interface.
  13. Buka Command Promt dan masuk ke direktori dimana Android SDK anda tempatkan, atau jika menggunakan AIO Toolkit, buka program tersebut dan ikuti instruksi yang diberikan. Jika tidak, ikuti langkah 14 dibawah.
  14. Ketik kode fastboot.exe-i 0x0fce getvar version, dan pastikan bahwa Anda mendapatkan respon (misalnya "0.3"). Ini berarti perangkat Anda sudah terhubung dengan baik.
    15. Peringatan! Jika Anda melakukan unlock bootloader ini, Anda dapat membatalkan garansi resmi dan/atau garansi dari operator Anda. Jika Anda masih ingin membuka boot loader dari perangkat Anda, ketik-kan fastboot.exe-i 0x0fce oem unlock 0xKEY, di mana KEY adalah kunci unlock boot loader yang Anda dapatkan di awal tutorial ini.
  15. Jika sudah, silahkan lepaskan perangkat anda dan hidupkan kembali
