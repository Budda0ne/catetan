Cara Unsecure Boot Galaxy J1

  1. Ekstrak kedua file (odin dan file unsecure) yang tadi anda download dalam satu folder.
  2. Jalankan Odin3 v3.09.exe dari folder tersebut.
  3. Sekarang, matikan ponsel Galaxy J1 sekitar 5 - 10 detik
  4. Boot Galaxy J1 ke recovery mode dengan menekan tombol power dan volume down dan home secara bersamaan hingga muncul peringatan lepas ketiga tombol dan tekan lagi tombol volume up untuk masuk mode download.
  5. Kemudian, hubungkan Galaxy J1 dengan komputer menggunakan kabel data USB.
  6. Jika sudah terhubung, salah satu kolom ID:COM pada tool Odin akan berubah warna menjadi biru muda.
  7. Pilih file rooting dengan mengklik tombol "PDA" dan arahkan ke lokasi file "boot.tar.md5".
  8. Pastikan bahwa hanya pilihan "F. Reset Time" dan "Auto Reboot" yang tercentang di tool Odin.
  9. Setelah semua siap maka Anda bisa menekan tombol "Start" untuk melakukan unsecure.
  10. Jika Anda suda melihat kata "PASS" pada kolom ID:COM Odin (ponsel akan mati dan menyala kembali secara otomatis) artinya Samsung Galaxy J1 sudah unsecure.

Lepaskan Samsung Galaxy J1 dari PC dan setelah ponsel Anda menyala, lanjutkan dengan proses root.
