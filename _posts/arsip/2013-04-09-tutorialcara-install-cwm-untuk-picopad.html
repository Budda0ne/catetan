Tetapi ada juga cara untuk memasang/meng-install ClockWorkMod tanpa harus melakukan flashing.
Sebelum mengikuti Tutorial berikut, pastikan anda sudah melakukan Rooti-ng terlebih dahulu. Pada beberapa device dengan prosesor tertentu memerlukan perlakuan khusus dalam proses rooting, misalnya rooting Axioo PicoPad 5.

Jika Anda sudah melakukan rooting, yang perlu anda siapkan adalah 2 file berikut:
- CMW recovery
- Aplikasi MobileUncle bisa anda download di PlayStore atau dari sini

Sebelum melakukan proses selanjutnya, Pastikan baterai pada konsisi Fullcharge/Penuh.

Sekarang lanjutkan dengan mengikuti Cara Install CWM Recovery berikut:
  1. Install aplikasi MobileUncle
  2. Copy file CWM Recovery ke \mnt\Sdcard\ (di akar folder)
  3. Jalankan MobileUncle, pilih menu "Flash Recovery From SdCard"
  4. Pilih file CWM Recovery yang anda copy tadi.
  5. Jika muncul popup konfirmasi, pilih OK.
  6. Pilih Allow di pop-up Superuser request yang muncul.
  7. Proses installasi CWM memerlukan waktu beberapa saat.
  8. HH akan reboot ke menu Recovery yg Baru!
  9. Proses instalasi selesai.. CWM recovery sudah terinstal..
  10. Untuk menghidupkan HH Pilih "Reboot System Now"

Petunjuk untuk yang belum pernah menggunakan CWM : Jika anda sudah menginstal CWM, untuk masuk ke mode Recovery anda bisa melakukan langkah berikut:
  1. Matikan HH
  2. Tekan tombol power kemudian tekan tombol volume up scr bersamaan. .
  3. Silahkan dipilih menu yg akan digunakan.
  4. Tombol operasional : volume up, vulume down, power, back
