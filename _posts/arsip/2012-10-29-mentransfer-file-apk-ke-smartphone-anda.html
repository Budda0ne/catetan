Setelah anda berhasil download file APK lewat PC, Anda dapat memilih untuk mentransfer file APK untuk smartphone Android Anda melalui kabel USB. Itu tidak "keren", jadi sekarang saya akan mengajarkan Anda bagaimana menggunakan Lite WebSharing untuk dengan cepat mentransfer file (termasuk file APK) dari dan ke komputer Anda melalui setiap jaringan Wi-Fi.

Langkah 1:
Buka Google Play pada smartphone Anda dan download WebSharing Lite (versi lengkap, harga $ 2.99 memungkinkan Anda untuk meng-upload beberapa file, menelusuri media dan hal-hal seperti itu, tapi versi Lite benar-benar semua yang kita butuhkan dalam situasi ini) .

Langkah 2:
Aktifkan Wi-Fi pada smartphone Anda dan meluncurkan aplikasi Lite WebSharing. Setelah aplikasi di-load, masuk ke Menu -> Settings, lalu ketuk sandi Pemilik akun. Masukkan password baru Anda dan klik OK (password yang sama akan digunakan untuk semua koneksi masa depan, jadi pastikan itu sesuatu yang mudah diingat).

Langkah 3:
Kembali pada layar utama, tekan Start dan WebSharing akan menampilkan URL. Buka PC Anda, masukkan URL tersebut, maka password yang telah ditetapkan pada langkah sebelumnya. Jika Anda pernah menggunakan Windows Explorer sebelumnya (mana mungkin Anda tidak pernah?), Anda akan cukup kompeten bagaimana cara mentransfer file bolak-balik antara desktop dan smartphone Android.

Langkah 4:
Ingat file APK yang telah anda download pada Tahap 1? Hanya mentransfer file tersebut ke smartphone Anda dan Anda akan ditetapkan untuk pergi.

Anda berhasil!! Sekarang Anda tahu cara men-download aplikasi apapun dari Google Play pertama pada komputer Anda, kemudian ke smartphone Android Anda, sehingga melewati geo-lokalisasi, serta pembatasan lain yang mungkin Anda alami dalam apa yang pernah disebut Android Market. Mungkin Anda tertarik menggunakan Google Play Store 3.5.16).APK
