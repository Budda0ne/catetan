Pagi ini, saya kembali mendapatkan email newsletter dari microsoft --yang lagi-lagi terlambat saya baca :), yang isinya tentang jadwal rilis Windows 8. Microsoft secara resmi (seperti yang ada di email dan blog windows 8) mengumumkan jadwal perilisan sistem operasi Windows 8, pada bulan Oktober 2012.

Dalam Konferensi Partner Sedunia (Worldwide Partner Conference) pada hari Senin (9/7/2012) malam, Microsoft menyatakan bahwa jadwal perilisan Windows 8 untuk mitra pembuat hardware akan dilakukan pada bulan Agustus.

Sedangkan Windows 8 untuk konsumen baru dirilis akhir Oktober, bersamaan dengan peluncuran komputer perdana yang menggunakan Windows 8.

Di situs resminya, Microsoft juga memberikan kesempatan bagi pengguna sistem operasi Windows 7 ASLI bisa melakukan update ke Windows 8 Pro hanya dengan membayar 14,99 dollar AS.

Windows 8 versi beta atau disebut Consumer Preview mendapat sambutan yang cukup baik. Dalam sehari, Windows 8 Consumer Preview diunduh sampai 1 juta kali. Setelah itu Microsoft mengeluarkan Windows 8 Release Preview sekitar Juni lalu (untuk yang versi release preview ini saya memang tidak membuat artikel ataupun review). Silahkan men-download versi Release Preview dari Windows 8, berupa ISO Image Fileh dari halaman ini saja.

Ada banyak perubahan-perubahan yang terjadi/ditambahkan di Windows 8 seperti saya singgung di artikel saya sebelumnya. Seperti menggunakan interface (antarmuka/tampilan) Metro yang identik dengan tampilan petak-petak berwarna-warni, yang mewakili software maupun fitur. Windows 8 juga membuang tombol start, yang pada Windows versi sebelumnya selalu diletakkan di pojok kiri bawah monitor.

Microsoft juga merancang agar Windows 8 bisa berjalan di tablet. Hanya saja, Windows 8 untuk tablet ini akan dibuat dalam dua versi, yakni yang menggunakan prosesor berarsitektur x86, dan yang berarsitektur ARM.
