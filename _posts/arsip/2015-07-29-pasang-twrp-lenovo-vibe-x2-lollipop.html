Lenovo Vibe X2 ini tidak bisa masuk ke recovery mode. Masalah tidak bisa masuk recovery mode setelah upgrade ke Android Lollipop ini terjadi pada ponsel Lenovo Vibe X2 yang sudah root, sehingga update ke Lollipop hanya bisa dilakukan melalui TWRP. Hampir sama dengan yang terjadi di Lenovo A6000, dimana TWRP untuk Kitkat tidak bisa dipakai untuk Lollipop. Setelah upgrade ke Lollipop seolah olah custom recovery TWRP yang sebelumnya sudah pernah diinstall menjadi hilang.


Masalah ini tidak bisa diselesaikan dengan cara install recovery hasil back-up sebelumnya karena recovery tersebut juga hanya mendukung OS Kitkat saja. Jadi, mau tidak mau jika ingin menggunakan recovery bawaan pun harus meminta recovery bawaan dari Lenovo X2 yang sudah Lollipop. Meskipun sebenarnya tidak perlu karena yang anda butuhkan adalah TWRP yang sudah mendukung OS Lollipop

Anda bisa melakukan instalasi menggunakan SP Flash Tool untuk install custom recovery TWRP Lenovo Vibe X2 Android Lollipop. Anda bisa mengikuti turorial dibawah jika belum bisa malakukan flashing menggunakan SP Flashtool.

Sebelum itu, persiapkan dulu file dan tool yang diperlukan.

Driver Auto Installer

SP Flashtool Terbaru

TWRP 2.8.6.0

Setelah semua file diatas lengkap, install Driver Auto Installer di PC/laptop dengan cara ekstrak file driver yang telah anda download dan dobel klik di file install.bat, ikuti saja proses yang ada di CMD yang muncul sampai driver terpasang.

Ekstrak juga file TWRP_v2.8.6.0_X2-EU_5.0.zip. Yang anda butuhkan adalah file MT6595_Android_scatter.txt dan file recovery.img

Ekstrak SPFT yang sudah anda download, buka dan dobel klik di spflashtool.exe dengan klik kanan run as administrator

Setelah SP Flash Tool terbuka, klik menu Options - Option - Download dan Centang DA DL All with Checksum

Kembali ke jendela utam SP Flash Tool, masih di tab Download dan mode Download Only, klik tombol Scatter-loading dan cari file MT6595_Android_scatter.txt yang sudah anda ekstrak sebelumnya.

Jika sudah ter-load, seharusnya pilihan Recovery juga ikut dicentang dan lainnya tidak. jika tidak tercentang, klik di Recovery dan arahkan ke file recovery.img dan klik tombol Download.

Lanjutkan dengan mematikan Lenovo Vibe X2 anda. Untuk memastikan, lepas baterai senentar dan pasang lagi, jangan nyalakan.

Dalam keadaan mati, hubungkan ponsel ke komputer menggunakan kabel data. Maka proses pemasangan TWRP Lenovo Vibe X2 akan berjalan. Tunggu hingga muncul pemberitahuan sukses dengan munculnya tanda centang hijau (OK) tanda proses selesai.

Kini lepaskan sambungan kabel data, lepaskan baterai kemudian pasang lagi, jangan nyalakan.

Dari sini (masih dalam keadaan ponsel mati) masuk ke recovery mode dengan cara menekan tombol [power] + [volume up] + [volume down] secara bersamaan dan jangan sampai dilepas. Tunggu hingga muncul logo Lenovo, lepas ketiga tombol tersebut.

Seharusnya anda sudah masuk recovery mode TWRP. Thats it, Lenovo Vibe X2 anda kini sudah bisa masuk recovery mode dengan lancar kembali. Pilih menu Reboot dan pilih reboot System untuk menyalakan ponsel android kesayangan anda.
