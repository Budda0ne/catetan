Pusing karena hape china anda bootloop gara-gara kesringan oprek dan salah langkah? Solusinya ya mengembalikan ke kondisi sebelum bootloop kalau menggunakan CWM Recovery, atau flash ulang Custom ROM. Masalahnya kalau sudah terlamapu berat jalan terakhir sebelum ke SC ya Flashing Firmware ori menggunakakan SP Flashtool.

Di sini, anda bisa mendownload Stock ROM Stock ROM Evercoss A26C. File ini di read back dan dumping dari Evercoss A26C yang belum di root, jadi masih perawan ting-ting. Dengan melakukan Flasing file stock firmware ini akan menjadikan Evercoss A26C anda kembali menjadi sepeti baru beli.

Karena Firmware ini masih:
- NON UNSECBOOT alias STOCK BOOT
- NON CR alias STOCK RECOVERY
- NON ROOT alias bebas dari segala macam bau root (su binary, busybox, init.d dll.)
- ODEX NOT DEODEXED

Keterangan:
- File ini hanya untuk digunakan dengan software SP FLASHTOOL
- Apa itu unsecboot? unsecboot adalah istilah untuk partisi boot yang sudah Unsecure. dengan unsecboot, HH anda akan mudah di root dengan aplikasi rooting apapun.
- Apa itu CR? CR stand for Custom Recovery, bisa TWRP atau CWMR.

Tutorial Flasing Firmware Evercoss A26C:

  • Download bahan dibagian download
  • Extract firmware ke dalam satu folder, misalnya Flahsing A26C
  • Extract SP Flashtool
  • Install driver MT6572
  • Sambungkan HH ke PC tanpa baterai
  • Kmudian buka SP flashtool, jika sdh kedetek akan ada tulisan MT6572 pada Flashtool
  • Pilih scatter loading dan cari file scatter yg ada di folder Flahsing A26C
  • Otomatis flashtool akan meload file file lain yg ada di folder tersebut
  • Gunakan opsi Download Only
  • Centang semuanya KECUALI PRELOADER
  • Cek sekali lagi apa sudah benar semua, dan klik tombol Download
  • Tunggu sampai muncul gambar lingkaran hijau (OK) sekitar 10-15 menit
  • Sekarang cabut Kabel data dan Baterai, diamkan 5 detik.
  • Masukkan kembali baterai, dan nyalakan seperti biasa.

Opsi Saat Flashing dengan SP Flashtool:

1. Tetap menggunakan Recovery sebelumnya, maka pilihan RECOVERY jangan di centang.
2. Ingin menjadikan Custom Recovery menjadi Stock Recovery (bawaan), centang opsi RECOVERY
2. Ingin Unsecboot, rename unsecboot.img menjadi boot.img dan centang pilihan BOOT
3. Ingin Stock Boot dan Stock Recovery, file unsecboot.img jangan direname dan centang pilihan BOOT, (berguna untuk klaim garansi hardware)
