Cara Mengatasi Andromax G Bootloop dengan TF Update
Jika mengalami kejadian bootloop atau stuck di logo pada Andromax G ini, ada kok cara untuk memperbaikinya. Kesal boleh tapi jangan keterusan. Ikuti panduan singkat yang bisa anda praktekkan berikut:

Ingat, dengan mengikuti panduan ini berarti anda setuju dengan disclaimer dan dmca blog ini.
Sebelum memulai, download dulu File TF-Update Andromax G di bawah ini.

Untuk mendownload file TF Update Andromax G, pilih link mirror yang anda sukai dari link dibawah, file sama saja.
4shared

mediafire

google drive
  1. Setelah di download, ekstrak file tersebut. Nantinya akan ada folder dengan nama AD687G v.1
  2. Buka dan lihat isi folder tersebut, jika ada ada folder bernama AD687G v.1 lagi, cut dan pindahkan ke luar sehingga natinya urutan folder seperti ini /sdcard/AD687G v.1/system
  3. Pasang kembali sdcard anda ke dalam Andromax G dalam keadaan mati
  4. Jika sudah terpasang, tekan tombol volume up, volume down dan power secara bersamaan
  5. Sekitar 5-10 detik akan ada proses yang berjalan secara otomatis. Biarkan sampai proses selesai/li>
  6. Jika di bagian bawah layar Andromax anda muncul pertanda instalasi sukses, silahkan reboot ponsel anda dengan menekan tombol power agak lama, sekitar 5 detik

Jika proses yang anda lakukan sesuai urutan, dijamin ponsel anda akan sembuh seperti sedia kala tetapi dengan versi OS yang lebih baru. Semoga panduan yang saya berikan bermanfaat buat anda. Cara ini sebenarnya hampir sama dengan cara mengatasi bootloop di Andromax C Jika ada permasalahan saat mengikuti tutorial ini, sebaiknya sertakan kronologisnya (termasuk kode kesalahan atau hal hal lainnya) di kolom komentar agar saya bisa membantu, karena saya bukan dukun.
Catatan: Solusi ini bisa dilakukan jika anda belum pernah mengupgrade firmware atau sudah pernah upgrade tetapi versinya dibawah file TF Update ini.
