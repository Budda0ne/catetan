Di sini kamu bisa download Sony Flashtool 0.9.23.2 terbaru yang sekarang sepenuhnya mendukung Nougat FTF. Alat luar biasa ini memang sangat populer sebagai alat utama untuk mem-flash file FTF ke smartphone Android merk Sony Xperia. Update versi Flashtool-0.9.25.0 untuk Windows bisa di lihat di sini: Download Sony Flashtool 0.9.25.0.

Kamu dapat menggunakannya untuk mem-flash (install ulang) ROM stok atau ROM kustom apa pun yang dibuat untuk Sony Xperia. Sony Flashtool memungkinkan flashing dan memperbarui (upgrade) atau menurunkan (downgrade) versi ponsel Xperia kamu secara manual. Agar Sony Flashtool dapat dipakai, kamu harus sudah punya file FTF dari firmware sesuai tipe Xperia kamu dan mem-flashnya dengan mudah ke ponsel cerdas kamu.

Selama beberapa tahun terakhir, alat ini sudah mendapatkan pembaruan yang konsisten dengan semua firmware Android baru diluncurkan. Pembaruan Android Nougat dan Oreo (juga Pie) untuk smartphone Sony sudah diluncurkan. Dan sejak sebelum update Nougat dimulai, versi baru Sony Flashtool ini juga dirilis.

Versi baru Sony Flashtool 0.9.23.0 mendukung file Android Nougat FTF. Ini berarti Anda sekarang dapat memperoleh file FTF dari firmware Nougat dan mem-flash-nya menggunakan Sony Flashtool. Mem-flash file Nougat FTF tidak dimungkinkan pada versi Sony Flashtool sebelumnya.

Versi terbaru ini juga telah mengintegrasikan XperiFirm tempat Anda dapat mengunduh firmware terbaru Xperia Anda langsung dari dalam Flashtool.

Selain menginstal pembaruan Nougat, Sony Flashtool sekarang dapat mencadangkan partisi TA dengan memanfaatkan penggunaan dirtycow oleh rayman. Sekarang pengurai log USB juga menjadi semakin akurat.

change log:

  • Fix for Nougat sin files parsing. FT should now be able to flash Nougat FTF
  • A raw backup for all devices exposed to dirtycow exploit. TA backup using dirtycow by rayman
  • More accurate USB log parser for creating fsc scripts
  • In advanced mode, some new TA features: Viewer and custom TA file generator or flashing Added a new device property to tell if fsc is mandatory or not. Check of this property is done before flashing
