Rilis IGOS Nusantara 9

IGOS Nusantara S9 atau IGN S9 adalah nama baru untuk mengganti kode rilis X9 yang dianggap kurang cocok. Kode S9 merupakan singkatan untuk Server versi 9. IGN S9 adalah distro IGOS Nusantara yang dibuat dengan memakai basis distro versi Enterprise.

Distro untuk kebutuhan enterprise, memiliki ciri jadwal rilis yang panjang/lama, dukungan yang jauh lebih lama, memiliki fitur keamanan sistem yang baik, dan disertai penerapan teknologi yang telah teruji dan stabil. IGN S9 memakai basis Redhat Enterprise Linux 6.4 dan CentOS 6.4.

Sebagaimana IGOS Nusantara X9, Fitur-fitur tidak banyak berubah hanya ada beberapa penambahan dan fix bug dan update. Fitur keamanan di IGN S9 juga masih menggunakan Security-Enhanced Linux (SELinux). IGOS Nusantara S9 juga tetap mendukung penggunaan virtualisasi sVirt. (deatail penjelasan bisa dibaca di artikel IGOS Nusantara X9 [IGN X9] : OS Handal Untuk Server Perusahaan)

IGOS Nusantara S9 [ IGN S9 ] dapat digunakan untuk:

1. Server

Pemakaian basis RHEL/CentOS akan membuat IGN S9 memiliki karakteristik yang sama sebagai distro untuk enterprise. IGN S9 cocok dipakai untuk server. Service atau layanan yang dipasang di IGN S9 antara lain lain: Web Server, Mail Server, Proxy Server, OpenGeo, OpenStack, CloudStack, Virtualisasi dan sebagainya.

2. Desktop (khusus)

Pemberian dukungan update yang lebih lama (dibanding IGN Desktop) cocok untuk pengguna yang memakai satu desktop dalam waktu lama. Pengguna tidak memerlukan banyak update. Pembaruan/update hanya dilakukan untuk beberapa aplikasi saja.

Varian IGN S9


IGN S9 tersedia dalam bentuk CD instalasi dan LIVE. Terdiri dari:
IGN S9 CD instalasi menyediakan paket minimal (untuk masuk terminal saja)
IGN S9 LIVE32 untuk prosesor 32 bit
IGN S9 LIVE64 untuk prosesor 64 bit
IGN S9 full berisi Desktop Gnome, Firefox dan LibreOffice
IGN S9 mini berisi Desktop Gnome dan Firefox
IGN S9 openbox berisi OpenBox dan Firefox

Pengguna yang akan melakukan instalasi ke harddisk harus membakar ISO image IGN S9 CD instalasi ke CD kosong terlebih dahulu, baru kemudian melakukan booting dari CD dan instalasi ke harddisk. Selain itu, anda dapat juga memakai IGN S9 LIVE, caranya dengan mengunduh ISO image, kemudian membakar ISO image ke CD/DVD kosong. Agar proses lebih lancar dan cepat, disarankan ISO image LIVE dimasukkan/dibuat menjadi LiveUSB.
