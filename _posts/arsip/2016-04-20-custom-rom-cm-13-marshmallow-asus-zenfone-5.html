Tetapi sudah ada Custom ROM CM 13 Android Marshmallow untuk Asus Zenfone 5. ROM CyanogenMod UnOfficial yang di bangun oleh senior member xda developer: dgadelha dan masih dalam tahap experimental.



Karena ROM ini masih Experimental build, maka masih sangat banyak sekali Bugs yang cukup mengganggu. Dan yang paling mengganggu adalah bug Camera dan Suara tidak muncul saat melakukan panggilan. Selain itu masih ada beberapa bugs kecil lainnya.

Meskipun demikian, hal ini cukup menggembirakan. Setidaknya ada harapan untuk merasakan Android Marshmallow di Zenfone 5. Jika anda tertarik untuk ikut merasakan CyanogenMod 13 untuk Asus Zenfone 5 ini ada beberapa hal yang harus anda siapkan diantaranya:

1. ADB Tool, untuk meng-install ROM CyanogenMod karena ROM berupa 2 file image (boot.img dan system.img)
2. Download CyanogenMod 13 For ZenFone 5 ROM. Ada dua file, System dan Kernel.
3. Download GAPPS X86 Android 6.0 (rekomendasi: modul pico)
4. TWRP Recovery. Jika belum terpasang silahkan baca Tutorial Install TWRP Recovery Untuk Zenfone 5. Ini sekaligus Unlock Bootloader.
5. Jangan lupa PC dengan OS Windows 7 (recomended) dan juga kabel data.

How to Install CyanogenMod 13 on Asus ZenFone 5


Dengan mengikuti tutorial yang saya berikan di bawah berarti anda sudah membaca dan setuju dengan Disclaimer. Ingat, melakukan customasi rom selalu beresiko, mulai bootloop hingga brick.

Ok, setelah semua sudah bahan dan alat disiapkan, silahkan ikuti step by step berikut:

Install ADB Tool di PC/Laptop seperti menginstall aplikasi lain di Windows.

Ekstrak ROM CM 13 yang sudah di-download, anda akan mrndapatkan file system.img dan copy 2 file image tersebut (system dan boot) ke folder ADB Tool yang sudah anda pasang.

Rename masing masing menjadi system.img dan boot.img untuk mempermudah.

Copy GAPPS Marshmallow ke SDcard dan pasang lagi ke Zenfone 5 anda.

Turn Off your device!! Maksudnya matikan hapenya.

Nyalakan ke mode fastboot (bootloader) dengan cara menekan tombol Power bersamaan dengan tombol volume atas.

Tancapkan Zenfone ke PC saat sudah dalam fastboot mode.

Buka folder ADB Tool dan tekan tombol [ctrl] + [shift] jangan lepas dan Klik kanan di ruang kosong. Pilih Open command window here saat muncul menu.

Dalam jendela CMD, ketik perintah
fastboot devices
untuk melihat apakah sudah terhubung. Jika muncul serial number berarti sudah terhubung.

Lanjutkan dengan mengetik
fastboot flash boot boot.img
untuk update partisi boot.

Lanjutkan lagi dengan mengetik
fastboot flash system system.img
untuk melakukan pemasangan ROM CM 13.

Ketikkan lagi
fastboot erase data
untuk menghapus isi partisi data (wipe data)

Lanjut dengan
fastboot erase cache
untuk menghapus isi partisi cache (wipe cache)

Setelah selesai (setiap menekan enter dan sukses akan ditandai dengan OKAY) reboot ke system dengan perintah
fastboot reboot
tunggu hingga Zenfone 5 selesai booting ke system CM 13.

Aktifkan USB Debugging.

Kembali ke PC anda dan Reboot Zenfone 5 ke recovery mode dengan mengetik:
adb reboot recovery

Saat sudah masuk TWRP Recovery, pilih install zip dan cari file GAPPS dan swipe untuk mengkonfirmasi.

Tunggu sampai flashing selesai baru kembali ke menu utama. Dan pilih Format data dan cache.

Ingat, FORMAT bukan WIPE.



Rebooot to system dan rasakan Android Marshmallow di Zenfone 5 anda.
