TUTORIAL INI KHUSUS BLOG DENGAN PLATFORM BLOGGER/BLOGSPOT
Tujuan modifikasi script open id ini adalah menggunakan semua link dari artikel blog kita bisa digunakan sebagai Open ID.

Caranya cukup singkat. Pada halaman Dasbor Blogger anda masuk ke bagian Template dan Klik Tombol Edit HTML. Cari tag <head> dengan menekan keyboard [ctrl] + [f] dan ketik <head> kemudian tekan [enter].

Pada artikel tutorial Credential Could Not Be Verified di blog lain, biasanya anda akan disuruh untuk menyalin kode berikut tepat dibawahnya:
<link href='http://www.blogger.com/openid-server.g' rel='openid.server'/>
<link href='http://blog-anda.blogspot.com' rel='openid.delegate'/>

Memang tidak ada yang salah dan berfungsi dengan baik. Tetapi kali ini Anda tidak perlu bersusah payah mengetik ulang alamat blog anda, anda tinggal copy-paste-kan kode berikut tepat dibawah tag <head>:
<link href='http://www.blogger.com/openid-server.g' rel='openid.server'/>
<link expr:href='data:blog.homepageUrl' rel='openid.delegate'/>
Anda sudah lihat bedanya kan? pada bagian href='...' kita ganti dengan expr:href='data:blog.homepageUrl'.
Dalam platform blogger, expr mewakili expression untuk memanggil data XML yang sudah di definisikan dalam tag html yakni xmlns:expr='http://www.google.com/2005/gml/expr' dalam data tersebut, data:blog.homepageUrl dikembalikan nilainya ke alamat blog masing-masing dimana kode tersebut dipasang. Sehingga anda bisa mem-backup template anda saat ini dan mengaplikasikannya ke blog yang lain tanpa harus merubah link pada bagian href='...'.

Lalu apa single link dari postingan blog kita sudah bisa digunakan untuk berkomentar pada blog lain yang menerima komentar dengan OpenID? Anda coba lihat komentar di bawah. Terlihat ada yang mencoba untuk berkomentar dengan single link tetapi tidak berhasil.
Jika anda sudah merubah script openID delegate dengan kode XML Blogger, maka anda bisa berkomentar menggunakan OpenID yang berasal dari URL postingan anda, bukan URL halaman depan blog anda.

Triknya adalah dengan menghilangkan http:// baru anda masukkan sebagai OpenID.
