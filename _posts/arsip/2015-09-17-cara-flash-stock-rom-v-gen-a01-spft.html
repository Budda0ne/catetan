Tutorial Flashing Firmware Asli V-Gen A01

Sebelum melakukan flashing, pastikan sudah meng-install driver VCOM (MT65xx Preloader) dengan benar.
Download dan ekstrak Software SPFlashtool di lokasi yang mudah anda ingat.
Download juga file stock rom V-Gen A01 dan ekstak di komputer/laptop, di dalam folder SPFlashtool agar mudah di temukan.

Jika persiapan diatas sudah selesai, silahkan ikuti langkah-langkah mudah berikut untuk melakukan flashing.

1. Matikan ponsel dan cabut baterai, kemudian pasang lagi baterai tetapi jangan dinyalakan.
2. Buka program SP Flashtool SPFlashtool-xxx.exe dengan klik kanan run as administrator jika menggunakan Windows 7.
3. Setelah terbuka, klik di tomnbol scatter loading dan arahkan ke lokasi anda meng-ekstrak firmware V-Gen A01 dan pilih file mt6572-scatter.txt.
4. Setelah semua ter-load sempurna, pilh menu download dan tancapkan ke komputer/laptop ponsel V-gen A01 dalam keadaan mati dengan kabel data usb.
5. Jika driver VCOM (preloader) ter-install dengan baik, dan ponsel tidak berganti ke mode charging maka proses flashing akan berjalan.
6. Tunggo hingga proses flashing menunjukkan angka 100% dan memberikan pop-up lingkaran hijau tanda flashing sukses.
7. Setelah proses flashing selesai, cabut ponsel dari komputer, lepas baterai sebentar.
8. Pasang lagi baterai dan nyalakan seperti biasa.

Langkah-langkah diatas seharusnya bisa berjalan dengan baik jika anda teliti. Namun terkadang terkadang pada langkah ke 6 muncul notifikasi preloader has changed atau hal peringatan lainnya. Untuk mengatasinya, pilih menu Firmware Upgrade dilangkah ke 4, bukan Donwload Only. Dalam hal ini anda harus berhati-hati karena bila salah langkah ponsel anda bisa hardbrick alias dead boot.
