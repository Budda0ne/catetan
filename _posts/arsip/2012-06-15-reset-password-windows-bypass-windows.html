Metode 1: Bypass password Windows dengan Command Prompt


  • Nyalakan Komputer(PC, Laptop) Windows.
  • Sebelum masuk ke Windows tekan tombol F8 secepatnya hingga muncul Booting Options.
  • Pilih opsi Safe Mode With Command Promt, kemudian tekan Enter.
  • Tunggu hingga seluruh file yang dibutuhkan windows selesai diproses, dan muncul layar untuk login.
  • Masuk dengan akun yang setingkat dengan Administrator.
  • Setelah muncul Command Promt. ketikkan: net user kemudian tekan Enter. Semua akun yang ada di komputer akan ditampilkan.
  • Misalkan username yang akan di-reset: kolorijo, maka ketikkan: kolorijo 123456. (123456 adalah password baru untuk akun kolorijo)
  • Tunggu sampai proses yang berjalan selesai, maka password windows sudah di reset.
  • Login kembali dengan user kolorijo dan password 123456.
Nah dari sini kita bisa lihat bahwa yang kita butuhkan untuk me-reset password adalah AKUN SETINGKAT ADMINISTRATOR. Jadi jika akun yang ingin di-reset adalah akun administrator dan tidak ada akun lain (user lain) dengan tingkat otorisasi administrator, maka cara ini tidak bisa dilakukan.

Metode 2: Bypass password Windows dengan Tool Windows Password Unlocker Professional


Cara ini terbilang mudah untuk dilakuakan karena menggunakan tool untuk me-reset password. Kali ini saya akan menggunakan tool atau software yang bernama Windows Password Unlocker Professional.
Berikut ini adalah step by step untuk menggunakannya:
  • Download Windows Password Unlocker Professional dan install tool ini di komputer lain
  • Setelah selesai di-install, bakarlah (buatlah) sebuah CD Bootable atau USB bootable dengan ISO image dari tool ini.
  • Nyalakan Komputer yang ingin di reset menggunakan CD bootable atau USB Bootable yang sudah di buat tadi.
  • Selesai, password windows kamu sudah di-reset.
