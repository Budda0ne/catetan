Sudah lumayan lama tidak membagikan Custom ROM baru, sekarang ada ROM Color OS yang bisa anda cicipi. Custom ROM Color OS ini di porting oleh agan Herman Koeman (dev. Indonesia) dari Stock ROM Oppo Neo 7 (A33F). ROM ini sendiri berdasarkan Android versi 5.1.1 arsitektur 32 bit dan sudah dalam keadaan root. Jadi sangat pas dengan RAM 1giga-an milik Lenovo A6000. Meskipun demikian, ROM Color OS ini bisa juga dipakai untuk A6000 plus. Tentu dengan performance yang lebih baik mengingat kapasitas RAM 2x lipat.

ROM Color OS hasil port ini bekerja baik dan bisa dikatakan sempurna tanpa ada bug yang menggangu. Untuk kinerja ponsel juga lumayan oke. Kapasitas RAM bebas sekitar 400 MB, hampir sama dengan Stock ROM Lenovo A6000 Lollipop.

Tertarik untuk memasang ROM ini ke Lenovo A6000 anda? Download ROM Color OS dibawah:

ROM Color OS

GAPPS 5.0 32bit
Setelah file yang dibutuhkan (ROM dan GAPPS) siap, langsung ikuti Tutorial di bawah, syaratnya sudah memakai Stock ROM 5.0.2 Lollipop atau custom ROM lain berbasis Lollipop. Dan juga sudah meng-install custom recovery (rekomendasi: TWRP sevenmaxx Lollipop).

Ingat, Selalu lakukan backup dan baca disclaimer! Saya anggap anda sudah mengerti dan setuju jika mengikuti langkah instalasi si halaman ini.
  1. Download ROM Color OS 2.1.0i Lollipo Lenovo A6000/plus di atas.
  2. Letakkan di sdcard (eksternal). Sebaiknya diluar folder agar mudah ditemukan.
  3. Reboot A6000/plus Anda ke Custom Recovery (rekomendasi: gunakan TWRP Lollipop) dengan menekan tombol power + vol up dan pilih menu recovery atau bisa juga dengan menekan power + vol up + vol down bersamaan.
  4. Lakukan Backup via TWRP ROM Anda saat ini untuk jaga-jaga ada yang salah. Jangan abaikan langkah ini, tapi tidak perlu menulis amin dikolom komentar :p
Oke, sekarang eksekusi dengan melakukan langkah berikut:
  1. Pilih menu Wipe data dan konfirmasi penghapusan data dengan memilih Yes.
  2. Sekarang flash rom ini dengan TWRP dengan memilih menu Install Zip from SD card.
  3. Cari dan pilih file zip yang tadi anda salin ke sdcard.
  4. Konfirmasi permintaan instalasi dan tungg hingga proses pemasangan Color OS pada Lenovo A6000 Anda.
  5. Jika sudah selesai, pilih menu reboot to system dan konfirmasikan penghapusan cache dan dalvik-cache jika diminta.
Oke, Lenovo A6000 anda kini sudah menggunakan Custom ROM Color OS 2.1.0i yang di porting dari OPPO Neo 7 ini.

Langkah terakhir adalah pemasangan GAPPS agar anda bisa melakukan sinkronasi data yang terback-up di akun google anda. Caranya sama seperti sebelumnya, hanya file zip yang digunakan. File bisa di unduh di bawah. Atau bisa juga cari GAPPS Lollipop 32bit disitus lain.
