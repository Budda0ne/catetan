Cara Instalasi Billing Explorer ver DeskPro8 Windows 7 & Vista 2010

Instalasi di Windows7 atau Vista

Sebelum Instal Billing, Pastikan Untuk Merubah dahulu setting User.
  1. Masuk Ke Start – Control Panel
  2. Pilih User Account and Family Safety
  3. Pilih Lagi User Accounts
  4. Klik User Account yang ada, Misalkan Admin atau Apa saja yang akan menjalankan Windows ini.
  5. Pilih “Change User Account Control Settings”
  6. Scrool ke bawah sampai pilihan ada di Never Notify (tanda kotak ada di bawah)
  7. Tekan OK
Setelah itu, Baru anda Install Billing Client dan Server seperti instalasi di wondows XP seperti biasa.x20

Instalasi di Windows XP sp3

Cara instalasi sesuai dengan peruntukan/fungsi pc yang akan di install UNTUK SERVER Sebelum Instal Billing, Pastikan Untuk Merubah dahulu setting User. Masuk Ke Start – Control Panel Pilih Security Center – Windows Firewall Ubahlah menJadi Off Kemudian Install seperti biasa UNTUK CLIENT Install seperti biasa Pastikan semua fitur keamana aktif guna mencegah upaya pembobolan. Gunakan Software tambahan bila perlu, Seperti : Deep Freeze, Anti Exe, Winlock, dsb.
