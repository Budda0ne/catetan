Jubler Subtitle Editor
software untuk mengedit subtitle berbasis teks. Jubler dapat digunakan sebagai software untuk membuat subtitle baru atau sebagai alat untuk mengubah, memperbaiki dan menyempurnakan subtitle yang telah ada. Format-format subtitle yang populer digunakan dapat diedit dengan Jubler. Preview sub judul secara realtime atau pada saat peng-editan subtitle, spell-checker, mode terjemahan dan gaya beberapa pengeditan fitur utama.

Persyaratan/System Requrement
• JRE versi terakhir
• MPlayer untuk melihat sub judul
• aspell untuk memeriksa ejaan sub judul

Jubler sendiri adalah software open source di bawah lisensi General Public License (GPL). Jubler ditulis menggunakan Java 5.0 (alias Java 1.5.0) agar benar-benar multi-platform (digunakan dalam berbagai sistem operasi). Telah diuji digunakan untuk Linux, Windows XP (juga compatible dengan Windows 7) dan Mac OS X.


Tertarik untuk menggunakan Jubler sebagai editor subtitle? Anda bisa mendownload Jubler untuk Sistem operasi Windows, sekaligus juga Java Runtime-nya. sebelum meng-install Jubler, jangan lupa meng-install JRE terlebih dahulu, agar installer Jubler bisa di berjalan lancar tanpa harus mendownload JRE dalam proses instalasi.
