"Downgrade firmware Zenfone 6 setelah melakukan update KitKat untuk apa?" Mungkin akan ada yang bertanya seperti itu. Bisa jadi karena merasa bahwa OS KitKat lebih baik dari Jellybean. Tetapi bagi yang sudah terlanjur update ke KitKat dan ingin melakukan rooting akan merasa terbantu dengan tutorial yang akan saya berikan ini. Selain untuk tujuan agar rooting menjadi lebih mudah, beberapa user juga mengeluhkan masalah masalah yang muncul di update KitKat maupun Lollipop.

Seperti anda ketahui, untuk saat ini masih sulit melakukan rooting android kitkat. Meskipun pada beberapa device tertentu bisa dilakukan dengan bantuan tool rooting cina via PC ( rooting langsung dari HH dengan app rooting cina paling mentok sampai versi KK 4.4.2) Untuk KK 4.4.4 masih sulit di tembus meskipun menggunakan tool rooting via PC.

Dalam kasus Zenfone 6 (juga Zenfone 5 dan 4) malah lebih rumit lagi karena jika gagal melakukan rooting akan masuk mode demo. Meskipun bisa diatasi, masalah ini cukup menggangu juga. Dan aplikasi yang bisa digunakan adalah aplikasi RootZenFone.apk.

Selesai penjelasannya, sekarang ikuti langkah langkah untuk melakukan downgrade Zenfone 6 menjadi Jellybean 4.3 agar anda bisa lebih mudah melakukan rooting. Meskipun sebenarnya sudah ada cara untuk melakukan rooting Firmware V2.21.40.39

Langkah Downgrade Menjadi Jellybean 4.3

Setelah semuanya di download, lakukan langkah langkah dibawah secara urut dan teliti.

  1. Install Driver USB (bahan 1) dan ekstrak ADB Fastboot (bahan 2) di drive C PC/laptop anda. Agar lebih mudah beri nama folder ADB Fastboot dengan nama ADBF ( C:\ADBF )
  2. Pindahkan file image recovery (bahan 3) ke C:\ADBF dan rename menjadi recovery.img
  3. Ekstrak Firmware JB 4.3 yang anda download ke lokasi C:\ADBF dan rename menjadi update_jb43.zip
  4. Aktifkan USB Debugging Zebfone 6 anda di developers option, jika tidak ada menu developer option buka Settings - About - Software information kemudian ketuk 7x di build number untuk memunculkan.
  5. Colokkan Zenfone 6 ke PC/Laptop dengan kabel data
  6. Buka folder C:\ADBF. Tekan dan tahan tombol Shift dan klik kanan mouse, pilih Open command window here.
  7. Dari command prompt, ketik perintah: adb reboot bootloader. Ponsel akan mati dan masuk mode fastboot
  8. Dari sini, ketik lagi perintah: fastboot flash recovery recovery.img
  9. Jika proses selesai dan sukse masuk mode recovery, pilih menu Recovery dan tekan volume up dan down bersaman kemudian pilih menu aply update from adb
  10. Kembali ke command prompt di PC, ketik perintah adb sideload update_jb43.zip
  11. Jika proses flasing selesai, lakukan wipe data dan wipe cache dengan mengetik perintah adb wipe data dan adb wipe cache secara berurutan.
Selesai. Dan langkah terakhir adalah me-reboot Zenfone 6 anda. Ketik adb reboot. Proses booting akan memakan waktu yang lumayan lama.

Jika ponsel Zenfone 6 anda sudah menyala, anda bisa mengecek versi firmware anda yang sudah Downgrade ke versi 4.3 Jellybean.
