Contoh Salah Satu Boot Animation Custom

Mengganti boot animation adalah hal menarik utnuk anda yang doyan men-dandan-i handheld android anda. Animasi boot android secara default hanya menampilkan text-logo "android" dalam font khas-nya. Tetapi, sekarang ini rata-rata produsen gadget sudah mengganti boot animation sesuai dengan merek yang mereka pakai.

Nah, jika anda tidak "sreg" dengan tampilan animasi default tersebut, anda bisa menggantinya sendiri dengan boot animasi custom yang bisa anda cari di google atau di forum/group yang sesuai dengan handheld yangg anda pakai. Untuk anda yang belum tahu, jika anda melihat animasi yang tampil setelah boot logo dan sebelum android anda masuk ke system itulah boot animation.

Ingin tahu cara mudah dan cepat untuk mengganti animasi booting handheld android anda? Jika anda mengikti tutorial yang saya berikan di bwah ini, berarti anda sudah menyetujui disclaimer blog dan Hak Cipta blog ini.

Cara Mudah Ganti Boot Animation Android


Penuhi dulu syaratnya:

== Hadheld anda harus sudah Root
== Sudah punya file bootanimation.zip (usahakan kurang dari 5mb)

Langkah-langkah instalasi:

1. Install Root Explorer/Root Browser
2. Buka aplikasi Root Explorer/Root Browser
3. Buka folder /system/media/ cari file dengn nama bootanimation.zip dan rename menjadi bootanimation.zip.backup
4. Pindahkan bootanimation.zip.backup (cut/potong) ke folder di sdcard anda untuk cadangan.
4. Buka lokasi file bootanimation.zip pengganti yang anda punya.
5. Kopi-kan ke folder /system/media/--disini--.
6. Rubah owner menjadi: root:root
7. Rubah permission menjadi: rw-r--r--
8. Reboot dan lihat hasilnya.

Mudah kan? Cara ini lebih efektif daripada anda meng-install melalui CWM Recovery.

Jika anda mendownload file yang dalam instruksinya mengunakan CWM recovery, tentu saja anda harus meng-ekstrak file bootanimation.zip dari file zip yang anda download. Anda bisa mencoba cara di atas, terutama jika anda meng-install bootanimation via CWM dan mengalami kegagalan.
