Artikel cara menambahkan aplikasi di android emulator ini adalah lanjutan dari artikel sebelumnya mengenai cara menginstall Android emulator pada PC. Pada bagian ini, saya akan menjelaskan bagaimana menginstall/menambahkan aplikasi android (yang berekstensi *.apk) ke dalam gadget Android Virtual kita. Kenapa hal ini penting? Khususnya developer, hal ini bermanfaat untuk mencoba aplikasi buatannya sebelum dicoba ke gadget Android langsung. Untuk end-user, mungkin dipakai untuk mencoba aplikasi-non-Market (sekarang Google Play Store) sebelum dicoba ke gadgetnya. Berikut step by step cara install file APK ke Android Emulator.



1. Jalankan Android Emulator.


Tampilan Android Gingerbread dari Android Emulator

2. Perhatikan :
lokasi folder dan nama folder Android SDK kamu. ( contoh : F:\android\)
letak adb.exe (defaultnya terletak di folder platform-tools, lengkapnya di F:\android\platform-tools), klo tidak ketemu, atau males nyari, silakan download saja dulu. Lalu ekstrak di folder Tools Android SDK.
lokasi di mana file apk berada (misalkan anda letakkan di F:\android\iQuran_Pro_v2.1.2.apk )

3. Jalankan Command Prompt Windows ( Windows + R > ketik cmd ), Coba menuju lokasi adb.exe tadi, dengan cara :
F:\>android\platform-tools\adb.exe
4. Install APK, dengan perintah :
<lokasi file adb.exe>adb install <lokasi file apk\nama_file.apk>
misalnya kita ingin install APK dari aplikasi iQuran Pro, maka perintahnya :
F:\android\platform-tools\adb install F:\android\iQuran_Pro_v2.1.2.apk
akan muncul notifikasi :
175 KB/s (24496914 bytes in 135.964s)
pkg: /data/local/tmp/iQuran_Pro_v2.1.2.apk
Success
5. Cek di Android Emulator, akan muncul aplikasi iQuran.


Aplikasi iQuran Pro berhasil terpasang di Android Emulator.


Aplikasi iQuran Pro berhasil dijalankan
