Pada umumnya, template blog di blogspot default mempunyai fitur untuk menambahkan Add Widget Footer di dalam pengaturan Tata Letak. Namun apa jadinya jika anda sudah mengganti template bawaan blogspot tersebut dengan template custom, dan tidak menemukan tombol 'Add Widget' dalam mode pengeditan 'Tata Letak'? Jika anda sudah mahir meng-edit template dalam mode HTML editing, mungkin akan mudah untuk menambahkan footer atau komponen apapun dalam blog anda.

Untuk mempermudah anda yang tidak begitu mahir dalam meng-edit template dalam mode HTML, perlu membuat menu Add Widget Footer sendiri yang muncul dalam mode peg-editan Tata Letak.

Tetapi untuk membuat ini, anda perlu bersusah payah dulu sebelum menikmatinya. Anda harus menggunakan edit template dalam mode HTML, tetapi akan saya buat tutorial ini sesedikit mungkin menggunakan mode HTML.

Step by Step

Untuk menambahkan menu add widget footer dalam mode edit Tata Letak, ikuti langkah-langka berikut:

Langkah pertama

Ppertama kali, kita definisikan dulu posisi munculnya widget tersebut dengan kode-kode CSS.
Buka pengaturan>>template>>ubahsesuaikan>>tingkatlanjut>>[lihat gambar]
dan tuliskan kode CSS berikut
#widget-footer {
clear:both;
height:auto;
margin-top:10px;
padding-top:5px;
background:#1A1A1A;
border:1px solid #333;
}
#content-footer {
float:left;
padding-left:10px;
}
Simpan terlebih dahulu.

Langkah kedua

Sekarang untuk membuat Add Gadget di dalam pengaturan Tata Letak Blogger.
sebelumnya buka dulu edit template,>>template>>edit html>>lanjutkan
dan cari 
<!- - end content-wrapper -->
karena biasanya dalam template yang umum beredar selalu di sertakan komentar ini. Jika template anda tidak menyertakan komentar akan sedikit merepotkan, tapi kita anggap ada terlebih dahulu.
kemudian letakkan kode berikut di bawahnya, ingat! Tepat di bawahnya.
<div id='content-wrapper'>
<b:section class='widget-footer' id='content-footer' preferred='yes' showaddelement='yes'>
<b:widget id='HTML12' locked='false' title='test' type='HTML'/>
</b:section>
</div>
Simpan template anda.
setelah ini, anda tidak perlu lagi membuka edit template untuk menambahkan footer

Sekarang coba masuk Menu Tata Letak, sudah muncul menu Add Widget di bagian footer bukan??

Selamat mencoba.
