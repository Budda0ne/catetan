MultiSystem adalah sebuah alat yang sangat bagus. Tool ini diciptakan oleh LiveUSB.info, yang cara kerjanya hampirsama dengan tool berbasis Windows: MultiBootISOs USB, tetappi dibuat untuk pengguna Linux. MultiSystem juga menggunakan grub2 bukan GRUB, dan dapat dioperasikan dari OS Ubuntu Linux untuk membuat Multiboot UFD Bootable Linux Live kustom yang berisi distro-distro linux favorit Anda.

Perhatikan, MultiSystem adalah tool untuk OS linux. Bukan untuk digunakan di windows. Jika Anda menggunakan OS Windows, anda bisa menggunakan XBOOT utnuk membuat Multiboot Live USB.

Membuat Multiboot ISO USB dari Windows dengan Tool MultiSystem

  • Download and Open the Script with Ubuntu"s Archive Manager
  • Extract the install-depot-multiboot.sh Script to your Desktop
  • Double Click install-depot-multiboot.sh from your desktop and select the option to "Run in Terminal"

Cara menggunakan Multisystem LiveUSB Tool

  • Insert a USB Drive
  • Launch the Script via Applications > Accessories > MultiBoot
  • Select your USB Device from the list and click Validate
  • Now simply Drag and Drop an ISO into the Window to add the Distribution (repeat the process to add more ISOs)

Ketika sudah selesai, restart PC anda, rubah settingan BIOS agar boot dari USB.
Nah, ketika PC sudah di bootinng daru USB Pilih Distro Linux yang anda inginkan.
