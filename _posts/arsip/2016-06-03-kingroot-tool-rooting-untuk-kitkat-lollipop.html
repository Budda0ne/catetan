Versi terbaru Kingroot saat ini versi 5.2.2 Release 14 Semptember 2017. Kingroot kini mengklaim bisa melakukan root Marshmallow dan Nougat (beberapa device) setelah sebelumnya meng-klaim bisa melakukan rooting Android Kitkat hingga Marshmallow di versi 4.9.0 terutama untuk yang Kingroot versi PC (untuk hasil yang terjamin, sebaiknya menggunakan versi PC).

Untuk versi yang lama (versi 4.1.0 dan sebelumnya) sebenarnya bisa juga melakukan rooting di device dengan OS Kitkat (hanya gagal di beberapa device) namun di OS Lollipop sering tidak berhasil.

Kingroot sendiri merupakan tool rooting yang paling banyak prosentase kesuksesannya saat root android kitkat (framaroot seringkali gagal, framaroot versi terbaru sekalipun). Tool rooting ini mempunyai interface yang simple dan sangat mudah di gunakan meskipun oleh pengguna baru.

Meskipun menggunakan bahasa cina, saya yakin anda tidak akan bingung karena desain interface aplikasi Kingroot ini sangat sederhana.

Up# date: untuk versi 4.0.0 tombol root berbentuk kotak berwarna biru muda di bawah lingkaran. Saat tombol root di-klik/sentuh lingkaran akan menampilkan prosentase proses root. Sudah saya coba gunakan untuk root Lenovo A6000 Kitkat 4.4.4 dengan firmware update terbaru.

Daripada anda memendam perasaan dongkol kepada saya, lebih baik saya kasih screenshoot agar lebih jelas dan terang cara rooting dengan kingroot.

Sangat mudah, semudah ketika anda buang hajat (piss gan..), tinggal buka celana dan jongkok, proses pun berjlan secara otomatis. Aplikasi Kingroot meng-install Kinguser sebagai manajemen hak akses root. Setelah Aplikasi Kinguser ter-instal anda bisa menghapus aplikasi Kingroot. kasusnya sama seperti ketika anda melakukan rooting dengan Framaroot dan setelah Aplikasi SuperSU ter-install anda bisa menghapus Aplikasi Framaroot.

Beberapa hal yang perlu diperhatikan:

  • Kingroot App tidak membutuhkan persyaratan khusus untuk digunakan dalam proses root Ponsel/Gadget anda. Cukup install aplikasi ini dan buka kemudian tekan tombol ROOT (dalam lingkaran besar berwarna hijau) Selesai.
  • Anda harus menyalakan koneksi internet, karena proses root sekaligus men-download Aplikasi Kinguser Dan exploit system yang cocok. Terkadang root strategy unavailable juga Karena koneksi data tidak stabil.
  • Tidak akan ada yang namanya resiko kehilangan data dalam proses rooting. Jika anda menggunakan KingRoot App untuk root ponsel anda, maka tidak satupun dari data ponsel Android (semisal kontak, gambar, maupun video, musik dll) terpengaruh.
  • Garansi Hangus: Bukan hanya karena menggunakan Kingroot App. Tetapi saat melakukan rooting dengan Tool Root lain juga hangus karena masalahnya bukan tool yang digunakan tetapi proses rootingnya.
  • Credit: Aplikasi Kingroot dibuat dan dipublikasikan oleh Kingroot studio secara Gratis. Jadi semua kredit dan terimakasih hanya untuk mereka.
  • Jika anda mencoba menggunakan tool Kingroot dan berhasil, silahkan tulis nama dan type device anda beserta versi android anda di kolom komentar. Sama halnya jika ternyata gagal
Sedangkan untuk instalasinya, pastikan anda mencentang pilihan "Unknown Source" atau "Sumber tak dikenal" dalam Security settings handheld anda. Dan juga aktifkan USB Debugging agar meningkatkan rasio keberhasilan proses root.

Namun sekali lagi saya ingatkan: Tidak semua device bisa di root tanpa PC dengan mudah. Jadi, what's next? yah.. Silahkan request dengan mencantumkan Brand dan type ponsel anda. Jika memang ada dan bisa di root, akan saya buatkan tutorialnya di posting baru.

Atau bisa mencoba root menggunakan Kingroot versi PC, cara root juga mudah seperti versi android-nya.
