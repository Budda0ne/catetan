Racun-racun yang ditawarkan ViperOS sebenarnya tidak banyak berbeda dengan fitur RR. Beberapa fitur terletak pada Features Bar Customization misalnya: Customizable icons, Quick pulldown, Smart pulldown, Notification count, Clock position, Seconds in clock, Date design juga ada pada Ressurrection Remix.

Sementara pada bagian Quick Settings ada pengaturan seperti: Brightness slider, Brightness icon, Tile tap customization, juga Header customization. Dalam System Viper OS juga ada pilihan untuk kustomasi Power menu.

Ada juga ViperOS untuk Lenovo A7000 dan ViperOS untuk Xiaomi Redmi 3.

ViperOS untuk Redmi Note 3 bisa dipasang melalui TWRP dan hanya bisa dipasang pada device dengan codename KENZO. Mengenai codename, bisa anda baca pada artikel Daftar Codename Xiaomi dan Fungsinya. INGAT! Jangan install pada Redmi Note 3 Mediatek!

Cara Install ROM ViperOS untuk Redmi Note 3

Jika anda ingin mencoba "racun" dari ViperOS, anda bisa mengikuti tutorial instal ViperOS Redmi Note 3 dibawah. Sebelumnya, anda harus sudah melakukan unlock bootloader dan memasang twrp untuk redmi note 3 terlebih dahulu.

Download lebih dulu ROM ViperOS Nougat. Silahkan mengunduh file rom zip dari sini terlebih dulu. Pilih link download ViperOS terbaru yang terletak paling atas. Versi paling baru biasanya paling stabil karena sudah update dengan perbaikan-perbaikan bug.

Reboot ke recovery mode, caranya dengan mematikan Redmi Note 3 anda terlebih dulu. Kemudian tekan tombol Volume Bawah, Volume Atas dan Power secara bersamaan hingga bergetar dan lepaskan tombol Power saja. Tunggu hingga logo TWRP muncul, baru lepaskan semua tombol. Kini anda sudah masuk TWRP. Dengan beberapa menu utama.

Dari sini, pilih menu wipe dan geser Swipe to factory reset ke arah kanan. Tunggu hingga selesai dan kembali ke menu utama.

Sekarang pilih menu Install, cari file zip ROM ViperOS di sdcard (external) kemudian geser slider Swipe to Install ke arah kanan.

Tunggu hingga proses instalasi selesai. Selesai, pilih Reboot System.

Setelah menunggu sebentar (agak lama sebenarnya), ViperOS pun terpasang pada Xiaomi Redmi Note 3 anda.

Selanjutnya anda bisa melakukan beberapa konfigurasi akun dan pengaturan tambahan lainnya. Anda juga bisa melakukan kustomasi tampilan agar mantapnya pas. Selain itu, agar performa makin smooth, anda bisa melakukan berbagai penyesuaian Redmi Note 3 anda melalui berbagai menu tweaking yang disediakan oleh ViperOS.

Sedangkan bagi penggemar Xposed Framework, silahkan gigit jari, karena Android Nougat belum sepenuhnya didukung Xposed secara resmi. Jangan kuatir, anda bisa mencoba Magisk Systemless Root yang kini sudah punya banyak modul yang bisa si unduh langsung dari repositori. Contohnya Greenify dan Dolby Atmos.
