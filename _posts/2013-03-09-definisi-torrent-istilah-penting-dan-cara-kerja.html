Dalam arti dia tidak mempunyai server yang dedicated, maksudnya tidak mempunyai server hosting untuk file yang di download. Tracker sama sekali tidak mempunyai file yang dicari, tidak mempunyai data tentang isi file, karena itu walaupun dengan bandwidth tracker terbatas pun, koordinasi dapat tetap jalan. Filosofi dasarnya adalah user harus juga meng-upload data ketika mereka mendownload. Dengan demikian, kita dalam saat yang sama bertindak sebagai server data dengan kata lain semua user yang mendownload, akan otomatis menjadi server juga. Tentu saja kecepatannya akan berbeda dari download melalui HTTP langsung.

Torrent sendiri adalah file yang kecil, yang hanya sekitar beberapa kilobyte saja dengan ekstensi *.torrent. File ini berisi informasi yang dibutuhkan untuk mendownload file tertentu. Informasi yang terkandung dalam file torrent meliputi nama file, ukurannya, tempat downloadnya dan sebagainya. Perlu diingat bahwa cara kerja torrent adalah berbagi. Jadi kita perlu tetap membuka koneksi agar orang lain juga dapat mendownload file yang baru saja kita download melalui komputer kita. Lama koneksi ini bisa kita set, misalnya 1 jam setelah itu (100% terdownload) koneksi akan terputus secara otomatis.

Ada beberapa istilah dalam torrent yang perlu anda tahu, yakni SEEDER dan LEECHER. Seeder adalah mereka yang sudah mendownload 100% (lengkap) dan masih tetap membiarkan aplikasi torrent-nya berjalan agar pendownload lain yang belum selesai (yang disebut leecher) bisa mendownload dari komputernya, dan Leecher adalah mereka yang belum selesai mendownload (belum 100%). Meskipun belum mencapai 100%, file yang sudah didownload oleh Leecher bisa juga di download oleh Leecher lain.

Untuk lebih jelasnya, berikut ini pengertian istilah-istilah dalam torrent:
  • Seed adalah pengguna, yang telah selesai mendownload file dari torrent dan melakukan sharing.
  • Peer adalah semua orang yang sedang mendownload file, setelah download selesai, ia menjadi seed (seeder).
  • Leech adalah sama dengan peer, sering digunakan untuk menamai peer dengan UD (Upload/Download) ratio yang rendah. Sedangkan istilah Leecher ditujukan kepada mereka yang telah selesai mendownload sebuah file dan begitu selesai mendownload file tersebut mereka langsung disconnect atau menghentikan proses sharing file tersebut.
  • UD ratio adalah rasio download dan upload. Rasio upload dan download termasuk sedang, bila bernilai 1 (yang anda download sama dengan yang anda upload).
  • Tracker adalah server, yang mengkoordinasikan proses download semua user. Server tersebut menyimpan data tentang berapa banyak seeder, leecher, peers dan info lainnya dari sebuah file torrent. Tracker mengidentifikasi lokasi network dari orang-orang yang mendownload, upload/sharing sebuah file.
  • Swarm adalah simpulan dari semua seed dan peers (dengan demikian tracker mempunyai swarm untuk semua komputer yang pernah mendownload file torrent darinya).
File Torrent, berfungsi sebagai perantara agar file yang berada di lain-lain komputer dikenali. Mirip seperti limewire dan bearshare, hanya saja semua orang yang mendownload file dari torrent yang sama akan saling berbagi. Saling melengkapi file yang kurang atau belum terdownload.

Proses downloadnya sendiri ada 2 tahap. Yang pertama adalah mengambil file dengan ekstensi .torrent terlebih dahulu. File .torrent itu adalah file berisi informasi saja, seperti saya sebutkan sebelumnya. Nantinya, file .torrent itu bisa kamu load (open) dari aplikasi torrent client (misalnya; utorrent, bittorent, atau vuze/azureus). Kemudian, aplikasi torrent client itu yang akan mendownload file aslinya berdasakan informasi yang ada dalam file torrent tersebut.

Dari situs-situs penyedia file torrent (misalnya isohunt.com), ketika anda mau mengabil file .torrent tersebut, biasanya akan tertulis jumlah seed dan leech. Nah jika jumlah seed nya sedikit, kemungkinan besar speed downloadnya lambat. Apalagi kalau tidak ada seed sama sekali alias 0 (nol), file tersebut tidak akan pernah selesai anda download.

Setelah mengetahui apa itu torrent, tentu anda perlu tahu bagaimana cara menggunakan file torrent.
