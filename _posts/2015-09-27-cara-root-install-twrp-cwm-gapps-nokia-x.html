Baru ingat lagi kalau saya belum pernah posting artikel yang agak dalem berkaitan dengan ponsel android buatan Nokia ini. Satu-saunya artikel yang pernah saya posting hanya artikel ini, yang tergolong ringan dan menjemukan.

Nokia X tanpa Google Playstore memang masih bisa meng-install aplikasi selain dari nokia market. Tetapi bagaimana jika pengguna adalah penggila game yang membutuhkan akun google unuk back-up data permainannya? Misalnya saja, game populer semacam Clash of Clans. Tetapi sebelum itu, anda harus melakukan flashing GAPPS 4.1.2 ke dalam Nokia X anda. Yang tentu saja harus melakukan rooting atau minimal bisa bootke custom recovery agar bisa melakukan flashing GAPPS untuk Nokia X anda.

Jadi, secara garis besar yang bisa anda lakukan (pilih) adalah:

1. Melakukan rooting, dan meng-install paket aplikasi google secara manual.
2. Mem-boot kedalam custom recovery (sementara) dan melakukan flashing GAPPS 4.1.2 untuk Nokia X.

Untuk opsi pertama, bisa anda lakukan dengan cara:
1. Melakukan rooting dengan bantuan Kingroot di sini (gunakan versi 4.1.0) atau Framaroot dari sini (gunakan versi 1.9.2 atau 1.9.3).
2. Setelah berhasil rooting, dan Install Root Explorer dari sini.
3. Download paket GAPPS manual di bagian download, ekstrak di sdcard/internal dengan Root Explorer.
4. Buka lokasi ekstrak dan select semua file apk kemudian kopi ke /system/app.
5. Setelah terkopi semua, select all lagi dalam folder /system/app/ tersebut dan rubah permission-nya menjadi rwx r-x r-x
6. Sekarang matikan Nokia X anda dan lepas baterai agar benar-benar mati.
7. Nyalakan seperti biasa.

Seharusnya, System akan mengenali paket aplikasi baru yang belum dibuat file dalvik-cachenya dan melakukan ekstraksi (android is upgrading bla..bla..).
Dan tara... Google Playstore ter-nstall di Nokia X anda. Kini anda bisa melakukan login ke Playstore dan juga mendownload aplikasi-aplikasi langsung dari Playstore.

Dan dengan sudah dalam keadaan root, anda bisa lansung meng-install custom recovery (TWRP atau CWM Recovery) yang bisa anda gunakan untuk mem-back-up dan merestore System Android anda. (file recovery ada dibagian download)

Namun, ada sedikit celah di sini, anda tidak mempunyai back-up system asli (non rooted) yang bisa di restore saat nanti ada apa-apa dengan Nokia X anda. Karena itulah saya lebih suka opsi kedua, karena proses-nya di balik. Back-up dulu, baru root dan dan install GAPPS. Bahkan instalasi GAPPS tidak perlu manual.

Tetapi untuk opsi kedua, anda membutuhkan PC, caranya sebagai berikut:
1. Download dan Install paket Minimal ADB & Fastboot dari sini (ada panduannya)
2. Install juga driver adb untuk Nokia X di bagian download.
3. Download dan letakkan file custom recovery di lokasi instalasi Minimal ADB Fastboot, rename menjadi recovery.img.
4. Matikan ponsel dan lepas baterai agar benar-benar mati.
5. Pasang baterai dan nyalakan kembali dalam mode fastboot atau bootloader dengan cara menekan secara bersamaan tombol power dan volume down (-) hingga muncul logo Nokia
6. Dalam folder Minimal ADB & Fastboot, klik kanan mouse bersamaan dengan menekan tombol [ctrl] + [shift] keyboard PC/Laptop anda, pilih Open command promt here.
7. Tancapkan ponsel dengan kabel data ke laptop/PC dan ketik perintah fastboot -i 0x0421 devices lihat apakah sudah muncul di list.
8. Jika sudah lanjut dengan mengetik perintah fastboot -i 0x0421 boot recovery.img agar Nokia X masuk custom recovery sementara. Jadi recovery asli tidak tersentuh.

Nah dari sini anda bisa melakukan back-up boot, recovery, system yang belum tersentuh (ori) kemudian melakukan instalasi flashable zip (baik Flashable-GAPPS.zip maupun rooting dengan Update-SuperSU.zip).

Sebagai tambahan karena custom recovery ini sementara anda perlu mem-boot dari mode fastboot lagi jika ingin restore hasil backup system dan data anda. Agar custom recovery ter-install permanen, anda bisa menggunakan perintah fastboot -i 0x0421 flash recovery recovery.img.
