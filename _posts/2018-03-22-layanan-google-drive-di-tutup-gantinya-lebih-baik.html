Sudah tahu Google Drive? Saya yakin jika kamu sudah akrab dengan cloud storage (layanan penyimpanan online) gratis yang disediakan oleh Google. Bahkan kamu mungkin sudah sering memakainya untuk sharing (berbagi file) dengan teman -teman kamu. Atau sekedar memakai Google Drive sebagai media penyimpanan cloud untuk mem-backup data aplikasi Android kamu. Jika begitu, bersiaplah untuk berpindah, karena layanan Google Drive akan ditutup Tanggal 12 Mei 2018! Waduh? Tenang...

Google sendiri telah mengumumkan aplikasi desktop terbaru bernama Drive File Stream. Ternyata aplikasi ini yang disiapkan untuk menggantikan aplikasi desktop Google Drive. Namun untuk saat ini, layanan aplikasi Drive File Stream baru tersedia untuk pelanggan G Suites yang merupakan layanan untuk korporat alias layanan Berbayar. Setidaknya hingga dukungan untuk aplikasi Google Drive ditutup.

Google Drive sudah secara resmi ditutup. Google sudah mengingatkan para pengguna aplikasi Google Drive tidak didukung lagi sejak 11 Desember 2017 lalu. Meskipun demikian aplikasi ini masih bisa beroperasi hingga Google benar-benar menutupnya pada tanggal 12 Mei 2018 setelah sebelimnya mengumumkan penutupan tanggal 12 Maret 2018 lalu. Namun, sebenarnya Google Drive tidak kemana-kemana, hanya berubah nama saja.

“Drive File Stream adalah aplikasi desktop baru yang memungkinkan kamu mengakses semua file Google Drive secara cepat sesuai permintaan, langsung dari komputer Anda dan menghemat waktu saat file akan disinkronkan,” begitu kata Google. Jadi bagi pengguna aplikasi Google Drive mobile tidak akan ada perbedaan karena ini hanya pergantian aplikasi yang dipakai untuk desktop. Ah, bikin panik saja.

Drive File Stream dulu hanya bisa digunakan oleh user G Suites. Kabar baiknya adalah aplikasi tersebut akan diaktifkan untuk semua user. Namun ada perbedaan mendasar antara aplikasi konsumen dan versi perusahaan, yakni pilihan untuk kontrol administratif.

Kekurangan dari Backup dan Sync tidak mendukung Google kolaboratif Tim, tetapi dapat digunakan bersama dengan File Drive Streaming untuk melakukan sinkronisasi folder lain dengan Drive, seperti Documents dan Desktop.

Berikut ini adalah perbedaan fitur pada Drive File Stream dan Backup and Sync

Jika dilihat dari perbandingan diatas, tidak terlalu banyak perbedaan sepertinya. Maka sebagai alternatif File Drive Stream, Anda dapat meng-upgrade ke versi baru dari Google Drive untuk Mac/PC, yang disebut Backup and Sync.
