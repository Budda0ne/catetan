Sistem yang dibangun untuk memberikan sistem yang didasarkan pada stabilitas dan kecepatan.



UNE (Linux Ubuntu Netbook) telah dioptimalkan dengan benar dan diatur agar dapat digunakan dalam perangkat mobile. Linux Ubuntu Netbook menawarkan hampir semua fitur yang dirancang khusus untuk netbook.

UNE dapat menelusuri web denagn cepat dan mudah, cek email Anda, memutar file media, atau menggunakan instant messaging. Semua ini ter-paket dengan rapi di balik suatu lingkungan desktop intuitif dan transparan (GNOME yang dimodifikasi). Lebih ringan dari pada Ubuntu dekstop, sehingga sangat cocok juga jika di jadikan Sistem Operasi Portabel.

Di antara aplikasi standar yang ada dalam Linux Ubuntu Netbook:
  • Web browser - Firefox 3
  • E-mail client - Evolusi
  • Messenger - Pidgin
  • Media player - Rhythmbox
  • EBook reader - FBReader
  • Image Viewer - F-Spot
  • Paket Office - OpenOffice.org
Persyaratan Minimum Hardware untuk menjalankan Linux Ubuntu Netbook: Intel Atom Processor, 384 MB RAM, 4 GB HDD
