All in One tweaking untuk semua android yang ada dalam zip installer ini di ambil dari xda forum. Tweak yang dijalankan melalui init.d ini sudah di-customize berulang sampai jadi seperti yang sekarang. Tweak-tweak yang ada merupakan tweak yang selalu saya pakai di setiap Stock ROM/Custom ROM.

Sebelumnya, saya memasukkannya satu persatu setiap kali meng-install Custom ROM baru maupun setiap berganti Handheld. Berhubung memakan waktu, akhirnya saya packing jadi satu berupa file flashable zip. Tujuannya hanya untuk mempermudah saja saat ingin menggunakan script-script init.d favorit saya.

Dengan script AIO ini, setidaknya bisa meningkatkan responsifitas Android dengan RAM cekak dan mengoptimalkan ponsel dengan RAM besar. Tweaking koneksi internet dan memperbesar buffer media saat mengakses video secara online sehingga tidak putus putus saat streming video online semacam youtube. Juga mengurangi lag saat main game dan multi tasking.

NiceTweak selain mengandalkan script init.d untuk memanggil konfigurasi tweak internet di file sysctl.conf. Selain itu fitur-fitur yang ada antara lain:

Init.d runner dan fixer.
Jadi, jika anda belum mempunyai dukungan init.d di handheld anda, NiceTweak juga bisa meng-aktifkan dukungan init.d di ponsel anda.

Script ota survival untuk superSU
Beberapa user superSU, terkadang kehilangan binary SU dari aplikasi superSU ketika melakukan update karena tidak mengaktifkan opsi OTA Survial di aplikasi tersebut. dengan script ini, maka diharapkan bisa meng-aktifkan dukungan ota survival di ponsel yang sudah root.

Tweak untuk Internet dan baterai dalam sysctl.conf.
Tweak internet jarang saya tulis di dalam build.prop, tetapi di dalam file sysctl.conf yang kemudian di panggil dengan init.d script setelah boot complete.

ZipAlign apk system dan data.
Untuk lebih jelasnya silahkan ke forum seperti XDA untuk melihat fungsi Zipalign. Disana sudah sangat banyak sekali di bahas.

Defrag db sqlite3
file binary sqlite3 yang dipakai merupakan versi stabil terbaru yang bisa lebih cepat 50% dalam defrag db

Swap Cache 350mb swappines 60
Swap bertujuan untuk membantu ponsel dengan ram 512mb yang sering OOM (out of memory) agar tetap bisa multitasking dengan lancar tanpa sering mengalami OOM, swappiness di set 60 (moderate swap) agar tidak terlalu lag saat berpindah aplikasi.

Seeder v.7
Meningkatkan entropy, berguna saat membuka aplikasi berat. Banyak dipakai oleh pengguna ponsel android yang mengoleksi game besar dan berat semacam Aspalt Urban ataupun RR3.

zRam swap 64mb
Fungsinya hampir sama dengan Swap hanya saja media yang digunakan merupakan RAM itu sendiri dengan cara mengambil ruang sekitar 64mb yang kemudian di mampatkan.

Untuk menggunakan tweak init.d seperti diatas, syarat paling utama adalah sudah terinstall busybox yang lengkap. jadi jika tweak tidak berjalan semestinya, kemungkinan besar busybox anda tidak lengkap. Tetapi dengan meng-install tweak ini, masalah busybox tidak lengkap bisa di selesaikan.

Tetapi jika ternyata masih tidak bisa jalan , maka sebaiknya anda membersihkan busybox yang ter-install dengan Busybox Cleaner di bawah. Kemudian install kembali dengan versi busybox yang lengkap ada di bawah juga.

Untuk pengguna yang expert, berikut keterangan lanjutan dari tweak ini:

Init.d runner ada di install-recovery-2.sh karena install-recovery.sh sudah di pakai untuk superSU untuk meload daemonsu.
Script install-recovery.sh dan otasurvival.sh symlink ke /system/bin. Jadi dapet garansi dobel.

Swap cache hanya sebesar 350mb (max). Itu sudah cukup karena nilai swappiness 60.

Untuk binary sqlite3, ane pake versi terbaru (3.8.7.4) yang lebih cepat 50% dari binary lama.

Note:

- NiceTweak, nama dari tweak AIO ini, sedangkan isi tweak bukan karya ane sendiri, ane cuma memodifikasi nilai/value nya saja.

- NiceTweak untuk General devices, dg RAM 512mb - 1,5gb. Dalam kasus tertentu, ada yg mesti dirubah lagi value dan parameter-nya.

- Install dari recovery, wipe davik-cache dan cache.

- Buat yang suka dengan script, silahkan di bongkar dan di pelajari. Kalau ada yg kurang kasih masukan ke saya
