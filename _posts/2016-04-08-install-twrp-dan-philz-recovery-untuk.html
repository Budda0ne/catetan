Kini sudah ada tool yang mempermudah proses instalasi TWRP maupun Philz Recovery tanpa ribet di awal-awal dulu.


Dengan Tool TWRP-Philz AutoInstaller, anda tinggal menghubungkan ke PC dan sekali klik saja sudah terpasang custom recovery pilihan anda di Asus Zenfone 5 anda.


Auto Installer

Aktifkan usb debugging Asus Zenfone 5 anda.

Ekstrak file di atas dan buka folder auto installer.

Cara instalasi TWRP Recovery:

Sambungkan Zenfone 5 ke PC/Laptop dengan kabel data USB.
Double klik file TWRPRecovery-install.bat
Proses instalasi akan berjalan otomatis.

Cara instalasi Philz Recovery:

Sambungkan Zenfone 5 ke PC/Laptop dengan kabel data USB.
Double klik file PhilZTouchRecovery-installer.bat
Proses instalasi akan berjalan otomatis.

Setelah menggunakan custom recovery diatas anda tidak akan bisa mendapatkan OTA update. Agar bisa mendapatkan OTA update kembali anda harus mengembalikan Recovery Zenfone 5 anda dengan stock recovery.

Cara kembali ke stock recovery:
Sambungkan Zenfone 5 ke PC/Laptop dengan kabel data USB.
Double klik file lockbootloader-forOTA.bat

Selesai.
