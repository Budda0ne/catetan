Perlu diperhatikan, hampir semua ponsel Xiaomi tipe baru dirilis dengan Bootloader dalam kondisi terkunci atau locked bootloader. Jadi silahkan unlock bootloader terlebih dahulu sebelum melakukan pemasangan TWRP Recovery untuk Redmi 3 PRO. TWRP 3.1.0-0 adalah versi terbaru update 8 Maret 2017

Setelah TWRP Recovery terpasang dengan sukses, silahkan lakukan nandroid back-up partisi-partisi penting seperti aboot, boot, system, data, cust dan EFS jika ada. Jangan lupa simpan file hasil nandroid backup di lokasi yang aman seperti PC, USB OTG Drive atau micro SDCARD. Jangan simpan di penyimpanan internal.

Cara untuk mmasuk mode TWRP sendiri bisa dengan cara mematikan Redmi Note 3 PRO terlebih dahulu kemudian nyalakan kembali dengan menekan tombol power + vol + + vol - secara bersamaan selama beberapa detik hingga muncul logo MI. Ketika Logo MI sudah muncul, lepaskan tombol power, tetapi jangan lepaskan tombol vol + dan vol - hingga muncul logo TWRP Recovery.
