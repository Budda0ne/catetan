Anda pasti sudah tahu apa itu boot logo. Boot Logo adalah gambar diam yang muncul sebelum boot animation ketika anda menyalakan ponsel Android pertama kali. Mungkin anda pernah merasa bosan dengan tampilan yang itu-itu saja. Mungkin anda sudah pernah mengganti boot logo Android anda dengan bootlogo buatan orang lain, sekarang anda bisa membuat boot logo sendiri. Dan mengganti boot logo (logo.bin) di ponsel android anda dengan bootlogo buatan anda sendiri.

Meskipun boot logo ini hanya gambar diam, tetapi ternyata boot_logo juga berisi indikator baterai saat ponsel di-charge dalam keadaan mati. Jadi tidak boleh sembarangan dalam mengedit file boot logo. Dalam tutorial kali ini, saya akan memberitahukan kepada anda cara membuat bootlogo menggunakan tool logo builder.

Sebelum memulai, sebaiknya anda melakukan backup file/data anda yang terdapat di ponsel anda ke PC atau media penyimpanan lain. Ini adalah tindakan jaga-jaga agar jika ada sesuatu hal yang tidak di-inginkan, anda masih mempunyai back-up data anda. Jika anda mengikuti tutorial yang saya berikan, berarti anda sudah menyetujui disclaimer dan copyright blog blog ini.

Untuk membuat dan mengganti Boot Logo di Android dengan tool boot logo builder, ikuti tutorial berikut:

Persiapan:
  1. Siapkan file logo.bin atau file boot_logo yang bisa anda cari di internet atau bisa juga dengan mengambilnya dari stock ROM atau custom ROM agar sesuai dengan resolusi layar ponsel android anda.
  2. Siapkan juga software Logo Builder untuk meng-edit boot logo. Anda bisa mendownloadnya di bagian download. Kemudian ekstrak isinya di tempat yang anda sukai (sebaiknya di desktop), yang penting anda tidak lupa.
Setelah kedua bahan diatas anda siapkan, lanjutkan dengan langkah-langkah berikut:
  • Letakkan file logo.bin atau file boot_logo di folder LogoBuilder hasil ekstrak sebelumnya
  • Jalankan tool Logo Builder 1.5.1 dengan meng-klik LogoBuilder.exe
  • Pada jendela yang terbuka, pilih menu Create New Project dan arahkan ke lokasi file file logo.bin atau file boot_logo yang anda persiapkan sebelumnya.
  • Tentukan lokasi folder untuk temoat mengedit gambar yang ada dalam file logo.bin atau bisa juga membuat folder baru.
  • Dalam Jendela aplikasi Logo Builder akan muncul beberapa gambar yang ada di dalam file boot_logo anda.
  • Disana juga terdapat beberapa gambar baterai yang berbeda-beda selain boot logo yang biasa tampil.
  • Dari sini anda bisa mengganti gambar-gambar tersebut sesuai selera anda, yang penting resolusinya harus sesuai dengan resolusi layar Android anda.
  • Pilih gambar yang ingin anda ganti dengan MS Paint (atau Photosop jika anda punya) Lihat ukuran gambar tadi.
  • Siapkan Gambar yang anda sukai dan resize dengan software editor foto agar sesuai dengan resolusi gambar yang akan anda ganti, kemudian simpan dengan format PNG.
  • Rename gambar yang anda edit tadi sesuai dengan nama gambar yang akan anda ganti kemudian replace.
  • Jika anda sudah merasa puas meng-edit boot logo, klik saja tombol Make dan tool ini akan segera membuatkan anda bootlogo sekaligus akan membuat flashable zip yang bisa anda install/flash menggunakan CWMR/TWRP.
  • Selesai.
Untuk mengganti boot logo, matikan ponsel dan nyalakan dalam mode recovery kemudian flash dengan file flashable yang sudah dibuatkan secara otomatis oleh Logo Builder. Atau, jika anda punya aplikasi Root Explorer, anda bisa mengambil file logo.bin dari dalam file zip dan dan rename menjadi boot_logo kemudian langsung copy ke dalam folder /system/media/images
