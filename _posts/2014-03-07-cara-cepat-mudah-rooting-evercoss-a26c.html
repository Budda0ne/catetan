Sudah punya Evercoss A26C tapi merasa kurang puas dengan performanya dan ingin mengutak atik system-nya? Anda bisa melakukan rooting terlebih dulu agar anda bisa melakukan tweak-ing. Dengan melakukan rooting Evercoss A26C, anda akan mempunyai hak akses superuser/root sehingga anda bebas ber-experimen dengan handheld anda.
Cukup basa basinya, sekarang akan saya tunjuk-kan cara paling cepat dan paling mudah untuk melakukan rooting pada smartphone Evercoss A26C anda tanpa PC.
Ingat, dengan mengikuti tutorial yag saya berikan di blog ini, saya anggap anda setuju dengan disclaimer blog ini dan setuju untuk mematuhi dan memahami ketentuan Copyright blog ini.

Cara untuk rooting Evercoss A26C


Download terlebih dahulu Framaroot 1.9.1 (link ada di bawah)

Buka menu Pengaturan -> Pengaturan Keamanan dan scroll kebawah sampai anda melihat Sumber tak dikenal dan Centang.
Cari file Framaroot-1.9.1.apk yang sudah anda download dan Install Framaroot 1.9.1
Install FramarootKonfirmasi
Setelah Framaroot ter-install, buka dan pilih opsi Install SuperSU dan metode rooting gunaka Opsi Barahir.
Tunggu sebentar, hingga muncul notifikasi success. Maka aplikasi SuperSU pun ter-install.
Rooting SuccessSuperSU

Voila.. Evercross A26C anda sudah rooted. untuk mengecek apakah Evercocc A26C anda sudah rooted atau belum, gunakan aplikasi Root Checker yang bisa anda download di playstore.

Untuk berjaga-jaga jika anda melakukan kesalahan dan mengalami bootloop atau kerusakan system yang lain, anda sebaiknya mempunyai CWM Recovery yang bisa anda gunakan untuk membackup ROM Evercross A26C anda. Baca tutorial install CWM Recovery untuk A26C.
