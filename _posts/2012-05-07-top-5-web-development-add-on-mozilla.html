Kawan, bagi saya, Browser yang paling saya sukai itu Mozilla Firefox. Karena browser inilah yang saya kenal pertamakali selain Internet Explorer milik Microsoft yang selalu disertakan dalam setiap OS Windows-nya. --sekarang saya juga menggunakan Chrome, :)

Apa yang membuat saya masih menyukai Firefox sampai sekarang? Add-On! Tak banyak browser yang didukung begitu banyak developer third party. Hingga saat ini, sudah ada ribuan Add-On yang di kembangkan oleh developer-developer software untuk di pasang di Firefox. Mulai dari sekedar toolbar situs pertemanan hingga tools untuk Web Development.


Kali ini saya akan membahs beberapa tools Web Development yang menurut saya bagus dan banyak mendapatkan pujian dari penggunanya akan saya coba me-ranking dari posisi 1 s/d posisi 5...


Inilah Top Five Rank of Web Development Tool Firefox Add-On

1. Firebug.
Home Page : http://getfirebug.com/
Deskripsi dari developer :
Firebug integrates with Firefox to put a wealth of development tools at your fingertips while you browse. You can edit, debug, and monitor CSS, HTML, and JavaScript live in any web page
Screen Shoot Firebug
Catatan saya :
Jika kamu adalah developer (entah pengembang website, blogger, atau apalah itu) kamu tentu membutuhkan alat debuging semacam Firebug. Add-On ini adalah alat debugging paling bagus menurut saya, firebug menyediakan akses yang mudah untuk melihat/edit DOM, style, dan informasi skrip, dan firebug tidak hanya mempunyai fitur itu-itu saja, firebug juga dibekali konsol yang membuat kita dengan mudah menambahka kode-kode langsung ke halaman yang sedang di-debug untuk mengecek apakah sudah bekerja dengan baik.

Saya sering menggunakan Firebug untuk membangun prototipe skema baru. Daripada terus memodifikasi satu set argumen javascript dan jQuery yang panjang, kemudian menyimpan dan me-reload/refresh halaman, saya cukup mengedit dari konsol secara langsung, tweaking pada halaman secara langsung, dan membuang/menambahi kode yang relevan ketika saya akhirnya mendapatkan visualisasi yang bagus.

Firebug pernah saya gunakan ketika berhadapandengan beberapa situasi yang "merepotkan" saat mencoba untuk mengakses halaman sebuah website yang dirancang dengan "tidak baik" (hehehe). Saya pernah harus memasukkan nilai tertentu ke dalam kolom formulir yang max-length-nya terlalu pendek untuk sebuah task.
Firebug membuatnya menjadi mudah untuk menemukan elemen yang tepat, memeriksa properties-nya, dan menambah max-length dengan cepat.
[Download dan Install]

2. NoScript
Homepage : http://noscript.net/
Deskripsi dari developer :
The best security you can get in a web browser!
Allow active content to run only from sites you trust, and protect yourself against XSS and Clickjacking attacks.
Screen Shoot NoScript
Catatan saya :
Saya suka pengaya ini. Saya selalu memastikan NoScript terpasang setelah saya melakukan instalasi Firefox! Saya tidak bisa browsing tanpa Add-On ini, sudah menjadi sesuatu yang penting bagi saya. Aplikasi ini memblok skrip-skrip yang tidak saya inginkan untuk muncul di halaman yang saya buka dan jika NoSript memblok sesuatu yang saya ingin melihatnya, sangat mudah untuk mengembalikannya, cukup dengan cara mengklik kanan halaman (atau icon-nya) dan taraa... terbuka. :)
Add-On ini memblokir fungsi (script) tertentu, sehingga bila tidak terinstal saya akan pusing dengan besarnya jumlah peng-engeksekusi-an skrip pop-up dan iklan yang membuat browsing menjadi lambat dan 'lemot'. :(
Kita sudah mulai dibombardir oleh iklan yang begitu banyak di internet yang pada kondisi tertentu membuat darah saya mendidih, hahaha. Sudah terlalu banyak iklan di TV!!!! (juga billboard, majalah, dll.) Saya merasa tidak perlu membayar biaya internet 'hanya' untuk melihat iklan dan skrip berbahaya yang bisa saja menginfeksi dan mmporak-porandakan komputer saya.

[Download dan Install]
Add-On yang hampir sama : AdBlock Plus

3. Element Hiding Helper for Adblock Plus
Homepage: http://adblockplus.org/elemhidehelper
Deskripsi dari developer:
Fight the text ads! Element Hiding Helper is a companion extension for Adblock Plus meant to make creating element hiding rules easier. You simply select the element you want to be hidden and then choose which attributes of this element should be taken into account when hiding it in future. The element hiding rule is generated and added automatically.
Screen Shoot EHE untuk ABP

Catatan saya :
Element Hiding Helper adalah Add-On must-have jika kamu memiliki AdBlock Plus yang sudah terinstal (syaratnya memang harus memasang AdBlock Plus dulu.
Add-On ini banyak membantu saya, karena saya dapat dengan cepat mem-block elemen tertentu setiap kali saya menemukan iklan yang unhidden. Selanjutnya saya dapat melaporkannya ke daftar filter dari .
Bila kamu menggunakan Web Development Tool yang terintegrasi dalam Firefox untuk menjelajahi website (Ctrl + Shift + I), penggunaan Element Hiding Helper ini sangat intuitif.
[Download dan Install]

4. FireFTP
Homepage: http://fireftp.mozdev.org/
Deskripsi dari developer:
Along with transferring your files quickly and efficiently, FireFTP also includes more advanced features such as: directory comparison, syncing directories while navigating, SFTP, SSL encryption, search/filtering, integrity checks, remote editing, drag & drop, file hashing, and much more!
Screen Shoot FireFTP
Catatan saya :
Singkat saja. Pernah menggunakan FileZilla? Ya, fungsinya hampir sama, hanya saja ini versi Add-On.
Dengan Add-On ini kamu bisa dengan mudah mengupload dan mengatur file-file dari web-server(web-host) kamu.
[Download dan Install]

5. Web Developer
Homepage : http://chrispederick.com/work/web-developer/
Deskripsi dari developer :
The Web Developer extension adds various web developer tools to a browser. The extension is available for Firefox and Chrome, and will run on any platform that these browsers support including Windows, OS X and Linux.

Screen Shoot Web Developer

Catatan saya:
Hampir sama dengan Add-On urutan 1 (satu).  Jika kamu baru memulai (belajar) membuat website, Add-On ini termasuk dalalm daftar tool pertama yang mesti kamu pakai. Sekilas, melalui berbagai pilihan dan kamu akan tahu cara membuat web yang lebih baik. dengan melihat dan mencoba-coba 'membongkar' website orang lain :)
[Download dan Install]
