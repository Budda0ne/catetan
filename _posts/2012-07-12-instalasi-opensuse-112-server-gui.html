Tutorial Instalasi openSUSE 11.2 server berbasis GUI ini akan menitikberatkan penggunaan tampilan GUI untuk manajemen server. Saya sangat faham sepenuhnya bahwa sistem yang didedikasikan untuk server sebaiknya menggunakan konfigurasi seminimal mungkin dan direkomendasikan untuk menggunakan tampilan Text Mode. Meski demikian, dengan mempertimbangkan kemudahan migrasi, pemahaman dan transisi, saya lebih condong untuk menggunakan tampilan GUI, meski semua caranya bisa dilakukan melalui tampilan konsole/text mode. Karena itulah saya mencari panduan atau tutorial instalasi openSUSE berbasis GUI, dan akhirnya mendapatkannya dari vavai.com

Dengan mempublikasikan ulang tutorial instalasi opensuse server GUI ini, saya berharap dapat membantu sesama pemula, terutama para Administrator Windows yang berniat melakukan migrasi sistem ke Linux namun masih belum terbiasa dengan konfigurasi berbasis teks.

Panduan yang saya dapatkan dari vavai.com ini menggunakan distro Linux openSUSE 11.2 namun bisa digunakan dengan sedikit modifikasi dan perubahan jika diterapkan pada openSUSE 11.0, 11.1, 11.3 maupun SUSE Linux Enterprise Server 11.

Instalasi openSUSE bisa dilakukan dengan berbagai cara, baik menggunakan CD, DVD, file ISO, melalui jaringan dan lain sebagainya. Panduan yang nantinya akan kamu download ini menggunakan instalasi berbasis DVD. Instalasi berbasis jaringan, file iso atau media instalasi lainnya menurut vavali.com akan dibahas pada bab terpisah dari seri PDF miliknya.

Bagi rekan-rekan yang ingin menggunakan tutorial ini, silakan men-download Tutorial Instalasi openSUSE 11.2 Versi Server Berbasis GUI di tautan ini.
