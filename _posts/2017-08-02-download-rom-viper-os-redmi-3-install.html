Custom ROM yang dikembangkan oleh developer Brazil ini juga berbasis Android Nougat versi 7.1.2 (saat menulis tutorial ini) dan baru saja saya coba setelah agak bosan dengan ROM RR.

Racun-racun ViperOS sebenarnya tidak banyak berbeda dengan fitur RR. Beberapa fitur pada Features Bar Customization misalnya: Customizable icons, Quick pulldown, Smart pulldown, Notification count, Clock position, Seconds in clock, Date design juga ada pada Ressurrection Remix.

Sementara pada bagian Quick Settings ada pengaturan seperti: Brightness slider, Brightness icon, Tile tap customization, juga Header customization. Dalam System Viper OS juga ada pilihan untuk kustomasi Power menu.

ViperOS untuk Redmi 3 bisa dipasang pada Redmi 3 standar maupun Redmi 3 Prime atau PRO karena sama-sama punya codename IDO. Mengenai codename, bisa anda baca pada artikel Daftar Codename Xiaomi dan Fungsinya.

Cara Install ROM ViperOS untuk Redmi 3

Jika anda ingin mencoba "racun" dari ViperOS, anda bisa mengikuti tutorial instal ViperOS Redmi 3 Prime dibawah. Sebelumnya, anda harus sudah melakukan unlock bootloader dan memasang twrp untuk redmi 3 prime terlebih dahulu.

Download lebih dulu ROM ViperOS Nougat. Silahkan mengunduh file rom zip dari sini terlebih dulu. Pilih link download ViperOS terbaru yang terletak paling atas. Versi paling baru biasanya paling stabil karena sudah update dengan perbaikan-perbaikan bug.

Reboot ke recovery mode, caranya dengan mematikan Redmi 3 anda terlebih dulu. Kemudian tekan tombol Volume Bawah, Volume Atas dan Power secara bersamaan hingga bergetar dan lepaskan tombol Power saja. Tunggu hingga logo TWRP muncul, baru lepaskan semua tombol. Kini anda sudah masuk TWRP. Dengan beberapa menu utama.

Dari sini, pilih menu wipe dan geser Swipe to factory reset ke arah kanan. Tunggu hingga selesai dan kembali ke menu utama.

Sekarang pilih menu Install, cari file zip ROM ViperOS di sdcard (external) kemudian geser slider Swipe to Install ke arah kanan.

Tunggu hingga proses instalasi selesai. Selesai, pilih Reboot System.

Setelah menunggu sebentar (agak lama sebenarnya), ViperOS pun terpasang pada Xiaomi Redmi 3 anda, baik itu Redmi 3 standar (2/16) maupun Redmi 3 pro/prime (3/16).
