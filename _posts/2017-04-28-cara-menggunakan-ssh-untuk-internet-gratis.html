Persiapan Bahan dan Software untuk Menggunakan SSH


Untuk menggunakan koneksi SSH, anda perlu menggunakan beberapa aplikasi. Silahkan persiapkan bahan dan download alat atau aplikasi di bawah ini:

1. Pastikan punya koneksi ke internet.

2. Konfigurasi Inject, bisa dicari via Google. Cari yang masih work karena buasanya konfig semacam ini cepat mati, caranya cari dengan kata kunci konfig terbaru sesuai simcard yang anda pakai. Misalnya: `konfig inject indosat unlimit` atau yang senada.

3. Gunakan Bitvise karena menggunakan komputer PC. Bitvise bisa diunduh di sini.

4. Unduh dan pasang juga tool Proxifier dari situs resmi.

5. Jangan lupa, untuk dial up gunakan tool IP Hunter yang bisa didapatkan dari link ini. Usahakan mendapatkan IP 100.xx.xx.xx melalui form search.

Untuk selanjutnya gunakan Bitvise dan Proxifier seperti biasa.

Cara Menggunakan Bitvise dan Proxifier dengan SSH


1. Hubungkan modem ke internet, pada tutorial ini menggunakan simcard Im3(indosat)

2. Setelah terhubung, silahkan gunakan konfigurasi inject. Untuk tutorial ini menggunakan IHQ(Inject Header Query), cara menggunakannya bisa baca artikel sebelumnya.

3. Buka Bitvise, sebagai contoh bisa lihat konfig berikut ini. Jika tidak berhasil saat anda mencoba, silahkan cari konfig sendiri, dan unduh. Kembali ke Bitvise dan pilih Load profile dan pilih file config yang anda unduh tadi.

Atur Bitvise pada tab Login, pilih Proxy Setting.
Pada proxy setting centang pilihan Use Proxy dan isikan sesuai ketrangan berikut:

Server : 127.0.0.1 (harus menggunakan IP localhost ini)
Port : 8888 (Samakan pada listen port inject) lalu klik OK
4. Masih di Bitvise, sekarang beralih ke tab Service. Pada tab ini centang opsi Enable dan isi kolom sesuai keterangan di bawah:

Listen Interface : 127.0.0.1 (IP localhost, harus IP ini)
Listen port : 1080 <~ boleh berbeda, tapi ini harus sama dengan yang di atur di tool proxifier. 5. Sekarang klik Login. Jika berhasil akan seperti di bawah ini 6. Sekarang buka Proxifiernya. Pilih Profile pilih Proxy setting pilih Add lalu isikan: address : 127.0.0.1
port: 1080 <~ sama seperti di Bitvise tab service. Selanjutnya klik Ok dan Ok lagi. 7. Langkah selanjutnya Klik icon Proxifications Rules, pada action pilih proxy profile yang sudah di buat sebelumnya lalu Klik Ok.

8. Tahap terakhir yaitu setting proxy pada browser (mozila atau Google Chrome) tanpa proxy

9. Selesai sudah Tutorial internet gratis unlimited menggunakan SSH Jika kurang paham bisa Tanya di komentar Selamat bertinternetan Murah bahkan geratis.

Pro Tips : Jika mau gratis akun ssh, bisa download tiap hari di situs tertentu, cari dengan kata kunci `ssh gratis` atau `free ssh`. Biasanya akun gratisan ini punya batasan jumlah data dan waktu aktif selama kurang dari 3 hari. Kalau mau yang bebas bisa membeli 1 akun SSH premium dengan harga sekitar 20rb-50rb sudah dapat jangka waktu hingga 1 bulan.
