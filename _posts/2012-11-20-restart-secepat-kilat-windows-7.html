Trik Restart secepat kilat pada Windows 7 ini sangat berguna untuk anda yang sering melakukan tweaking registry atau Install/uninstall program pada Windows. Tindakan restart sering diperlukan untuk melihat perubahan dari proses tweaking yang anda lakukan. Sementara me-restart windows sendiri memakan waktu yang lama jika dilakukan secara normal.

Ada beberapa yang menyiasatinya dengan melakukan Log-Off saja, tetapi cara ini masih memakan waktu yang agak lamajuga.

Untuk menghindari proses restart yang memakan waktu lama, anda dapat mematikan proses explorer.exe sebagai alternatif restart, sehingga tidak perlu lagi melakukan proses restart berulang-ulang.

Di windows 7 (sebenarnya ini juga bisa di lakukan di Windows versi sebelumnya) ada sedikit trik untuk mempercepat proses restart ini seperti dengan melakukan step-by step berikut ini:

1. Klik menu Start, arahkan kursor pada tanda panah di samping tombol Shut-down.


2. Tekan dan tahan tombol kombinasi [Ctrl] + [Shift] pada keyboard, disertai klik kanan pada mouse. Dari menu pilihan yang muncul, pilih Exit Explorer.


3. Setelah explorer dimatikan, akan ditandai dengan hilangnya taskbar windows, icon desktop dan tertutupnya semua jendela windows explorer.

4. Untuk mengaktifkan kembali windows explorer, buka task manager. Tekan kombinasi [Ctrl] + [Shift] + [Esc]. Buka menu File > new task (run..).


5. Setelah jendela create new task muncul, ketikkan explorer.exe (atau explorer saja), Klik OK.

Explorer akan berjalan kembali dengan fungsi perubahan aktif. Serasa seperti telah melakukan restart komputer.

Cara diatas juga bisa digunakan saat komputer mengalami hang. Anda tidak perlu melakukan proses restart yang lama. Dengan mematikan explorer terkadang masalah hang bisa teratasi. Selamat mencoba.

Dan selamat tinggal restart-merestart yang lama! Nikmati proses restart secepat kilat di Windows 7 anda.
