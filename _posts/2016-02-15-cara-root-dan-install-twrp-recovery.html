Karena saat ini belum bisa dilakukan root instan dengan aplikasi.
Root Samsung Galaxy J7 tidak bisa dilakukan dengan beberapa aplikasi root instant seperti, Framaroot, Towelroot, Kingroot, KingoApp dan aplikasi lainnya. Untuk melakukan root, tapi bisa gunakan cara di bawah ini.

Samsung Galaxy J7 sendiri ialah smartphone Android Samsung yang memiliki layar 5.5 inci Super Amoled, RAM 1.5 GB, ROM 16 GB, serta kamera depan 5 MP dan belakang 13 MP.

Jadi proses rooting membutuhkan bantuan PC/Laptop dengan install TWRP Custom Recovery baru bisa melakukan rooting. Cara root ini bisa dilakukan untuk Samsung Galaxy J7, J700F, J700M, J700H, dan seri J7 lainnya.

Sebelum memulai melakukan Root Samsung Galaxy J7, yang pertama kali harus dilakukan adalah install Custom Recovery untuk Galaxy J7, Custom Recovery yang digunakan adalah TWRP. Jika TWRP sudah terinstall, barulah root bisa dilakukan.

Berikut ini bagaimana cara root dan install TWRP di Samsung Galaxy J7

Persipan Rooting


Backup semua data dan pastikan baterai mencapai 60% lebih.

Download Samsung Galaxy J7 USB Driver dan install di komputer

Samsung Galaxy J7 USB Driver

Aktifkan USB debugging di Samsung Galaxy J7.

Caranya Setting > Developer Options > USB Debugging.

Jika tidak ada Developer Option, kamu bisa memunculkannya dengan cara: Setting > About Phone > Build Number (tekan 7x untuk memunculkan developer option).

Aktifkan juga OEM Unlock: Setting > Developer Options > Enable OEM Unlock.

Hubungkan smartphone ke komputer, pastikan semua Driver terinstall dengan baik.

Cara Install TWRP Recovery Samsung Galaxy J7


Download Odin3, ekstrak di folder yang mudah ditemukan.

Odin3 3.10.7

Download TWRP Recovery untuk Samsung Galaxy J7 dan letakan dalam folder Odin3.

TWRP Recovery Samsung Galaxy J7

Matikan Galaxy J7 dan reboot ke download mode. Caranya, tekan tombol Power + Home + Volume Up bersamaan.

Jika sudah masuk ke Download mode. Buka Odin3 di komputer lalu hubungkan Galaxy J7 ke komputer/laptop. Akan muncul pesan Added di Odin3.

Centang tombol AP di Odin3 lalu pilih file "twrp-2.8.7.0-j7elte.tar" yang sudah di-download sebelumnya.

Buka tab Option. Pastikan Auto Reboot dan F.Reset Time telah tercentang. Pastikan juga Re-Partition tidak tercentang. Jika sudah yakin, klik tombol Start untuk memulai proses flashing.

Jika sudah selesai, akan muncul pesan PASS! di Odin3 dan TWRP pun sudah berhasil di-install di Samsung Galaxy J7.

Cara Root Samsung Galaxy J7


Download file UPDATE SuperSU dan letakkan di memori internal agar mudah ditemukan.

Update SuperSU 2.46

Matikan smartphone-nya, lalu nyalakan kembali dalam mode Recovery. Caranya tekan tombol Power + Home + Volume Down bersamaan.

Saat sudah masuk ke TWRP Recovery, flash SuperSU.zip tadi. Caranya: Install > Select Zip.

Jika sudah terinstal, pilih Reboot System.

Jika file SuperSU.zip sudah terinstall, secara otomatis Samsung Galaxy J7 sudah root.
