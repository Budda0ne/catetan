MIUI (ROM) yang ter-install di Xiaomi Redmi 1S masih MIUI v32.0 (MIUI-JHCMIBF32.0) saat pertama kali anda operasikan. Untuk saat ini, sudah ada big OTA update untuk Xiaomi Redmi 1S yakni versi MIUI v45.0 (MIUI-JHCMIBH45.0). Dala update terbaru ini ada beberapa bug-fix. Sehingga performa ponsel Redmi 1S anda akan bekerja lebih maksimal.

Dengan update Redmi 1S dari v32.0 ke MIUI ke v45.0 katanya agar ponsel tidak mengalami overheat dan kekurangan RAM lagi. Update Xiaomi Redmi 1S kali ini lebih diutamakan pada masalah penggunaan RAM yang agak boros, dan juga ponsel cepat panas saat digunakan. Anda sebaiknya segera melakukan update. Update v45.0 ini juga ditujukan untuk memperbaiki beberapa fitur seperti System, Messaging, File Explorer, Email, Calendar, dan Mi Account.

Ada 2 cara untuk melakukan Update, yang bisa anda lakukan.

1. Langsung update dari ponsel Xiaomi Redmi 1S anda (OTA), caranya:
  • Pastikan ponsel anda sudah terhubung ke internet, sebaiknya gunakan jaringan Wi-Fi.
  • Buka menu Tools dan pilih Updater.
  • Ponsel anda akan secara otomatis mencari update versi terbaru MIUI.
  • Klik pada versi terbaru yang sudah ditemukan MIUI-JHCMIBH45.0 (Build Stabil) kemudian klik Perbarui, maka ponsel anda akan otomatis mendownload versi terbaru MIUI.
  • Tunggu hingga selesai download dan ponsel melakukan update.
2. Install Manual dari SDcard dengan mendownload file zip update
Untuk mendownload file update, anda bisa memilih mendownload langsung dari ponsel maupun komputer.

  • Download file zip di bawah pilih sesuai dengan tipe Xiaomi Redmi 1S anda (lihat belakang baterai/dosbox)
  • Letakkan di SDcard (di Internal Storage juga bisa) boleh di akar folder atau dalam folder.
  • Masuk ke menu Tools dan pilih Updater. Tekan tombol menu (kiri tombol home) untuk masuk ke Options, dan pilih Select update package.
  • Cari file update yang sudah anda salin di SD Card tadi (misal: miui_H2AGlobal_JHCMIBH45.0_8ae836eff3_4.3.zip), pilih file tersebut
  • Pilih Update dan lanjtukan dengan Start Update
  • Ponsel akan melakukan update secarara otomatis dan restart sendiri.
  • Tunggu proses update selesai dan ponsel Xiaomi Redmi 1S selesai boot pertama.
Jika ada notifikasi update berhasil, berarti anda sukses meng-update Xiaomi Redmi 1S . Beberapa user yang sudah meng-update OS MIUI ke paket 45.0 mengatakan ponsel lebih terasa ringan saat dipakai. Dan yang paling kelihatan adalah kapasitas free RAM yang bertambah, sebelumnya hanya di kisaran 190MB kini bisa sampai 350MB. Dan daya tahan baterai menjadi lebih lama.
