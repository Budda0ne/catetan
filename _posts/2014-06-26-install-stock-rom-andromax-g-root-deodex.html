Setelah mencoba berbagai macam custom ROM di handheld Andromax G anda, biasanya akan ada rasa jenuh dan muncul keinginan untuk mengembalikan MaxG anda kembali ke ROM asli. Saya mendapatkannya di blog Iwan Uzumaki (lihat credit di bawah).

ROM ini adalah Stock ROM yang sudah DEODEX. Dan dapat di instal via CWM. Stock ROM Andromax G ini sudah rooted dan sudah dimodifikasi sedikit. Tujuannya agar anda bisa lebih nyaman menggunakan MaxG kesayangan anda karena tidak perlu melakukan rooting ulang dan mengcustomasi terlalu banyak.

Untuk Rooting Andromax G, gunakan tutorial rooting dengan v-root. bisa juga dengan tutorial rooting untuk Samsung Galaxy S3 dan Untuk Install CWM Recovery bisa menggunakan Andromax Tool.

Stock ROM customized untuk Andromax G ini mempunyai fitur-fitur asik yg sudah di tambahkan. Diantaranya:

- Any Cut For Gsm Internet. Untuk penfaturan jaringan GSM.
- File Manager dengan interface Sony Xperia
- Kitkat Launcher
- Root Explorer
- Super SU
- Muzhiwan Market (?)
- Boot Animation XPERIA

Selain penambahan fitur-fitur, Stock Rom ini juga ada beberapa fitur uang dibuang.

- Applikasi milik Smartfren, alasannya: menggangu dan menambah beban RAM karena berjalan di latar belakang.
- Dolby FX, karena tidak begitu terasa efeknya. Jika anda tidak suka anda bisa meng-instal aplikasi Dolby sendiri.

Jangan kuatir. Untuk menggantikan efek suara Dolby, ada aplikasi lain yan sudah ditambahkan yakni:

- Viper Android FX
- Viper Android XHIFI

Cara Install Stock ROM untuk Andromax G Deodex.

Sebelum meneruskan Tutorial Instalasi Stock ROM ini, anda harus setuju dengan Disclaimer dan patuh dengan artikel DMCA Copyright di blog ini. Dan anda juga paham bahwa: dengan menginstall Custom ROM di handheld anda menyebabkan garansi anda gugur, resiko kehilangan data, jadi backup data penting anda.

1. Download terlebih dulu Stock ROM nya (link dibawah)
Link Download Stock ROM Andromax G dan File Tambahannya

Stock ROM

Driver Viper

App Viper Android

2. Sepweti biasa, letakkan di /sdcard/--disini--
3. Reboot ke mode recovery, bisa melalui terminal emulator (ketik 'su' dan 'reboot recovery') atau matikan dan nyalakan lagi dengam menekan tombol vol up + power.
4. Setelah masuk dalam mode recovery, lakukan:
5. Pilih menu Wipe Format/Factory Reset
7. Advance > Wipe Dalvik Cache
8. Install From SDCARD
9. Pilih file stock rom yang anda letakkan di sdcard tadi.
10. CWM Recovery akan melakukan instalasi runggu hingga selesai dan
11. Pilih menu Reboot system now
12. Jika ada pertanyaan Fix Root, pilih Yes.

Sukses. Ingat, proses booting pertama setelah install memakan waktu agak lama karena Android sedang menyiapkan file dalvik aplikasi dan konfigurasi awal system.

Setelah masuk system. Ada beberapa konfigurasi/setting yang harus anda lakukan agar Viper Android bisa berfungsi.

Download dulu Driver Viper dan aplikasi Viper (link download ada di bawah).

Setelah itu, lakukan:

- Instal Viper FX setelah ter-instal Pilih COrtex9 With Neon
- Istall Viper XHIFI kemudian pilih COrtex Juga (jangan NVIDIA dan Tegra)
- Extract Dan Copy Driver Viper Ke SDCARD
- Reboot Andromax G.

Selesai.
