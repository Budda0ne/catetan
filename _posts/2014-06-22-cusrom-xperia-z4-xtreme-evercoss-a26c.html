Selain Tampilannya yang sangat mirip dengan aslinya, Banyak fitur-fitur asli dari XperiaZ yang bisa berfungsi sebagaimana aslinya walaupun di pasang di Evercoss A26C.

Istimewanya, Custom ROM ini di bangun untuk para maniak game istilahnya Xperia Z4 Xtreme adalah ROM Gamer karena ada fitur khusus untuk gaming mode. Walaupun tetap saja tak semulus smartphone dengan spesifikasi tinggi karena keterbatasan hardware dari Evercoss A26C sendiri (RAM hanya 512MB).

Custom ROM Xperia Z4 Extreme ini dibangun oleh member (developer) dari group Facebook Evercoss A28A yang kemudian di porting oleh member Group Facebook Evercoss A26C. Sebelumnya, rom ini juga sudah di porting ke handheld Evercoss A26C (lihat bagian credit).

Fitur yang ada di versi Evercoss A26C juga tidak bayak berubah/berbeda dengan versi untuk Evercoss A7S antara lain:

- Adanya mode Game yang tentunya sudah di tweak.
- RAM 1Gb dan Internal Storage 3 Gb hanya 'hiasan' saja alias palsu.
- Super Smooth, dan Camera 8 MP fix.
- Notifikasi LED (Flash LED) untuk panggilan masuk.
- GPS bisa berfungsi dengan baik dan akurat
- Ditambahkannya Auto Swap
- Kernel sdah mendukung init.d untuk bash scripting dan Busybox binary.

Cara Pasang/Instalasi Custom ROM Xperia Z4 Xtreme

Sebelum meneruskan tutorial Instalasi ini, anda harus setuju dengan Disclaimer dan patuh dengan artikel DMCA Copyright di blog ini. Dan anda juga paham bahwa: dengan menginstall Custom ROM di handheld anda menyebabkan garansi anda gugur, resiko kehilangan data, jadi backup data penting anda.

Syarat: harus sudah ter-install CWM Recovery, jika belum, baca di tutorial install CWM Recovery.

Untuk memasang ROM ini, anda sebaiknya melakukan Backup IMEI melalui aplikasi Mobile Uncle atau aplikasi backup IMEI lainnya, sebagai antisipasi jika terjadi INVALID IMEI. Buat juga Nandroid Backup melalui CWM Recovery sebelum melakukan flashing/install.

Langkah-langkah instalasi:

  1. Download Xperia_Z4_Xtreme_Port_by.Azmy.zip dan letakkan di sdcard tapi jangan di dalam folder. Letakkan di /sdcard/--disini-- (menjadi /sdcard/Xperia_Z4_Xtreme_Port_by.Azmy.zip)
  2. Matikan Evercoss A26C anda, dan hidupkan lagi dalam mode CWM Recovery dengan menekan tombol [Power] dan [Volume Up]
  3. Ketika sudah masuk mode CWM Recovery, gunakan tombol [Volume Up] dan [Volume Down] untuk navigasi dan tombol [Power] untuk memilih menu:
    • pilih menu Wipe Data/Factory Reset pilih yes
    • pilih lagi menu Wipe Cache Partition pilih yes lagi
    • pilih menu Advaced, pilih Wipe Dalvik Cache
  4. Pilih Install Zip From SdCard kemudian pilih file Xperia_Z4_Xtreme_Port_by.Azmy.zip
  5. CWM Recovery akan melakukan proses flashing, tunggu hingga selesai.
  6. Setalah Selesai, pilih GO BACK (menu paling bawah) dan pilih Reboot System Now

Jika Proses Instalasi berjalan mulus, Custom ROM Xperia Z4 Xtreme akan ter-install dengan baik di Evercoss A26C anda.

Untuk fix sinkronasi kontak download dan letakkan GoogleContactsSyncAdapter.apk ke folder /system/app/--disini--

Agar anda bisa menggunakan gaming mode, setelah boot pertama buka aplikasi Terminal Emulator dan
- ketik "su" (tanpa tanda petik) kemudian enter
- ketik "arch" (tanpa tanda petik) lalu enter
Jika muncul notifikasi, abaikan saja. Mode gaming pun sudah aktif, reboot lagi Evercoss A26C anda.

Untuk memilih antara Gaming mode dan Normal mode, masuk ke Setting, Developer options dan pilih opsi yang ada.
