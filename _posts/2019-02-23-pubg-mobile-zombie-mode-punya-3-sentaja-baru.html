Tencent sebagai pengembang game PUBG sudah merilis update patch 0.11.0 pada tanggal 19 Februari 2019. Patch ini awalnya disebut sebagai update zombie mode, namun sekarang sudah memiliki nama resmi yakni Survive 'Till Dawn. Pada mode zombie ini walaupun jumlah pemainnya lebih sedikit, namun para pemain tetap akan dibuat sibuk.

Karena selain harus menghindari serangan pemain lain, dan mengumpulkan loot. Di mode zombie ini para pemain juga harus menghadapi serangan para zombie dari Resident Evil 2. Para zombie ini akan muncul terus tanpa henti, selain itu ada juga mini bos zombie dan bos tyrant yang harus kamu kalahkan.

Senjata dari Resident Evil 2

Dalam update yang berkolaborasi dengan game zombie paling laris ini ada 3 senjata baru yang diadaptasi dari Resident Evil 2. Senjata baru ini bisa kamu pakai untuk melawan para zombie tersebut.

1. Combat Knife

Di PUBG selain senjata untuk menyerang jarak jauh, ada senjata lain yang bisa kamu pakai untuk menyerang dari jarak dekat. Nah di mode zombie ini senjata jarak dekat yang bisa kamu pakai adalah combat knife.

Bentuknya yang kecil membuatnya praktis untuk dibawa-bawa saat lari dikejar zombie. Pisau tempur ini juga sangat berguna ketika kamu kehabisan peluru. Kamu masih bisa membunuh zombie dengan memakai senjata ini.

2. Flamethrower

Flamethrower atau pelontar api adalah senjata kedua yang wajib ada dalam permainan mode zombie dalam game apapun. Senjata ini memakai gas bottle sebagai amunisinya. Kalau dibandingkan, senjata ini memiliki efek yang mirip dengan molotov. Yakni musuh akan terbakar dan mendapatkan damage secara terus menerus kalau terkena api dari senjata ini. Sangat berguna untuk membakar para zombie.

3. M134

M134 adalah senjata jenis senapan mesin. Senjata ini punya kelebihan mampu menembakkan ratusan peluru secara bersamaan. Jenis senjata yang satu ini memang senjata wajib ada dalam permainan yang punya zombie mode karena sangat buerguna sekali kalau kamu harus menghadapi banyak zombie sekaligus. Yang sulit itu membawa amunisi cadangannya.

Nah, itulah 3 senjata baru yang ada di PUBG Mobile mode zombie. Kamu bisa memilih senjata mana yang cocok sesuai dengan kebutuhanmu. Semoga saja ketiga senjata ini bisa membantu kamu mengalahkan para zombie. Selamat bermain mode zombie...

Tapi ingat! Kamu tidak sedang bermain Resident Evil 2. Karena ini game PUBG kamu harus tetap waspada dengan pemain lain yang juga sedang bertahan hidup di map yang sama.
