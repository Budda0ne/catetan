Lagi lagi, ada yang bertanya cara menginstal dan menggunakan Adobe Flash Player di versi terbaru Android, seperti Android 4.2 atau bahkan yang paling baru Android 4.3. Install Flash Player di perangkat Android sebenarnya mudah, dan saya akan menunjukkan kepada Anda carnya dalam beberapa langkah mudah. Siap? Ayo kita mulai.

Persiapkan Perangkat Anda


Sebelum Anda memulai, Anda harus memastikan bahwa Anda mengatur pada bagian "pasang aplikasi dari sumber yang tidak diketahui" kotaknyadalam keadaan ter-centang. Ada dalam pengaturan Android Anda. Karena Anda tidak sedang menginstal Flash Player dari Google Play Store, tetapi dari sumber lain (sdcard). Ingatlah untuk menghilangkan centang di kotak ini setelah Anda selesai menginstal nanti.
  • Buka Pengaturan Android dan buka menu "Security".
  • Aktifkan kotak centang "Sumber diketahui" di bawah Administrator Perangkat.
Setelah langkah ini dilakukan, langkah instalasi yang sebenarnya dapat dimulai.

Download Flash Player

Adobe Flash Player sudah dihapus dari Google Play Store sejak lama, tapi masih ada beberapa diarsipkan versi pada homepage Adobe. Buka halaman Arsip Flash Player lewat PC Anda dan gulir ke bawah di arsip Flash Player untuk Android 4.0. Download versi terbaru (yang di bagian paling atas dari daftar) dan kemudian salin file APK ke memori/sdcard perangkat Anda.

Install Flash Player

Buka file explorer / File Manager file milik Anda dan arahkan ke lokasi file APK Flash Player dan anda dapat menginstalnya. Sederhana dan mudah bukan?

Menggunakan Flash Player

Untuk memutar konten Flash di situs web seluler, Anda masih memerlukan web browser yang mendukung Flash. Google Chrome for Android tidak mendukung Flash, tetapi browser alternatif lain seperti Mozilla Firefox atau Dolphin Browser bisa memutar Flash dengan Flash Player Plugin. Untuk menggunakan Flash Player dengan Firefox, tidak ada pengaturan lain yang perlu dilakukankan. Di Dolphin Browser Flash plug-in harus diaktifkan dalam pengaturan terlebih dulu. Anda bisa mengaturnya di bagian Pengaturan > Pengaturan konten Halaman > Flash Player > Selalu aktif.

Sekarang Anda dapat memainkan konten Flash dengN Flash Player pada smartphone atau tablet Android Anda!
