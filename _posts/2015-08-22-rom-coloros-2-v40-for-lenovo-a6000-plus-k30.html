Jika iya, anda bisa menggunakan ROM Color OS tersebut untuk ponsel Lenovo A6000 atau A6000 plus. Meskipun ROM ini sebenarnya diporting untuk ponsel Lenovo K30T dan K30W, tetapi ROM ini bisa digunakan juga untuk Lenovo A6000 dan keluarga Kraft lainnya.

Color OS ini berdasarkan rilis resmi pembaruan terbaru dari versi ColorOS V150417. Fix bug di versi terbaru untuk masalah restart berulang tiba-tiba, blank hitam dsb. Meskipun rom ini merupakan rom porting, ColorOS versi ini mempertahankan hampir semua fitur bawaan, tidak banyak aplikasi yang dibuang.

Rilis official versi ini dilengkapi dengan aplikasi pihak ketiga untuk merampingkan ROM ColorOS yang aslinya hampir 1G menjadi 800M saja.

Performa Color OS v.4.0 0522 secara keseluruhan lebih stabil, meskipun penilaian mengenai performa itu relatifsesuai dengan kebutuhan pengguna itu sendiri.

Bug yang terdeteksi di ROM Color OS ini hanya fitur black gesture dan double tap di sensor cahaya karena kernel yang dipakai keluarga Kraft tidak mendukung hal ini. Sementara itu beberapa fitur yang bisa berjalan normal antara lain:
  1. Multi-finger to activate the camera
  2. Double-click on the lock screen
  3. Three fingers screenshots
  4. Two fingers to adjust the volume
Dan rom ini sudah terinstall SuperSU didalamnya, jadi tidak perlu melakukan root lagi.
Opsional: Gunakan Flashable SuperSU untuk menambahkan akses root

Tutorial Instalasi Color OS 2 v4 ke Lenovo A6000/plus dan Lenovo K30T/W

Sebelum mulai menginstall custom rom Color OS, ponsel anda harus sudah terpasang custom recovery. Baca tutorialnya di sini.
  1. Download file Kraft_Color2.0.1_V4.0_0522.zip dan letakkan di sdcard
  2. Masuk ke recovery mode dengan menekan tombol volume up (+), volume down (-) dan tombol power bersaman beberapa detik setelah ponsel dimatikan.
  3. Lakukan Back-up jika belum melakukan back-up
  4. Lakukan Wipe System, Data, dan Cache (atau opsi Wipe to Install new ROM)
  5. Pilih menu Install from sdcard dan arahkan ke lokasi file Kraft_Color2.0.1_V4.0_0522.zip dan pilih install, tunggu hingga proses selesai.
  6. Kembali ke menu utama dan pilih Reboot system.
Saat booting awal setelah instalasi akan memerlukan waktu yang agak lama. Ini karena system memebuat dalvik cache dari aplikasi bawaan dan system, juga menyelesaikan konfigurasi awal system android.

Opsional: Install Google apps untuk kitkat 4.4.4
