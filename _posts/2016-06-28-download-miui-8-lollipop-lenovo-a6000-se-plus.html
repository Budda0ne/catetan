Langkah Instalasi ROM MIUI di Lenovo A6000.

Perhatikan: MIUI 8 Russia 6.7.21 (dan versi sebelumnya) sudah menyediakan custom recovery TWRP dalam paket zip. Jadi setelah Instalasi, custom recovery akan berganti menjadi TWRP 3.0.0.

Untuk memasang ROM, pastikan sudah mempunyai Custom Recovery, rekomendasi saya, paling baik menggunakan TWRP meskipun menggunakan CWM Recovery maupun PhilzTouch Recovery juga bisa.

OS sebelum instalasi bebas, baik Stock ROM atau Custom ROM, kitkat maupun lollipop bisa langsung melakukan instalasi.

Download juga SuperSU.zip jika ingin melakukan rooting ulang dengan versi SuperSU terbaru. Sebenarnya tidak perlu dan tidak wajib, karena selama proses Instalasi anda juga melakukan root secara otomatis.

Download GAPPS, juga tidak perlu. Kecuali jika anda ingin memperbaiki paket aplikasi Google dengan versi paling update daripada yang terpasang. Bisa mengunduh OpenGapps 5.1.1 ARM pico atau jika anda maniak Google Apps, anda bisa menggunakan paket aplikasi OpenGapps 5.1.1 ARM Full, tetapi jangan mengeluh kalau internal storage jadi penuh.

Setelah semua hal diatas terpenuhi, boot ke mode Recovery dengan menekan tombol [power] + [vol up] + [vol down]. Jika belum bisa masuk TWRP Recovery secara langsung, pilih menu Recovery di layar sela yang muncul.

Jika sudah masuk TWRP, lewati langkah ini jika sebelumnya anda sudah menggunakan MIUI 7 Russian Lollipop, jika tidak anda bisa melakukan wipe Cache dan Dalvik-cache. Tidak perlu wipe System karena nantinya ROM installer akan memformat partisi System. Sementara untuk Data, sebaiknya anda wipe juga jika anda menggunakan ROM selain MIUI Lollipop. Karena sesuai pengalaman, tidak bisa masuk System setelah instalasi selesai. Tetapi jika menggunakan ROM MIUI lain yang sudah menggunakan base Lollipop 5.1.1 anda boleh tidak melakukan wipe Data agar data dan aplikasi yang sudah anda install tidak hilang.

Setelah proses wipe selesai, kembali ke layar utama TWRP dan pilih menu Reboot Recovery. Alasannya untuk member kesempatan TWRP membaca ulang tabel partisi System, Cache dan Data agar tidak muncul pesan kesalahan saat proses instalasi atau flashing.

Setelah masuk mode recovery lagi, pilih menu Install dari layar utama TWRP. Arahkan ke lokasi dimana anda meletakkan file ROM MIUI Rusia. Pilih dan swipe (geser bar indikator) untuk mengkonfirmasi pemasangan ROM. Proses install ini mungkin terlihat stag (mandeg) tetapi santai saja. Flashing ROM kan memakan waktu sekitar 15 hingga 20 menitan.

Setelah sukses (done), wipe cache dan Dalvik-cache lagi. Kemudian pilih menu Reboot pilih lagi menu System. Ingat, sebaiknya biarkan boot system selesai sebelum melakukan flashing file zip lain (mis: GAPPS/SuperSU).

Sampai disini proses instal ROM MIUI 8 Rusia Lenovo A6000 sudah selesai.
