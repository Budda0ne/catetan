Umumnya sebuah distro dibuat untuk di jalankan lewat mode live cd atau di install di hardisk, namun slax memberikan anda opsi ketiga yaitu dapat dijalankan di flashdisk anda sebagai OS portable.

Bagi anda yang belum faham apa itu linux dan apa manfaatnya, berikut ringkasnya

Mengenal dan mencoba linux tanpa harus install di HD, anda tidak perlu takut lagi windows ga mau boot atau partisi terhapus karena kegagalan saat instalasi linux.

Linux bisa menjadi Rescue System ketika windows anda ga bisa di boot, anda bisa booting lewat linux dan membackup data anda di partisi windows

Linux Live USB bisa anda gunakan untuk menghapus virus dipartisi windows anda dan masih banyak lagi manfaat lainnya.

Berikut ini adalah cara instalasi Linux Slax di Flashdisk anda. Instalasi Slax di flasdisk cukup dilakukan kurang dari 5 menit dan instalasinya lewat windows. Nah, mudah sekali kan.

Namun sebelumnya, berikut syarat yang harus dipenuhi agar Slax bisa berjalan di flasdisk

  1. 1. Flasdisk minimal 512MB
  2. 2. Bios komputer/Notebook anda mendukung boot lewat USB

Komputer era Pentium 4 sampai era saat ini (multicore) sudah mendukungnya.

Langkahnya :

  1. Download file slax untuk USB http://www.slax.org/get_slax.php?download=tar
  2. Ekstak file tar tersebut ke flashdisk anda, nanti ada dua folder yang pertama bernam slax dan folder yang kedua bernama boot
  3. Masuk kedalam direktori X:\boot dengan X adalah drive label flashdisk anda. Cari file bootinst.bat dan klik dua kali file tersebut untuk menginstall boot record slax di flashdik. INGAT: JANGAN MENJALANKAN (atau mengklik 2x) filebootinst.bat diluar flashdrive, hal ini bisa membuat windows anda tidak mau booting.
  4. Silahkan komputer di restart dan jangan lupa ubah urutan boot device melalui BIOS anda agar first boot nya mengarah ke flashdisk anda.

Jika berhasil akan muncul tampilan menu slax beriktu ini:

Pilih menu Slax graphic mode (KDE) atau Slax Always fresh untuk masuk ke Distro slax.

Berikut ini adalah tampilan distro Slax 6.x
