Jika menggunakan PC, Proses Full Rooting bisa dilakukan dengan Tool MTK Droid. Lain halnya Jika anda melakukan langsung dari handheld. Oh ya, perlu di perhatikan bahwa Seri Advan S5E mempunyai banyak versi, diantaranya ada Advan S5E Pro KitKat. Untuk KitKat (dan versi baru lainnya) biasanya hanya bisa melakukan Half Root di Advan S5E versi ini. Untuk itu perlu langkah lanjutan agar hasil rooting bisa 100%, alias Full Root.

List versi Advan S5E antara lain :
  • - Versi lama: Ver. B ada 2 versi Versi B1V02, Versi B2V01
  • - Versi baru: Versi K1V02, Versi K1V04, C1, S5E+, dan S5E Pro
Untuk melihat versi ponsel milik anda bisa melihat di belakang baterai.

Untuk Rooting semua Advan S5E langsung dari Ponsel, sebenarnya cukup menggunakan Framarot versi 1.91 atau 1.93 menggunakan pilihan dropdown Boromir. Tetapi hasilnya bergantung dengan versi ponsel milik anda. Untuk lebih jelasnya silahkan ikuti langkah langkah berikut:

Melakukan Rooting dengan Framaroot:

Langkah langkahnya cukup mudah dan gampang sekali, ikuti saja instruksi berikut
  1. Pastikan opsi USB debugging di Developer Options aktif.
  2. Install aplikasi Framaroot di ponsel anda.
  3. Buka aplikasi framaroot dan lakukan rooting dengan memilih opsi Install SuperSU dan pilih Borormir

Sampai disini biasanya anda akan mendapati pesan sukses. Tetapi jika anda mengalami Half Root (di tandai dengan notifikasi half success) maka anda perlu menginstall SuperSU dari Playstore dan ulangi proses diatas agar bisa Full Root.

Jika solusi diatas masih tidak bisa maka sebaiknya anda menggunakan CWM Recovery untuk mendapatkan akses Full Root. Tentu saja anda harus melakukan instalasi CMW terlebih dahulu dengan keadaan Half Root., Kemudian anda bisa melakukan instal Update SuperSU zip via CWM untuk fixing

Memasang CWM Recovery Advan S5E


Entah anda sukses melakukan rooting pada Advan S5E anda (apapun tipenya), adalah sebuah keharusan untuk meng-install custom recovery. Hal ini agar anda bisa melakukan full back-up system anda sehingga jika suatu saat ada masalah dengan ponsel anda, anda bisa melakukan restore.

Untuk meng-install CWM Recovery / Carliv Touch Recovery, anda harus tahu versi ponsel Advan S5E anda. Sehingga bisa meminimalkan resiko kerusakan (baca disclaimer).

  1. Download file custom recovery sesuai versi anda di bagian bawah (download section) artikel ini.
  2. Rename file recovery yang anda download menjadi recovery.img agar lebih mudah.
  3. Download dan install aplikasi Mobile uncle tool, bisa juga install via Playsotre.
  4. Jika sudah terinstall, buka mobile uncle dan pilih menu install recovery pilih from sdcard/eksternal sdcard.
  5. Pilih reboot to recovery jika ada notifikasi pilihan reboot.
  6. Jika ponsel anda menyala kembali dalam mode recovery dengan tampilan yang seperti gambar dibawah atau mirip(yang jelas buka android system recovery 3e/stock recovery) proses instalasi yang anda lakukan sudah berhasil. Dan anda bisa melakukan back-up system.

Jika anda kebetulan mengalami half success saat rooting anda bisa menginstall Update-SuperSU.zip seperti yang sudah ditulis sebelumnya.
