Android! Ini pertama kalinya saya menggunakan smartphone android, tangan sudah terasa gatal, langsung saya obrak abrik si Android dengan berbagai cara. Mulai dengan meng-install berbagai aplikasi android dari market --yang saya share di blog ini dengan label android dan apk.

Hingga akhirnya merasa kekurangan resource karena android saya termasuk jadul (LG Optimus GT540), saya pun menambahkan beberapa apps untuk meningkatkan RAM. Kemudian mengoprek isi celana dalam si android dengan mengubah partisi untuk system dan data juga menambahkan partisi swap dan cache di memory external (sdcard). Semua tutorial berbahasa inggris saya lahap habis. Sebagian besar (95%) saya dapatkan dari forum XDA Developer. Namun itu malah membuat saya semakin lapar.

Seminggu penuh saya pun mengoprek isi celana dalam Miss Android. Berhasil, meningkatkan performa LG Optimus GT540 yang notabene beroperasi di OS Android Donut (1.6) --tetapi kemudian saya update ke Android Eclair (2.1-update1), membuat saya ingin mencicipi Android JellyBean (4.1.x). Saya pun berusaha mencari tutorial cara meng-upgrade GT540 menjadi Android JellyBean di forum yang sama. Yang merupakan CyanogenMod 10

Singkatnya, saya berhasil upgrade ke JellyBean via CyanogenMod 10. Ada 2 versi yang saya gumnakan yakni JellyBean CM10 forGT540 ALPHA 2 dan XDA CM 10.0.0 RC0 JellyBeanSwift keduanya ClockModWork 10. Untuk tutorialnya sedang saya buat baru saya buatkan satu dengan alur se-simpel mungkin dalam meng-upgrade android yakni yang menggunakan file JellyBean CM10 forGT540 ALPHA 2

Untuk yang pertama agak bermasalah, masih banyak bug yang muncul. Seperti kamera tidak bisa di akses, Setting-nya menutup sendiri ketika mengakses seksi Storage, dan beberapa aplikasi bawaan tidak berjalan sebagaimana mestinya. Maklum masih Alpha.

Kemudian saya Flash ulang dengan yang ke-dua, yaitu XDA CM 10.0.0 RC0 JellyBeanSwift. Hasilnya lumayan memuaskan. Tetapi saya sempat kebingungan mencari tempat mensetting ke bahasa indonesia, karena XDA CM 10.0.0 RC0 JellyBeanSwift secara default menggunakan bahasa rusia dengan teks rusia yang aneh itu. Saya berhasil merubahnya menjadi berbahasa indonesia berdasar ingatan saat menggunakan JellyBean CM10 forGT540 ALPHA 2. Secara keseluruhan hasilnya sudah bagus. Tetapi saya masih berencana untuk menggunakan CyanogenMod 7 saja alias downgrade dari CyanogenMod 10
