Fungsi utamanya adalah untuk memperbaiki sistem dan memulihkan data yang hilang sebagai akibat dari kegagalan sistem (Linux dan Windows). SystemRescueCd mendukung semua sistem file utama: ext2/ext3/ext4, reiserfs, Reiser4, btrfs, xfs, jfs, vfat, ntfs, iso9660.


Distribusi Linux ini dilengkapi dengan sejumlah tool antara lain:
· GNU Parted, GParted - manajemen partisi
· Partimage - menyimpan dan memuat image partisi
· FSArchiver tool yang fleksibel untuk menyimpan / mengembalikan sistem ke file arsip terkompresi.
· Ntfs3g - membaca dan menulis partisi NTFS
· Sfdisk - menyimpan dan mengembalikan citra tabel partisi
· Ddrescue - Data Recovery
· Uji-disk - menemukan dan memulihkan partisi (format yang didukung termasuk reiserfs, ntfs, FAT32 dan ext2 / 3)
· Memtest + - RAM pengujian
· System Tools untuk file sistem Linux dan Windows
· Utilitas Jaringan (seperti Samba, NFS, ping, dan nslookup)

SystemRescueCd paling baru (3.2.0) di bangun menggunakan versi kernel Linux 3.0.8, lingkungan desktop default Xfce.
