Untuk instalasi Xposed Framework Lollipop, caranya sedikit lebih ribet karena anda harus mendownload framework dan aplikasinya secara terpisah. Dan tentu saja harus punya sedikit pemahaman tambahan mengenai SDK dan Arsitektur Android yang dipakai device anda saat ini.

Pemasangan Xposed Framework untuk Lollipop agak berbeda dengan pemasangan Xposed untuk Kitkat kebawah yang tinggal menginstall aplikasi-nya dan melakukan langkah selanjutnya dari aplikasi tersebut.

Singkatnya, Android Lollipop (saat ini) mempunyai dua SDK, Lollipop 5.0 masih menggunakan SDK 21 dan Lollipop 5.1 sudah menggunakan SDK 22. Sedangkan Arsitektur Android Lollipop yang dipakai adalah 32bit untuk device dengan chipset ARMv7 (Qualcomm maupun Mediatek) sedangkan untuk chipset Intel lebih sering disebut sebagai X86 saja dan tidak ada pembedaan arsitektur seperti di ARM (CMIIW).

  • [ 10 Juli 2016 ] Touchwiz UI, Untuk device baru buatan Samsung rata-rata mulai menggunakan cipset dengan arsitektur 64 bit ARMv8. Untuk ROM stock sepertinya tidak semua device didukung terutama untuk device baru tersebut, dan beresiko terjadi bootloop. Meskipun begitu, seperti halnya Xposed spesial untuk ROM MIUI, saat ini sudah tersedia Xposed Framework untuk Samsung dengan Stock ROM (Touchwiz UI), baik untuk yang odexed maupun deodexed. Lebih jelasnya akan saya bahas di artikel lain.

Cara Install Xposed Framework, Step by step.

Syarat agar bisa memasang Xposed tentu saja harus sudah root dan memiliki custom recovery. Jangan lupa melakukan nandroid backup dengan custom revovery ya. Anda sudah saya ingatkan.

Oke, anggap saja anda sudah memenuhi syarat diatas dan tahu SDK dan Arsitektur yang dipakai oleh ponsel android anda. Misalkan: Karena anda memakai Android Lollipop 5.1.1 dengan arsitektur 64bit. Maka anda tinggal men-download file SDK 22 dan ARM64. Silahkan cari di bawah ini:

Xposed Framework Lollipop terbaru saat ini (saat saya upate artikel ini tgl. 10 Juli 2016) sudah versi 86, baik untuk sdk21 (LL 5.0) maupun sdk22 (LL 5.x)

Penting: Sebaiknya sebelum melakukan instalasi, anda melakukan Full Nandroid Back-up partisi system mengingat resiko bootloop.

Install terlebih dahulu Aplikasi Xposed Framework 3.0 alpha 4 yang berfungsi untuk mengatur modul-modul xposed yang akan anda install.

Kemudian, letakkan file yang sudah anda dowload di sdcard. Matikan ponsel dan reboot ke menu recovery.

Cara masuk recovery mode itu variatif, tergantung model ponsel anda. Atau gunakan tool reboot recovery untuk mempermudah.

Dalam mode recovery, pilih install update dan pilih install zip from sdcard (CWM) atau Install dan pilih Install zip (TWRP) dan arahkan ke lokasi file *.zip

Jika sudah selesai, pilih menu reboot to system dari recovery mode.

Buka kembali Aplikasi Xposed Framework. Coba lihat apakah Xposed Framework sudah terpasang.

Jika sudah terpasang, anda bisa mulai meng-install modul xposed favorit anda, dan mengaktifkan modul tersebut dari Aplikasi Xposed Framework.
