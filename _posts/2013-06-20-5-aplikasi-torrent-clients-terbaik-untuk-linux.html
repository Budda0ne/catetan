Anda bisa membaca di sini. Jika Anda suka men-download distribusi Linux yang besar sekedar untuk 'mencoba-sebelum-membeli', Anda akan membutuhkan torrent client yang bagus untuk men-download distro-distro tersebut dalam keaadaan 'OKE'.

Pengguna Windows memiliki aplikasi uTorrent, yang (menurut saya, setidaknya) tak tertandingi jika Anda menggunakan sistem operasi keluarga Microsoft. Sayangnya, tidak ada uTorrent untuk Linux, tetapi ada beberapa alternatif yang layak digunakan.

Berikut ini adalah lima klien Torrent terbaik untuk sistem operasi Linux.

1. qBitTorrent

qBitTorrent, Seperti yang saya sudah sebutkan tidak ada klien uTorrent untuk Linux, namun qBittorrent berudaha untuk mengisi kekosongan. Tujuan dari proyek sukarela ini adalah untuk menyediakan klien uTorrent untuk platform Linux.
Dibuat menggunakan C++, qBittorrent menyediakan antarmuka yang failier dan cepat. Lengkap dengan mesin pencari built-in untuk mereka yang kesulitan dalam mencari link download. Ada berbagai macam fitur yang sudah biasa, termasuk dukungan untuk enkripsi, UPnP, IPv6 dan RSS.

Anda juga dapat me-remote qBittorrent dengan web UI (sama dengan uTorrent). Bagi Anda yang mencari pengganti uTorrent pada platform Linux, rasanya Anda tidak perlu mencari yang lain.

Untuk menginstal qBittorrent pada Ubuntu masukkan kode/perintah berikut di Terminal:
sudo apt-add-repository ppa:hydr0g3n/ppa
sudo apt-get update
sudo apt-get install qbittorrent

2. Transmission (GNOME)

Jika anda menggunakan distribusi Linux Ubuntu, rasnya anda akan kenal dengan Transmission. Karena paket ini adalah paket default klien, dan performanya memang tidak mengecewakan.

Transmission (yang juga dibuat dengan C++) adalah klien torrent yang memiliki sejumlah fitur canggih. Klien dapat dikonfigurasi untuk melihat direktori tertentu, yang berguna dalam hubungannya dengan Dropbox. Mendukung Enkripsi dan juga daftar hitam alamat IP yang berda dalam kategori "bad".

Kurangnya dukungan RSS dan UI yang lumayan rumit adalah satu-satunya kelemahan paket ini. Seharusnya aplikasi torrent klien dibangun dalam prinsip kesederhanaan.

3. Vuze


Sebelumnya dikenal sebagai Azureus, Vuze adalah satu-satunya klien torrnet dalam daftar ini yang ditulis menggunakan JAVA. Sayangnya, aplikasi Java terkenal intensif dalam menggunakan resource alias BOROS TENAGA dan Vuze mungkin tidak cocok bagi anda yang masih menggunakan komputer jadul.

Tak jadi masalah jika spek komputer anda mampu melayani permintaan sumber daya Vuze yang besar. Maka Vuze bisa anda gunakan, karena klien torrent yang satu ini menyatakan dirinya sebagai "aplikasi BitTorrent paling kuat di bumi". Developer Vuze benar-benar membanjirinya dengan segala macam fitur dengan full-web UI, enkripsi, dukungan RSS, IPv6, juga mesin pencari, HD-video playback yang fantastis dan banyak lagi fitur keren lainnya. Sebading kan?

Vuze lebih dari sekedar klien BitTorrent, karena juga mempunyai dukungan untuk mentransfer ke perangkat mobile, termasuk dukungan untuk Android, iOS dan perangkat Blackberry serta Xbox 360, PS3, TiVO dan Apple TV.

Untuk menginstal Vuze di Ubuntu masukkan berikut ke Terminal:
sudo apt-get install vuze

4. Deluge (GNOME)


Aplikasi ini populer di kalangan pengguna Mac, Deluge adalah torrent klien cross-platform yang juga kompatibel dengan Linux dan Windows. Deluge menyediakan UI/antarmuka ringan yang ditulis dengan bahasa Python dan C++ untuk GTK + window manager yang mengingatkan saya klien torrent utnuk Windows: uTorrent.

Menggunakan libtorrent untuk backend, dan pilihan antarmuka frontend termasuk GUI, web UI dan konsol, Deluge adalah aplikasi klien yang 'satu paket'. Ada dukungan penuh untuk enkripsi, link magnet, UPnP, IPv6 dan dukungan untuk plugin.

Deluge kuat, rapi dan menarik. Anda bisa memilihnya jika anda suka dengan fitur Plugin yang dimilikinya :).

Untuk menginstal Deluge pada Ubuntu masukkan berikut ke Terminal:
sudo add-apt-repository ppa:deluge-team/ppa
sudo apt-get update
sudo apt-get install deluge

5. KTorrent (KDE)


Ditulis untuk distro Linux ber interface KDE, KTorrent adalah klien yang banyak digunakn untuk Kubuntu (Ubuntu dengan desktop KDE)dan banyak distribusi berbasis KDE lainnya. Umumnya dianggap klien torrent KDE paling kaya fitur, KTorrent memiliki fitur yang biasanya ada untuk ukuran klien torrent papan atas.

Ada enkripsi, dukungan RSS, pencarian, link magnet, UPnP, dan kontrol torrent remote melalui web UI. Jika Anda suka tampilan Delge yang rapi, tetapi lebih memilih Linux KDE (maksud saya Linx degan desktop KDE) daripada Linux GNOME maka KTorrent mungkin bisa jadi pilihan.

KTorrent memang untuk Kubuntu dan merupakan aplikasi default. Untuk menginstal-nya di Ubuntu gunakan perintah ini di Terminal:
sudo apt-get install KTorrent

Selain 5 Aplikasi Torrent Clients Terbaik untuk Linux di atas, masih ada lagi yang lain di antaranya:

rTorrent

Torrent client untuk Linux berbasis konsol (perintah text, katakanlah DOS di windows), sangat berguna jika Anda menghabiskan banyak waktu Anda untuk berkutat dalam baris-baris kode.

BitStorm Lite

Torrent client yang ringan yang tidak memiliki fitur sebanyak yang dimiliki aplikasi yang disebutkan dalam artikel ini. mengingatkan pada aplikasi klien bittorent yang lama.

Selain klien yang tersebut di atas, rasanya anda juga punya aplikasi klien bitTorrent favorit sendiri. Berikan komentar anda :)
