dsiXDA Kitchen (Android Kitchen) dioperasikan di Windows via Cygwin Terminal

dsiXDA Kitchen, tool yang ampuh untuk membuat custom rom. Sejak hape android pertama saya [LG GT540] yang masih bersiste operasi froyo, sampai era kitkat saat ini, tool ini masih bisa diandalkan. Selain pemakaiannya yang mudah, alias user-friendly, juga banyak fitur-fitur yang berguna.

Tutorial ini saya bagi menjadi 2 bagian. Mempersiapkan Lingkungan Kerja untuk dsiXDA Kitchen dan Instalasinya dan Cara Menggunakan Tool dsiXDA Kitchen. Oh ya hampir lupa, tutorial ini untuk user Windows. Jika anda pengguna linux, anda bisa melewatkan tutorial ini :)

Dengan mengikuti tutorial ini anda setuju dengan DMCA dan Disclaimer di blog ini.

Tutorial Membuat Custom ROM dengan dsiXDA Kitchen [1]: Mempersiapkan Lingkungan Kerja untuk dsiXDA Kitchen

Usahakan membaca dengan teliti tutorial di bagian pertama ini karena ini adalah pondasi untuk dsiXDA Kitchen dan instalasinya. Jangan kuatir, saya sertakan gambar untuk memudahkan yang baru mencoba meracik custom rom sendiri.

Siapkan dulu peralatan tempurnya, yakni:
Java 6 JDK (Java Development Kit)
Cygwin Installer Offline (untuk dsiXDA)
dsiXDA Kitchen
Ketiganya bisa anda dapatkan di bagian bawah postingan (bagian download)
Setelah anda download ketiga bahan di atas, lanjutkan dengan instalasi Java 6 SDK dan instalasi Cygwin baru instalasi dsiXDA Kitchen.

Instalasi Java 6 JDK

Install Java 6 JDK, pilih customize dan hanya pilih Java 6 JDK, anda tidak perlu meng-instal JRE (Java Runtime Environment). Jika anda pernah/sudah menginstall java versi lainnya, silahkan di un-install dulu sebelum meng-install versi ini.
Buat settingan "JAVA_HOME" di Environment Variables (start - klik kanan computer - properties - andvaced system setting - environment variables) dan arahkan ke C:\Program Files\Java\jdk1.6.0_45

Edit juga "Path" dan tambahkan ;C:\Program Files\Java\jdk1.6.0_45


Oke, anda sudah menyelesaikan pondasi utama dalam developing android, yakni Java. Jika anda bertanya: kenapa tidak menggunakan Java 7 yang lebih baru. Jawabannya: Google sendiri masih menggunakan java 6 untuk Developing Android. Jika anda ingin mencoba menggunakan Java 7, silahkan. Tapi jangan komplain ke saya kalau ada masalah. :)



Instalasi Cygwin (for windows)

Ekstrak cygwin_installer_x86_offline.rar atau cygwin_installer_x64_offline.rar dan dobel klik setup-x86.exe atau setup-x64.exe. Pilih "install from local directory", klik next.
Tentukan lokasi dimana anda akan menginstall cygwin (sebaiknya di c:\cygwin\), pilih All Users, klik Next.
Arahkan ke folder "packages" hasil ekstrak file cygwin offline installer yg anda download sebelumnya
klik "default" disebelah kanan "all" 1x agar berubah jadi "install" dan lanjutkan.


Lanjutkan instalasi sampai "Finish", lalu buka aplikasi Cygwin Terminal. Lihat gambar dibawah ini:


Note: Jika anda melakukan kesalahan di tahap Instalasi Java 6 JDK, maka anda akan menemukan pesan error saat membuka Cygwin Terminal.

Selesai sudah pondasi untuk dsiXDA Kitchen. Anda bisa melanjutkannya dengan melekukan instalasi dsiXDA Kitchen.

Instalasi dsiXDA Kitchen

Buka dsiXDA Kitchen (Android-Kitchen-0.224.zip) via winrar/winzip dan lakukan drag/drop ke folder user cygwin.
Kemudian masuk ke folder Android-Kitchen-0.224 dengan perintah "cd". Untuk lebih jelasnya, lihat gambar dibawah:


Selamat!! Anda sudah menyelesaikan instalasi dsiXDA kitchen yang nantinya akan anda gunakan untuk membuat custom rom sendiri. Agar mudah dalam berpindah-pindah folder, sebaiknya rename folder Android-Kitchen-0.224 menjadi kitchen saja. Anda bisa merubah dari windows ecplorer atau dari terminal dengan perintah "mv" seperti gambar paling atas di artikel ini.
