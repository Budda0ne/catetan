Samsung Galaxy J1 2016 mempunyai beberapa varian, diantaranya: SM-J120FN, SM-J120F, SM-J120H, SM-J120DS. Spesifikasi ketiga varian ini beda-beda tipis. Untuk melakukan root Galaxy J1 2016, anda harus mengetahui ponsel anda termasuk variant yang mana. Karena anda harus memasang Custom Recovery pada Galaxy J1 2016 lebih dahulu.

Custom Recovery yang digunakan adalah TWRP. Setelah TWRP terpasang, baru anda bisa melakukan rooting dengan melakukan flashing file SuperSU.zip untk melakukan rooting. Sedangkan varian SM-J120FN otomatis root setelah flashing dengan Odin.

Sebelum melakukan root Galaxy J1 2016, silahkan download file yang dibutuhkan di bawah sesuai tipe (varian) Galaxy J1 anda.

Root J120FN Root J120F
Root J120DS Root J120H

Berikutnya ekstrak file root-sm-Jxxx.rar yang sudah anda download dan buka folder tersebut. Ambil file SuperSU.zip dan letakkan di sdcard/internal memori ponsel anda.

Peringatan!
Pastikan anda mengaktifkan pilihan Enable OEM Unlock di pengaturan ponsel jika anda menggunakan TWRP. Karena ada kasus dimana saat saya mencoba melakukan factory reset, Samsung Galaxy J1 2016 tidak bisa boot ke system dan tidak bisa di flash dengan Odin dengan status binary blocked.

Jika anda sudah yakin ingin melakukan root,siap kehilangan garansi, kehilangan data dan resiko lain yng lebih parah silahkan proses dengan mengikuti langkah berikut dengan semua konsekuensi anda tanggung sendiri.

Selanjutnya anda tinggal mengikuti cara root Samsung Galaxy J1 2016 langkah demi langkah berikut:

Install TWRP Recovery J120H / Auto Root J120FN

1. Install samsung drivers, jika ada masalah lihat bagian troubleshoot
2. Buka Odin3_v3.11.1.exe dan klik AP dan pilih file ***.tar.md5.
3. Untuk J120H saja, klik options dan hilangkan centang auto restart.
4. Matikan ponsel dan nyalakan dengan menekan tombol [Vol down] + [Home] + [Power] untuk masuk download mode
5. Jika muncul peringatan, lanjut dengan menekan tombol [Vol up] untuk melanjutkan.
6. Pasang kabel data pada PC dan sambungkan dengan Samsung Galaxy J1 2016 anda.
7. Lihat status koneksi dengan melihat kotak biru di bagian atas Odin. Jika ID:COM muncul dan berwarna biru muda berarti sudah terdeteksi.
8. Klik Start untuk melakukan install TWRP (J120H, J120F, J120DS) dan autoroot untuk J120FN.
9. Jika di kota ID:COM berubah warna menjadi hijau dengan teks PASS berarti sudah terpasang.

Ingat! Kecuali J120FN, setelah flashing selesai jangan reboot ke system terlebih dahulu. Jika anda langsung menyalakan ponsel, partisi recovery akan kembali menjadi stock/bawaan dan yang anda lakukan sebelumnya menjadi sia-sia. Jadi, cabut kabel data dan matikan dengan mencabut baterai. Kemudian lanjutkan langkah root dengan TWRP berikut.

Langkah Root dengan TWRP untuk varian J120F, J120H, J120DS

1. Nyalakan Galaxy J1 2016 anda langsung ke recovery mode. Caranya dengan menekan tombol [vol up] + [home] + [power]
2. Jika anda gagal masuk TWRP dan boot ke system, anda harus mengulang proses instal TWRP lagi.
3. Dalam mode TWRP recovery, lakukan flashing SuperSU.zip dimana sebelumnya sudah anda copy di sdcard/internal memori anda.
4. Pilih meno reboot dan tunggu. proses boot pertama akan memaka waktu lumayan lama.

Setelah menyala, cek apakah aplikasi SuperSU sudah muncul di menu aplikasi. Jika sudah muncul, artinya anda sudah berhasil melakukan root pada Samsung Galaxy J1 2016 anda. Selamat.

Untuk memastikan bahwa Samsung Galaxy J1 2016 sudah 100% root, anda bisa menggunakan aplikasi root checker.

Troubleshoot:

1. Ada masalah dengan Odin? Cari artikel di google dengan keyword "stuck at pit file mapping".
2. Ponsel tidak terdeteksi? Download dan install smartswitch from samsung.com jalankan dan jika sudah terdeteksi silahkan buka kembali Odin.
3. Alternatif cara root untuk SM-J120F bisa dengan cara memasang Stock ROM versi ini dan melakukan dengan tool roting seoerti biasa
