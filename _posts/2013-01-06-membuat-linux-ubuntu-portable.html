Bagi anda yang pengguna Linux dan sering menggunakan OS open source ini, apa yang terjadi bila anda ingin memakai Linux akan tetapi mendapati komputer yang ber-OS lain? Tentu sangat kerepotan ketika ingin mengerjakan sesuatu diluar rumah (saya sangat mengerti bahwa pengguna linux cenderung fanatik dan lebih menikmati menggunakan Linux daripada Windows). Dan bagi anda pengguna Windows anda pasti sudah tahu juga ada windows portable (dan mungkin pernah menggunakannya). Tapi saya beri tahu anda: Portable Linux lebih luwes daripada Portable Windows.

Jika anda pengguna mempunyai minat untuk mencoba Portable Linux, anda bisa mencoba membuat Linux [Ubuntu] Portable sendiri. Hanya dengan berbekal dengan USB flashdisk, anda bisa menjalankan Ubuntu dari flashdisk PC ber OS apapun (mungkin dulu kita memakai Live CD untuk bisa menikmati Ubuntu di PC lain tapi sekarang rasanya bukan jamannya lagi kemana-mana membawa CD).

Kenapa harus Ubuntu sementara masih banyak distro lain bertebaran diluar sana? Salah satu alasan terkuat adalah Ubuntu merupakan distro yang kompatibel di segala jenis komputer dan tampilannya pun mudah dipahami dan sederhana.

Apabila anda memiliki USB flashdisk berkapasitas minimal 1GB (semakin besar lebih baik karena anda bisa menginstall bermacam-macam aplikasi) anda bisa menginstall OS Ubuntu langsung dan menjalankannya di PC manapun. Dan satu tool yang bisa digunakan untuk membuat Ubuntu Portable ialah Unetbootin yang bisa diunduh di http://unetbootin.sourceforge.net/.

Pilih aplikasi berbasis Windows, Linux atau Mac OS yang sesuai dengan PC yang akan digunakan sebagai basis saat membuat Linux [Ubuntu] Portable. DI tutorial kali ini kita memakai Windows sebagai basis PC kita. Lalu pilih Download (for Windows).

Jika anda belum memiliki file ISO Ubuntu silahkan mendownload versi terbaru Ubuntu di www.ubuntu.com. Versi teranyar Ubuntu saat ini Ubuntu Naty Narwal 12.10 lalu pilih sesuai chipset anda, i386 atau amd64. Setelah file ISO selesai didownload, Silahkan ikuti langkah-langkah berikut ini:

Langkah 1: Siapkan file ISO Ubuntu yang telah diunduh. Letakkan file tersebut di folder tertentu agar lebih mudah menemukannya.

NOTE: Sebelumnya format flashdisk anda dengan format FAT atau FAT32. Karena FAT mampu dikenal oleh BIOS komputer-komputer lama.

Langkah 2: Jalankan aplikasi Unetbootin yang sudah diunduh terlebih dahulu klik 2x bagi WIndows XP dan klik kanan lalu klik run as administrator bagi WIndows Vista dan WIndows 7. Lalu akan muncul jendela munculan seperti dibawah ini.



Langkah 3 : Ada dua pilihan untuk membuat Ubuntu portabel, yakni.
Pilihan pertama : Distribution jika anda ingin memilih distro linux lain dan langsung mendownload file ISO dari internet.
Pilihan kedua : Diskimage. Yakni menggunakan file ISO atau Disc Image.

Gunakan pilihan ini jika anda sudah memiliki file ISO di dalam Harddisk, lalu klik browse untuk menentukan lokasi file iso yang sudah anda Download sebelumnya.

Kemudian di pilihan Type pilih USB Drive.

Langkah 4: Jika sudah menentukan semua pilihan di window tersebut, klik tombol OK. Maka proses pembuatan Linux [Ubuntu] Portable akan segera dimulai seperti gambar berikut:



Langkah 5: Tunggu sampai proses selesai. Sampai muncul tampilan seperti di bawah ini. Untuk menge-test hasil instalasi OS Linux [Ubuntu] Portable, silahkan reboot komputer anda. dengan klik Reboot Now



Langkah 6: Pada saat proses reboot pastikan anda bisa masuk ke BIOS anda, biasanya dengan menekan tombol Delete di keyboard. Lakukan perubahan pada boot setting. Pilih USB Drive sebagai pilihan booting pertama, jika sudah anda tinggal simpan settingan BIOS dan lakukan reboot (restart). Dan pastikan USB Drive anda tetap terpasang.

Langkah 7: Proses booting akan dilakukan dengan menggunakan flashdisk anda dan jika anda sudah masuk ke menu login Ubuntu, saya ucapkan: Selamat anda sudah sukses menginstall Ubuntu di flashdisk, tanpa perlu membuat partisi baru di harddisk anda.

Cukup Praktis bukan? Sekarang anda bisa memakai Linux di komputer manapun. Selamat mencoba.
