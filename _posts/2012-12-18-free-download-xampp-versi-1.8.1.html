Maksudnya menjadikan komputer kita sebagai server yang mampu melayani permintaan seperti sebuah server online. XAMPP merupakan Apache yang dilengkapi dengan aplikasi pihak ketiga diantaranya MySQL, PHP dan Perl. XAMPP sendiri dirancang sebagai web server yang dapat menjadi aplikasi localhost dalam pembuatan website. XAMPP biasa digunakan untuk mengembangkan Website dengan cara yang mudah dan nyaman, juga untuk menguji halaman Web Dinamis (menggunakan PHP)tanpa akses ke Internet.

XAMPP versi 1.8.1 ini mencakup versi terbaru dari Merkurius dan Tomcat.Dengan XAMPP 1.8.1, Anda dapat mengelola database MySQL dan SQLite dan Anda dapat membuat sebuah remote host menggunakan FileZilla FTP server. Ini berguna untuk administrasi website yang menggunakan Content Management System (CMS, misalnya Joomla atau Wordpress).
