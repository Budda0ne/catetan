Apalagi jika kamu masih pelajar yang belum punya penghasilan sendiri harus berapa bulan menabung dari uang saku? Karena itulah kita akan membahas bagaimana cara untuk melakukan instalasi Mikrotik Router OS pada Komputer (PC) karena bisa dipakai untuk belajar secara gratis dengan lisensi level 0.

Seperti yang sudah kita pelajari pada artikel sebelumnya, jika sudah membeli mikrotik routerboard, maka OS sudah include didalamnya dengan Lisensi Level tertentu. Installasi OS ini adalah jika belum membeli routerboard. Maka kamu perlu menyiapkan 1 PC untuk difungsikan sebagai Router dengan OS Mikrotik.

Perlu kamu tahu, PC ini khusus untuk dipasang OS Mikrotik saja, dengan spek yang minim pun sudah dapat dipergunakan dengan baik. Kelebihan jika mikrotik diinstall pada PC adalah kamu bisa melakukan upgrading pada RAM maupun Processor juga memiliki penyimpanan data yang lebih besar yaitu hard disk. Kekurangannya adalah, konsumsi listrik yang lumayan besar, karena menggunakan sebuah PC Desktop sebagai Router yang mana harus beroperasi terus sebagai server jaringan.

Tutorial ini dari pengalaman saya, sebelum menggunakan Mikrotik Routerboard. Saat itu masih mengelola Warnetu menggunakan PC Mikrotik Router OS dalam waktu yang lama dengan spek PC standar saja (Pentium 4 dengan RAM 1Gb dan HDD 40Gb dan 2 LAN card). Dengan spek seperti itu saja sudah mencukupi untuk menjalankan berbagai fungsi router. Jika membahas ketahanan, hasilnya tergantung pada spek OC dan kelistrikan yang ada.

Langkah-langkah instalasi Mikrotik Router OS


Sebelum memulai, download terlebih dahulu Mikrotik Router OS nya di halaman resminya di sini http://www.routeros.co.id/. Kemudian cari versi OS yang ingin kamu pakai, sebaiknya pilih versi paling baru. Untuk artikel ini, saya memilih versi 6.0rc9 dimana merupakan versi terbaru Mikrotik OS saat membuat artikel ini. Atau bisa juga mengunduh langsung file mikrotik-6.40.6.iso (24.93 MB) dari server mikrotik.com

Kamu hanya perlu mengunduh OS-nya saja, lisensi-nya masih level 0 alias free atau trial hanya dalam waktu 24 jam saja. Jika ingin menggunakan dalam jangka waktu yang lama, kamu bisa membeli lisensinya di mikrotik.co.id sesuai dengan kebutuhan kamu.

Setelah didownload, silahkan bakar file iso tersebut pada cd kosong atau bisa juga denagn membuat usb bootable dengan software rufus. Setelah selesai mebuat salah satunya, boot PC yang akan dipakai sebagai Router OS menggunakan CD atau USB yang sudah kamu buat, sebelumnya.

Lihat gambar berikut:

Gambar diatas adalah fitur-fitur mikrotik os, jika kamu ingin menggunakans semua fitur, maka tekan tombol a untuk memilih semuanya, lalu tekan tombol i untuk memulai instalasi mikrotik.

Setelah menekan tombol i, maka akan muncul pilihan apakah akan memakai konfigurasi default? Default merupakan konfigurasi dasar yang sudah disediakan oleh mikrotik. Jika ingin memulainya dari 0 maka tekan tombol N untuk no/tidak kemudian akan muncul pertanyaan tambahan tekan tombol Y untuk Ya.

Pemasangan Mikrotik OS sudah selesai, tinggal tekan tombol Enter untuk melakukan restart, lepaskan media instalasi (usb/cd) agar boot berjalan langsung dari hdd yang sudah terpasang Mikrotik Router OS.

Setelah berhasil booting, akan ada halaman login administrator. Saat ini masih menggunakan username default mikrotik yaitu:
  • username = admin
  • password =

Password memang masih kosong atau cukup tekan enter saja. Hingga sampai halaman ini, mikrotik sudah siap untuk digunakan.

Untuk mengatur atau melakukan konfigurasi, kamu bosa memakai CLI (Command Promt pada OS Windows) ataupun memanfaatkan tool Winbox untuk mempermudah. Untuk mempelajari Cara setting Mikrotik, silahkan lanjutkan ke artikel lain yang berkaitan dengan konfigurasi Mikrotik.
