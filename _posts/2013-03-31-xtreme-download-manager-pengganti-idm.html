Xtreme Download Manager adalah tool/software downloader yang mumpuni untuk meningkatkan kecepatan download hingga 500%, melanjutkan sesi download yang belum terselesaikan sebelumnya, juga meng-capture video FLV dari situs semacam Youtube dan juga mampu me-jadwal-kan sesi download (queue). Istimewanya, XDM bisa berjalan di OS Windows, Linux dan Mac.

Xtreme Download Manager menggunakan algoritma yang terbaru dalam sitem pemecahan file secara dinamis (Dynamics Files Sgmentation), kompresi data dan penggunaan kembali koneksi untuk mempercepat proses download.

Xtreme Download Manager mendukung protokol HTTP, HTTPS dan FTP, firewall, proxy server, Halaman Pengalihan (redirect), Cookies, dll.


Otorisasi XDM dengan mulus dapat terintegrasi dengan Firefox, XDM juga terintegrasi dengan baik di browser lainnya. Xtreme Download Manager berfungsi untuk menangani download secara otomatis menggunakan integrasi browser.(Chrome, Internet Explorer, Opera, Safari) pada sistem operasi apapun. Jadi bukan hanya untuk OS Windows saja, melainkan bisa digunakan untuk OS Linux.


Xtreme Download Manager sepenuhnya dibuat menggunakan bahasa pemrograman JAVA, sehingga dapat berjalan pada semua Sistem Operasi. Bahkan tanpa perlu diinstal alias portable.

Jadi, Xtreme Download Manager adalah pengganti Internet Download Manager yang lebih baik. Kenapa? karena Xtreme Download Manager bisa berjalan di multi platform.

Nah, Berikut ini adalah panduan singkat untuk penggunaan Xtreme Download Manager di Linux:

1. Unduh file zip dari sini.
2. Unpack (unzip nama file atau klik kanan via ark atau program peng-arsip yang lain)
3. Berikan perintah chmod dan x xdman.sh
4. Jalankan dengan ./xdman.sh
5. Langkah selanjutnya seperti menggunakan Internet Download Manager.
