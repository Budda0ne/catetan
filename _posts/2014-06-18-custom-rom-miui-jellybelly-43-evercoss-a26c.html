Jika sebelumnya sudah ada ROM ColorOS untuk Evercoss A26C, kali ini adalah MIUI ROM yang sukses di porting. Siapa yang tak kenal MIUI ROM racikan developer Xiaomi untuk handheld buatannya ini memang jenis ROM cantik yang pertama.

Nah, kali ini, porter dari group Facebook Evercoss A26C berhasil mem-porting MIUI OS untuk Evercoss A26C (lihat bagian kredit). Menurut porter-nya, ROM ini bug-nya belum terlihat. Hal ini mungkin karena ROM basenya juga tidak.sedikit mempunyai bug. ROM ini di porting dari Group Symphony W68 yang sama-sama ber-chipset MTK 6572, dengan spesifikasi yang hampir sama dengan Evercoss A26C.

Seperti namanya, ROM MIUI JellyBelly 4.3 dibangun berdasarkan Android versi 4.3 (Jelly Bean 4.3). Support Bahasa Inggris meskipun belum ada Bahasa Indonesia. Fitur lainnya adalah:

- Android Assistance menggunakan versi terbaru.
- MIUI Home sudah di upgrade.
- Keyboard menggunakan Ridmik sekaligus Ridmik Dictionary-nya.
- Ada tambahan Freestore untuk mendownload Aplikasi berbayar secara gratis.
- MIUI Compas sudah ditambahkan.
- Ditambahkannya link download Mi File Explore
- GAPPS sudah ada dalam file flashable zip yang sudah menggunakan Aroma Installer

Selain fitur tersebut di atas, juga sudah ditambahkan beberapa tweak untuk meningkatkan performa dan tampilan Evercross A26C. diantaranya:

- Custom Boot Logo baru yang dibuat oleh Eshanjahan
- Full Modifikasi
- Deodexed
- Pre-Rooted (sudah dalam keadaan di root)
- Tweak agar baterai lebih irit
- Tampilan UI juga di percantik (Custom Framework)
- Super Smooth UI
- Peningkatan performa
- Perbaikan build.prop untuk meningkatkan kinerja.
- RAM Management sudah diperbaiki sehingga lebih baik dari Stock ROM Evercoss A26C
- Peningkatan kualitas kamera (software).
- Tweak Audio dan peningkatan kualitas Suara
- Modifikasi di bagian Statusbar
- Tweak Zipalign saat booting, sehingga aplikasi-aplikasi lebih ringan saat di buka.
- Kernel sudah mendukung init.d (untuk menjalankan bash script secara otomatis)
- Peningkatan koneksi internet
- Sudah mendukung Gravity Box untuk merubah tanpilan dan framework ui.

Tidak ada custom ROM yang sempurna, apalagi ROM portingan. Jadi, jangan kaget jika suatu saat anda menemukan sesuatu yang mungkin sedikit menggangu.

Cara Pasang/Instalasi Custom ROM MIUI JellyBelly 4.3

Sebelum meneruskan tutorial Instalasi ini, anda harus setuju dengan Disclaimer dan patuh dengan artikel DMCA Copyright di blog ini. Dan anda juga paham bahwa: dengan menginstall Custom ROM di handheld anda menyebabkan garansi anda gugur, resiko kehilangan data, jadi backup data penting anda.

Syarat: harus sudah ter-install CWM Recovery, jika belum, baca di tutorial install CWM Recovery.

Untuk memasang ROM ini, anda sebaiknya melakukan Backup IMEI melalui aplikasi Mobile Uncle atau aplikasi backup IMEI lainnya, sebagai antisipasi jika terjadi INVALID IMEI. Buat juga Nandroid Backup melalui CWM Recovery sebelum melakukan flashing/install.

Langkah-langkah instalasi:

  1. Download MIUI JELLYBELLY 4.3 ROM.zip dan letakkan di sdcard tapi jangan di dalam folder. Letakkan di /sdcard/--disini-- (menjadi /sdcard/MIUI JELLYBELLY 4.3 ROM.zip)
  2. Matikan Evercoss A26C anda, dan hidupkan lagi dalam mode CWM Recovery dengan menekan tombol [Power] dan [Volume Up]
  3. Ketika sudah masuk mode CWM Recovery, gunakan tombol [Volume Up] dan [Volume Down] untuk navigasi dan tombol [Power] untuk memilih menu:
    • pilih menu Wipe Data/Factory Reset pilih yes
    • pilih lagi menu Wipe Cache Partition pilih yes lagi
    • pilih menu Advaced, pilih Wipe Dalvik Cache
  4. Pilih Install Zip From SdCard kemudian pilih file MIUI JELLYBELLY 4.3 ROM.zip
  5. CWM Recovery akan melakukan proses flashing, tunggu hingga selesai.
  6. Setalah Selesai, pilih GO BACK (menu paling bawah) dan pilih Reboot System Now

Jika Proses Instalasi berjalan mulus, Custom ROM MIUI JellyBelly 4.3 akan ter-install dengan baik di Evercross A26C anda.
