--- layout: post title: Cara Fix Evercoss A26C Blank Putih saat Install CWM [Update Link] date: '2014-07-11T05:51:00.000+07:00' author: Phate Holloway tags: - CWM - Evercoss A26C - StockROM - Recovery - Android last_modified_at: '2015-06-16T00:33:46.072+07:00' blogger_id: tag:blogger.com,1999:blog-8168996750712797250.post-2187342911859297064 redirect_from: /2014/07/cara-fix-evercoss-a26c-blank-putih-install-cwm.html --- Beberapa waktu yang lalu, di Group FB Evercoss A26C ada member yang mengeluhkan Evercoss-nya tidak bisa di root menggunakan framaroot. Dan Semakin parah ketika di Install CWM malah blank Putih. Dari beberapa solusi yang diberikan ternyata belum bisa menyelesaikan masalah. Kemudian ada dugaan bahwa ini terjadi karena firmware Evercoss A26C sudah di perbarui kernelnya.

Adanya Firmware Baru untuk Evercoss ini bisa diketahui dari beberapa Screenshoot yang diberikan agan Nizamudin yang pertamakali mengalami hal ini. Setelah beberapa diskusi dan trial error akhirnya baru di ketahui bahwa hal ini (blank putih saat install CWM) terjadi karena adanya update kernel tersebut. Apa saja perbedaan antara Firmware baru dengan firmware lama? Lihat listing permasalahan berikut.

Di firmware Lama A26C:

- Kernel: 3.4.5 eng.wangyan (138xxxxx) (Identifikasi CPU Z)
- Aplikasi kamera, hanya mempunyai dua mode: mode otomatis dan mode malam

Di firmware Baru A26C

- Kernel: 3.4.5 eng.wangyan (139xxxxx) (Identifikasi CPU Z)
- Aplikasi kamera, hanya mempunyai beberapa mode tambahan: mode otomatis, mode malam, mode aksi, mode lanskap dan beberapa fitur yang baru.

Sebenarnya Firmware baru ini bisa di root dengan menggunakan Root Master, dan bisa install CWM menggunakan MTK Droid Tools (PC).

Masalahnya, tidak semua pengguna A26C mempunyai PC/Laptop. Ada beberapa user yang memaksakan menggunakan CWM untuk firmware lama mengalami blank putih saat mencoba meng-install via Handheld. Bagi anda yang sudah terlanjur mengalami blank putih, harus melakukan flashing menggunakan Stock firmware Evercoss A26C lama.

Sedangkan untuk yang belum melakukan mengalami blank putih, sebaiknya menggunakan MTK Droid Tools untuk membuat CWM Recovery sendiri.

Sebagai catatan, jika anda masih menggunakan firmware baru, saya tidak merekomendasikan anda menggunakan Custom ROM yang ada. Karena pada dasarnya Custom ROM tersebut di desain dan dibuat untuk Firmware lama.

Agar bisa menggunakan Custom ROM tersebut anda harus melakukan flashing firmware lama terlebih dahulu untuk keamaanan. Meskipun sebenarnya anda bisa langsung menggunakan Custom ROM yang ada hanya dengan melakukan flashing USECBOOT saja melalui SP Flashtool menggunakan stock firmware A26C lama tersebut. Bukan melakukan flashing keseluruhan.

Jika anda tidak melakukan Flashing dengan Firmware lama, maka biasanya kamera akan force close sendiri atau memberikan peringatan "kamera tidak terhubung" dan masalah yang berhubungan dengan kamera lainnya selain blank putih setelah install CWM.

Lihat Tutorial Flashing Firmware Evercross A26C disini

Atau ikuti Tutor Flashing untuk repair Blank Putih sekaligus Downgrade:

- Buka SP Flashtool.
- Pilih Scatter Loading dan arahkan ke file MT6572_A26C.txt
- Otomatis semua opsi terpilih.
- Hilangkan semua Centang kecuali MBR, EBR1, BOOTIMG dan RECOVERY
- Pilih Download
- Matikan A26C dan cabut baterai selama 5 detik
- Pasang Baterai, jangan nyalakan dan hubungkan ke PC dengan kabel data
- Tunggu hingga muncul Lingkaran Hijau (OK)
- Cabut kabel data dan Baterai
- Pasang Baterai, Nyalakan.
