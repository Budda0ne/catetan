Update: game Castle Clash versi 1.2.791 juga tidak bisa di skip lagi. Yang butuh silahkan download di bawah.

Game yang cukup populer sebagai pesaing game Clash of Clans ini melakukan update hari ini. Ukuran apk dan file .obb yang cukup besar membuat beberapa gamer agak malas untuk melakukan update melalui ponsel android-nya, termasuk saya.

Di versi versi sebelumnya, Castle Clash tidak mewajibkan gamer untuk meng-update game ini. Seperti yang saya alami sendiri, saya masih tetap menggunakan versi 1.2.75 dari game ini saat sudah ada update versi 1.2.77. Dengan adanya tombol skip untuk mengabaikan update game Castle Clash.

Sedangkan untuk update kali ini (Castle Clash v 1.2.78) tidak ada lagi tombol skip. Olala...

Dengan mengasumsikan bahwa sebagian besar pengguna (gamer) juga malas melakukan update via playstore seperti saya, maka saya bela-belain upload file apk dan data obb Castle Clash buat kamu penyuka game Castle Clash yang bisa didowload di bagian bawah.

Untuk instalasinya seperti biasa, install game menggunakab "base.apk" yang sudah di di-download kemudian letakkan file .obb di folder obb. Kalau masih bingung, ikuti petunjuk di bawah:

1. Buka menu Setting > Security dan centrang pilihan Unknown Source.
2. Buka file base.apk dari folder download dan pilih install. Jangan jalankan dulu, tutup saja.
3. Ekstrak file main.1200791.com.igg.castleclash.obb dari zip, copy ke lokasi sdcard default (internal sd) di dalam folder Android > obb > com.igg.castleclash > di sini.
4. Jalankan game Castle Clash seperti biasa.

Untuk update kali ini Game Castle sepertinya hanya memberikan update tampilan bertema Cristmass (sorry kalo salah tulis, saya Muslim ) Untuk update fitur-fitur baru sepertinya sudah dilakukan di saat update versi 1.2.77 yang lalu.
