Umumnya, aplikasi pengolah berkas (file manager) yang disertakan secara default sudah bisa menangani file terkompresi dengan ekstensi *.zip. Namun sebagian besar masih terbatas untuk melakukan ekstraksi saja. Jika ingin melakukan kompresi juga sudah banyak aplikasi pihak ketiga yang mampu melakukannya.

Akan jadi masalah saat anda ingin meng-kompres atau meng-ekstrak file dengan extensi RAR, TAR, 7Z, TAR.GZ dan file arsip dengan ekstensi aneh lainnya. Anda membutuhkan aplikasi khusus untuk menagani berkas-berkas tersebut.

Kasus lainnya, anda menginginkan meng-edit isi file terkompresi tanpa meng-ekstrak terlebih dahulu. Entah itu menambahkan file baru atau menghapus file di dalam file ZIP/RAR. File manager default tak bisa melakukannya, bahkan aplikasi file manager pihak ketiga jarang yang bisa.

Karena sebab-sebab itulah anda membutuhkan aplikasi khusus untuk menangani compressed file, entah itu ber-ekstensi *.zip, *.rar, *.tar dan lainnya. Ada banyak aplikasi semacam ini yang ada di playstore, namun hanya beberapa saja yang menurut saya sangat efektif dan efisien dengan fitur lengkap untuk menangani file terpadatkan alias archive (arsip).

Aplikasi pengelola file terkompresi tersebut bisa anda download atau install langsung dari Plyastore atau link di bawah. Berikut adalah aplikasi yang saya rekomendasikan:

5. Root Explorer dan ES Explorer

Jangan salah sangka, kedua aplikasi ini tak hanya oke saat mengatur file yang anda miliki. Juga tak hanya diperuntukkan ponsel yang sudah di-root.
Aplikasi Root Explorer bisa melakukan kompres file dan ekstrak file dengan ekstendi ZIP, RAR, TAR dan GZ. Aplikasi ini bisa juga di gunakan untuk ponsel non-rooted. Begitu juga ES Explorer meskipun saya agak benci dengan user interfacenya.

Root Explorer: download install

ES Explorer: download install

Kedua aplikasi ini tidak bisa meng-ekstrak file RAR ber-password. Dan yang bikin keki adalah anda harus beli jika ingin menggunakan aplikasi ini. Mungkin anda akan bilang: "Bodo amat, di google banyak kok." Terserah anda deh..

4. RAR for Android

Aplikasi ini punya fitur yang sangat lengkap untuk menangani berkas dengan ekstensi RAR. Mulai mengkompres, mengekstrak dan mengedit file di dalam arsip RAR. Opsi kompresi/ekstraksi juga mencakup opsi pemberian password dan pengaturan kompresi lainnya. Tentu saja, karena ini adalah aplikasi buatan developer asli dari aplikasi WinRAR, rarlab.

RAR for Android: download install

Aplikasi RAR untuk Android ini mempunyai fitur hampir sama dengan versi desktopnya. Sayangnya aplikasi ini lemah dukungannya untuk file terkompresi berekstensi selain ZIP dan RAR. Yang menyebalkan adalah adanya IKLAN, padahal di versi awal-awal aplikasi ini tidak ber-iklan.

3. 7zipper dan 7zipper 2.0

Hampir sama dengan RAR for Android, aplikasi ini juga mempunyai kemampuan yang handal untuk menangani berkas terkompresi *.7z yang jadi format kompresi file default-nya.

7Zippper: download install

7zip Lebih baik dari RAR for Android karena kemampuannya menangani berkas-berkas lainnya seperti *.tar dan bzip dan juga mampu menangani splited zip arcvive selain mampu mengani file arsip populer.

Selain itu, yang patut diberi apresiasi adalah kedua aplikasi pengelola arsip aneh ini BEBAS IKLAN.

Oh iya, untuk 7zipper 2.0 ada tambahan fitur akses file ke cloud drive seperti dropbox dan google drive. Kalau tidak begitu butuh, cukup pasang yang 7zipper.

7Zipper 2.0: download install

2. B1 Archiver

Aplikasi ini handal dalam menangani file-file berekstensi B1, ZIP, RAR, TAR bahkan 7Z juga mampu di tangani. File *.rar ber-password juga mampu ditangani aplikasi B1 Archiver.

B1 Archiver: download install

Aplikasi ini dulu menjadi aplikasi wajib saya karena dukungannya atad berbagai macam format arsip. Kurang lebih seperti aplikasi yang saya pakai saat ini: ZArchiver.

Diluar kelebihan kelebihan di atas, aplikasi ini sekarang punya nilai minus: IKLAN dan pembatasan fitur. Tetapi jika anda tak keberatan, aplikasi ini layak anda gunakan.

1. ZArchiver


Ini dia aplikasi favorit saya pribadi. Selain kemanpuan untuk mengolah (menambah, menghapus) isi file terkompresi dari arsip lzip, 7zip, tar, APK, mtz.

Dukungan untuk file berekstensi aneh yang tidak umum dan jarang digunakan juga sangat mumpuni. File arsip yang bisa dibuat: 7z (7zip), zip, bzip2 (bz2), gzip (gz), XZ, tar. Sedangkan arsip yang bisa di-ekstrak: 7z (7zip), zip, rar, rar5, bzip2, gzip, XZ, iso, tar, arj, cab, LZH, lha, LZMA, XAR, tgz, TBZ, Z, deb, rpm, Zipx, mtz, chm, dmg, cpio, cramfs, img (fat, ntfs, UBF), wim, ECM, arc (FreeArc). Kemampuannya setara dengan B1 Archiver, bahkan lebih baik lagi. Bahkan bisa meng-ekstrak file arsip terbagi (parted) dari zip (zip.001, dst), rar (part1.rar, dst) dan 7z (7z.001, z01, dst).

Diluar semua itu, aplikasi ini BEBAS IKLAN. Ini yang saya suka.

ZArchiver: download install

Aplikasi ini bahkan bisa membuka file hasil back-up partisi system dan data dari custom recovery seperti TWRP Recovery dan CWM Recovery. Untuk file yang bisa di buka antar lain: 7z (7zip), zip, rar, rar5, bzip2, gzip, XZ, iso, tar, arj, cab, LZH, lha, LZMA, XAR, tgz, TBZ, Z, deb, rpm, Zipx, mtz, chm, dmg, cpio, cramfs, img (fat, ntfs, UBF), wim, ECM, arc (FreeArc).

Mungkin informasi ini tidak begitu berguna jika anda tidak atau jarang mengolah berkas-berkas terkompresi. Tetapi akan sangat anda butuhkan jika anda sering berhubungan dengan kompres mengompres file. Misalnya saja mumbuat flashable zip untuk Custom ROM anda sendiri, mengambil dan meng-edit file back-up system android anda, atau melihat isi file *.tar.gz hasil back-up aplikasi Titanium Backup.
