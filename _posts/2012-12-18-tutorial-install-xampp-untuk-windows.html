Bagi anda yang belum tahu, XAMPP merupakan perangkat lunak untuk membuat server sendiri (localhost). Maksudnya menjadikan komputer kita sebagai server yang mampu melayani permintaan seperti sebuah server online. XAMPP didistribusikan secara opensource / perangkat lunak sumber terbuka yang didalamnya sudah ada paket program untuk sebuah server dalam satu kesatuan. Antara lain: Apache, MySQL, PHP, phpMyAdmin, FileZilla FTP Server, Perl. Saat ini XAMPP terbaru untuk Windows adalah versi XAMPP 1.8.1
Untuk informasi saja arti kata XAMPP sebenarnya merupakan suau singkatan :
X = dapat di jalankan di banyak sistem operasi, seperti Windows, Linux, Mac Os dan Solaris.
A = Apache, merupakan aplikasi web server yang berguna untuk menproses halaman web yang langsung untuk user berdasarkan kode PHP yang dituliskan.
M = MySQL, merupakan aplikasi database server yang berguna untuk mengelola database.
P = PHP, merupakan bahasa pemrograman web yang populer saat ini.
P = Perl, merupakan bahasa pemrograman.

Untuk meng-install XAMPP di komputer windows, anda perlu mengunduh / download installer XAMPP terlebih dahulu di apachefriends.org atau langsung dari sourceforge xampp dan mirrror ini. Saran saya, gunakan versi stabil, untuk saat ini saya sendiri menggunakan XAMPP 1.7.4. Sedangkan versi terbaru saat saya membuat tutorial ini adalah XAMPP versi 1.8.1.

Setelah selasai mengunduh (download) XAMPP, buka folder lokasi installer XAMPP. Klik kanan pada file, dan pilih Run as Administrator. Jika muncul notifikasi UAC klik YES saja lihat gambar 1 dan 2 berikut:



Setelah itu akan muncul jendela pilihan bahasa untuk meng-install, pilih English saja dengan mengabaikannya dan langsung pilih OK. yang di susul dengan munculnya notifikasi dari XAMPP segera klik OK lagi. lihat gambar 3 dan 4 di bawah:



Selanjutnya Monitor akan ter-blok secara penuh menampilkan proses instalasi XAMMP dan menampilkan dialog-dialog. Ikuti saja seperti gambar-gambar berikut:






Setelah anda meng-klik Finish, proses instalasi di komputer windows anda pun selesai. jika muncul notifikasi lain anda tinggal pilih YES, NEXT atau OK saja. biasanya XAMPP akan meng-konfigurasi sendiri secara otomatis, ditandai dengan munculnya command promt yang menjalankan batch-batch/script yang di buat XAMPP yang baru anda install.

Secara default, semua layanan dihentikan, sehingga Anda perlu menyalaknnya secara manual apa saja yang Anda butuhkan dan mengaturnya pada saat itu juga. Dalam Pengaturan Layanan Anda dapat memilih untuk Control Panel XAMPP untuk menjalankan sebagai layanan yang berjalan otomatis aktif pada startup. Selain itu, Anda dapat membuat account dan password untuk akses yang aman. XAMPP juga mendapatkan update secara reguler untuk mengikuti versi terbaru dari komponen yang disertakan.
