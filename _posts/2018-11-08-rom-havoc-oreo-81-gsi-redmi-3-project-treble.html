Havoc OS adalah Custom ROM baru dengan banyak fitur yang tidak anda temukan di custom rom lain. Layaknya ROM Bootleggers, rom ini dibekali banyak sekali fitur kustomisasi. Lebih banyak dari yang ada pada custom rom populer Resurrection Remix.

Saat mencobanya pertama kali, saya dibuat kagum oleh performanya. Boleh dikatakan bahwa Havoc OS adalah ROM terbaik dalam hal Performa & Kustomisasi saat ini.

Baru-baru ini Google telah meluncurkan versi android terbaru yakni Android 9.0 Pie. Begitupun Havoc OS, juga telah merilis versi Pie-nya. Namun sayang, sebagai pemakai Xiaomi Redmi 3, belum bisa menikmatinya. Ini dikarenakan custom Kernel & Vendor buatan Winkarbik untuk project treble belum kompatibel untuk versi Pie.

Tapi kamu masih bisa instal rom/GSI (General System Image) apa saja asal memakai Android Oreo, termasuk Havoc.

Khusus Havoc OS, memiliki tampilan yang sudah diintegrasikan dari Android Pie, seperti tombol visual volume yang berada di kanan layar, panel Quick Setting bergaya Pie, bahkan sudut visual layar yang bisa diatur bentuknya runcing atau melengkung, dll.

Sekarang kita menuju bahan-bahannya. Jangan lupa untuk selalu backup Internal Storage terlebih dahulu tiap kali kita akan menginstal rom. Menambahkan internal storage dalam memformat partisi sangat disarankan. Ini untuk menghindari segala kemungkinan error yang bisa muncul saat penginstalan dan akan memudahkan segala aplikasi untuk berjalan dengan baik.

  • TWRP Treble
  • Kernel Vendor Treble
  • Fix Camera Delay & Lags (fix cam saya ambil dari vendor buatan Ariafan)
  • Havoc Oreo GSI
  • Boot Animation
  • GApps (arm64 8.1 nano)
  • Magisk (root)
  • MagiskHide Props Config (fix vendor mismatch)

Setelah bahan-bahan diatas selesai anda siapkan pada external sdcard, saatnya melakukan penginstalan. Harap perhatikan baik-baik urutannya karena ini akan sangat berpengaruh pada keberhasilan anda menginstal.

  1. Extract fix cam delay n lag.zip pada external sdcard.
  2. Extract Havoc-arm64_aonly_vanilla_nosu-0819.zip pada external sdcard.
  3. Install TWRP Treble. Jika tanpa PC, masuk Recovery, pada menu TWRP pilih Install > klik tombol Install Image > pilih twrp-3.2.3-0-ido-treble.img > pilih Recovery dan swipe untuk menginstall.
  4. Reboot recovery lagi.
  5. Wipe > Format Data. Ini akan menghapus semua data dan enkripsi pada Internal Storage (hati-hati).
  6. Reboot recovery lagi.
  7. Wipe dalvik, system, data, cache, vendor.
  8. Flash Vendor Treble (kernel-vendor-from-lineage-15.1-20180923-UNOFFICIAL-ido.zip.zip).
  9. Mount > Vendor. Jika sudah tercentang, hilangkan centang dan klik lagi untuk mencentang.
  10. Advanced > File Manager > pilih sdcard1/vendor/bin/mm-qcamera-daemon > copy file > vendor/bin > swipe.
  11. Advanced > File Manager > pilih sdcard1/vendor/etc/thermal-engine.conf > copy file > vendor/etc > swipe.
  12. Flash GSI. Pada menu TWRP pilih Install > Install Image > Choose Storage > MicroSD > pilih Havoc-arm64_aonly_vanilla_nosu-0819.img > pilih System > swipe.
  13. Mount > System. Jika sudah tercentang, hilangkan centang dan klik lagi untuk mencentang.
  14. Instal animasi boot. Advanced > File Manager > pilih sdcard1/bootanimation.zip > copy file > system/media > swipe.
  15. Reboot recovery lagi.
  16. Flash Magisk.
  17. Reboot.
  18. Buat sebuah file dengan nama printslist pada root Internal Storage dan ketikkan "Xiaomi Redmi 3=Xiaomi/ido/ido:5.1.1/LMY47V/V9.2.1.0.LAIMIEK:user/release-keys" didalamnya. Ini untuk membuat fingerprint agar device certified di Play Protect, menghilangkan error vendor mismatch dan SafetyNet Check sukses. (lihat gambar dibawah)
  19. Install Magisk module. Buka Magisk > Modules > klik + > pilih MagiskHide_Props_Config-v2.4.3-v30.zip.
  20. Reboot.
  21. Buka Settings > System > About phone > ketuk 8x pada Build number (hingga Developer options aktif).
  22. Kembali ke Settings > System > Developer options > aktifkan Local terminal
  23. Buka Terminal > Ketik "su props" [enter] & grant permission, "props" [enter], pilih 1, pilih f, pilih 15 (Xiaomi), pilih 24 (Xiaomi Redmi 3), ketik "y" [enter] & "y" lagi [enter] kemudian Redmi 3 akan reboot.
  24. Setelah reboot, error popup vendor mismatch hilang, silakan reboot recovery lagi untuk install gapps.
  25. Reboot & nikmati Havoc OS.

Sampai sini penginstalan selesai dan anda siap menggunakan Havoc Oreo GSI di Redmi 3 kesayangan anda.
