Tutorial flashing menggunakan QFIL umumnya serupa dan sudah banyak yang membagikan. Namun, agar anda tidak bingung, akan tetap saya berikan langkah flashing firmware secara khusus untuk Hisense B16C2G alias Andromax E2+. Silahkan simak di bawah ini :

Persiapan Sebelum Flashing Firmware

Lengkapi dulu file-file dan software yang dibutuhkan dengan megunduh dari link yang tersedia di bawah. Selain itu, siapkan juga PC/Komputer dengan OS Windows dan kabel data yang bagus:

1. Firmware Hisense B16C2G branding Smartfren. Download file firmware dari sini. Ekstrak file di folder baru dengan nama Andromax untuk memudahkan mencari.

2. Download juga Qualcomm Flasher Qfil QPST 2.7.460 di sini. Jika file installer dalam bentuk kompresan (zip/rar) ekstrak duku dengan winzip atau winrar di folder yang lain, kemudian install di PC seperti biasanya.

3. Download Driver HS-USB Qualcomm dari sini yang sudah bisa dipakai untuk Windows 10. Boleh juga cari sendiri driver-nya dari pencarian google .

Langkah-Langkah Flashing/Install ulang menggunakan QFIL

Setelah hal yang dibutuhkan sudah lengkap. Silahkan buka software QFIL. Biasanya ada di C:\Program Files (x86)\Qualcomm\QPST\bin\QFIL.exe untuk Windows 64Bit atau C:\Program Files\Qualcomm\QPST\bin\QFIL.exe untuk 32Bit.

Jika sudah terbuka, klik tombol Browse pada bagian Progammer Path.

Silahkan arahkan ke folder hasil ekstrak firmware sebelumnya. Kemudian pilih file "prog_emmc_firehose_8909_ddr.mbn" yang ada dalam folder firmware.

Lanjutkan klik "Load XML" kemudian pilih file 3 file berikut; "rawprogram0.xml", "rawprogram_unspare.xml" dan "patch0.xml".

Hubungkan ponsel dengan kabel data dalam mode download, pastikan terdeteksi pada QFIL sebagai HS-USB QDLoader 9008. Jika belum terdeteksi cek lagi instalasi driver, pastikan, pastikan driver sudah terinstal dengan benar agar QFIL bisa mengenali ponsel.

Jika sudah terdeteksi, lanjut saja klik menu "Download" dan tunggu hingga proses selesai.

Jika proses sudah selesai, sekarang anda boleh mencabut ponsel dari komputer kemudian nyalakan seoerti biasa.

Flashing dengan QFIL sudah selesai. Kini yang perlu anda lakukan hanya menunggu hingga logo andromax muncul yang berarti proses flash firmware berhasil. Proses boot pertama biasanya memang lebih lama daripada booting normal, jadi sabar saja.

Catatan:Sekedar untuk diketahui QFIL juga bisa dipakai untuk flashing firmware fastboot ponsel Xiaomi dan Lenovo yang ber-cipset Qualcomm juga loh. Langkah-langkah tutorial flashing menggunakan QFIL / QPST ini juga bisa dipakai untuk ponsel lain dengan Chipset Qualcomm dengan beberapa penyesuaian
