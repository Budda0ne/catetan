IGOS Nusantara 8.1 atau IGN8.1 diluncurkan tanggal 27 Pebruari 2013. Pengembangan IGN8.1 dilakukan oleh Pusat Penelitian Informatika LIPI dan dibantu oleh pengembang/kontributor dari komunitas open source di Indonesia. IGN8.1 merupakan perbaikan dan pembaruan terhadap IGN8.0. Varian atau jenis yg telah dibuat ada empat belas ISO image. Banyaknya varian dibuat untuk memenuhi kebutuhan pengguna yang beragam.

Ada beragam fitur IGOS Nusantara versi 8.1, antara lain:

  • Beragam perbaikan terhadap IGN8.0
  • Pembaruan aplikasi: kernel 3.7.9, Firefox 19, Wine, MySQL dan aplikasi lainnya.
  • Tersedia IGN8.1 LiveDVD, LiveCD micro/mini/full, CD mini instalasi: versi 32bit dan 64bit
  • MATE Desktop bergaya khas IGOS Nusantara
  • IGN8.1 LiveDVD dengan delapan Desktop Environment (DE): Cinnamon, GNOME3, GNOME2/fallback, KDE, LXDE, Mate, OpenBox, XFCE
  • Satu set ikon IGNdelapan untuk semua DE
  • Dukungan boot untuk perangkat UEFI, EFI: MacBook/iMac, Laptop/PC yang memakai EFI
  • Tersedia 23GB repositori 32bit, dan 29GB repositori 64bit
  • Tersedia Download via torrent 32bit dan 64bit

IGN8.1 terdiri dari versi LiveCD dan LiveDVD. LiveCD terdiri dari varian/jenis micro, mini dan full. LiveCD dan LiveDVD memiliki perbedaan dalam jumlah aplikasi yg dimasukkan:

  • IGN8.1 LiveCD Micro berfungsi untuk browsing, aplikasi utama yang ada didalamnya adalah Desktop Mate + Firefox.
  • IGN8.1 LiveCD Mini = isi Micro + Pemutar audio/video
  • IGN8.1 LiveCD Full = isi Mini + Pemutar audio/video + LibreOffice + beberapa aplikasi lain
  • IGN8.1 LiveDVD = isi Full + 8 Desktop Environment + banyak aplikasi.

IGN8.1 CD mini adalah versi yang harus dipasang lebih dulu (instaler). CD mini hanya akan memasang aplikasi minimal, setelah booting ulang akan masuk ke terminal. Versi ini direkomensasikan untuk pengguna yang akan membuat IGN menjadi server.
