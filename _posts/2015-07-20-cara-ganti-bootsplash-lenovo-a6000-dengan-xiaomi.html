Meskipun demikian, bootsplash MIUI ini bisa juga anda pakai di ROM lain. Bahkan file bootsplash ini juga bisa dipakai ponsel Redmi 2 untuk mengembalikan bootsplash bawaan, karena file bootsplash ini memang milik Redmi 2.

Cara Install Bootsplash dengan Custom Recovery

Untuk mengganti bootsplash atau bootlogo, anda membutuhkan custom recovery. Jika belum silahkan pilih salah satu custom recovery yang anda sukai (CWM, PTR, atau TWRP) di posting ini. Di artikel tersebut juga sudah disertakan cara-cara install-nya. update untuk yang sudah upgrade menjadi lollipoop bisa gunakan TWRP Lollipop untuk Lenovo A6000/plus Tapi, Jika anda sudah mempunyai salah satu custom recovery tersebut ikuti langkah-langkah berikut:
  1. Download file bootsplash di bawah
  2. Letakkan di sdcard (internal juga bisa), sebaiknya jangan di dalam folder agar mudah di temukan.
  3. Matikan Lenovo A6000 anda dan nyalakan kembali dalam recovery mode dengan menekan tombol [power], [vol up] dan [vol down] secara bersamaan.
  4. Dalam mode recovery, pilih install zip from sdcard atau internal sd
  5. Arahkan ke file MIUI6-Bootsplash-signed.zip dan konfirmasi pilihan anda.
  6. Tunggu hingga instalasi sukses.
  7. Langsung reboot ponsel anda tanpa wipe.
Lihat, sekarang bootslpash bawaan anda sudah berubah seperti ponsel Redmi 2. Jika anda ingin mengembalikan bootsplash bawaan Lenovo A6000, gunakan file Bootsplash-Lenovo-signed.zip dengan langkah yang sama.
