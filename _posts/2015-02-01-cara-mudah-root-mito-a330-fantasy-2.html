Ponsel ini sudah menggunakan OS Android Kitkat sama dengan Mito Fantasy 2 A75. Sehingga saat akan melakukan rooting tanpa PC (root kitkat langsung dari hape) banyak yang mengalami kesulitan.

Solusinya adalah melakukan rooting melalu komputer dengan V-ROOT/i-Root. Cara ini juga lebih mudah dan cepat. Jika sebelumnya tool yang digunakan adalah VROOT /i-ROOT. (baca post lama: root samsung dengan vroot dan root mito a75 kitkat, caranya hampir sama). Maka kali ini saya akan tunjukkan alternatif rooting lain dengan Kingo Root App.

Sebelum melakukan rooting, persiapkan dulu alat dan bahan yang akan digunakan:

  1. ADB dan Fastboot driver installer, download di bawah
  2. Kingoroot App link ada di bawah juga.

Sebelum melanjutkan, saya ingatkan untuk melakukan back-up data anda seperti kontak (sinkronkan dengan akun google), sms, data di sdcard dll. Dengan mengikuti tutorial ini, saya anggap anda sudah setuju dengan disclaimer blog ini.

Tutorial Rooting MITO Fantasy 2 A330 (KitKat)

Ikuti langkah langkah berikut dengan runut dan teliti agar rooting anda berhasil:

  1. Ekstrak dan Install ADB Fastboot driver yang sudah di download sebelumnya, DPInst_x86.exe untuk windows 32bit atau DPInst_x64.exe untuk windows64bit
  2. Install tool Kingoroot App yang tadi anda download. ikuti saja petunjuk atau instruksi yang diberikan
  3. Sekarang aktifkan usb debugging mito a330 anda di developer option
  4. Kembali ke laptop dan jalankan tool Kingoroot App yang sudah terinstall.
  5. Sambungkan Mito A330 ke komputer dengan kabel data usb.
  6. Mito A330 akan masuk mode FASTBOOT, dan secara otomatis akan reboot beberapa kali.
  7. Jika dalam mode fastboot Kingoroot meminta untuk meng-install driver, jika tidak meng-install driver dengan benar, anda perlu terkoneksi internet agar kingoroot mendownload driver secara otomatis.
  8. Tunggu hingga Kingoroot App mendeteksi ponsel anda. Kingo Root akan mengenali MITO A330 sebagai HTC Device (wt97057).
  9. Ponsel akan reboot beberapa kali saat proses root berlangsung.

Jaga agar kondisi layar selalu menyala, karena akan muncul notifikasi untuk menginstall applikasi superuser yang harus dikonfirmasi ijin root-nya.
Tekan ijinkan atau allow jika muncul notifikasi di layar.

Setelah proses selesai, restart ponsel anda dan lihat di app drawer. Jika ada aplikasi SuperSU berarti MITO Fantasy 2 A330 anda sudah berhasil di root.
