Meskipun demikian, Stock ROM MIUI 8.2 untuk Redmi 3 sendiri tak kalah populer mengingat tak semua pemilik ponsel Xiaomi bisa melakukan instalasi Custom ROM RR Nougat v.5.8.2 ini dengan sukses. Ini terjadi karena belum bisa melakukan unlock bootloader secara aman. Akibatnya tidak bisa memasang TWRP Recovery pada Redmi 3 yang selanjutnya akan dipakai untuk memasang ROM Ressurrection Remix ini.

Baca beberapa referensi berikut agar bisa dengan mudah install custom ROM RR v.5.8.2 Nougat pada Xiaomi Redmi 3 atau Redmi 3 Pro/Prime.
1. Cara mudah Unlock Bootloader Xiaomi.
2. Pasang TWRP pada Redmi 3 dengan Aman di sini

Setelah bootloader Redmi 3 sudah unlocked dan TWRP recovery terpasang, barulah bisa memasang ROM RR dengan sukses. Silahkan unduh terlebih dahulu rom zip di bawah ini.

File size: 515.9 MB
Download:
RR-N-v5.8.2-20170316-ido-OFFICIAL.zip

Abaikan link diatas dan ambil versi Update Terbaru Juli 2017 dari sini

Setelah terunduh, ikuti cara install rom dengan langkah-langkah mudah berikut ini:

Langkah 1. Pindahkan file RR-N-v5.8.2-20170316-ido-OFFICIAL.zip dari folder download ke sdcard atau internal storage sehingga berada di luar folder.

Langkah 2. Reboot ke mode recovery dengan cara mematikan daya terlebih dahulu. Kemudian, tekan ketiga tombol power + volume + + volume - secara bersamaan hingga muncul logo MI lepaskan tombol power tetapi jangan lepaskan tombol volume + + volume - hingga muncul menu TWRP.

Langkah 3. Dalam mode Recovery (TWRP) sebaiknya anda melakukan full back-up sebelum melakukan pemasangan rom. Ini bisa anda gunakan untuk mengembalilan ponsel anda seperti sediakala. Sehingga ponsel anda tetap aman walaupun anda tidak sukses dalam proses pemasangan. Pilih menu Backup dan pilih lokasi yang anda sukai. Namun saya sarankan untuk menggunakan external sdcard (microSD).

Langkah 4. Karena anda akan melakukan pemasangan untuk pertama kalinya, anda perlu melakukan full wipe. Untuk itu, pilih menu Wipe dan geser indikator ke arah kanan untuk konfirmasi. Ini untuk memastikan tidak ada data yang tersisa dari ROM sebelumnya (baik stock/MIUI maupun custom).

Langkah 5. Setelah selesai melakukan Wipe, lanjutkan dengan proses install ROM Ressurrection Remix. Caranya dengan kembali ke menu utama dan memilih menu Install. Arahkan ke lokasi dimana anda memindahkan file RR-N-v5.8.2-20170316-ido-OFFICIAL.zip sebelumnya. Pilih file tersebut. Slide (geser) indikator ke arah kanan untuk melakukan flashing. Proses mungkin akan berjalan lambat, sabar saja. Tunggu hingga ada pemberitahuan success

Langkah 6. Dari sini, anda bisa melakukan flashing beberapa file zip lain sebagai tambahan fitur menggunakan langkah yang sama (Langkah 5). Terutama GAPPS (Google Apps) agar bisa memanfaatkan fitur-fitur Google seperti Playstore atau Gmail. Anda juga busa sekaligus melakukan rooting menggunakan SuperSU.zip dalam langkah ini tidak perlu karena SuperSU sudah include dalam Magisk.

Langkah 7. Silahkan kembali ke menu utama TWRP dan pili menu Reboot dan pilih System untuk nenyalakan ponsel. Proses boot pertama setelah instalasi biasanya akan SANGAT LAMA. Jadi JANGAN PANIK. Boot animasi dengan logo RR biasanya berkisar antara 5 hingga 15 menit. Paling lama sekitar 30 menit.

Langkah 8. Silahkan lakukan pengaturan awal seperti yang diminta jika anda juga melakukan instalasi GAPPS, sebaiknya langsung masukkan akun google anda agar data kontak dan aplikasi lainnya bisa di sinkronisasi. Anda bisa mengambil GAPPS dari sini

Selesai. Anda bisa menggunakan ROM Ressurrection Remix yang sudah terpasang dengan sukses diponsel Xiaomi Redmi 3 anda. Anda juga bisa mencoba menggunakan tema-tema substratum untuk mempercantik tampilan. Salah satu tema substratum yang saya sukai adalah MonoArt. Kalau ingin mencobanya, silahkan download apk-nya dari sini
