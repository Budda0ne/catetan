KnoAcc.org - Redmi 3 adalah ponsel paling duluan dihapus dari daftar dukungan update MIUI oleh Xiaomi. Berhubung saya masih memakai ponsel ini untuk daily use dan memainkan beberapa game termasuk game populer PUBG Mobile dan PES 2019, mau tidak mau harus mencari rom alternatif yang mampu menampung kedua game tersebut. Sebelumnya, saya menggunakan Havoc OS (Oreo 8.1) treble GSI ¹ yang lumayan stabil. Sayangnya ada beberapa bug yang cukup mengganggu.

Setelah beberapa waktu, saya mencoba beberapa ROM GSI lain. Diantaranya (yang lumayan lama) adalah ROM Pixel Experience ², namun masih belum sreg juga. Akhirnya kembali memakai Havoc OS beberapa minggu. Yang pada akhirnya menemukan fix (trial error) agar tidak terlalu buggy. Fokus fixing ini ada di partisi vendor, sayang caranya sudah lupa mana yang benar karena terlalu banyak trial error. Jadi saya backup saja partisi vendor-nya menjadi file IMG yang bisa kamu unduh di sini. Dan bisa kamu gunakan untuk memasang ROM GSI manapun termasuk Android Pie, asal file image tidak melampaui besar partisi system Redmi 3.

Nah, karena sudah fix di Oreo 8.1, maka saya pun mencoba untuk memasang Custom ROM berbasis Android Pie. Tentu saja masih memakai project treble karena pilihan cusrom untuk Redmi 3 sudah makin terbatas, sedangkan project treble sedang hangat-hangatnya. Setelah menimbang beberapa ROM GSI berbasis Android PIE, pilihan pun jatuh pada Arrow OS yang menyertakan image untuk device treble A-only dan ukuran file image (.img) hanya sebesar 1.8 Gb-an setelah diekstrak untuk versi yang sudah ada Gapps-nya, dan 1.5 Gb-an untuk versi non-Gapps. Selain itu, Develepoer ArrowOS masih sangat aktif memperbarui ROM GSI-nya.

Mencicipi Android 9.0 (Pie) di Redmi 3 Tanpa Bug!

Baca sampai akhir! Jangan skip. Karena bisa jadi ada yang terlewat, dan saya tidak bertanggung jawab jika ada kerusakan baik software maupun hardware atau kerugian lain yang disebabkan oleh pemasangan ROM ini.

Oke, Jika kamu tertarik, kini saatnya untuk mempersiapkan Ponsel Redmi 3 (baik ram 2Gb maupun ram 3Gb) agar bisa menggunakan Android Pie tanpa bugs ³ yang mengganggu.


Sebelum itu, pastikan kamu sudah familiar dengan cara memasang custom rom dengan custom recovery! Dan lebih bagus lagi sudah pernah memasang ROM berbasis Project Treble, lebih spesifik lagi GSI.

Pastikan kamu sudah memasang TWRP yang mendukung project treble ⁴. Cirinya, tak hanya bisa meng-install file zip, tetapi juga bisa langsung flashing file image tertentu ke partisi tertentu. Mislanya: partisi system, boot dan vendor. Kau bisa mengunduh TWRP treble di situs official berikut ini. Pastikan bootloader Redmi 3 sudah kamu unlock sebelum memasang TWRP.

Jika kamu belum pernah memasang ROM GSI, kamu perlu mengunduh image boot.img di sini, ekstrak dan melakukan flashing ke partisi boot. Jika sudah pernah memakai rom treble, kamu bisa langsung mengunduh file vendor di sini, ekstrak dan lakukan flashing vendor.img ke partisi vendor agar Xiaomi Redmi mendukung semua ROM GSI dari treble project

File GSI untuk partisi system milik ROM ArrowOS Pie tersedia dua jenis. Yakni untuk device yang mendukung A-B dan A-only. Karena aslinya Redmi 3 tidak mendukung project treble, maka yang kamu perlukan adalah jenis GSI A-only. GSI A-only Arrow OS ini juga mempunyai 2 pilihan: 1. Dengan gapps embedded (gapps included), bisa download di sini (sekitar 800-an MB berupa file zip), dan 2. tanpa gapps (sekitar 600-an MB dalam bentuk file zip) download dari sini. Untuk yang terakhir kamu harus memasang gapps sendiri semisal opengapps. Apapun yang kamu pilih, ekstrak file zip, install file system.img ke partisi system

Magisk! Ini adalah hal paling penting menurut saya. Selain sebagai pengganti SuperSU sebagai manager superuser, sejak adanya magisk systemless interface, hampir semua kebutuhan modifikasi saya lakukan dari sini. Kamu bisa mengunduh Magisk Manager di sini dan Magisk Installer dari sini dan memasangnya dengan tutorial ini.

Lanjut dengan memasang modul MagiskHide Props Config. Jika kamu tidak begitu merasa perlu menggunakan aplikasi perbankan, rasanya modul ini tidak kamu butuhkan. Tetapi kamu akan tetap membutuhkan modul Busybox for Android NDK jika kamu berencana menggunakan ponsel dengan akses root dan modifikasi lainnya. Kamu bisa memasang keduanya dari menu unduhan yang tersedia di aplikasi Magisk Manager. Atau unduh saja keduanya dari sini dan pasang dari TWRP.

Jika ada beberapa hal yang belum kamu pahami dalam memasang ROM ArrowOS untuk Redmi 3, silahkan tanyakan dalam kolom komentar di bawah. Semoga bermanfaat, dan selamat menikmati Android Pie pada Xiaomi Redmi 3. Asyeeek...

Jika kamu memilih untuk tidak memasang Magisk Hide Props maka tabahkan hatimu dengan gangguan pop-up "A vendor image mismatch has been detected. Typically this means your vendor image is out of date. Please ensure your vendor image matches XXX" setiap kali boot-up. Untuk itu sebaiknya memasangnya dan lakukan pemalsuan identitas seperti di tutorial memasang rom Havoc OS. Ingat! Beberapa menu yang muncul di terminal mungkin berbeda karena modul ini selalu diperbarui.

Ringkasan:
Berikut urutan singkat proses flashing ArrowOS Pie GSI untuk Redmi 3:

  1. Pasang TWRP Treble.
  2. Instal image boot dan vendor atau vendor saja.
  3. Instal image system ArrowOS GSI
  4. Pasang Magisk
  5. Pasang modul Busybox for Android NDK dan Magisk Hide props
  6. Rubah props sesuai keinginan untuk bypass safetynet.
