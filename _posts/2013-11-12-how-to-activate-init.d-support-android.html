Sebagian dari anda yang berkunjung ke blog IDC mungkin sudah paham benar aa itu init.d support dan apa fungsinya. Apalagi kalau sidah sering utak-atik smartphone android-nya.

Nah, bagi anda yang belum tahu arti dan fungsi init.d, ada baiknya anda simak dulu penjelasan apa init.d support itu dan apa fungsi init.d support.

Jadi, apa sih init.d support itu? Init.d support yang kalau di-Bahasa Indonesia-kan menjadi "dukungan init.d" adalah kondisi dimana device anda dalam hal ini adalah Smartphone Android anda, sudah bisa mengeksekusi script [bash] tertentu yang dijadwalkan untuk dijalankan secara rutin setiap kali device anda booting.


Mungkin anda akan bertanya: "Init.d itu sendiri apa? Dan apa fungsi init.d ini?" Init.d sendiri merupakan sebuah plug-in untuk menambahkan fungsionalitas device android anda yang berfungsi untuk menampung script berisi perintah tertentu yang akan di eksekusi secara rutin setiap kali device anda booting.

Script yang ditanamkan bisa bermacam-macam. Misalkan saja script untuk menambahkan fungsi swap atau membalik memori internal menjadi memori eksternal, bahkan anda bisa membatasi aplikasi background yang berjalan atau men-tweak system dengan menambahkan script di folder system/etc/init.d jika gadget anda sudah mendukung-nya.

How to Activating init.d support? Ikuti Tutorial berikut ini:

Sebelum masuk ke tutorial, syaratnya smartphone anda harus sudah:
- Root.
- Sudah terinstal Busybox
- Custom Recovery (CWM/TWRP) jika anda ingin menggunakan cara ribet-nya. Tanpa itu juga tak apa, anda bisa menggunakan cara mudah-nya, jika menggunakan cara ribet akan semakin ribet.

Sesudah syarat diatas anda penuhi, anda perlu mempersiapkan bahan-bahan yang diperlukan untuk mengaktifkan init.d suport di Smartphone Android anda.

1. Root Explorer atau Root Browser, yang jelas bisa membuka folder-folder sistem dengan hak akses root.
2. file Init.d-autorunner-stock.zip jika anda punya custom recovery [saya memakai cwm untuk tutorial ini].
3. Init.d Installer [aplikasi] download dari sini dan letakkan di sdcard anda.

Setelah syarat dan bahan sudah terpenuhi, anda bisa melanjutkan instalasi init.d ke android anda dengan mengikuti 2 tutorial dengan tingkat kesulitan yang berbeda dibawah:

Cara Ribet Install Init.d

Tutorial ini cocok untuk anda yang ingin meng-aktifkan dukungan init.d sembari meng-eksplorasi "jeroan" sistem android. Tapi harap di-ingat, cara ini cukup beresiko jika anda tidak ber-hati-hati :).
Anda membutuhkan bahan 1 sampai bahan 2 saja. Caranya:
- Buka folder /system/etc/ dengan root explorer dan tekan "mount R/W" dipojok kanan atas dan cari file dengan nama "install-recovery.sh".
- Jika ada tekan lama pada file tersebut dan buka dengan text-editor dan copy-paste kode berikut di bagian paling bawah sendiri:
# init.d support
busybox run-parts /system/etc/init.d/
- Jika tidak ada, buat file baru. tekan pilh Menu > Create new file. dan beri nama "install-recovery.sh". Buka dan copy-paste kode
#!/system/bin/sh
# init.d support
busybox run-parts /system/etc/init.d/
kemudian simpan dan hapus semua file yang ber-ekstensi .bak
- Ingat, selau tambahkan satu baris kosong setelah kode diatas [tekan enter 2 kali pada saat selai copas kode]
- Reboot Android anda dan masuk Recovery Mode dan flash file init.d-autorunner-stock.zip dengan memilih install zip from sdcard - chose zip from sdcard pilih init.d-autorunner-stock.zip
- Setelah complete, kembali ke menu awal dan reboot system now.
- Cek apakah init.d sudah aktif dengan membuka /data/local/tmp menggunakan Root Explorer. Jika di sana ada file "init.d_log_test.txt" maka init.d sudah aktif.

Cara Mudah Install Init.d

.
Cara paling mudah adalah dengan menginstall init.d dengan menggunakan aplikasi init.d installer yang sudah anda download sebelumnya (bahan 3). Caranya:
- Install Universal Init.d Support seperti aplikasi biasa. Jangan lupa untuk mencentang opsi from unkown source di pengaturan aplikasi.
- Setelah selesai proses instalasi, tap/sentuh tombol Buka atau Open.
- Dalam aplikasi ini hanya ada 2 tombol aktif. Yakni tombol Activate dan Verify. Jika anda ingin segera meng-aktif-kan dukungan init.d, anda tinggal tekan tombol Activate. Setelah itu reboot smartphone anda.
- Jika sudah menyala, silahkan cek apakah android anda sudah mendukung init.d atau belum, sentuh di tombol Verify. Jika berhasil akan muncul notifikasi "Success You have init.d support!"
- Untuk mengecek ulang apakah init.d sudah didukung/aktif buka folder /data dan cari file "Test.log". kalau file ini ada, berarti Smartphone/Tablet android anda sudah mendukung init.d support.

Cara inilah yang paling mudah untuk mengaktifkan init.d, tanpa harus repot-repot membuka-buka folder-folder sistem sehingga resikonya lebih kecil.

Dengan aktifnya dukungan init.d di Smartphone anda, maka andapun bisa menambahkan berbagai script untuk tweaking mulai dari meningkatkan kemampuan baterai hingga performa CPU dan RAM Android anda.
