Panduan instalasi Cross A7star: MIUI ROM via CWM Recovery

Namun sebelum mencoba menggunakan MIUI ROM for Cross A7star ini, anda harus mempartisi sdcard menjadi 2 partisi terlebih dulu. Partisi pertama akan di gunakan sebagai media penyimpanan utama (/sdcard) dan partisi kedua akan digunakan sebagai internal memori (/data). Baru lanjutkan tutorial berikut:
  1. Download Cross A7star: MIUI ROM dari sini
  2. Letakkan file MIUI.zip di /sdcard
  3. Charge A7star anda setidaknya 80% untuk jaga-jaga.
  4. Reboot dan masuk ke mode CWM Recovery. (jika belum anda bisa meng-install terlebih dulu) dalam mode recovery lakukan:
    • wipe data/factory reset
    • wipe cache partition
    • kemudian pilih menu
    • pilih install zip from sdcard
    • pilih chose zip from sdcard ( cari file MIUI.zip )
  5. install tunggu hingga selesai pilih rebot system now.
Booting pertama akan lama, tunggu hingga masuk ke home screen. Proses ini bisa memakan waktu 3-5 menit. Biasanya ROM akan terload dengan baik, tetapi jika anda menunggu hingga lebih dari 15 menit tapi belum masuk ke home screen. maka sebaiknya anda cek lagi partisi kedua di sdcard anda dengan minitool apakah ext4, jika iya rubah dulu menjadi ext3.
