Distro pasaran (dalam arti positif loh ya) ini secara default menggunakan repository (repo) luar. Sehingga kadang kala saat pengguna ingin melakukan pasangan aplikasi maupun update terkendala jaringan internet yang lola mengingat server repo berada di luar negri.


Untuk mengatasi hal ini, beberapa komunitas pengguna (baca: Universitas) dan bahkan ISP memberikan layanan untuk memperpendek jarak tempuh pengambilan aplikasi-aplikasi ini dengan membuat repository lokal. Maksudnya, menyediakan server repo mirror yang menggunakan server dalam negri. Alasan ISP memberikan layana repo lokal bisa jadi karena tatif bandwith ke luar negri yang lebih mahal daripada tarif bandwith lokal, selain alasan responsifitas koneksi yang lebih cepat.



Beberpa repository lokal untuk pengguna OS Ubuntu 14.04 yang diberi nama Trusty Tahr ini antara lain

1. www.foss-id.web.id (Telkom)


Repository ini dibuat oleh Telkom sang ISP pelat merah sejak lama. Repo milik Telkom tak hanya melayani OS Ubuntu saja, melainkan juga distro linux lain.

Repository yang perlu di tambahkan di file konfigurasi repo adalah:

#www.foss-id.web.id (Telkom)
deb http://dl2.foss-id.web.id/ubuntu/ trusty main restricted universe multiverse deb http://dl2.foss-id.web.id/ubuntu/ trusty-updates main restricted universe multiverse deb http://dl2.foss-id.web.id/ubuntu/ trusty-security main restricted universe multiverse deb http://dl2.foss-id.web.id/ubuntu/ trusty-backports main restricted universe multiverse deb http://dl2.foss-id.web.id/ubuntu/ trusty-proposed main restricted universe multiverse

2. ftp.itb.ac.id (ITB, INHERENT)


Sama dengan Telkom, repo milik Universitas di bandung ini juga tak hanya menyediakan repo untuk Ubuntu saja. Untuk menambahkan server repository ITB sebagai repo lokal Ubuntu, tambahkan alamat berikut dalam konfigurasi repo Ubuntu 14.04:

#ftp.itb.ac.id (ITB, INHERENT)
deb ftp://ftp.itb.ac.id/pub/ubuntu trusty main restricted universe multiverse deb ftp://ftp.itb.ac.id/pub/ubuntu trusty-updates main restricted universe multiverse deb ftp://ftp.itb.ac.id/pub/ubuntu trusty-security main restricted universe multiverse deb ftp://ftp.itb.ac.id/pub/ubuntu trusty-backports main restricted universe multiverse deb ftp://ftp.itb.ac.id/pub/ubuntu trusty-proposed main restricted universe multiverse

3. komo.vlsm.org (si Komo)

Sudah ada sejak lama, repo si Komo yang namanya mungkin terinspirasi lagu si komo lewat(?) agar memperlancar trafik data saatelakukan apt-get instll / apt-get update :).

Tambahkan baris berikut di file konfigurasi repository untuk menambahkan repo si Komo sebagai repo lokal.


#komo.vlsm.org
deb http://komo.padinet.com/ubuntu/ trusty main restricted universe multiverse deb http://komo.padinet.com/ubuntu/ trusty-updates main restricted universe multiverse deb http://komo.padinet.com/ubuntu/ trusty-security main restricted universe multiverse deb http://komo.padinet.com/ubuntu/ trusty-backports main restricted universe multiverse deb http://komo.padinet.com/ubuntu/ trusty-proposed main restricted universe multiverse

4.kambing.ui.ac.id (UI, Telkom, Indosat, OpenIXP, INHERENT)


Nah kalau repo ini paling banyak digunakanengingat gemuk-nya repo lokal satu ini. Munkin karena dirawat secara keroyokan, koleksi yang dimiliki repo kambing bisa dibilang sangat lengkap.

Tambahkan saja kode dibawah ke dalam file konfigurasi repo ubuntu anda:


#kambing.ui.ac.id (UI, Telkom, Indosat, OpenIXP, INHERENT)
deb http://kambing.ui.ac.id/ubuntu/ trusty main restricted universe multiverse deb http://kambing.ui.ac.id/ubuntu/ trusty-updates main restricted universe multiverse deb http://kambing.ui.ac.id/ubuntu/ trusty-security main restricted universe multiverse deb http://kambing.ui.ac.id/ubuntu/ trusty-backports main restricted universe multiverse deb http://kambing.ui.ac.id/ubuntu/ trusty-proposed main restricted universe multiverse

Cara melakukan update menggunakan terminal, ketik perintah berikut untuk membula file konfigurasi repo.

$ sudo nano /etc/apt/sources.list
Salin kode repo satu persatu (maksudnya semua) diatas. Dan lakukan pembaruan setelah psemua daftar repo lokal dimasukkan.
Lanjut dengan mengetik perintah:

$ sudo apt-get update
Sebagai langkah akhir umtuk pembaruan repo agar saat melakukan apt-get install aplikasi bisa mngunduh dari repositiry lokal yanh naru saja dimasukkan.
