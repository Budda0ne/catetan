Beberapa tool yang ada biasanya hanya mapu membuat usb bootable untuk satu OS saja.
Saat ini, sejauh yang saya tahu ada Yumi Multiboot USB Creator, XBOOT dan Multisystem saja yang memberikan pilihan untuk membuat beberapa file ISO installer sistem operasi dapat di gabung menjadi satu dalam satu Bootable USB. Saya mencoba menggunakan Yumi Multiboot USB Creator untuk membuat file iso installer Windows 7 dan file iso Windows 8 berda dala satu buah Flashdisk ber kapasitas 8 Gb. Keduanya bisa berjalan dengan baik ketika saya gunakan untuk mem-booting dan menginstall Windows 7 dan Windows 8 dengan komputer yang berbeda.

Yumi Multiboot USB Creator ini merupakan salah satu dari sekian banyak tool di pendrive linux. Bisa digunakan dalam lingkungan sistem operasi windows dengan baik. YUMI sendiri merupakan singkatan dari Your Universal Multiboot Installer
