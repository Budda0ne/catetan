Intermezo : Lebih mudah melakukan instalasi windows dari pendrive (USB flashdisk)
Dulu saya seringkali 'manyun' ketika meng-install komputer (PC/Laptop) tiba-tiba berhenti dalam waktu yang lama gara-gara CD/DVD tergores, Apalagi ini memang pekerjaan saya yang jadi Teknisi Komputer lepas. Kalau sudah begini membuat saya mati kutu di depan klien, apalagi jika pengerjaannya di rumah/kantor klien tersebut.

Tanpa CD atau DVD Menginstall Windows melalui USB flashdisk ini ternyata banyak keuntungannya dibanding melalui CD atau DVD disk, Kita tidak perlu khawatir akan CD atau DVD yang tidak terbaca oleh ROM Drive. CD atau DVD yang sudah tergores (baret), memang menyulitkan dalam proses instalasi, sedangkan dengan media Flashdisk itu sendiri, hal itu bisa di singkirkan.

Karena itulah, sebagai teknisi yang baik, saya akan menunjukkan dua cara (manual dan otomatis) dalam menginstall Windows 7 melalui USB FlashDisk

Langkah pertama selalu Preparation : Sebelum kita mulai, perlu disiapkan adalah: Setidaknya 1 (satu) Flashdisk dengan kapasitas 4Gb atau lebih, karena File Installer Windows 7 setidaknya membutuhkan ruang sebesar 3Gb.

Do The Job

Spoiler : Dua cara ini sebenarnya juga bisa digunakan untuk versi Windows sebelum Windows 7

Pertama : Cara Manual


  • Tancapkan USB Flashdisk anda. --masukkan ke USB port ya, jangan ke tempat lain :)
  • Tekan Tombol [WIN]+[R], ketik 'cmd' dan klik OK.
  • Ketik 'diskpart' dan tekan [ENTER].
  • Ketik 'list disk', tekan [ENTER] dan pilih yang mana USB flashdisk kamu, hati-hati, jangan salah pilih, salah-salah yang terformat adalah disk yang lain. jika kamu hanya 1 harddisk maka USB flashdisk-nya adalah disk1.
  • Ketik 'select disk 1' dan tekan [ENTER].
  • Ketik 'clean' dan tekan [ENTER].
  • Ketik 'create partition primary' lalu tekan [ENTER].
  • Ketik 'select partition 1' dan tekan [ENTER]
  • Ketik 'active' dan tekan [ENTER].
  • Ketik 'format fs=fat32' dan tekan [ENTER]
  • Ketik 'assign' dan tekan [ENTER]
  • Ketik 'exit' dan tekan [ENTER].
  • Masukkan disk DVD windows 7 dan copy-lah semua isi DVD tadi ke USB Flashdisk kamu.
  • Booting komputer kamu dengan USB Flashdisk. (setting melalui bios komputer kamu) pastikan boot-nya melalui USB Flashdisk,

Apabila langkah-langkahnya tepat, maka proses instalasi windows akan berjalan melalui USB Flashdisk.

Kedua : Cara Otomatis


Cara otomatis lebih saya sarankan untuk anda, tetapi membutuhkan tool/program yang namanya WinToFlash, program ini akan melalukan langkah-langkah seperti cara manual tadi, yang kita perlu lakukan hanyalah menjalankan programnya.
  • Download dulu program WinToFlash versi terbaru. [download link]
  • Ekstrak dan jalankan file WinToFlash.exe
  • Klik tombol [Check] dan mulailah proses 'Windows setup transfer wizard'.
  • Klik [Next]
  • Pilih lokasi file Windows 7 dan lokasi USB Flashdisk anda. klik [Next].
  • Pilih 'I Accepted the terms of the license agreement' dan klik [Continue]
  • Klik [OK] untuk memulai memformat USB Flashdisk dan file-file Windows 7 secara otomatis akan ditransfer ke USB Flashdisk.
  • Klik [Next] apabila peng-copy-an selesai , dan booting-lah komputer kamu melalui USB Flashdisk tadi.
Apabila langkah-langkahnya tepat, maka proses instalasi windows akan berjalan melalui USB Flashdisk.
