Color OS V3 untuk Evercoss A7S. Lebaran gak cuma baju yang baru, Evercoss A7S anda juga mesti di bikin baru dong... Untuk itu, saya bagikan Custom ROM Color OS V3 (Amber) yang di porting salah satu member group facebook (lihat bag. kredit rom) agar Evas anda terlihat baru.

ColorOS ini merupakan ROM yang digunakan di Handheld Oppo. Untuk ColorOS V3 yang dikenal juga sebagai Color OS Amber ini merpakan versi upgrade dari ColorOS V2 untuk A7S yang pernah saya posting bulan lalu (Juni 2014). Tetapi, porter ColorOS V3 untuk a7s kali ini berbeda dengan porter ColorOS V2.

Baca juga: Custom ROM ColorOS R831T untuk Andromax V2

Note: Work fine di Evas milik saya sendiri.

ColorOS Amber ini masih tetap cantik, dengan beberapa perbaikan tampilan di bagian UI-nya

Fitur-fitur masih tetap sama dengan versi sebelumnya, tetapi ada beberapa aplikasi yang mendapatkan update seperti toko aplikasi Nearme.

Performa Custom ROM ini termasuk biasa saja, sehingga anda bisa menunjang kinerja Evercoss A7S dengan menambahkan Turbo Boost Swap karena init.d support sudah aktif. Bisa juga di tambahkan tweak Pure Graphic untuk meningkatkan kualitas grafis-nya.

Cara Pasang/Instalasi Custom ROM Oppo COLOR OS V3

Sebelum meneruskan tutorial Instalasi ini, anda berarti sudah setuju dengan artikel Disclaimer dan mengerti serta patuh dengan artikel DMCA Copyright di blog ini. Dan anda juga paham bahwa dengan menginstall Custom ROM di handheld anda bisa menyebabkan garansi anda gugur resiko kehilangan data, backup data penting anda.

Syarat: harus sudah ter-install CWM Recovery, jika belum, baca di tutorial install CWM Recovery.

Untuk memasang ROM ini, anda harus melakukan Backup IMEI melalui aplikasi Mobile Uncle atau aplikasi backup IMEI lainnya, sebagai antisipasi jika terjadi INVALID IMEI. Buat juga Nandroid Backup melalui CWM Recovery sebelum melakukan flashing.

Langkah-langkah instalasi:

1. Download ColorOS v3 Amber By Ainur_Bejad.zip dan letakkan di sdcard tapi jangan di dalam folder. Letakkan di /sdcard/--disini-- (menjadi /sdcard/ColorOS v3 Amber By Ainur_Bejad.zip)
2. Matikan Evercoss A7s anda, dan hidupkan lagi dalam mode CWM Recovery dengan menekan tombol [Power] dan [Volume Up]
3. Ketika sudah masuk mode CWM Recovery, gunakan tombol [Volume Up] dan [Volume Down] untuk navigasi dan tombol [Power] untuk memilih menu:
  • pilih menu Wipe Data/Factory Reset pilih yes
  • pilih lagi menu Wipe Cache Partition pilih yes lagi
  • pilih menu Advaced, pilih Wipe Dalvik Cache
  • Pilih Install Zip From SdCard kemudian pilih file ColorOS v3 Amber By Ainur_Bejad.zip
5. CWM Recovery akan melakukan proses flashing, tunggu hingga selesai.
6. Setalah Selesai, pilih GO BACK (menu paling bawah) dan pilih Reboot System Now

Jika Proses Instalasi berjalan mulus, anda akan mendapati Evercross A7S anda sudah terpasang/terinstall ColorOS V3.

Melakukan Fix Bug Audio dan Internet ColorOS v3 A7S

1. Masuk Recovery mode, langkahnya sama.
2. Langsung pilih menu Install from sdcard kemudian pilih file Fix Audio tanpa wipe.
3. CWM Recovery akan memproses update, tunggu sampai selesai.
4. Jika sudah pilih GO BACK dan pilih Reboot System Now

Ulangi proses diatas untuk Fix Internet-nya
