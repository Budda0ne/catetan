Tutorial Install CWM Recovery Galaxy Ace GT-S5830
CWM Recovery yang akan dipasang ini adalah CWM versi 5.0.2.6. Anda bisa download terlebih dulu file recovery-clockwork-5.0.2.6-galaxyace-fix.zip (XDA) atau recovery5.0.2.6.zip (MF) di bagian download di bawah posting. Selanjutnya, ikuti tutorial dibawah langkah demi langkah.
  1. Letak-kan file recovery-clockwork-5.0.2.6-galaxyace-fix.zip di sdcard, jangan di dalam folder tetapi diluar seperti ini /sdcard/recovery-clockwork-5.0.2.6-galaxyace-fix.zip untuk mempermudah saat instalasi.
  2. Matikan Galaxy Ace anda, tunggu sekitar 5 detik.
  3. Sekarang nyalakan Galaxy Ace anda dalam mode recovery (stock recovery/recovery bawaan) dengan cara menekan tombol [power] dan [home] bersamaan agak lama.
  4. Setelah masuk recovery mode, pilih menu "Apply update from zip"
  5. Tunggu hingga proses instalasi selesai.
  6. Kembali ke menu awal dengan menekan tombol [back]
  7. pilih "Reboot now"
Selesai. Kini anda sudah mempunya CWM Recovery di Samsung Galaxy Ace GT-S5830 anda.

Menginstall CWM recovery penting di lakukan untuk Galaxy Ace [GT-S5830] anda karena jika terjadi bootloop ataupun hal-hal yang berakibat tidak dapat masuk ke system anda tidak bisa melakukan apa-apa. Selain itu, dengan instalasi CWM Recovery ini anda bisa melakukan instalasi custom rom.
