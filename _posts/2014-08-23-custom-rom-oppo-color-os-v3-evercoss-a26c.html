About Phone Color OS A26C
Berbeda dari Color OS V2 Evercoss A26C sebelumnya yang masih banyak Bug, Hasil Port kali ini lebih sedikit bug-nya. Dan kalaupun ada bug, sudah di sediakan file zip update-nya.

Syarat untuk Install Custom ROM OPPO (Color OS V3)
1. Root (baca cara rooting a26c)
2. Terinstall CWM Recovery
Untuk memasang ROM ini, sebaiknya melakukan Backup IMEI melalui aplikasi Mobile Uncle atau aplikasi backup IMEI lainnya, sebagai antisipasi jika terjadi INVALID IMEI. Buat juga Backup melalui CWM Recovery sebelum melakukan flashing/install.

Langkah-langkah instalasi:
  1. Download [ROM]ColorOS_v3_A26C_bonzes.zip (link) dan letakkan di sdcard tapi jangan di dalam folder. Letakkan di /sdcard/--disini-- (menjadi /sdcard/[ROM]ColorOS_v3_A26C_bonzes.zip)
  2. Matikan Handheld anda, dan hidupkan lagi dalam mode CWM Recovery dengan menekan tombol [Power] dan [Volume Up]
  3. Ketika sudah masuk mode CWM Recovery, gunakan tombol [Volume Up] dan [Volume Down] untuk navigasi dan tombol [Power] untuk memilih menu:
    • pilih menu Wipe Data/Factory Reset pilih yes
    • pilih lagi menu Wipe Cache Partition pilih yes lagi
    • pilih menu Advaced, pilih Wipe Dalvik Cache
  4. Pilih Install Zip From SdCard kemudian pilih file [ROM]ColorOS_v3_A26C_bonzes.zip
  5. Tunggu CWM Recovery melakukan proses flashing, tunggu hingga selesai.
  6. Setelah Selesai, pilih GO BACK (menu paling bawah) dan pilih Reboot System Now
Jika Proses Instalasi berjalan lancar, Custom ROM ColorOS v3 A26C ported by Bonzes akan ter-install dengan baik di Evercoss A26C anda.
