Hal ini karena ROM MIUI menggunakan framework UI tambahan. Sehingga diperlukan modifikasi oleh developer lain (unofficial xposed) agar bisa digunakan di ROM MIUI Based.

Cara Install Xposed Framework untuk MIUI 7 dan MIUI 8

Langkah di bawah seharusnya bisa diaplikasikan baik untuk ROM MIUI Native (ponsel Xiaomi) maupun ROM MIUI Porting (ponsel non-Xiaomi).

r Anda juga perlu tahu sedikit soal SDK dan Arsitektur yang dipakai. Pengetahuan ini lumayan berguna untuk ROM MIUI hasil porting. Singkatnya:
Android Lollipop mempunyai dua SDK:
1. Lollipop 5.0 = SDK 21
2. Lollipop 5.1 = SDK 22.
Sedangkan Marshmallow = SDK 23.

Xposed Framework untuk Nougat sudah dirilis baca di sini untuk keterangan lebih lanjut.

Arsitektur Android (mulai Lolipop) juga ada 2 yakni 32bit dan 64bit untuk device ARM (Qualcomm maupun Mediatek). Sedangkan untuk chipset Intel lebih sering disebut sebagai X86 untuk 32 bit dan X86_64 untuk 64 bit. Untuk yang menggunakan X86 (Intel) 64 bit sangat jarang, bisa dikatakan hampir tidak ada ponsel yang memakai. Hanya ditemui di device semacam Android TV. X86 tidak disediakan mengingat Xiaomi tidak mempunyai device dengan cipset Intel.

Untuk instalasi Xposed Framework support MIUI 8 maupun MIUI 7 caranya sedikit lebih ribet karena anda harus mendownload framework dan aplikasinya secara terpisah.

Cara Install Xposed Framework MIUI, Step by step.


Syarat agar bisa memasang Xposed MIUI 7 dan MIUI 8 tentu saja harus sudah root dan memiliki custom recovery.

Jangan lupa melakukan nandroid backup dengan custom revovery ya. Anda sudah saya ingatkan.

Oke, anggap saja anda sudah memenuhi syarat diatas dan tahu SDK dan Arsitektur yang dipakai oleh ROM MIUI anda.

Misalkan saja, anda sedang memakai MIUI 7 berbasis Lollipop 5.1.1 yang berarsitektur 32 bit. Maka anda tinggal men-download file yang sesuai SDK 22 ARM. Misalkan MIUI 8 Marshmallow 64 bit maka download yang SDK 23 ARM64.

Install Aplikasi Xposed Framework 3.0 alpha 4 agar anda bisa mengatur modul-modul xposed yang akan anda install.

Kemudian, letakkan file zip yang sudah anda dowload di sdcard. Matikan ponsel dan reboot ke menu recovery. Caranya variatif tergantung ponsel anda, atau gunakan tool reboot recovery untuk mempermudah.

Dalam mode recovery, pilih install update dan pilih install zip from sdcard (CWM) atau Install dan pilih Install zip (TWRP) dan arahkan ke lokasi file *.zip

Jika sudah selesai, pilih menu reboot to system dari recovery mode.

Ternyata cara pasang Xposed Framework untuk MIUI Lollipop based ini tidak terlalu ribet. Yang penting anda tahu SDK dan Arsitektur yang dipakai oleh ROM MIUI sebagai base rom. Boleh kok tanya-tanya di kolom komentar kalau masih agak ragu.
Attention!
Starting from v86 official version of Xposed become more compatibly with ROMs that use a Resources subclass for e.g. theming, like MIUI.
So, try firstly official Xposed on your device. If it will works well (take a look at themes, if they works well, then all Ok), then you doesn't need this version of Xposed.
But I suppose that on some devices official Xposed still may cause bootloops.
