Untuk meng-install TWRP sebagai pengganti Stock Recovery (native) Redmi 2, bisa melalui dua cara.

Tutor Redmi 2, Install TWRP 2.8.6.1 dan Restore Stock RecoveryCara yang pertama menggunakan PC/Laptop (fastboot) dan cara kedua langsung dari ponsel, baik menggunakan aplikasi TWRP Manager maupun flashing melalui Mi recovery (stock).

Sedikit informasi mengenai TWRP, TWRP Recovery merupakan salah satu dari Custom Recovery populer selain CWM Recovery dan tutrunannya. Banyak user yang suka dengan TWRP karena fitur-fitur yang lebih banyak dari CWM maupun turunannya (seperti CTR dan PTR). Selain fungsi, TWRP juga mempunyai tampilan yang bisa di customize.

Fitur yang ditambahkan di versi 2.8.6.1 ini antara lain:
  1. Back-up partisi Modem (/modem) untuk memback-up Baseband
  2. Back-up partisi Efs (/fsg, /modemst1, /modemst2) untuk memback-up IMEI
  3. Back-up partisi Persist (/persist)
  4. Back-up partisi Aboot (/aboot, /abootbak) untuk memback-up bootloader
  5. Membuat Firmware-update /sbl1(/sbl1bak), /rpm(/rpmbak), /tz(/tzbak)
  6. Back-up partisi Splash (/splash) untuk memback-up bootsplash (bootlogo)

Instalasi TWRP 2.8.6.1 dengan PC/Laptop


Download file TWRP PC sesuai tipe (HM2014811 atau HM2014813) di bagian download
1. Ekstrak file zip yang sudah di download dan buka folder tersebut.
2. Jalankan "Flash-TWRP.bat" dengan cara klik kanan "Run as Administrator" setelah Redmi 2 anda terhubung ke PC/Laptop.
3. Jika semua driver sudah ter-instal dengan baik, selanjutnya tinggal mengikuti petunjuk yang muncul.
4. Jika anda mengikuti petunjuk dengan baik, maka TWRP akan ter-install di Xiaomi Redmi 2 anda.

Instalasi TWRP 2.8.6.1 Langsung dari HH/Ponsel

Download file TWRP Mobile Install sesuai tipe (HM2014811 atau HM2014813) di bawah dan lanjutkan dengan metode yang anda suka berikut:

1. Instalasi TWRP menggunakan TWRP Manager

Jika menggunakan cara ini, Redmi 2 harus root, (lihat cara root lollipop/kitkat tanpa pc)
  1. Ekstrak dulu **_TWRP2.8.6.1_signed.zip kemudian letakkan file recovery.img di sdcard
  2. Install dan jalankan aplikasi TWRP Manager, ijinkan jika ada notifikasi superuser
  3. Pilih menu Install Recovery dan arahkan ke lokasi file recovery.img
  4. Selanjutnya tinggal menunggu proses selesai.
Setelah proses install selesai, pilihan menu Install menjadi tidak bisa di sentuh (disabled) dan Recovery Installed menunjukkan versi TWRP terinstall.

2. Instalasi TWRP 2.8.6.1 via Mi Recovery

Opsi pertama:
  • Buka Settings dan pilih About Phone pilih System Update
  • Lihat pojok kanan atas layar, tekan menu (...) dan pilih update package
  • Arahkan ke lokasi file **_TWRP2.8.6.1_signed.zip
    • .
  • Redmi 2 akan otomatis reboot dan TWRP akan terinstall.
atau bisa juga dengan cara:
Rename file **_TWRP2.8.6.1_signed.zip menjadi update.zip
  • Buka Settings dan pilih About Phone pilih System Update
  • Lihat pojok kanan atas layar, tekan menu (...) dan pilih "Reboot to Recovery mode"
  • Dalam mode recovery, pilih english dan pilih instal update.zip
  • Pilih reboot to system kalau sudah selesai.

Mengembalikan Stock Recovery Setelah Install TWRP

Jika anda ingin menganti TWRP yang sudah ter-install, download file **_stock_recovery.img sesuai type Redmi 2.
  • Kopikan file tersebut ke sdcard
  • Masuk ke recovery mode (TWRP)
  • Pilih menu Install, tap di Install zip kemudian pilih Install Image.
  • Arahkan ke lokasi file **_stock_recovery.img yang tadi anda download
  • Pilih opsi Recovery, kemudian tekan dan geser dari kiri kekanan bagian "Swipe to Flash Image"
  • Reboot setelah proses selesai.
