Menu penting TWRP Recovery SevenMaxx:
  • Modem : untuk melakukan backup partisi - /modem
  • Efs : untuk melakukan backup partisi - /fsg, /modemst1, /modemst2
  • Persist : untuk melakukan backup partisi - /persist
  • Aboot : (bootloader) untuk melakukan backup partisi - /aboot, /abootbak
  • Firmware-update : untuk melakukan backup partisi - /sbl1 (/sbl1bak), /rpm (/rpmbak), /tz (/tzbak)
  • Splash : makes backups partitions - / splash

Letakkan dalam folder dimana anda meng-install adb fastboot.

Matikan Lenovo A6000 atau A6000plus

Nyalakan dalam mode fastboot dengan cara menekan tombol [volume down] + [power] bersamaan.

Tancapkan ponsel ke komputer atau laptop dengan kabel data usb.

Buka folder Minimal ADB Fastboot dan tekan tombol [CTRL] + [SHIFT] jangan lepaskan dan [klik kanan] di ruang kosong dalam folder tersebut.

Pilih Open command window here.

Dalam Command Promt yang muncul ketik 

fastboot devices

Jika muncul list device maka ponsel sudah terhubung.

Selanjutnya untuk meng-akses TWRP recovery sementara (hot boot) tinggal mengetik perintah

fastboot boot recovery.img

Sedangkan untuk melakukan instalasi permanen ketik

fastboot flash recovery recovery.img

Saran saya, menggunakan hot boot terlebih dahulu untuk mengambil back-up yang benar-benar ASLI dan belum terjamah. Berikut adalah Screenshoot TWRP Recovery setelah ter-install di Lenovo A6010. Untuk melihat gambar penuh, silahkan klik di masing-masing header: teaser.


Setelah anda mendapatkan back-up TWRP seperti list yang ada di menu penting, baru anda bisa melakukan install TWRP permanen dan sekaligus melakukan rooting

Untuk melakukan rooting Lenovo A6010, anda tinggal mendownload file Supersu.zip dari link diatas dan flash dari TWRP Recovery.

Update: daripada menggunakan supersu, lebih baik melakukan root dengan magisk systemless root. lihat caranya di sini
