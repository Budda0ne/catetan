Salah satu jenis kuota baru yang diluncurkan Telkomsel adalah kuota MAXstream. Bisa dikatakan bahwa kuota MAXstream ini hampir sama dengan kuota Videomax. Pada awalnya MAXstream adalah sebuah aplikasi layanan hiburan yang punya katalog film-film box office dari beberapa aplikasi seperti Tribe, HOOQ, Viu, Nickelodeon Play dan CathPlay. lambat laun seiring berjalannya waktu, Telkomsel terus meningkatkan aplikasi buatannya ini hingga saat ini menyediakan juga beberapa channel TV kabel yang boleh dikata cukup lengkap. Sekarang ini Telkomsel pun meng-khususkan paket internet yang sebelumnya untuk videoMAX menjadi khusus untuk mengakses MAXStream saja. Jika kamu sering menggunakan aplikasi resmi dari Telkomsel ini bisa membeli kuota MAXstream agar bisa lebih hemat ketika menonton film ataupun semua channel.

Kuota MAXstream sendiri di bandrol dengan harga yang cukup terjangkau yaitu dari mulai Rp. 10rb / 10GB hingga Rp. 30rb / 30GB. Murah sih, sayangnya kuota ini hanya bisa dipakai untuk mengakses layanan MAXstream saja, tidak bisa untuk aplikasi lainnya. Maka untuk pengguna yang tidak bisa menghabiskan kuota ini, banyak yang ingin mengubahnya menjadi kuota flash biasa 24 jam agar bisa digunakan untuk mengakses semua aplikasi online seperti WhatsApp, Facebook, Youtube dan lainnya.

Jika kamu pernah memakai aplikasi Anonytun untuk merubah kuota Vodeomax menjadi kuota flash dan belum berhasil, bosa jadi karena BUG MAXstream tidak bisa dipakai di beberapa wilayah. Untuk itu saya akan membagikan BUG MAXstream terbaru yang mungkin bisa kamu coba. Baiklah silahkan simak dibawah ini.

Agar Bisa Memakai BUG MAXStream Telkomsel

Seperti yang sudah disampaikan di atas, bahwa kuota MAXstream tidak jauh beda dengan kuota Videomax, maka BUG-nya pun sama-sama membutuhkan IP dewa, yakni IP dengan awalan 100. Cara untuk mencari IP 100 mudah saja. kamu buka saja aplikasi Anonytun. Setelah terbuka perhatikan pada bagian header: teaser, di sana akan tertera IP yang didapat ponsel kamu. Jika masih berawalan 10.xx.xx.xx, ON OFF kan mode pesawat hingga kamu mendapatkan IP 100.xx.xx.xx. Jika kamu merasa kesulitan, gunakan bantuan aplikasi IP hunter yang bisa kamu unduh dari sini.

Setelah kamu menemukan IP 100 dari langkah sebelumnya, kamu bisa menggunakan BUG Telkomsel untuk MAXStream ataupun videoMAX. Berikut ini URL BUG untuk mengisi Spoof Host Anonytun.

URL BUG MAXstream Terbaru

  • api.tribenow.tv
  • prod-in.viu.com
  • www.maxstream.co.id
  • play.hooq.tv
  • telkomsel.com
  • www.cathplay.com
  • nickelodeon.com.au
  • video.nickelodeon.com.au

Nah, itulah URL BUG yang bisa kamu coba satu-satu persatu hingga annonytun (atau aplikasi internet gratisan lainnya) benar-benar terhubung dan sukses menjadikan kuota MAXstream sebagai kuota flash biasa.

APN MAXstream

Jika kamu sudah menggunakan IP 100 dan sudah mencoba semua URL tetapi kuota MAXstream tidak terpotong, maka kamu bisa mencoba mengubah APN di ponsel kamu dengan beberapa APN di bawah ini.

  • Nama : terserahkamu
  • APN : sabarbrn / sabartbs (pilih salah satu)
  • Proxy : 10.1.89.130
  • Port : 8000
  • Untuk bagian ke bawah kosongkan / tidak perlu diisi
  • Nama : terserahdeh
  • APN : aha
  • Proxy : kosong
  • Port : kosong
  • Nama : terserah
  • APN : sabartbs7
  • Proxy : 157.240.7.32

Cara untuk menambahkan APN baru di Android sangatlah mudah, silahkan masuk ke Pengaturan Ponsel > Jaringan Seluler > Nama Titik Akses (APN) > selanjutnya silahkan kamu tekan tombol Tambahkan yang berada di sudut kanan atas.
