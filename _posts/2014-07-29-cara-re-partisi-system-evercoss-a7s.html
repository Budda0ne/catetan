Dalam kasus ini, mau tidak mau, anda harus melakukan re-partisi system agar bisa meng-install Custom ROM yang lebih besar daripada partisi system.

Begitu juga Evercoss A7S (MT6572) yang "hanya" mempunyai partisis system sekitar 600mb. Ketika anda meng-install Custom ROM berukuran lebih besar dari 600mb, repartisi system adalah keharusan. Meskipun sebenarnya tidak banyak ROM yang besar dari 600mb, saya rasa tidak akan rugi, toh nantinya (sesuai dengan perkembangan) akan muncul ponsel-ponsel dengan partisi system besar untuk menampung aplikasi-aplikasi yang semakin besar) dimana bisa saja developer Evas A7s ada yang memporting-nya :). Selain alasan itu, repartisi system juga berguna saat annda melakukan modifikasi dengan menambahkan fitur-fitur tertentu atau aplikasi tertentu kedalam partisi system.

Untuk saat ini, repartisi System 1Gb akan lebih dari cukup, entah nanti. Karena itu yang saya gunakan juga file untuk merepartisi sebesar 1Gb saja. Lihat saja ruang kosong yang tersedia.
Hasil Repartisi System 1Gb

Untuk melakukan Re-partisi, caranya cukup mudah. Meskipun saya tahu cara membuat file erpartisi juga tidak gampang, berterimakasihlah kepada agan Rahman Maulana. Caranya, ikuti langkah-langkah berikut:

Cara Mem-partisi System 1Gb lansung dari HH:


Sebelum melakukan repartisi, backup dulu semua data penting seperti aplikasi, sms, no. Tlp, atau apa saja yang anda anggap penting, karena proses re-partisi ini akan menghapus semua data di ponsel (wipe data/factory reset).
  1. Download dulu file untuk repartisi di bawah
  2. Letakk-kan di akar sdcard seperti ini (/sdcard/Partition-System_1GB-CWM.zip)
  3. Flash melalui Recovery Mode dengan cara:
    • Pastikan sudah meng-install Custom recovery semisal CWM atau TWRP dan Reboot ke recovery mode.
    • Pilih install zip from sdcard
    • Pilih file Partition-System_1GB-CWM.zip dan install
  4. Tunggu hingga proses selesai
  5. Reboot.

Jika setelah booting anda mendapati notifikasi Enkripsi gagal Sebaiknya langsung cabut baterai untuk mematikan paksa.

Untuk mengatasi hal ini, masuk ke recovery mode dan lakukan wipe data/factory reset dari mode recovery kemudian reboot.
