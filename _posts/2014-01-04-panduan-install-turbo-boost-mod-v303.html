Di tutorial Tutorial meng-edit script swap internal memori sebelumnya, saya hanya menjelaskan masing masing kegunaan parameter yang ada di dalam init.d script TurboBost Mod V 3.0 dan sehingga anda bisa meng-edit script swap dengan settingan yang sesuai dengan HH (Handheld/device) anda.

Saya rasa tidak adil kalau saya membuat tutorial mengedit script tetapi tidak memberikan dimana bisa mendapatkan dan bagaimana meng-install init.d script TURBO BOOST MOD V3.03

Script ini bisa anda temukan di forum XDA-Developers di treath ini.
script-nya bisa anda download di treath tersebut atau langsung dari link ini

Anda bisa meng-installnya dengan CWM Recovery. Instalasinya pun cukup mudah karena sudah disediakan aroma installer (flashable zip dengan interaktif menu). Caranya sama persis seperti ketika meng-instal Custom ROM.

Namun, jika ternyata TURBO BOOST MOD V3.03 ini tidak terinstall dengan baik di HH anda, maka anda bisa meng-install-nya secara manual dengan mengikuti tutorial yang akan di berikan nantinya.

Sebelum meng-install pastikan sudah mempunyai tool/alat/bahan berikut:
  • Handheld yang sudah kodisi ROOTED.
  • Sudah support init.d, jika belum anda bisa membaca tutorial init.d script
  • Aplikasi Root Explorer atau sejenis (File manager dengan hak root)
  • Terminal Emulator (bisa di download di playstore)
  • Dan file .zip TURBO BOOST MOD V3.03 disini
Jika sudah sudah memenuhi hal-hal diatas anda bisa melanjutkan dengan mengikuti panduan berikut:
  1. buka Root Explorer dan ekstrak file TURBO BOOST MOD V3.03.zip ke /sdcard (tidak harus, lokasinya terserah asal anda ingat)
  2. Cek free space/partisi yg tersedia dengan aplikasi disk-info atau buka Terminal Emulator ketik :
    df /system
    df /cache
    df /data
  3. Setelah itu tinggal menentukan besaran swap yg akan diinstall di partisi /system, /data, atau /cache
  4. Masuk di folder swap hasil ekstrakan file tadi (point 1)
  5. Pilih script sesuai besaran yg diinginkan dan dimana akan diinstall sesuai dengan free scpace yang tersedia
  6. Kopikan ke dalam folder /system/etc/init.d/[file], dan jangan langsung di reboot.
  7. Setting permission dari file init.d script (dalam contoh ini filenya 98swap_data_v300) terlebih dulu menggunakan terminal emulator:

    su
    cd /etc/init.d
    chown root.shell 98swap_system_v300
    chmod 775 98swap_data_v300
    Jika anda ingin menggunakan beberapa partisi sekaligus, anda tinggal merubah namanya dengan urutan tertentu. Misalnya 97swap_cache_v300, 98swap_system_v300, 99swap_data_v300 (sesuai partisi yang digunakan). sehingga script akan dijalankan secara berurutan.
  8. Setelah selesai, silahkan di-reboot.
  9. Untuk mengecek script berfungsi atau belum, buka Root Explore dan lihat di folder tempat swap di-instal (mis. /data/swap/) jika ada file swap.img berarti script berhasil dipasang dan berjalan dengan baik.
  10. Jika tidak ada, cek dulumelalui Terminal Emulator ketik :

    free -m
    atau

    cat /proc/swaps


Untuk bisa berjalan dengan baik, biasanya memerlukan reboot dua kali agar script swap bisa berjalan.
