Cara Install Backtrack 5 di Android

Dalam dokumen ini akan dijelaskan bagaimana cara install backtrack di Android. Menginstall dalam hal ini tidak mengistall dalam arti memasang karena kita hanya menggunakan backtrack layaknya live cd yang diakses menggunakan VNC.

Syarat untuk Install Backtrack 5:

  1. Rooted
  2. Resolusi Layar HDPI (480x800)
  3. Processor 1GHz
  4. RAM 512MB
  5. Memory Kosong/Sisa 4GB

Alat dan Bahan yang perlu disiapkan:

  1. File Image Backtrack 5 ARM | via GDrive http://bitly.com/bt5a26c [ 512 MB ]
  2. Aplikasi VNC Viewer Android | via PS http://bitly.com/vncA26c
  3. Aplikasi Terminal Emulator | via PS http://bitly.com/TEA6c
  4. Aplikasi Busybox | setelah terinstal, pilih menu

Semua ada di bawah

Langkah langkah Italasi-nya:

1. Download file image Backtrack kemudian ekstrak sdcard sehingga lokasinya akan menjadi /sdcard/bt5/.

2. Buka terminal emulator kemudian ketik su dan tekan enter.

3. Setujui akses root.

4. Kemudian berpindah working direktori ke lokasi file backtrack tadi dengan mengetik cd /sdcard/bt5/ dan tekan enter.

5. Lanjutkan dengan mengetik sh bt dan tekan enter. Jika muncul pertanyaan, ketik y dan enter.

6. Jika user di terminal menjadi root@localhost# maka kita lanjut ke tahap selanjutnya, biarkan terminal emulator berjalan (tekan tombol home saja dan jangan di close).

7. Buka Android VNC kemudian isi kolom isian masing-masing dengan parameter berikut.

  • Nama: Backtrack 5 (atau terserah)
  • Password : root
  • Address: localhost (atau Address:127.0.0.1)
  • Port : 5061

8. Klik connect.

9. Tunggu sebentar hingga anda masuk desktop Backtrack 5.

Back Track 5 pun bisa anda nikmati.
