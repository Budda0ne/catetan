Untuk mengaktifkan USB debugging di Android sebenarnya cukup mudah. Namun ada beberapa vendor yang memang sengaja mengunci USB debugging device-nya dengan password. Masalah ini banyak dijumpai pada ponsel bundling Smartfren, Andromax. Tak terkecuali Andromax I3 (Hisense AD689G) dan Andromax I3S (Haier AD682H).


Untuk meng-aktifkan USB di dua ponsel Andromax ini, anda harus membuka kunci dengan memasukkan pasword usb debugging terlebih dulu. Untuk mengetahui paswordd, caranya adalah dengan menggunakan aplikasi tertentu karena password tiap ponsel berbeda sesuai dengan MEID dan IMEI. Lebih jelasnya, silahkan ikuti langkah-langkah berikut:

Mencari Pasword USB Debugging

  1. Download aplikasi USB debug password yang bisa anda download di bawah
  2. Install Andromax i3 dan i3s Password Generator
  3. Buka aplikasi setelah terinstall, aplikasi akan menunjukkan IMEI dan MEID Andromax anda.
  4. Klik menu Calculate Password untuk menggenerat pasword usb debugg-ing.
  5. Tekan lama di password yang di tampilkan untuk meng-copy.

Membuka Kunci USB debugging

Setelah anda mendapatkan password usb debugging, lanjutkan dengan:

  1. Buka menu Settings scroll kebawah dan pilih menu About Phone
  2. Pada baris Build number, ketuk beberapa kali dengan cepat hingga muncul toast "You're now Developer"
  3. Pilih menu back, dan sekarang sudah muncul pilihan Developer Options. Klik di situ.
  4. Cari dan centang pilihan USB debugging.
  5. Pda pop up yang muncul, tekan lama di kotak isian dan pilih paste.
  6. Klik yes, Selesai

USB Andromax I3 atau Andromax I3S sekarang sudah aktif.
