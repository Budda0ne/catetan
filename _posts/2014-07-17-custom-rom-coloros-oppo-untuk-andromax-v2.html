Cara Pasang/Instalasi Custom ROM ColorOS (OPPO) untuk Andromax New V [N986D]

Sebelum meneruskan tutorial Instalasi ini, anda berarti sudah setuju dengan artikel Disclaimer dan mengerti serta patuh dengan artikel DMCA Copyright di blog ini. Dan anda juga paham bahwa dengan menginstall Custom ROM di handheld anda bisa menyebabkan garansi anda gugur resiko kehilangan data. Ingat, backup data penting anda!

Syarat: harus sudah ter-install CWM Recovery atau TWRP Recovery.

Untuk memasang ROM ini, anda harus melakukan Backup IMEI melalui aplikasi Mobile Uncle atau aplikasi backup IMEI lainnya, sebagai antisipasi jika terjadi INVALID IMEI. Buat juga Nandroid Backup melalui CWM Recovery (TWRP Recovery) sebelum melakukan flashing.

Langkah-langkah instalasi:

  1. Download ColorOS_N986D_140524.zip File size: 579.56 MB dan letakkan di sdcard tapi jangan di dalam folder. Letakkan di /sdcard/--disini-- (menjadi /sdcard/ColorOS_N986D_140524.zip)
  2. Matikan Andromax New V [N986D] anda, dan hidupkan lagi dalam mode Recovery dengan menekan tombol [Power] dan [Volume Up]
  3. Ketika sudah masuk mode CWM Recovery, gunakan tombol [Volume Up] dan [Volume Down] untuk navigasi dan tombol [Power] untuk memilih menu (kecuali anda menggunakan TWRP):
    • pilih menu Wipe Data/Factory Reset pilih yes
    • pilih lagi menu Wipe Cache Partition pilih yes lagi
    • pilih menu Advanced, pilih Wipe Dalvik Cache (opsional)
    • Pilih Install Zip From SdCard kemudian pilih file ColorOS_N986D_140524.zip
  4. CWM Recovery akan melakukan proses flashing, tunggu hingga selesai.
  5. Setelah Selesai, pilih GO BACK (menu paling bawah) dan pilih Reboot System Now

Jika proses instalasi custom rom berjalan mulus, anda akan mendapati Andromax New V (ZTE N986D) anda sudah terpasang/terinstall ColorOS OPPO.

Catatan:
- Untuk booting pertama kali lumayan memakan waktu (lama), jadi bersabar saja dulu.
- Untuk SIM2 unlock, masuk ke ZTE setting dan lanjut ke simme unlock dengan password: 12345678.
- Untuk fix GAPPS download dan Flash dari recovery file miuiandroid_gapps-4.2.2-small_signed-8012.zip File size: 15.47 MB
