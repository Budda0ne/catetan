Samsung meluncurkan kembali seri smartphone mid-range (menengah) 'Samsung Galaxy A'. Saat ini, ada 3 ponsel dengan desain yang hampir sama dengan spesifikasi yang berbeda: Samsung Galaxy A3, A5 dan A7 . Dalam artikel kali ini, kita akan fokus pada Samsung Galaxy A5 type SM-A500G, SM-A500H, dan SM-A5000, setelah sebelumnya kita memberikan tutorial rooting Samsung Galaxy A3.

Samsung Galaxy A5 merupakan ponsel dengan layar Super AMOLED 5 inch, berprosesor Qualcomm Snapdragon 410 (MSM8916), CPU 1,2 GHz Quad-core dengan RAM 2 GB, kamera depan 5MP dan 13MP kamera belakang, 16 GB ROM, GPU Adreno 306, dan menggunakan OS Android 4.4.4 KitKat. Tidak perlu diragukan lagi bahwa Galaxy A5 mempunyai spesifikasi yang yahud hingga dapat menjalankan aplikasidan game-game dengan lancar. Namun, jika Anda masih merasa kurang, atau ingin mempunyai keleluasaan untuk mengkustomasi ponsel Galaxy A5 anda, silahkan melakukan rooting dengan panduan yang akan saya berikan.

Di sini, saya akan memberikan cara cepat dan sederhana rooting Galaxy A5. Metode ini bisa digunakan untuk root semua varian Samsung Galaxy A5 (SM-A500G, SM-A500H, SM-A5000) dengan OS Android 4.4.4 KitKat. Jadi, simak langkah-langkah Cara Root Galaxy A5 pada Android 4.4.4 KitKat [Semua Tipe A5].

Sebelum melakukan proses root, cek dan lakukan hal berikut ini:
- Cas baterai Galaxy A5 hingga minimal 65%.
- Aktifkan opsi "USB Debugging" pada Galaxy A5 Anda.
- Jangan lupa untuk mem-backup data penting yang ada di ponsel Anda.
- Instal "Samsung Galaxy A5 USB Driver" pada komputer Anda.

Langkah-langkah Root Galaxy A5 [SM-A500G, SM-A500H, SM-A5000]

  1. Download file rooting sesuai dengan nomor model perangkat Anda dan Tool Odin dari link di bawah untuk root Galaxy A5 xda.
  2. Ekstrak kedua file (odin dan file rooting) yang tadi anda download dalam satu folder.
  3. Jalankan Odin3 v3.09.exe dari folder tersebut.
  4. Sekarang, matikan ponsel Galaxy A5 selama 5 - 10 detik
  5. Boot Galaxy A5 ke recovery mode dengan menekan tombol power dan volume down dan home secara bersamaan hingga muncul peringatan.

    Lepas ketiga tombol dan tekan lagi tombol volume up untuk mengaktifkan download mode.
  6. Kemudian, hubungkan Samsung Galaxy A5 dengan komputer menggunakan kabel data
  7. Jika salah satu kolom ID:COM pada Odin berubah warna menjadi biru, berarti sudah terhubung dengan sukses
  8. Pilih file rooting dengan mengklik tombol "AP" dan arahkan ke lokasi file "CF_Auto_Root_xxxx.tar.md5" (dari folder tempat anda meng-ekstrak sebelumnya).
  9. Pastikan bahwa hanya pilihan "F. Reset Time" dan "Auto Reboot" yang tercentang di tool Odin.
  10. Setelah semua siap maka Anda bisa menekan tombol "Start" untuk memulai proses rooting, (jika anda menggunakan Odin yang anda download dari halaman ini, anda tinggal memilih file .tar.md5 dan klik start)
  11. Proses rooting ini tidak akan lama. Proses root sekitar 30 detik saja.
  12. Anda akan melihat kata "PASS" pada kolom ID:COM Odin dan ponsel akan reboot otomatis jika proses sudah selesai.
Nah kini anda bisa melepas Samsung Galaxy A5 dari PC dan setelah ponsel Anda menyala dengan normal, cek dan pastikan bahwa aplikasi "Super SU" sudah ada.
