Install CWM Recovery GT-I9100 menggunakan Teminal Emulator [metode 1]

Syarat menggunakan metode ini, Galaxy S2 anda harus sudah root. untuk rooting, anda bisa menggunakan tutorial Rooting Galaxy S3, caranya sama.
Untuk meng-install CWM Recovery melalui hape dengan cara ini anda harus:
  1. Install Terminal Emulator, download via Playstore jika belum punya.
  2. File Manager yang bisa meng-ekstrak file zip. anda bisa memilih Root Explorer atau Root Browser.

Setelah anda mempunya 2 aplikasi diatas dan mendownload file zip yang ada di bawah post, ikuti langkah berikut
  • Ekstrak file CWM-KitKatCompatible-i9100.zip menggunakan RE atau RB.
  • Cari file boot.img hasil ekstrak sebelumnya dan pindahkan ke /sdcard/disini
  • Buka Terminal Emulator dan ketik perintah
    su
    allow notifikasi yang muncull dan setelah tanda $ bergati menjadi # ketikkan lagi perintah:
    dd if=/sdcard/boot.img of=/dev/block/mmcblk0p5
  • Untuk mencoba apakah CWM Recovery sudah ter-install dengan benar, anda bisa mengetikkan perintah:
    reboot recovery
    Samsung Galaxy S2 anda akan reboot otomatis dan menyala dalam mode recovery.

Install CWM Recovery GT-I9100 menggunakan CWM Recovery lama[metode 2]

Untuk meng-install melalu mode recovery anda hanya membutuhkan file CWM-KitKatCompatible-i9100.zip yang sudah anda download.
  • Letakkan dulu file CWM-KitKatCompatible-i9100.zip di root sdcard (/sdcard/CWM-KitKatCompatible-i9100.zip)
  • Matikan Samsung Galaxy S2 anda dan nyalakan kembali dengan cara menekan tombol [volume up], [power] dan [home] secara bersamaan.
  • Setelah ponsel anda masuk mode recovery:
    • pilih menu Update from sdcard.
    • pilih lagi menu Choose zip from sdcard
    • pilih file CWM-KitKatCompatible-i9100.zip
    • konfirmasikan pilihan dengan memilih yes, tunggu hingga proses selesai.
  • Selesai. keluar ke menu utama, dan pilih reboot system now
Atau jika ingin melihat apakah CWM Recovery anda sudah terupdate, pilih saja menu reboot recovery setelah proses selesai.
