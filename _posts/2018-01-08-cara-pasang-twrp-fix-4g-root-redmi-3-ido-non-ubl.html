Cara Pasang TWRP, Fix 4G dan ROOT Redmi 3 / 3 Pro Tanpa UBL (Ido)

Tutorial ini untuk Redmi 3 / Pro (Prime) dengan codename IDO, bukan untuk Redmi 3S/Prime/3X (Land). .

Sebelum memasang TWRP Non UBL, siapkan bahan dan persyaratan sebagai berikut..

Persyaratan sebelum melakukan flashing TWRP Non UBL :

  • Pastikan baterai Redmi 3/3 Pro dalam keadaan full charged atau minimal 50% (Ini wajib, baca artikel ini Flashing Sukses HP Tetap Tidak Menyala? Hati-Hati Baterai Lemah!);
  • Kabel USB Original atau yang berkualitas baik;
  • PC/Laptop dengan OS Windows 7 ke atas dan menonaktifkan driver signature.

Bahan-bahan yang dibutuhkan :

  • MiFlashTools versi 20170425 (MiFlash ini sudah include dengan driver untuk Windows 32/64 bit, jadi kamu tidak perlu install driver lain lagi.)
  • Unduh TWRP_Patch_MiFlashV3.zip – 15 MB (sudah saya update menggunakan TWRP RedWolf untuk meminimalisir hardbrick, ekstrak dan letakkan di drive C:\ atau pada folder tanpa spasi);
  • Install Xiaomi Android USB Driver dan 15 Seconds ADB Installer berfungsi sebagai driver dan fastboot command;
  • Install Driver HS-USB QD Loader (EDL : Emergency Download Mode) (Windows 32 bit atau Windows 64 bit);
  • Anti-bootloop » no-verity-opt-encrypt-6.0.zip – 657 KB (untuk mencegah bootloop, letakkan di Internal Storage tanpa di ekstrak);
  • Anti-hardbrick + Fix 4G » Fix 4G NON UBL – 330 KB (untuk mencegah hardbrick, letakkan di Internal Storage tanpa di ekstrak);
  • Opsional, bahan ROOT SuperSU-v2.81_201705252021.zip (salin ke Internal Storage, jangan diekstrak);

Setelah memenuhi syarat dan mengunduh bahan-bahan di atas, ikuti langkah selanjutnya..

Install QDLoader HS-USB, klik Next hingga proses selesai.

Selanjutnya install MiFlash Tools.

Jika sudah, sambungkan Redmi 3/3 Pro ke PC/Laptop menggunakan kabel USB Original atau yang berkualitas baik.

Kemudian masuk ke mode EDL dengan cara mematikan smartphone dan nyalakan dengan menekan tombol Power + Vol Up hingga muncul mode recovery berbahasa China seperti berikut ini..

Pilih Download,

.. maka smartphone kamu akan mati. Jangan khawatir, HP kamu berada pada mode EDL (Emergency Download Mode).

Jika langkah ini malah membuat Redmi 3 Pro masuk ke Homescreen, coba tekan ketiga tombol Power + Vol Atas + Vol Bawah bersamaan dan tahan hingga muncul logo MI dan tetap tahan hingga muncul logo MI sesaat dan mati kembali, sekarang device kamu telah berada di Mode EDL.

Sebelum lanjut ke langkah berikutnya, pastikan bahwa driver Qualcom HS-USB QDLoader telah benar-benar berhasil terpasang ditandai dengan tidak adanya tanda pentung pada Port (COM & LPT) di Device Manager PC/Laptop kamu.

Jika ada tanda pentung, berarti driver Driver HS-USB QDLoader belum terpasang sempurna atau kamu belum menonaktifkan driver signature.

Jika sudah berhasil, kembali ke MiFlash Tools.

Klik Refresh,

Maka Redmi 3 Pro kamu akan terdeteksi sebagai COMX, tergantung dari PC/Laptop kamu.

Selanjutnya pilih select, dan arahkan ke folder dimana kamu menyimpan hasil ekstrak dari TWRP_Patch_Miflash tadi.

Klik OK.

Pada opsi di bagian bawah MiFlash, pilih Clean all untuk menghindari bug yang tidak diinginkan.

Jika sudah, baca bismillah terlebih dahulu kemudian klik Flash.

Tunggu beberapa saat hingga proses flashing selesai ditandai dengan status “flash done” dan result “success” yang memakan waktu kurang lebih 10 detik pada Redmi 3 Pro saya.

Setelah proses selesai, jangan masuk ke Homescreen dulu, lepaskan kabel USB dan tekan ketiga tombol Power + Vol Up + Vol Down sampai muncul logo MI.

Setelah logo MI muncul, lepaskan hanya tombol Power saja sementara tombol Vol Up + Vol Down tetap ditekan sampai muncul TWRP Recovery berikut..

Setelah masuk TWRP, wajib lakukan ini…!

Masuk ke Mount, hilangkan centang “Mount system partition read-only”;

.. agar TWRP tidak hilang setelah flashing ROM selesai.

Selanjutnya install no-verity-opt-encrypt-6.0.zip untuk mencegah bootloop.

Selanjutnya geser ke kanan pada “Swipe to confirm Flash”,

Setelah proses selesai, jangan Reboot System dulu..!

Untuk mencegah hardbrick dan mengaktifkan jaringan 4G / LTE, kembali lagi ke halaman depan TWRP kemudian pilih Install;

.. ketuk pada berkas HZQ_MIUI_Fix_4G_LTE_NO_Camera_Mod_IDO_170417_NONUBL.zip;

Selanjutnya geser ke kanan pada “Swipe to confirm Flash”,

Tunggu proses hingga selesai dan Reboot System.

Sampai tahap ini pemasangan TWRP sudah selesai.

Proses reboot pertama kali biasanya memakan waktu lama, jika ada animasi tiga titik dibawah tulisan Android, artinya smartphone kamu aman, tunggu saja.

Langkah Root Redmi 3 / 3 Pro dengan SuperSU (Opsional)

Root adalah opsional, boleh kamu lakukan atau tidak terserah saja.

Masuk ke TWRP dan pilih Install,

Pilih file SuperSU yang telah kamu salin sebelumnya,

Selanjutnya Swipe to Install;

Selanjutnya pilih Reboot System.

Setelah Redmi 3 / 3 Pro selesai reboot, pada halaman Home Screen kamu akan menemukan icon SuperSU.

Sampai pada tahap ini proses instalasi TWRP dan root Redmi 3 / 3 Pro telah selesai.

Bagaimana cara masuk ke TWRP di Redmi 3 / 3 Pro ini?

Untuk masuk ke TWRP Recovery di Redmi 3 / 3 Pro, ikuti panduan berikut ini..

#1. Melalui Mode EDL Berbahasa China

Matikan HP kemudian nyalakan dengan menekan tombol Power + Vol Up hingga muncul logo Mi kemudian lepaskan.

.. akan muncul seperti ini..

Pilih recovery,

Jika malah masuk ke Homescreen atau berhenti (stuck) di logo Mi, coba lakukan langkah kedua.

#2. Melalui Kombinasi Tombol (Recommended)

Setelah TWRP terpasang, terkadang masuk TWRP menggunakan langkah 1 di atas tidak berhasil, malah HP masuk ke Homescreen.

Jika terjadi seperti ini, coba matikan HP kamu dan nyalakan dengan menekan tombol ketiga tombol » Power + Vol Up + Vol Down secara bersamaan hingga muncul logo Mi. Setelah muncul logo MI, lepaskan hanya tombol Power saja sementara tombol Vol Up + Vol Down tetap ditekan sampai masuk ke TWRP Recovery.

#3. Jika Pemasangan TWRP Gagal, Tetap di Mi Recovery

Jika pemasangan TWRP gagal, misalnya malah masuk ke Mi Recovery atau kamu tidak berhasil masuk ke Mode TWRP Recovery, gunakan toolkit yang saya buat ini :

  • ToolKitTWRP_v3.0-Ido.zip – 27 MB | Mediafire | OneDrive | Google Drive (Ekstrak dan letakkan di drive C:\ atau pada folder tanpa spasi);

Toolkit ini bisa berjalan kalau kamu telah mem-flashing TWRP_Patch_Miflash pada bahan di atas tadi atau Redmi 3 Pro kamu telah unlock bootloader.

Sambungkan Redmi 3 / 3 Pro dalam mode fastboot ke PC/Laptop kemudian ekstrak ToolKitTWRP_v3.0-Ido.zip, letakkan hasil ekstrak pada folder tanpa spasi..

Klik kanan » Run as Administrator pada file DoubleKlikSaya.bat sehingga muncul kotak Command Prompt seperti berikut ini..

Ketik “1”, kemudian Enter;

Ikuti instruksi selanjutnya, proses pemasangan TWRP akan berjalan dan masuk ke TWRP Recovery secara otomatis. Jika tidak, lepaskan kabel dan masuk ke TWRP dengan menekan tombol seperti langkah masuk TWRP di atas tadi.
