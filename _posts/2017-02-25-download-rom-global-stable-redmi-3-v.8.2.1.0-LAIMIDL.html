A Detailed Review on What's New in MIUI 8.2 Update!


I would like to give you my own detailed review on what's new for MIUI 8.2 Update! Here we go:

New Toggle Setting

Left side = Old Toggle Setting, Right side = New Toggle Setting

The MIUI 8 toggle setting is now improved! We can now arrange the toggles and enable/disable them on the same setting page. Simply tap on the toggle in setting page, and it will be enable or activated. No more hassle of the need to swipe down the toggle page just to enable or activate those toggles.

Automated Task How could we call our device a smartphone, when we still need to do things manually? Introducing the new feature of MIUI 8.2 - Automated Task.

Now we can automate several simple tasks in MIUI 8.2 with very easy settings. Automated Task feature can be found in Security App after scroll the main page to the bottom. By default, we will have two default automated task, which are "Turn on airplane mode in specific time" and "Turn off several settings when battery reaches 20%".

To create a new automated task, just tap on the "+" icon at bottom right. First, choose a condition of the task, either by time range, battery level or start time. Second, choose one or several actions to be done on that condition. Once you have mastered how to use it, your smartphone will become the true "Smartphone"!

Home Screen Settings

Another new setting is added in MIUI 8.2 update - Home Screen Settings!

We can now find Home Screen setting in the general Settings. Inside home screen settings, we can turn on one of the new feature, Smart Assistant. Other than that, we can change our system launcher, settings for "Fill empty cells" and settings to lock home screen layout.

App fills in the empty cell of uninstalled app If we enable "Fill empty cells" setting, the apps will automatically fill up the empty cell of the uninstalled app. Above animation shows how the app fill up the empty cell. By enabling the "Lock Home Screen Layout" setting, we cannot move any of the apps in our home screen, except adding widget or install a new app.

Smart Assistant

Smart Assistant is a page which provide the most useful information for you at the most left home screen. By default, it displays our most used app at the top row, then several cards which provide quick info for us. As this feature is China ROM based, some of the cards don't really useful or work for us global users. By opening up the settings, we can find "Frequent places" and "Card Settings". The "Frequent places" is only for China address, so we can just ignore that. In Card Settings, we can choose the card to be displayed at Smart Assistant. My favorite cards are Mi Fit, Mobile data (Network Assistant) and 日程提醒 (Calendar Reminder). Mi Fit provides our steps, distance and calary burnt in today, calendar reminder reminds us about today's tasks, while the Network Assistant, which I found to be the most useful card of all, provide the remaining data, today used data and total used data in month.

For the shortcuts under Card Settings, we can choose to show either app recommendations (most used) or features. We can select 5 quick features to be displayed at Smart Assistant. I don't find the given features to be useful, so I choose App Recommendations instead.

left side shortcuts = App Recommendations, right side shortcuts = Features

All of these cards in one page, together with the most used apps, makes this page very informative and convenient. That's why we called it Smart Assistant.

Battery Saver Toggle

Did you know that we can enable Battery Saver mode in MIUI 8 Security App? In MIUI 8.2 update, we can now add a new toggle to easily enable Battery Saver Mode! By enabling batery saver mode, it will stop sync and optimize background processes to further reduce battery usage. For your information, we can find the Battery Saver setting in Security App > Battery > Battery Saver. The setting allows us to schedule the time for battery saver mode. If we enable it by default, battery saver will go off automatically when our battery reached 60% and above.

Virus Scanning on 3rd Party App

Not enough of security check on virus? Now MIUI 8.2 update brings you an automatic virus scanning on 3rd party apk files. Any app which is not downloaded from Google Playstore or Mi Market, will have the virus scan before installing it. This is a good protection against malware or virus which is downloaded via 3rd party link. However, this setting does not have an option to disable it. Nevertheless, this feature is great in reducing the chance of getting virus infected on our device!

New Icons USB Charging & Changing Keyboard In MIUI 8.2 update, There will be new icons added for "USB for charging" and "Change keyboard". This change might not affect to most of the users, but at least we have more icon varieties in MIUI 8, yay!!

Of course, that's not all. MIUI 8.2 update also provided the all new Feedback app, new system sound and ringtones, and many more. In this review, I just show the newly added features which I want to give my own review. But that does not mean MIUI 8.2 is bug free. MIUI 8.2 could be a great leap from MIUI 8.1, and MIUI users are always welcome to flash into this ROM and give your feedback as much as you could. Thus, are you excited with all these new features?
