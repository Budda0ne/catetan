Langkah Langkah Pasang TWRP Redmi 4


Jika belum punya Minimal ADB Fasboot unduh di sini, ekstrak dan instal Minimal ADB Fastboot. Baca (boleh skip) tutorial pasang adb fastboot jika anda belum tahu caranya.

*) Pilih file TWRP sesuai Varian Redmi 4 anda

Kalau anda ingin sekaligus melakukan rooting silahkan unduh juga SuperSU.zup di sini
  1. Aktifkan USB debugging di Pengaturan > Pilihan pengembang
  2. Slain file TWRP dan tempel di direktori Minimal ADB Fastboot ( biasanya di Drive C:/Program Files/Minimal ADB Fastboot atau di C:/Progam Files (x86)/Minimal ADB Fastboot
  3. Rename file TWRP Cofface tadi menjadi twrp.img
  4. Hubungkan Redmi 4/A/Prime ke PC dengan kabel data
  5. Masuk ke mode fastboot dengan cara mematikan ponsel dan nyalakan lagi dengan cara menekan tombol Vol - + Power secara bersamaan
  6. Dalam folder/direktori Minimal ADB Fastboot, Tekan Shift + Klik kanan dalam ruang kosong kemudian pilih Open command window here dari menu yang muncul
  7. Setelah masuk CMD, ketik perintah fastboot devices tekan enter. Pastikan muncul serial ponsel anda.
  8. kemudian ketik lagi perintah fastboot flash recovery twrp.img dan enter agar twrp.img di flash ke partisi recovery Redmi 4 anda
  9. Tunggu sebentar dan ketik lagi perintah fastboot reboot dan enter
  10. Untuk masuk mode recovery dalam posisi ponsel menyala bisa menggunakan perintah adb reboot recovery jika berhasil memasang TWRP, akan reboot otomatis masuk ke mode recovery TWRP
  11. Atau gunakan kombinasi tombol Power + Vol + + Vol - tekan bersamaan jangan dilepas ketiganya sampai muncul logo MI, baru lepaskan tombol Power saja dan jangan lepaskan tombol Vol + + Vol - hingga masuk menu TWRP.
Pemasangan Custom Recovery TWRP sudah berhasil, silahkan lakukan full nandroid backup san pindahkan hasil cadangan ke PC agar bisa digunakan jika dibutuhkan suatu saat. Credit: Cofface sebagai dev unofficial TWRP, silahkan kunjungi situs Cofface di sini

Langkah Untuk Melakukan Root Xiaomi Redmi 4

Anda bisa langsung mengganti ROM dengan Custom ROM yang anda suka dengan TWRP yang sudah terpasang pada Redmi 4A atau Redmi 4/Prime anda. Tetapi kalau hanya berniat untuk melakukan rooting, cukup ikuti langkah berikut:
  1. Salin file SuperSU .zip yang anda unduh sebelumnya dan simoan di penyimpanan internal ponsel
  2. Nyalakan ponsel ke mode Recovery TWRP
  3. Sesudah dalam mode TWRP, pilih menu Instal dan cari file SupeSU yang anda salin sebelumnya.
  4. pilih file zip tersebut dan konfirmasi proses flash dengancl menggeser tombol indikator di bawah ke arah kanan
  5. tunggu hinga proses selesai dan pilih reboot to system
Lihat apakah Aplikasi SuperSU sudah terpasang pada Redmi 4/prime. Jika aplikasi sudah ada salam daftar, artinya Redmi 4/Prime atau Redmi 4A anda sudah berhasil root.

Dengan demikian anda sudah berhasil memasang TWRP Cofface sekaligus melakikan rooting pada Xiaomi Redmi 4 series. Baik itu Redmi 4A, Redmi 4 maupun Redmi 4 Prime. Silahkan bagikan agar ilmunya bermanfaat. Silahkan ajukan pertanyaan dalam form komentar kalau masih ada yang belum dipahami.
