Tutorial Rooting Andromax I3 (Hisense)

Perhatian, anda harus setuju dengan Disclaimer di blog ini jika mengikuti Cara Rooting Andromax I3 ini. Dan anda juga paham bahwa: dengan melakukan rooting handheld anda menyebabkan garansi anda gugur, beresiko kehilangan data. Jadi backup data penting anda terlebih dulu dan simpan di tempat aman. JIka ingin share ulang baca tentang DMCA Copyright.

Sebelum melakukan rooting, anda harus sudah meng-aktifkan USB Debugging. jika belum ikuti tutorial mengaktifkan USB Debugging. Kemudian masuk ke mode fastboot baca tutorial fastboot 1 dan lanjut di tutorial fastboot 2.

Rooting Andromax I3 ini menggunakan perintah-perintah yang harus anda ketik melalui ADB Debugging jadi download dulu tool adb debug dan fastboot (link ada di bawah) kemudian ekstrak di drive C. download juga file Recovery (link dibawah) dan rename menjadi recovery.img dan letakkan dalam satu folder bersama adb.exe dan fastboot.exe yang anda ekstrak tadi. Download juga Aplikasi SuperSU (link dibawah) ekstark dan letakkan di drive C juga biar mudah.

Setelah semua siap, reboot Andromax I3 kedalam mode Fast Boot dan tancapkan ke komputer anda dengan kabel USB.
Masuk ke folder dimana adb.exe dan fastboot.exe beradaTekan tombol shift dan klik kanan mouse pilih menu open cmd here
Dalam jendela CMD ketik fastboot devices untuk memastikan bahwa Andromax I3 anda terdeteksi dan dalam keadaan fastboot..
Flash recovery white blank yang anda download tadi dengan mengetik fastboot flash recovery recovery.img
Setelah ada pemberitahuan sukses, cabut baterai dan nyalakan secara normal.
Sekaran tancapkan Andromax I3 anda ke komputer dengan kabel USB.
Masuk lagi ke folder adb buka cmd (caranya sama) dan ketik adb reboot recovery Andromax akan reboot ke dalam mode recovery dengan di tandai layar kosong putih.
Tutup dulu jendela CMD, kemudian masuk ke folder hasil ekstrak SuperSU, masuk ke lalu masuk folder armeabi dari sini buka CMD dengan cara tekan tombol shift dan klik kanan mouse pilih menu open cmd here.
Dalam CMD yang terbuka, ketik perintah dibawah ini:
    adb shell mount /system
  • adb push su /system/bin
  • adb push su /system/xbin
  • adb shell chmod 4755 /system/bin/su
  • adb shell chmod 4755 /system/xbin/su

Tutup CMD kemudian masuk ke folder lokasi Superuser.apk dan install-recovery.sh, tekan tombol shift bersamaan dengan klik kanan, pilih menu open CMD Here.
Pada jendela CMD, ketikkan perintah-perintah adb debug dibawah secara berurutan dan tekan emter setiap ganti baris.
  • adb push install-recovery.sh /system/etc
  • adb shell chmod 4755 /system/etc/install-recovery.sh
  • adb push Superuser.apk /system/app
  • adb reboot

Untuk mengecek apakah proses rooting yang ada lakukan berhasil atau tidak, anda bisa menggunakan terminal emulator (download dari Playstore) dan buka kemudian ketik su.

Jika tanda $ (dollar) berunah menjadi # (pagar)maka rooting Andromax I3 sudah berhasil.

Anda juga bisa menggunakan Root Checker untuk mempermudah.
