Lenovo A7000 plus (chipset mediatek) memang tak mendapat Vibe UI, meskipun sudah mendapat update Marshmallow. Sebagai obat rasa penasaran anda bagaimana rasanya menggunakan VIBE UI milik Lenovo yang paling baru, ada ROM Vibe UI 3.5 yang bisa anda cicipi.


Vibe UI 3.5 Marshmallow bisa anda install untuk A7000 sebagai pengganti ROM official alias Stock Firmware Lenovo A7000 plus anda. Dengan menggunakan ROM Vibe UI ini, anda bisa mencoba berbagai macam fitur keren yang sebelumnya tidak ada di Stock ROM.

Tampilan VIBE UI 3.5 ini sangat kental dengan aroma Marshmallow, berbeda dengan user interface MIUI yang tampak sama saja apapun basis OS Android yang dipakai. Untuk lebih jelasnya bisa dilihat di screenshot berikut:

Untuk mencoba dan mencicipi Android Marshmallow ala Vibe UI ini, silahkan download ROM di bawah. File nya lumayan besar, sekitar 1 GB. Silahkan download via Wi-Fi saja jika sedang miskin kuota atau jaringan data tak stabil.

Selanjutnya, anda bisa menggunakan TWRP recovery untuk install VIBE UI 3.5. Jika anda belum punya TWRP recovery untuk Lenovo A7000 plus, silahkan pasang terlebih dahulu. Panduannya bisa dibaca di tutorial pasang TWRP

Dengan mempraktekkan atau menggunakan tutorial maupun bahan yang ADA di halaman ini anda saya anggap sudah membaca dan setuju dengan disclaimer. Ingat, usahakan mempunyai back-up System dan Data anda, Karena ini beresiko bootloopp, soft brick bahkan ponsel anda mati total jika anda tidak hati-hati!!

Langkah Instalasinya, kopikan file zip (ROM Vibe UI 3.5) ke sdcard atau bisa juga di Internal Storage.

Reboot ke recovery mode. Dengan menekan tombol [power] + [vol down] bersamaan hingga masuk ke TWRP recovery.

Sebelumnya memasang ROM, bersihkan dulu partisi System, Data, Cache Dan Dalvk-cache dengan Cara memilih menu Wipe > Advance > centang semua opsi Dan swipe ke kanan indicator di bawahnya untuk konfirmasi. Proses ini memakan waktu, jadi bersabar jika proses seperti berhenti.

Setelah proses Wipe selesai, anda bisa kembali ke menu utama TWRP Dan pilih menu Install. Pilih lokasi penyimpanan yang anda pakai untuk meletakkan file ROM (Select Storage) Dan arahkan/pilih file ROM, Swipe kanan di indicator dibawah untuk konfirmasi instalasi. Proses ini juga memakan waktu lumayan lama.

Ingat, jangan sampai mati ditengah-tengah proses instalasi maupun Wipe. Karena bisa saja partisi menjadi rusak Dan tak terbaca.

Setelah instalasi selesai, pilih Wipe Cache dan dalvik jika di tawarkan. Kemudian pilih menu Reboot to System untuk menyelesaikan Instalasi.

Proses inisialisasi juga memakan waktu anda juga perlu bersabar. Setelah masuk Home screen anda bisa kembali ke recovery mode untuk melakukan rooting dengan flash file SuperSu.zip untuk memastikan manajemen root lebih baik.

Selesai... Jangan sungkan untuk bekomentar kalau anda punya tips (misal: tweak atau optimasi ROM) atau keluhan (misal: install Xposed Framework) di ROM VIBE UI 3.5 ini. Semoga bermanfaat.
