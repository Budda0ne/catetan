Sebelumnya beberapa platform lain sudah merilis fitur mode gelap semisal Facebook Messenger. Aplikasi pesan WhatsApp juga kabarnya akan punya mode gelap.

Meskipun fitur dark mode ini sudah stabil, pembaruan Chrome masih termasuk versi beta, jadi masih ada kemungkinan beberapa bug yang muncul. Untuk mengaktifkan fitur mode gelap, silahkan ikuti langkah berikut:

1. Perbarui Chrome ke versi terbaru v.74

Lakukan Pembaruan peramban Chrome ke versi 74 atau yang lebih tinggi. Jika tidak yakin berapa versi Chrome yang terpasang, buka Chrome dan pilih tiga titik dikanan atas untuk membuka menu, pilih Settings atau Setelan. Setelah itu scroll ke bawah, ketuk pilihan About Chrome atau Tentang Chrome yang berada pada baris paling bawah dan nomor versi akan terlihat di situ.

Jika ternyata belum menggunakan Chrome 74 atau lebih tinggi, silahkan perbarui aplikasi. Versi baru saat ini sudah diluncurkan di Play Store. Tapi jika belum muncul pilihan update, buka saja tautan ini Â¹ untuk memperbarui sendiri secara manual.

2. Mengaktifkan Flag Dark Mode

Setelah itu, buka aplikasi dan ketik chrome://flags ke bilah alamat, dan tekan tombol enter.

Dari sana, cari mode gelap Android Chrome UI dengan mengetik dark mode untuk mempercepat pencarian.

Ketuk kotak bertanda Default di bawah dan pilih 'Diaktifkan' dari menu yang muncul. Tekan "Luncurkan Ulang Sekarang" di pojok kanan bawah untuk meluncurkan ulang aplikasi.

3. Aktifkan dulu Mode Gelap di pengaturan

Setelah Chrome terbuka kembali, fitur mode gelap masih belum aktif. Jasih masih harus mengaktifkan mode gelap untuk aplikasi. Ketuk tombol menu di sudut kanan atas layar dan pilih Setelan. Dari sana, ketuk pilihan Dark Mode atau Mode Gelap. Aktifkan.

Sepengalaman saya, opsi Mode Gelap ini ternyata tidak muncul dalam pengaturan Chrome ketika pertamakali, namun setelah beberapa kali dibuka tutup, baru muncul. Jadi cobalah tutup paksa aplikasi jika belum muncul dan coba cek lagi pengaturan hingga Mode Gelap muncul. Pilihan Dark Mode ini baru muncul setelah membuka tutup paksa Chrome hingga tiga atau empat kali.

4. Mencoba Mode Gelap

Fitur mode gelap dalam peramban Chrome masih belum fix 100%, sehingga masih memiliki kekurangan yang perlu diperbaiki, sebelum diluncurkan secara resmi. Namun, dengan menerapkan langkah diatas bisa mengaktifkan tema gelap ke tampilan Chrome secara umum. Termasuk halaman tab baru, pengalih tab, bilah alamat, dan berbagai menu dalam aplikasi pun berwarna gelap.

Pada Chrome versi pra-stabil, tampaknya Google sedang menguji versi perpanjangan dari tema yang juga membuat beberapa situs web gelap. Ini dapat menyebabkan masalah teks gelap yang sama pada latar belakang gelap, dan ada masalah lain di mana gambar dapat muncul dalam warna terbalik dari waktu ke waktu.

4. Menonaktifkan Mode Gelap

Jika merasa bug ini adalah terlalu menggangu, silahkan menonaktifkan mode gelap dengan cara kembali ke menu pengaturan Chrome, ketuk menu Mode Gelap, dan matikan sakelar. Google kabarnya akan memperbaiki bug yang terlihat dalam fitur ini agar nantinya lebih nyaman dipakai.
