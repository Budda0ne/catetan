Ketiga pabrik lokal tersebut adalah Evercoss, Mito, dan Nexian. Evercoss One X dari Evercoss, Mito dengan ponsel Mito Impact-nya, dan Nexian memberi nama Nexian Journey untuk ponsel Lollipop ini.
3 Ponsel Android One di Indonesia
Ketiga vendor lokal ini diperbolehkan oleh Google untuk menetukan desain fisiknya saja. sedangkan untuk ketentuan hardware dan spesifikiasi HARUS sesuai permintaan Google. Jadi, bisa dibilang tiga ponsel ini adalah SAMA PERSIS kecuali tampilan luarnya. Sehingga cara root android one dan bahan-bahannya pun SAMA. Jadi jika anda ingin melakukan rooting pada Evercoss One X atau root Nexian Journey akan sama saja dengan cara root Mito Impact.

Untuk melakukan rooting Android One, anda membutuhkan Komputer atau Laptop karena tidak belum ada cara melakukan root android dari HH secara langsung. Jadi cara rooting lollipop ini akan menggunakan cara yang sedikit primitif dengan menggunakan adb dan fastboot untuk install Custom Recovery pada Android One anda baru anda bisa melakukan rooting. :D

Baiklah, persiapkan lakukan hal-hal berikut untuk melakukan rooting:

1. Cek file ZIP Update-SuperSU terbaru yang stabil atau bisa langsung download di bawah
2. Instal CWM Recovery atau TWRP, tutorialnya ada di link ini.
3. Rename file menjadi superuser.zip agar mudah mencarinya dan letakkan di sdcard internal
3. Masuk ke recovery mode dan back-up Stock ROM anda sebelum anda melakukan root (untuk jaga-jaga saja, boleh diabaikan)
4. Setelah selesai lakukan flashing file superuser.zip
5. Silahkan pilih menu reboot to system.

Android One anda sudah root. Anda bisa cek di drawer aplikasi anda akan muncul aplikasi SuperSU yang menandakan ponsel anda sudah root.

Catatan Penting:
- Anda juga bisa masuk mode fastboot (bootloader) dengan cara mematikan ponsel terlebih dahulu kemudian tekan bersamaan tombol volume up + tombol power dan pilih fastboot mode.
- Jika anda ingin melakukan root Android One tanpa melakukan Unlock Bootloader, Jangan Install Custom Recovery. Lakukan rooting dengan cara boot recovery temporer dan flash superuser.zip langsung untuk melakukan rooting.
