Apalagi akhir-akhir ini jika membeli notebook/laptop pasti akan ditawarkan OS Windows 7ini. Dari sekian sistem operasi milik Microsoft ini, seperti Windows XP misalnya, masing-masing memiliki cara yang berbeda dalam tahap instalasi ulangnya.

Cara Install Windows 7 ini bagi orang yang sudah terbiasa meng-instal, mungkin itu pekerjaan sangat mudah. Tapi, pastinya masih banyak juga yang belum biasa untuk menginstal Windows 7 sendiri. Mungkin karena takut error, takut mencoba atau masih pemula dan baru belajar Cara Install Windows 7, termasuk kamu.. Karena cara instalasi OS Windows 7 ini sedikit berbeda dengan sistem operasi yang lain, maka dibawah ini saya akan menuliskan tahap-tahap dalam melakukan instalasi Windows 7 yang mungkin bisa membantu kamu yang sedang belajar atau ingin bisa/mampu untuk menginstall windows 7-nya sendiri. Berikut adalah beberapa tahap Cara Install Windows 7 secara lengkap, perhatikan dengan teliti langkah-langkahnya.

1. Pastikan dari BIOS booting komputer Anda di setting untuk DVD
2. Masukkan DVD windows
3. Tekan sembarang tombol saat muncul boot from cd or dvd

4. Akan terlihat gambar seperti dibawah ini

5. Pilihlah Indonesian pada Language, time, currency, and location

6. Tekan tombol install now

7. Tunggulah beberapa saat proses ini

8. Centang pada I accept the license terms sebagai persetujuan penggunaan windows 7 kemudian klik next

9. Karena sedang melakukan clean install maka pilih saja custom (advanced) untuk memilih di drive mana windows 7 akan di install.

10. Di tahap ini kamu bisa melakukan partisi atau membagi Hardisk kedalam beberapa drive, dimana nanti untuk windows 7 berada pada drive (C) dan sisanya drive (D) untuk penyimpanan data kamu nantinya. Atau jika kamu mau partisi setelah instalasi windows 7 langsung saja klik "Next". Biasanya saya langsung klik next, untuk partisinya belakangan.

11. Tunggulah proses ini beberapa saat

12. Secara otomatis windows akan restart

13. Setelah restart akan muncul gambar berikut ini
14. Tunggulah proses Setting up the services hanya beberapa saat saja

15. Instalasi akan dilanjutkan secara otomatis

16. Masukkan Nama User dan Nama Komputer sesuka kamu

17. Jika perlu password ketikkan passwordnya 2 kali atau kosongkan saja jika kamu tidak ingin mem-password user kamu
18. Masukkan product key serial number windows 7. jika tidak ada kosongkan saja.

19. Pilihlah level proteksi keamanan dari Microsoft

20. Atur Zona waktunya(untuk Indonesia +7 dari GMT)

21. Selamat, windows 7 sudah siap digunakan

Nah, setelah proses install-nya selesai, saya menganjurkan suapaya meng-install anti mallware sesegera mungki dari sumber yang bersih. Soal anti mallware-nya apa silahkan tentukan sendiri, bisa dari Esset, Norton, Kaspersky atau lainnya.
