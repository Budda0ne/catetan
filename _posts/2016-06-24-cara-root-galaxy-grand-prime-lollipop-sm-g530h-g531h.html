Cara melakukan rooting Samsung Galaxy Grand Prime


Untuk SM-G530H[1]

- Instal TWRP menggunakan odin.
– Kopi SM-G530H-SuperSU-v2.46.zip ke sd card atau memory internal.
– Akses TWRP (vol up + home button + power button).
– Cari file supersu.zip tadi terus flash lewat TWRP

Untuk SM-G531H[2]

– instal TWRP yang ada di paketan zip lewat Odin.
– copy UPDATE-SuperSU-v2.46.Zip ke sd card atau memory internal
– akses TWRP (vol up+ home button + power button).
– cari file supersu.zip tadi terus flash lewat TWRP
