Meskipun tutorial edit init.d script swap menggunakan internal memori ini saya tulis untuk Cross A28, tips ini bisa juga digunakan untuk Smartphone selain Cross A28. Yang penting smartphone sudah rooted dan support script init.d.

Link Download Init.d Script Swap v 3.0.3 via dh.st [TURBO BOOST MOD V3.03-AROMA]

Untuk mengaktifkan swap di partisi /data atau /system menggunakan script swap v 3.00 sebenarnya ada beberapa hal yang perlu diketahui agar instalasi swap di Cross A28 berhasil dengan baik. Pertama, anda harus mengerti logika script yang digunakan dan kedua anda perlu menyesuaikan script tersebut sesuai dengan ruang yang akan anda gunakan sebagai tempat untuk menampung partisi/drive virtual yang bernama swap.img

Dalam script swap v 3.00 (swap internal memori) ini, yang perlu anda pahami agar bisa menyesuaikan dengan kemampuan ROM atau Custom ROM Cross A28 anda adalah 4 bagian baris script berikut:

bagian 1.
# partition /cache or /data or /system
partition="/data";
bagian 2.
# the requested swap file size in MB
swap_file_size_mb=75;
bagian 3
# min. free memory; below that
# value of free memory, the
# swap file will be resized -
# very nice for data partition,
# where 150MB min. is required
min_free_space_mb=100;
bagian 4.
# minimum size of the swap file
# if size is lower, the swap will
# not be created
min_swap_size_mb=20;
Selain 4 bagian di atas, silahkan di abaikan karena jika anda merubahnya malah akan membuat Cross A28 anda morat marit. Berikut adalah penjelasan dari fungsi - fungsi dari ke-empat baris script swap di atas

Lihat bagian 1

Script tu adalah dimana kalian akan menempatkan file swap.img yang akan berfungsi sebagai partisi virtual yang akan digunakan sebagai wadah untuk menempatkan file-file swap

Jika ingin menempatkan di partisi /system maka kalian perlu merubah baris kode:
partition="/data";
menjadi
partition="/system";
atau jika ingin menempatkan di partisi /cache, ya tinggal ganti saja menjadi:
partition="/cache";
atau bisa juga swap di memori internal yang 1 GB di Cross A28 (USB Storage atau EMMC). Cukup ganti menjadi:
partition="/mnt/emmc";
Lihat bagian 2

Baris kode ini adalah parameter untuk besaran file swap.img yang akan dibuat di partisi yang telah di tentukan di bagian 1. (/system, /data, /cache, atau /mnt/emmc)

Besar filenya dalam ukuran MB. Untuk menentukan berapa besar file yang bisa dibuat, check dulu menggunakan tool Disk Info dan lihat sisa memory/ruang kosong pada partisi yang akan digunakan.

Misalkan menggunakan partisi /system, kalau di /system tersisa 100MB, itu artinya file swap.img yang akan dibuat sehasrusnya tidak lebih dari sisa ruang tersebut. Jadi, set nilainya menjadi 75 atau 90 terserah asal masih dibawah 100MB seperti ini
swap_file_size_mb=75;
atau seperti ini
swap_file_size_mb=90;
Begitu juga jika akan menggunakan partisi /data dan /cache. Check dulu ruang yang tersisa dan baru tentukan nilainya.

Lihat bagian 3

Parameter dari baris kode ini memberi batasan, apakah file swap.img akan dibuat atau tidak. Kalau anda meminta 120MB dan ternyata di partisi yang ditentukan tidak sampai 120MB (yang kosong), katakanlah cuma 110MB maka file swap.img tidak akan dibuat. Misalkan saja anda set 100MB, maka kodenya menjadi seperti ini:
min_swap_size_mb=20;

Saya hanya akan membuat file swap.img (partisi virtual) di partisi /system sebesar 75MB kalau ada ruang sisa di partisi /system sebesar 100MB yang akan saya gunakan sebagai partisi swap. Tetapi saya akan tetap membuat file swap.img kalau masih tersedia ruang bebas dalam partisi tersebut sebesar 20MB.

Setelah memahami parameter parameter di atas, anda bisa meng-edit file script init.d menggunakan text editor dari root browser atau root explorer atau file manager lain dengan hak akses root.

Setelah selesai dan di simpan, anda bisa melanjutkan meng-install init.d script Swap v3.00 di Cross A28. Untuk memastikan Swap menggunakan memori internal ini berjalan dengan baik, reboot 2x.
