--- categories: [arsip] title: "Panduan Install Windows 8.1 dari USB Flash Drive" # date: '2013-07-06T13:51:00.001+07:00' # author: Unknown tags: - Tutorial - Windows 8.1 - Windows - USB Bootable last_modified_at: '2014-05-19T03:39:34.983+07:00' header: teaser: http://2.bp.blogspot.com/-qlyDe4pHi2Q/UdGEujxb0qI/AAAAAAAAAf8/YMqs9g31H_Y/s72-w500-c/Logo-Windows-8.1-Preview.jpg # blogger_id: tag:blogger.com,1999:blog-8168996750712797250.post-2964987372030156214 redirect_from: - /2013/07/panduan-install-windows-81-dari-usb.html --- Dalam proses Instalasi sistem operasi dari media USB Flasdisk, secara umum teradapat 3 langkah yaitu: menyiapkan bootable USB dengan memindahkan semua file sistem operasi ke USB, Mengatur pada PC anda agar booting dari USB, dan yang terakhir menginstal sistem operasi itu sendiri (dalam hal ini Windows 8.1).

Pengguna Windows yang sudah mahir dan tahu/bisa Command Prompt tentu akan lebih mudah ketika mempersiapkan bootable USB. Karena tidak semua pengguna terbiasa menggunakan CMD, dalam panduan ini, saya akan memberi tutorial untuk anda dengan dua metode. Langkah langkah untuk menginstal Windows 8.1 dari USB flash drive.

Metode 1 menjelaskan pembuatan bootable USB untuk Windows 8.1 dengan bantuan software pihak ketiga (tool). Sedangkan metode 2 diperuntukkan bagi pengguna yang ingin mempersiapkan bootable USB tanpa menggunakan software pihak ketiga (menggunakan Command Prompt). Kedua metode ini bisa digunkan untuk Windows versi 32-bit maupun 64-bit.

Baca juga: Cara Instal Windows 8/8.1 dari hard drive eksternal.

Metode 1: Membuat bootable USB dengan bantuan alat pihak ketiga.


CATATAN: Dalam panduan ini, anda belajar membuat USB bootable menggunakan PC Windows 8, tetapi Anda juga dapat menggunakan metode ini untuk membuat bootable USB dari PC Windows 7.

Ada banyak tool untuk membuat Botable USB yang bagus, termasuk milik Microsoft sendiri Windows 7 USB / DVD Tool, kali ini anda akan menggunakan salah satu tool tersebut, yakni software bernama Rufus. Ikuti panduan berikut:

Langkah 1: Hubungkan USB flash drive yang Anda ingin menginstal Windows pada PC Anda dan backup semua data dari USB drive sebelum melanjutkan lebih lanjut karena kami akan menghapus semua data dengan memformatnya.

Langkah 2: Buka halaman ini dan download versi terbaru dari Rufus. Ukuran file yang didownload sekitar 0,5 MB.

Langkah 3: Double-klik pada file Rufus.exe untuk menjalankan tool. Akan muncul notifikasi Do you want to allow Rufus to check for application updates?. Klik Yes atau No untuk melanjutkan.

Tool Rufus
Langkah 4: Pilih flash drive USB dalam menu drop-down Device, pilih Partition scheme and target system type sebagai MBR partition scheme for BIOS or UEFI computers, dan pilih file system sebagai NTFS.

CATATAN: Jika Anda sedang mempersiapkan bootable USB untuk PC UEFI, Anda harus memilih GPT partition scheme for UEFI computer untuk pilihan Partition scheme and target system type dan FAT32 untuk file system. Perlu diketahui bahwa UEFI tidak mendukung Windows 8.1 versi 32-bit.

Langkah 5: Selanjutnya, centak kotak Create a bootable disk using, dan pilih ISO Image lalu klik ikon CD / DVD untuk memilih file ISO 8.1 Windows. Dan jika Anda memiliki DVD Windows 8.1, lebih baik anda menggunakan metode 2.

Langkah 6:Tunggu hingga proses selesai.

USB flash drive bootable Windows 8.1 Anda sudah siap! Sekarang Anda dapat menyalakan PC, dan menyetting agar PC dapat di booting menggunakan USB melalui bios setting, dan anda bisa mulai menginstal Windows 8.1

Metode 2: Mempersiapkan USB bootable dengan menggunakan Command Prompt (tanpa menggunakan tool, untuk anvanded user saja).


Langkah 1: Buka Command Prompt sebagai administrator. Untuk melakukan hal ini, di Windows 8, secara bersamaan tekan tombol Windows + X untuk melihat Daya Menu di pojok kiri bawah layar, klik Command Prompt (admin), dan kemudian klik Ya untuk UAC prompt untuk memulai cepat ditinggikan.
Jika Anda pada Windows 7, menu Start terbuka, ketik CMD di kotak pencarian layar Start, dan kemudian secara bersamaan tekan Ctrl + Shift + Enter kunci untuk menjalankan Command Prompt sebagai administrator.

Langkah 2: Selanjutnya, masih dalam Command Prompt, Ketik DISKPART dan tekan Enter.

Langkah 3: Ketik LIST DISK kemudian tekan enter lagi untuk melihat daftar drive yang ada dalam PC Anda.
Di sini, Anda harus benar-benar mengetahui nomor flashdisk (USB flash drive) yang ingin Anda jadikan USB bootable. Misalnya, pada gambar di bawah, nomor Disk drive USB saya adalah "Disk 1". Kami sarankan Anda memeriksa nomor Disk USB drive Anda sebelum melanjutkan ke langkah berikutnya karena kita akan memformatnya. Dan jika anda salah nomor, akan menghapus drive (partisi) lainnya.

Langkah 4: Masukkan perintah berikut satu per satu dan tekan tombol Enter setelah memasukkan setiap perintah.
SELECT DISK 1 (ganti "1" dengan nomor disk USB drive Anda)
CLEAN
CREATE PARTITION PRIMARY
SELECT PARTITION 1
ACTIVE
FORMAT FS=NTFS QUICK
ASSIGN
EXIT

Langkah 5: Selanjutnya, Mount file iso Windows 8.1 yang sudah anda download. Jika Anda menggunakan Windows 8, mounting file ISO cukup sederhana, klik kanan pada file ISO, pilih opsi Mount. Ketika Anda me-mount file ISO, Anda akan melihat virtual drive baru di My Computer. Dan jika Anda berada di Windows 7, ikuti panduan untuk me-mount file ISO pada Windows 7 (gunakan Power ISO atau Virtual Drive).

Langkah 6: Dalam langkah ini, Anda perlu menyalin semua file termasuk autorun.ini file dari Windows 8.1 DVD atau mount drive (berisi file dari ISO). Itu saja! Bootable Windows 8.1 USB drive Anda siap!

Good luck! Anda dapat reboot atau menghubungkan USB drive ke PC di mana Anda ingin menginstal Windows 8.1 dan mulai menginstal versi terbaru dari Windows ini.
