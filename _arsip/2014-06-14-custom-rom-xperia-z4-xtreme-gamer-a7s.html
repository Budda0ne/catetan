Cara Pasang/Instalasi Custom ROM Xperia Z4 Xtreme

Sebelum meneruskan tutorial Instalasi ini, anda harus setuju dengan Disclaimer dan patuh dengan artikel DMCA Copyright di blog ini. Dan anda juga paham bahwa: dengan menginstall Custom ROM di handheld anda menyebabkan garansi anda gugur, resiko kehilangan data, jadi backup data penting anda.

Syarat: harus sudah ter-install CWM Recovery, jika belum, baca di tutorial install CWM Recovery.

Untuk memasang ROM ini, anda sebaiknya melakukan Backup IMEI melalui aplikasi Mobile Uncle atau aplikasi backup IMEI lainnya, sebagai antisipasi jika terjadi INVALID IMEI. Buat juga Nandroid Backup melalui CWM Recovery sebelum melakukan flashing/install.

Langkah-langkah instalasi:

  1. Download Xperia_Z4_Xtreme_by_Arch.zip dan letakkan di sdcard tapi jangan di dalam folder. Letakkan di /sdcard/--disini- - (menjadi /sdcard/Xperia_Z4_Xtreme_by_Arch.zip)
  2. Matikan Evercoss A26C anda, dan hidupkan lagi dalam mode CWM Recovery dengan menekan tombol [Power] dan [Volume Up]
  3. Ketika sudah masuk mode CWM Recovery, gunakan tombol [Volume Up] dan [Volume Down] untuk navigasi dan tombol [Power] untuk memilih menu:
    • pilih menu Wipe Data/Factory Reset pilih yes
    • pilih lagi menu Wipe Cache Partition pilih yes lagi
    • pilih menu Advaced, pilih Wipe Dalvik Cache
  4. Pilih Install Zip From SdCard kemudian pilih file Xperia_Z4_Xtreme_by_Arch.zip
  5. CWM Recovery akan melakukan proses flashing, tunggu hingga selesai.
  6. Setalah Selesai, pilih GO BACK (menu paling bawah) dan pilih Reboot System Now

Jika Proses Instalasi berjalan mulus, Custom ROM Xperia Z4 Xtreme akan ter-install dengan baik di Evercoss A7S anda.

Agar anda bisa menggunakan gaming mode, setelah boot pertama buka aplikasi Terminal Emulator dan
- ketik "su" (tanpa tanda petik) kemudian enter
- ketik "arch" (tanpa tanda petik) lalu enter
Jika muncul notifikasi, abaikan saja. Mode gaming pun sudah aktif, reboot lagi Evercoss A7S anda.

Untuk memilih antara Gaming mode dan Normal mode, masuk ke Setting, Developer options dan pilih opsi yang ada.
