Ketika kamu ingin memblokir situs tertentu, kamu bisa menyunting file hosts yang ada pada sistem Android dan otomatis situs tersebut tidak bisa diakses lagi. Sayangnya, kamu membutuhkan akses superuser a.k.a root untuk merubah isi file hosts tersebut. Kalau ponsel tidak dalam keadaan root, kamu tidak akan bisa membuka apalagi menyunting file hosts seperti yang kamu inginkan.

Akan tetapi, kamu bisa menggunakan VPN berbasis file hosts lokal, yang bisa megarahkan ulang atau memblokir alamat situs tertentu hanya dengan beberapa klik saja. Keuntungan memakai aplikasi sejenis VPN lokal (seterusnya akan kita sebut sebagai Local VPN) ini adalah kamu tidak perlu melakukan rooting, namun kamu bisa menyunting file hosts untuk memblokir situs (server iklan) yang kamu inginkan secara leluasa layaknya AdAway.

Seperti sudah dijelaskan secara singkat pada artikel sebelumnya, kedua jenis aplikasi; Native Adblock (mis: AdAway) dan Local VPN (mis: DNS66), mempunyai dasar yang sama namun dengan metode dan pendekatan yang berbeda. Keduanya sama-sama memanfaatkan file hosts untuk memblokir iklan.

Native Ad Blocker versus VPN AdBlocker

Jika AdAway melakukan modifikasi langsung terhadap isi file hosts android, maka VPN lokal membuat sever VPN secara lokal dan semua trafik data diteruskan melalui koneksi VPN tersebut. Dengan begitu, aplikasi VPN lokal ini bisa menyaring, apakah akan memblokir atau tidak dengan berpatokan pada daftar yang dibuat pada file hosts lokal yang sudah dibuat/ditentukan sebelumnya.

Singkatnya, baik Adaway maupun DNS66 merupakan aplikasi adblock yang bergantung pada file hosts. Yang membedakan adalah: Adaway membuat perubahan secara langsung pada file hosts, sedangkan DNS66 menambahkan file hosts lain untuk memfilter permintaan alamat IP domain tertentu.

Jika menilik dari performa, Adaway tetap memiliki kenggulan. Karena aplikasi ini langsung merubah isi file hosts yang ada di sistem Android, aplikasi adaway tidak perlu aktif terus menerus sehingga tentu lebih irit baterai. Untuk responsifitas tidak berbeda, Adaway hanya sedikit lebih gegas dibanding VPN lokal.

Tetapi, tidak semua orang bisa atau mau melakukan rooting. Jika kamu termasuk golongan yang tidak ingin dipusingkan masalah rooting dan ingin tetap memblokir iklan, maka pilihan terbaik adalah dengan menggunakan aplikasi VPN berbasis file hosts lokal alias VPN Lokal ini.

Kamu bisa memilih Salah Saturday dari 3 aplikasi VPN lokal untuk memblokir iklan pada ponsel non-root. Salah satu aplikasi berikut sudah digunakan untuk tutorial; memblokir semua iklan pada aplikasi android tanpa root. Silahkan kamu pilih mana yang kamu sukai.

1. AdGuard.

Aplikasi AdGuard sebenarnya tidak bisa dibilang sebagai VPN Lokal, karena aplilasi ini tidak menyediakan fitur untuk meng-edit file hosts. Daftar situs iklan disediakan secara otomatis oleh server AdGuard.

AdGuard lebih cocok untuk kamu yang tidak ingin repot. Cukup buka aplikasi, setujui permintaan akses koneksi VPN, iklan pun hilang. Selain AdGuard, adabanyak lagi aplilasi yang bisa kamu pakai seperti: NetGuard, Blockada, Block This dan masih banyak lagi.

2. DNS66.

Aplikasi inilah yang bisa dibilang pelopor VPN lokal karena menyediakan opsi untuk menentukan sumber file hosts yang akan digunakan untuk memblokir iklan layaknya AdAway. DNS66 menyediakan daftar alamat sumber hosts terpercaya yang sudah jadi sehingga kamu tidak perlu membuat sendiri atau mengunduh file hosts secara terpisah. Tetapi, tidak seperti jenis aplikasi pertama, kamu bebas menambahkan sumber file hosts yang tidak ada dalam daftar yang diberikan.

File hosts yang diunduh kemudian akan dipakai sebagai patokan untuk memblokir iklan. Karena tidak bisa mengedit file hosts secara lokal, jika kamu ingin menambahkan situs tertentu, maka kamu harus mengupload file yang sudah kamu edit ke file hosting seperti github, baru kamu unduh lagi.

3. Virtual Hosts.

Virtual Hosts, adalah aplikasi yang paling fleksibel. Tampilan dan interface-nya sangat sederhana. Untuk menggunakannya, kamu harus sudah punya file hosts. Aplikasi ini tersedia di Playstore dengan nama Virtual VPN yang dihargai sekitar 40.000, tetapi kamu bisa mengunduh gratis via github.

Aplikasi ini sangat bebas. Kamu bisa membuat file hosts dengan daftar situs yang ingin kamu blokir sesukamu. Bisa jugu mengunduh file hosts dari situs penyedia file hosts untuk memblokir iklan, kemudian menambahkan situs tertentu dalam daftar dan langsung memuatnya ke aplikasi VPN lokal ini tanpa harus repot mengupload ulang.

Yang praktis diapakai atau leluasa digunakan?

Dari ketiga aplikasi diatas, yang pertama adalah yang paling mudah dipakai. Namun jika diurutkan tingkat keleluasaannya, yang terakhir adalah pilihan terbaik walaupun agak repot dalam penggunaanya.
