--- categories: [arsip] title: "Mengaktifkan USB Debug Mode di Smartphone Android" # date: '2014-08-26T14:02:00.000+07:00' # author: Phate Holloway tags: - Tutorial - Samsung - USB Debugging - Android last_modified_at: '2015-08-17T14:11:47.704+07:00' header: teaser: http://1.bp.blogspot.com/-Oh3IkIKDH-Q/VdGItMXJe_I/AAAAAAAAInw/r-x9Gi0uxGY/w480/USB-Debugging-Option.jpg # blogger_id: tag:blogger.com,1999:blog-8168996750712797250.post-8633677369394040724 redirect_from: - /2014/08/mengaktifkan-usb-debug-mode-android.html --- Pilihan USB Debugging Mode sebenarnya dibutuhkan saat anda ingin menjadi pengembang atau ingin mengumpulkan data data tertentu berkaitan dengan aplikasi dan system android itu sendiri. Dalam banyak kasus, opsi ini sering digunakan saat akan melakukan rooting pada device tersebut. Walaupun sebenarnya USB Debugging Mode tidak melulu untuk keperluan rooting saja.


Secara umum, opsi ini ada di semua ponsel android dalam keadaan tersembunyi. Untuk memunculkannya, biasanya dengan men-aktifkan Menu Developer Option di dalam menu Pengaturan. Lokasi menu ini biasanya tidak berbeda jauh, hanya saja pabrikan seperti Samsung HTC atau Xiaomi mempunyai lokasi pengaturan sendiri-sendiri.

Pilihan USB debugging Mode untuk ponsel keluaran Samsung bisa diakses dengan cara:

1. Buka menu Setting (Pengaturan)
2. Gulung menu kebawah dan pilih menu About Phone (Tentang Ponsel)
3. Cari Build number (Nomor brntukan), tap 7 kali hingga anda mendapatkan notif Developer mode has been enabled!
4. Kembali ke menu Setting dan cari menu Developer Option
5. Aktifkan pilihan USB debugging dengan mencentangnya.
6. pilih OK dalam pop-up menu
