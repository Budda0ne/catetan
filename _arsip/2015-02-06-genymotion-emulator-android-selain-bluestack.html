Yang ada di pikiran anda tentu adalah Bluestack dan YouWave. Bluestack memang sudah dikenal luas sebagai emulator paling sering dipakai dan sedangkan Youwave anda mungkin juga baru dengar karena jarang dibahas. Tetapi, tahukah anda bahwa ada emulator selain Bluestack yang bisa memainkan Game dan aplikasi android?

Genymotion, dengan aplikasi emulator android ini anda juga bisa memainkan game android di PC anda dengan bantuan software yang bernama Genymotion. Emulator android seperti Bluestack dan Youwave, menurut saya terasa cukup berat dan lambat ketika dijalankan. Lain dengan Genymotion yang memiliki performa lebih cepat sehingga nyaman ketika digunakan. Anda bisa meng-install aplikasi seperti BBM, Line, Path dan game-game semacam Aspalt Urban atau Clash of Clan dan memainkannya di PC.

Memainkan COC di PC sambil online

Untuk meng-install software emulator Genymotion, setidaknya komputer/PC atau laptop anda mempunyai 2 GB RAM, 400 MB disk space, CPU yang mendukung VT-X/AMD-v, dan VGA yang mendukung OpenGL 2.0.

Genymotion sendiri membutuhkan Oracle VM VirtualBox dan tentu saja image sistem operasi Android virtual untuk di jalankan. jadi ada membutuhkan 3 bahan yang bisa anda download di bawah.

Cara Install Genymotion dan Menjalankan OS Android di Virtualbox


Setelah anda mendownload ketiga file/bahan yang disebutkan di atas, letakkan di dalam satu folder.
  1. Jalankan Genymotion 2.3.1.exe, dan install seperti biasa
  2. Jika sudah selesai, jangan di-buka dulu Genymotion-nya (close saja dulu)
  3. Jalankan Oracle VM VirtualBox 4.3.exe, install lagi seperti biasa
  4. Kalau instalasi sudah selesai, klik kanan Nexus 4.1.1.tar > Extract Here
  5. Buka folder hasil ekstrak, jalankan template_with_sdcard.ovf
  6. Secara otomatis, Oracle VM VistualBox terbuka, kemudian klik Import dan tunggu sampai proses import selesai
  7. Selanjutnya, pada Oracle VM Virtual Box klik menu: File > Preferences > Network > Host-only Networks, lalu hapus/remove VirtualBox Host-Only Ethernet Adapter (jika tidak ada, lanjutkan saja) > OK
  8. Sekarang buka Genymotion > klik gambar Kunci Pas (Configuration) pada salah satu virtual devices dan ubah resolusinya menjadi 480×800 – 240dpi > OK
  9. Sekarang Start Virtual Devices

Jika Bluestack hanya menyediakan toko aplikasi Mobile1 saja, dengan Genymotion, anda bisa mendownload aplikasi langsung dari Playstore. Pengoperasiannya pun semudah mengoperasikan ponsel android dan bisa di sinkronkan dengan ponsel Android anda punya.
