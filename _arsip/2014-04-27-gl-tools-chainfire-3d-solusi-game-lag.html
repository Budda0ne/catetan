GL Tools vs Chainfire 3D, Pakai yang mana?

Bagi para Android Gamer, untuk mengatasi game yang lag pasti sudah tidak asing lagi dengan GL Tools dan Chainfire 3D. Tapi bagi anda yang baru memegang Smartphone android, mungkin akan terasa asing dan bertanya-tanya, apa sih fungsi dan cara kerja dua tool/aplikasi ini hingga begitu populer di kalangan gamer.

Dua aplikasi ini (GL Tools dan Chainfire 3D) memang efektif meningkatkan performa gaming gadget android yang berfitur pas-pasan. GL Tools dan Chainfire 3D ini memang berfungsi untuk megurangi game lag dengan cara memalsukan informasi hardware gadget android se-olah-olah mempunyai hardware dengan spesifikasi tinggi dan menanamkan driver palsu sehingga game dipaksa untuk mengikuti dan menaikkan resolusi sesuai spesifikasi yang dipalsukan oleh aplikasi GL Tools dan Chainfire 3D.
Meskipun berfungsi sama, GL Tools dan Chainfire 3D mempunyai beberapa perbedaan. Perbedaan yang paling penting dan paling beresiko adalah OS Android yang didukung. Jika GL Tools juga mampu melayani OS Android ICS keatas maka Chainfire 3D hanya mendukung OS Android di bawah ICS. Jika anda mencoba-coba Aplikasi Chainfire 3D di perangkat ICS (misalnya) sudah dapat dipastikan Gadget/Handheld anda akan mengalami bootloop atau yang lebih parah gagal booting (softbrick)

Jadi jika anda mempunyai perangkat dengan OS ICS atau JellyBean atau Kitkat, jangan sekali kali menggunakan Chainfire 3D. Gunakan GL Tools saja untuk mengurangi game lag atau meningkatkan kualitas grafis game 3d favorit anda.

Tutorial Install Chainfire 3D untuk Android


Berbeda dengan GL Tools, Chainfire 3D ini hanya bisa anda install di perangkat android dengan OS versi dibawah ICS. Jika anda memasang Chainfire 3D di OS ICS, JB, atau KK, resiko bootlop dan softbrik menanti anda. Dan tentu saja android anda sudah harus rooted.

Cara install-nya:

1. Download Chainfire 3D dan GPU plugin-nya (ada di bawah)
2. Install Chainfire 3D melalui sdcard/memori internal.
3. Buka chainfire 3D, kemudian install CF3D driver dan reboot handheld anda.

Untuk menunjang performa game, anda juga bisa menambahkan plugin kedalam aplikasi Chainfire 3D yang sudah anda download sebelumnya. Lanjutkan:

4. Extract dan copy plugin libGLEMU_XXXX.zip ke sdcard
5. Reboot handheld anda
6. Setelah menyala kembali, buka aplikasi Chainfire 3D
7. Pilih install plugin, tunggu beberapa saat
8. Pilih libGLEMU_XXXX.zip
9. Pilih Reduce texture quality, Reduce texture size sesuai yg kebutuhkan
10. Back, pilih default openGL setting, pilih use plugin, pilih sesuai keinginan
Setelah ter-instal, Anda pun bisa memainkan game 3d favorit tanpa lag di handheld anda meskipun spesifikasi yang pas-pasan.
11. Back, lalu pilih fix market setting, dan reboot kembali handheld anda.

Meskipun aplikasi ini terbukti ampuh untuk mengurangi lag, tetapi saya pribadi tidak menganjurkan menggunakan apliksi ini.

Tutorial Install GL Tools untuk Android



Perbandingan game menggunakan GL Tools dan tanpa GL Tools

Syarat untuk meng-install GL Tools ini, android anda sudah harus rooted. Dan sebaiknya mempunyai OS ICS, JellyBean atau KitKat.

Cara install-nya:

1. Download GL Tools (ada di bawah)
2. Install aplikasi GL Tools dari sdcard/memori internal
3. Setelah ter-install, buka GL Tools dan centang semua pilihan yang ada.
4. Pilih Install, Gadget anda akan reboot secara otomatis. Jangan panik, Tunggu hingga proses rebot selesai dan handheld anda menyala kembali.

Jika handheld anda sudah menyala kembali, anda bisa membuka aplikasi GL Tools dan memainkan game favorit anda dari aplikasi GL Tools ini. Untuk memperoleh hasil yang memuaskan, anda bisa mencentang semua opsi dalam menu setting dan gunakan GPU Palsu yang anda mau. Anda bisa memilih Tegra 2, Tegra 3, Tegra 4, SGX 540, SGX 544MP, Adreno 320 maupun Mali T640.
