Setelah root kemudian instal alat-alat standar rooted device seperti : busybox, link2sd, dan teminal emulator.

Karena niatnya akan di pasang custom rom, maka saya install CWM Recovery Touch untuk Andromax C juga dan berhasil masuk CWM Recovery. Karena ingin cepat selesai, maka tanpa pikir panjang saya install salah satu Custom ROM saja tanpa rasa bersalah. Setelah selesai (di notifikasi-nya memang tertulis complete) saya reeboot. Ternyata masih berada dalam kondisi yang sama alias tetap stock dan tidak ter-install Custom ROM tadi.

Penasaran, saya 'reboot recovery' via terminal, rebooted tapi kok masuk system lagi? akhirnya saya matikan Andromax C dan saya nyalakan lagi dengan menekan Tombol Power dan Volume Min untuk masuk recovery, sama saja, tetep tidak bisa masuk recovery. Pikir saya, pasti partisi mmcblk0p16 nya yang hilang/terformat saat install custom rom yang tidak sesuai tadi. Install CWM Recovery lagi (karena system masih normal) dan akhirnya bisa masuk mode Recovery.
Perbaikan Bootloop tanpa CWM
Nah disinilah ketololan saya dimulai :). Karena masih penasaran, saya format partisi system, data dan cache. Kemudian saya ulangi install custom rom untuk Andromax C tadi tanpa menyadari bahwa Custom ROM tersebut BUKAN UNTUK ANDROMAX C ICS tetapi UNTUK ANDROMAX C JB. Selesai dan complete (sepeti sebelumnya). Reboot dan tara.... Andromax C pun bootloop karena Custom ROM belum ter-install dan partisi System, Data dan Cahce sudah terformat. Tidak bisa masuk recovery juga karena partisi recovery hilang saat melakukan flashing custom ROM. Nasib... bukannya cepat beres kerjaan malah jadi makin runyam.

Tutorial mengatasi Bootloop tanpa CWM:
  • Ekstrak file .rar
  • Kopikan folder EG902.V1 hasil ekstrak ke /sdcard menjadi /sdcard/EG902.V1/.
  • Dalam kondisi mati tekan dan tahan tombol power, vol up dan vol down secara bersamaan sampai keluar notification.
  • Jika pada tahap ini muncul peringatan failed to open file /mmc1/eg902.v1/conf/eg902.xml with error no -1! rubah file di dalam folder /eg902.v1/conf/ bernama ad686g.xml menjadi eg902.xml
  • Proses TF Update akan berjalan dan tunggu hingga selesai
  • Setalah Complete, tekan tombol power dan tunggu hingga menyala (sekitar 5 menit).
Jika proses diatas lancar, MaxC anda akan kembali ke kondisi seperti baru beli.

Tinggal satu masalah IMEI NULL dan MEID 00000000.
