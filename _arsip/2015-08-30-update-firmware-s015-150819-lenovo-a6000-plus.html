Versi OTA update terbaru adalah Kraft-A6000-s_S015_150819. Namun, bagi pengguna yang ponselnya sudah di root, banyak keadian bootloop atau paling ringan mengalami error update.
Manual Update Firmware S015 150819
Bagi anda yang belum bisa melakukan update secara otomatis via OTA baik karena ponsel A6000 plus yang sudah di root, bisa melakukan Update ke versi firmware terbaru saat ini secara manual dengan syarat downgrade Lenovo A6000 plus ke versi S010 terlebih dahulu. Baru kemudian melakukan update ke versi S015 secara manual dengan mengikuti tutorial berikut:

Manual Update Firmware Lenovo A6000 Kraft-A6000-s_S015_150819

Dengan mengikuti tutorial yang saya berikan berarti anda setuju dengan Disclaimer dan patuh dengan aturan DMCA di KnoAcc Blog
Seperti yang sudah saya bahas sebelumnya, lenovo A6000plus anda harus sudah dalam keadaan downgrade. Anda bisa mendowngrade ke Kraft-A6000-s_S010_150515 dengan menggunakan tutorial fix bootloop A6000plus, caranya sama. Gunakan file backup atau flashable zip terserah anda.

Setelah anda melakukan downgrade dan versi fiwmware sudah Kraft-A6000-s_S010_150515 anda bisa malanjutkan dengan mengikuti langkah berikut:

  1. Download file update to Kraft-A6000-s_S015_150819 di bagian download, letakkan di sdcard (internal/external)
  2. Install terlebih dahulu TWRP Recovery (karena saat proses downgrade anda menggunakan CWM Recovery). Caranya bisa anda baca di tutorial install custom recovery.
  3. Setelah TWRP Recovery ter-install, Masuk ke mode recovery dengan menekan 3 tombol secara bersamaan yakni tombol volume up, volume down dan power.
  4. Dalam mode recovery, pilih install zip dan pilih file yang sudah anda download tadi.
  5. Swipe (geser) kekanan seperti yang di instruksikan.
  6. Tunggu hingga proses selesai.
  7. Saat ditanyakan apakah ingin melakuka root dengan supersu pilih don't install.
  8. Ponsel akan reboot otomatis.

Proses booting up memakan waktu sekitar 5-10 menit paling lama. Semoga berhasil.
