Cara Root Mito A77 Fantasy Selfie KitKat Tanpa PC
Beberapa user yang belum melakukan rooting memang bisa melakukan upgrade ke KitKat. Ternyata, setelah upgrade Kitkat, proses root Mito A77 menjadi agak susah. Tidak seperti proses rooting sebelumnya yang mudah dilakukan. Tetapi jangan kuatir, masih ada kok cara rooting Mito A77 yang mudah dan tanpa bantuan PC.

Perhatian: baca disclaimer sebelum mengikuti tutorial. Dan yang ingin sharing ulang baca tulisan ini.

Langkah Root Mito A77 Fantasy Selfie KitKat

  1. Buka menu Pengaturan -> Pengaturan Keamanan dan scroll kebawah sampai anda melihat Sumber tak dikenal dan Centang/Contreng.
  2. Download dulu Aplikasi rooting di bawah
  3. Sesudah mendownload Root Genius segera install Root Genius, jika muncul peringatan membahayakan abaikan dan Install saja.
  4. Setelah selesai, tutup dulu.
  5. Aktifkan USB Debugging
  6. Buka aplikasinya
  7. Pilih Tulisan Sebelah Kanan
  8. Tekan menu Root
  9. Kalau berhasil, Handheld akan melakukan reboot sendiri.
Untuk mengecek apakah Mito A77 Fantasy Selfie sudah benar-benar root atau belum, gunakan aplikasi Root Checker yang bisa anda download di playstore.

Untuk berjaga-jaga jika anda melakukan kesalahan dan mengalami bootloop atau kerusakan system yang lain, anda sebaiknya mempunyai Custom Recovery yang bisa anda gunakan untuk membackup ROM Mito A77 Fantasy Selfie anda. Baca tutorial install TWRP Recovery untuk Mito A77 Fantasy Selfie.

Jika anda mengalami bootloop sebelum meng-instal Custom Recovery, anda bisa menggunakan Cara Upgrade Firmware Mto A77 sebagai solusi.
