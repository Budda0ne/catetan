Ini pula yang membuat Firmware keduanya bisa di pasang-silang-kan. Keduanya juga susah mendapatkan update Lollipop secara official melalui OTA Update. Meskipun demikian, Stock ROM yang FULL INSTALL juga ikut bergentayangan dengan cantiknya. Link download tersebut juga official dari server Lenovo. Karena itulah, sesuai dengan judul di atas, artikel ini akan membahas bagaimana cara men-downgrade ke OS Kitkat, baik Lenovo A6000 maupun Lenovo A6000 plus. Artikel ini juga saya tulis karena permintaan dari beberapa pengunjung blog yang merasa tidak nyaman dengan OS Lollipop. Sekaligus juga bisa dijadikan sebagai cara memperbaiki bootloop.

Untuk Downgrade menggunakan PC akan di gabung dengan Tutorial Unbrick Lenovo A6000 dan A6000 plus

Meskipun sebenarnya telat sekali, tetapi tidak masalah karena dalam tulisan cara downgrade ini akan saya coba tuliskan alternatif-alternatif yang bisa digunakan dan sebagai lokasi diskusi baru untuk permasalahan downgrade ke kitkat karena komentar di halaman fix bootloop sudah terlalu panjang. File Stock ROM yang digunakan dalam tutorial menggunakan versi S054 untuk A6000 polos dan S017 untuk A6000 plus. Berbeda dengan postingan sebelumnya yang menggunakan S032 (official) dan S010 (hasil back-up) yang sangat repot bin ribet.

Persiapan Untuk Downgrade Kitkat Lenovo A6000 dan A6000 plus

Dengan asumsi bahwa ponsel Lenovo anda dalam keadaan:
1. Normal dan tidak dalam keadaan bootloop
2. Bootloop tetapi sudah ter-install TWRP Recovery
Maka anda bisa mempersiapkan file berikut:
1. Install RAR for android atau 7Zip.
3. Install TWRP Recovery baca tutorial khusus untuk Instalasi TWRP Recovery di halaman ini.
4. Download Stock ROM sesuai tipe.
5. Setidaknya di SdCard anda masih ada ruang 4-5 GB. Jika hanya mempuyai internal storage, kosongkan!

Melakukan Downgrade Kitkat Lenovo A6000 dan A6000 plus

Dengan mengikuti tutorial flashing yang saya berikan berarti anda sudah setuju dengan Disclaimer dan harus memahami copyright notice di KnoAcc Blog jika ingin membagikan tutorial ini. Jika tidak setuju silahkan minggat dari sini.

Untuk melakukan downgrade, pastikan batterai sudah terisi karena (mungkin) akan memakan waktu yang lama.

1. Cari file zip Stock ROM yang sudah anda download dan buka dengan aplikasi RAR atau 7Zip

2. Ekstrak hanya file System.zip dan IndonesiaOTA.zip (atau sesuai Negara/Carrier anda. mis: India.zip atau Malaysia.zip) di lokasi yang sama. Selain itu buang!

3. Hapus juga file Stock ROM (Kraft-A6000-xxx.zip) tetapi jangan hapus system.zip dan indonesia.zip yang sudah di-ekstrak sebelumnya.

4. Buka file System.zip dan ekstrak file boot.img (nantinya akan berguna)

5. Dari sini silahkan boot ke mode recovery (TWRP sevenmaxx). Caranya dari kondisi mati, bisa dengan menekan tombol vol down + vol up + power atau vol up + power kemudian memilih menu recovery.

6. Lakukan back-up dan centang semua opsi (daro aboot sampai OEM) kecuali System, Cache dan Data. Tunggu hingga selesai.

7. Lakukan Wipe dengan memilih advanced wipe kemudian pilih opsi Dalvik Cache, System, Cache dan Data.
8. Kembali ke menu utama dan pilih Install dan pilih System.zip jangan lakukan flashing dulu.
9. Lihat tanda [+] dalam linkaran abu-abu? Tekan dan pilih lagi file Indonesia.zip (atau negara anda)
10. Ok, sekarang geser ke kanan bulatan biru sepanjang garis untuk melakukan flashing.

TWRP akan melkukan proses instalasi, silahkan santai sejenak untuk menunggu proses instalasi yang lumayan lama.

Jika Proses sudah seselai silahkan reboot Lenovo A6000 atau Lenovo A6000 plus anda. Booting pertama bisa memakan waktu hingga 15 menit paling lama.

Troubleshoot:
1. Jika berhenti karena pesan error code bla-bla-bla, ulangi dari langkah ke-7 dilanjutkan dengan langkah berikut:
- Pilih menu install di menu utama.
- Tekan menu Install Zip hingga muncul drop menu, pilih Install Image.
- Pilih menu boot untuk dan cari file boot.img yang anda ekstrak dilangkah ke-4
- Lanjutkan dengan memilih menu Reboot dan pilih menu Recovery.
Selanjutnya tinggal melanjutkan langkah ke-8 hingga akhir.

2. Jika berhenti lama (lebih dari 15 menit) di langkah ke-10, copot baterai dan diamkan selama 5 menit atau lebih. Ulangi lagi dari langkah ke-7.

3. Jika proses boot pertama sudah melebihi 15 menit dan masih belum masuk system (ditandai dengan munculnya pop-up Android is upgrading) copot baterai dan diamkan sekitar 5 menit atau lebih dan nyalakan kembali (jangan lupa pasang lagi baterainya).
