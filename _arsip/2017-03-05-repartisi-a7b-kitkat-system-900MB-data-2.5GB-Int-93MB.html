Spesifikasi pembagian partisi memori (ROM) untuk Data yang dipakai untuk melakukan instalasi aplikasi kadang terasa kurang besar. Apalagi jika anda gemar meng-koleksi game game besar semacam PES 2017, Need for Speed, FIFA atau game buatan Gameloft.

Untuk itu muncul ide untuk memperbasar ruang Data dengan mengurangi ruang InsternalSd (memori insternal/sdcard internal) sehingga menjadi Data: 2,5. GB dan Penyimpanan internal: 93 MB. Meskipun sebenarnya ide repartisi sudah ada sejak lama (baca disini). Namun cara repartisi sekarang sudah jauh lebih mudaj dengan hanya memanfaatkan flashable zip dan custom recovery.

Caranya:

Download dulu file untuk repartisi ROM Evercoss A7B Kitkat

Download Zip
Anda HARUS sudah memasang Custom Recovery, entah itu CWM atau TWRP atau yang lainnya.

Dan lakukan langkah berikut setelah masuk Recovery mode:

1. Sebaiknya lakukan full-backup rom asli dengan TWRP/CWM. Pindahkan semua file yang ada di dalam Internal memori (internal sd) mengingat semua data di dalamnya akan HILANG!
2. Format data, system, chache. Dengan langkah ini Custom ROM yang saat ini anda pakai akan HILANG juga. Jadi siapkan juga ROM di SD card untuk re-instalasi ROM.
3. Flash File Repartisi yang tadi anda download.
4. Setelah proses complete, biasanya akan otomatis reboot ke recovery lagi.
5. Lakukan lagi format System, Data dan Cache.
6. Rebot lagi ke mode Recovery.
7. Restore hasil backup partisi. Jika ini dilakukan, kemungkinan akan terjadi sedikit error. Jika berhasil, partisi akan berubah. Namun terkadang partisi akan kembali seperti semula jika melakukan full-restore.
8. Jika gagal, ulangi langkah sebelumnya dan di langkah ke 7 HANYA restore Data saja.

Langkah terakhir untuk finishing jika anda melakukan restore adalah melakukan instalasi ulang ROM. Flash Custom ROM yang sudah anda siapkan sebelumnya.

Selesai. Kini Evercoss A7B kitkat anda sudah mempunayi partisi data lebih besar. Dengan partisi Data 2,5 GB, anda tidak perlu kuatir kehabisan ruang (pesan kesalahan Penyimpanan Todak Mencukupi) saat melakukan instalasi Game atau Aplikasi Gajah dari Playstore.
