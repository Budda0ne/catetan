Kebetulan sekali karena akhir-akhir ini jarang ada melakukan root device asus zenfone. Tipe Asus Zenfone Laser sendiri ada beberapa varian diantaranya ZE550KL (Z00L), ZE550KG (Z00RD) dan ZE500KL (Z00E). Cara root ASUS Zenfone 2 Laser berikut ini bisa digunakan untuk Asus Zenfone 2 Laser untuk ketiga seri tersebut diatas. Pastikan terlebih dulu varian Zenfone 2 Laser anda di kolom komentar jika anda tidak yakin. Caranya di bcek agian Setting > About phone.

Seperti sudah anda ketahui, Asus Zenfone 2 Laser merupakan salah satu smartphone Asus yang dibekali dengan kamera terbaik. Asus Zenfone 2 Laser memiliki fitur Laser Autofocus yang berguna untuk menentukan fokus obyek dengan cepat.

Untuk tutorial rooting Zenfone 2 saat ini, device yang jadi korban adalah tipe ZE550KG (Z00RD) milik kawan saya.

Ingat!, proses root selalu beresiko. Jika anda mengikuti Tutorial Root Zenfone 2 Laser ini berarti anda sudah membaca dan setuju dengan disclaimer.

Persiapan Root Asus Zenfone 2 Laser


Sebelum melakukan Root Asus Zenfone 2 Laser, siapkan terlenih dulu beberapa hal berikut karena proses root membutuhkan PC.

PC atau Laptop dengan OS Windows 7 (rekomendasi) karena saat instalasi
driver tidak perlu disable verivy signature seperti Windows 8 keatas.

Baterai Asus Zenfone 2 Laser tersisa di atas 60% agar tidak kehabisan daya saat ditengah proses root.

Dan kabel USB untuk menghubungkan ke Zenfone 2 Laser ke PC, usahakan kabel berkualitas baik.

Download ADB Tool dan install di komputer Anda. Versi ADB Tool Terbaru saat ini adalah Versi 1.4.2

Download juga image partisi boot yang sudah di modifikasi (mod_boot.img) letakkan di folder yang mudah ditemukan.

mood_boot.img.zip

ADB Tool Ver. 1.4.2
Jika bahan-bahan sudah siap, aktifkan USB Debugging yang ada pada menu Developer Options. Jika menu Developer Options belum tersedia, cara memunculkannya dengan membuka menu Setting > About > dan tekan Build Number sebanyak 7x hingga muncul pemberitahuan "You are now a developer!"
Setelah itu, masuk ke Setting > Developer Options > centang opsi USB Debugging.

Step by step Rooting Zenfone 2 Laser.


Lanjutkan dengan langkah berikut untuk melakukan root:
Matikan Zenfone 2 Laser anda dan tunggu sekitar 5 detik.
Boot Zenfone 2 Laser ke mode fastboot, denga cara tekan dan tahan tombol Power + Volume Up.
Hubungkan Zenfone 2 Laser ke PC melalui kabel USB.
Buka folder ADB tool 1.4.2 yang di dalamnya sudah ada file mod_boot.img.
Tekan tombol Shift + Klik Kanan lalu pilih Open Command Window Here di dalam folder tersebut.
Cek duli apakah Zenfone 2 Laser sudah terdeteksi atau belum dengan perintah:

adb devices

Jika terdeteksi, akan muncul serial number di CMD

Lanjutkan mem-booting dengan partisi boot modifikasi dengan mengetik perintah:

fastboot boot mod_boot.img

Tunggu hingga proses boot selesai, Zenfone 2 Laser akan reboot secara otomatis.
