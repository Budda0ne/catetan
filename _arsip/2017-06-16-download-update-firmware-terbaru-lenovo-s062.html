Ini mungkin posting pertama saya setelah hape Lenovo saya hilang tak tentu rimbanya.
Jujur saja, saya sendiri tidak bisa menguji cobanya secara langsung firmware A6000_L_S062_161223_8G_ROW ini. Apakah bagus atau tidak, performanya bagus atau baterai boros tau malah lebih hemat, silahkan dicoba sendiri. Tetapi karena masih banyak yang menanyakan update firmware terbaru untuk Lenovo A6000 (yang artinya bisa dipakai untuk Lenovo A6000 Plus), maka saya bagikan link yang saya dapatkan dari forum Lenovo Rusia.

Apakah Firmware (stock rom) S062 ini berbahasa Indonesia karena didapat dari forum Rusia? Saya berani jamin ada bahasa Indonesia karena ini versi ROW.

Jadi silahkan gunakan file Full ROM ini jika anda merasa perlu untuk melakukan update firmware Lenovo A6000 anda. Untuk mengingatkan saja, anda sebaiknya mendownload stock ROM ini jika koneksi data anda lancar karena file-nya besar. Silahkan download dari link yang disediakan dibawah:
File name: A6000_L_S062_161223_8G_ROW.7z
Android Version: 5.0.2
File size: 1.01 GB
Password: lenovo-forums.ru
download

Untuk melakukan update, anda bisa menggunakan QFIL QPST dengan mengikuti cara yang ada di tutorial flash stock ROM Lenovo A6010 karena caranya mirip. Setelah itu, terserah apakah ingin memasang TWRP atau tidak. Jika anda ingin memasang TWRP, gunakan tutorial di artikel ini.

Meskipun Lollipop lebih aman, tetapi seingat saya, sebagian besar pengguna Lenovo A6000 lebih menyukai firmware Kitkat daripada firmware lollipop. Sedangkan saya (dulu) lebih menikmati stock ROM Lollipop dengan alasan lebih aman.

Terakhir kali saya memasang ROM MIUI Lollipop sebelum ponsel saya hilang. Jika anda juga suka dengan ROM MIUI, tetapi tidak suka Lollipop, silahkan pakai ROM MIUI 8 Kitkat.
