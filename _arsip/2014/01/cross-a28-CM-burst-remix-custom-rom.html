Panduan instalasi CM BURST Remix

Dengan Mengikuti Tutorial ini anda setuju dengan Disclaimer

Karena ada beberapa yang berbeda dalam segi konfigurasi system-nya, maka diperlukan beberapa penyesuaian jika memakai Custom ROM Cherry Mobile Burst Remix.

Agar bisa berfungsi dengan baik, anda perlu mengganti file boot.img dalam file flashable .zip recovery Cherry Mobile Burst Remix yang anda download dengan file boot.img dari ROM Cross A28 asli hasil back-up CWM Recovery.

Jika tidak punya boot.img, download di sini: via Mediafire.

Setelah boot.img diganti, maka anda bisa melanjutkan sesuai panduan instalasi berikutnya:

  1. Download ROM dan tenpatkan di SDcard (jangan di ekstrak).
  2. Matikan Handheld ke mode CWM Recovery (jika belum punya, silahkan install CWM Recovery terlebih dulu).
  3. Pilih Wipe Data, Cache dan Advanced -> Dalvik-Cache. Jika ada menu lagi pilih yes.
  4. Pilih Install zip from SD Card.
  5. Cari file .zip di SDCrad dan pilih. Jika ada menu lagi pilih yes.
  6. Tunggu hingga selesai.
  7. Reboot.

Untuk fix google maps A-GPS, gunakan file: agps.patch.google.build.v3.1.zip.
Untuk fix kamera 2MP, Camera 3MP fix:: camera 3MP a28.zip.
