Namun, ada cara yang lebih mudah untuk melakukan rooting pada Lenovo A6000 tanpa bantuan PC, alias langsung dari Handheld (HH). Tool yang digunakan adalah Kingroot versi 4.0.0, yang sudah bisa root Android Lollipop dan Kitkat.

Tutorial Rooting Lenovo A6000 & A6000+

Dalam proses rooting ini, anda membutuhkan jaringan internet yang lancar, karena saat proses rooting Kingroot juga mendownload aplikasi Kinguser sebagai pengganti Aplikasi SuperSU sebagai manager hak akses root. Sebagai gambaran untuk anda, saat anda melakukan root dengan framarot, framaroot juga meng-install Aplikasi SuperSU setelah selesai melakukan root.

Ikuti langkah-langkah berikut untuk melakukan rooting Lenovo A6000 atau Lenovo A6000+

  1. Unblock Security agar bisa meng-install aplikasi dari sdcard dengan cara: Buka menu Settings (setelan) dan pilih Security (Keamanan) dan aktifkan pilihan Unknown source (sumber tak dikenal).
  2. Aktifkan USB Debugging (opsional), caranya: Buka menu Settings (Setelan) dan pilih About Phone, pilih About Phone (Tentang Ponsel) dan lanjut Device Info (Informasi Perangkat), ketuk 7x di Build number (Nomor Bentukan) hingga muncul You are now developer (Anda sudah menjadi pengembang).
  3. Pilih menu back hingga di bagina Settings (setelan) pilih menu Developer option (opsi pengembang) yang baru muncul dan aktifkan, centang juga pilihan USB Debugging.
  4. Download Aplikasi Kingroot 4.0.0 (hanya versi ini yang bisa) lihat download section.
  5. Setelah terdownload, install Kingroot. Saat instalasi akan muncul peringatan. centang (I understand bla.. bla.. ) Saya mengerti dst.. dan pilih Install (tetap pasang).
  6. Setelah terinstall, Lansung buka aplikasi dengan memilih Open (buka) Anda akan melihat gembok dalam lingkaran dan dibawahnya ada kotak biru dengan teks Root. Klik saja di tombol tersebut.
  7. Proses root akan berjalan ditandai animasi prosentase proses root di lingkaran.

Jika sudah 100% berarti sudah berhasil root. Untuk mengecek buka drawer aplikasi dan lihat disamping app. Kingroot (ikon petir) ada app. Kinguser (ikon mahkota), sebagai pengganti SuperSU.

Selesai sudah proses rooting Lenovo A6000 dengan lancar, mudah dan tidak ribet meminjam PC/Laptop atau main ke warnet.

Sebagai tambahan saja, saat anda melakukan OTA Update, sebaiknya aktifkan pilihan upgrade versi binary su secara otomatis.
Caranya: buka Kinguser dan pilih menu settings (ikon gear pojok kanan atas) pilih Root authorization setting dan aktifkan Keep the root upgraded
