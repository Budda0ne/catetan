Sehingga jika terjadi bootloop ataupun hal-hal yang berakibat tidak dapat masuk ke system anda tidak bia melakukan apa-apa. Jalan terakhir adalah membawanya ke Gallery smartfren terdekat. Inilah kenapa Anda perlu meng-install CWM Recovery di Smartfreen Andromax C untuk yang versi JellyBean.


Dengan Mengikuti Tutorial ini anda setuju dengan Disclaimer

Sebelumnya, Anda tentu sudah membaca Cara Simple Rooting Andromax C [Jelly Bean]. Jika iya, maka anda sudah bisa melakukan Instalasi CWM untuk Andromax C JellyBean, karena proses ini membutuhkan hak akses root.

Instalasi CWM Recovery kali ini untuk Smartfreen Andromax C Jelly Bean, jadi jika Andromax C anda masih versi ICS anda bisa mengikuti tutorial untuk Andromax C ICS.

Untuk meng-install CWM Recovery, ada beberapa file yang harus anda persiapkan, link download ada di bawah:
  1. Terminal Emulator, download file-nya dari playstore dan install jika anda belum punya.
  2. Recovery.img, bisa anda dapatkan dari GDrive. dan letakkan file Recovery.img ke SDcard tapi jangan di dalam folder ( seperti ini /SDcard/Recovery.img )

25 Januari 2014 Menambahkan link mirror Mediafire: [Mediafire]

Setelah semua siap, Lanjutkan sesuai panduan Install CWM berikut:

Jalankan terminal emulator, ketik perintah dibawah dan tekan enter. Allow jika muncul notifikasi:
su
Masukkan lagi perintah berikut dan tekan enter:
dd if=/sdcard/recovery.img of=/dev/block/mmcblk0p13 bs=1024
Perhatikan, setelah dd sebelum if ada spasi, sesudah img sebelum of juga ada spasi, kemudian tekan enter. Proses sinkronasi antara file recovery.img dan partisi mmcblk0p13 sedang dilakukan
Setelah proses selesai masukkan lagi perintah berikut, dan tekan Enter :
reboot recovery

Otomatis Smartfreen Andromax C anda akan melakukan reboot dan masuk ke CWM recovery mode. Itu berarti anda sudah sukses memasang CWM recovery di Andromax C versi JellyBean. Selamat.

Catatan: Untuk versi ICS ikuti Tutorial Rooting + Install CWM Andromax C versi ICS.
