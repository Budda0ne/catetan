Dengan semakin banyak orang mengandalkan internet untuk membeli barang kebutuhan, ada peningkatan drastis dalam usaha toko online. Dengan nama-nama besar seperti Amazon dan eBay sebagai pelopor, Tokobagus.com dan Berniaga.com di Indonesia. Internet pun berkembang menjadi sebuah ajang untuk ber-wirausaha dengan prospek yang menjanjikan. Apalagi dengan adanya WordPress dan berbagai plugin yang tersedia, Anda bisa membuat sebuah situs toko online dengan cukup mudah.

Tanpa perlu basa basi, berikut ini saya sajikan 7 Plug-in Wordpress Gratis Terbaik untuk Toko Online.
1. Cart66 lite

Cart66 lite merupakan plugin toko online untuk blog WordPress. Tidak seperti plugin toko online lainnya, sangat ringan namun kuat. Anda dapat dengan mudah menjual digital serta produk fisik dengan itu. Dengan beberapa pengiriman dan pilihan mata uang, juga dapat membantu Anda dapat memberikan bisnis Anda keunggulan internasional. Anda dapat dengan mudah menempatkan iklan di halaman manapun dan juga menggunakan pemasaran afiliasi untuk keuntungan lebih.


Download plugin | Visit Cart66 lite website

2. eShop

eShop adalah plugin WordPress mudah diakses. Ini memiliki kontrol sederhana dan menyediakan berbagai metode untuk daftar produk. Menambah produk dan pengaturan menyesuaikan mudah pada eshop. Juga memiliki antarmuka yang sederhana untuk browsing cepat dan pembelian. Ini juga menyediakan beberapa gateway pembayaran merchant, statistik untuk pembelian, beberapa pilihan pengiriman (termasuk berat), dll


Download plugin | Visit eShop website

3. Jigoshop

Jigoshop menyediakan banyak fitur yang sangat penting untuk menjadi tuan rumah sebuah situs web toko online. Ini akan memakan waktu hanya beberapa menit untuk membuat toko penuh online Anda dengan berbagai produk dan layanan Jigoshop. Anda dapat mengelompokkan produk dan menyediakan berbagai atribut untuk produk Anda, (seperti S, M, L ukuran untuk pakaian) untuk membantu pengunjung Anda dengan mudah memperbaiki item sesuai kebutuhan mereka.


Download plugin | Visit Jigoshop website

4. Woocommerce Plugin

Plugin Woocommerce berasal dari pengembang terkenal WooThemes. Plugin Woocommerce ini memberikan fasilitas toko online yang kuat dan elegan. Woocommerce menyediakan banyak fitur termasuk widget dashboard sehingga Anda dapat dengan mudah memantau laju bisnis online Anda. Hal ini sangat mudah di atur dan dilengkapi dengan beberapa pilihan pembayaran termasuk Paypal, Cek, Cash on delivery dan banyak pilihan lainnya. Anda juga dapat menambahkan beberapa ekstensi plugin dari berbagai ekstensi Woocommerce.


Download plugin | Visit Woocommerce website


5. WP E-Commerce Plugin

Dengan lebih dari 1,5 juta download, WP e-commerce plugin toko-online WordPress yang paling populer sekarang. Sangat mudah untuk di-instal dan cocok digunakan dengan theme WordPress apa saja. WP E-Commerce menyediakan berbagai pilihan pembayaran termasuk Manual (pemeriksaan), PayPal dll. WP E-Commerce mudah di kustomasi dan Anda juga bisa dengan mudah mengubah tampilan dan memodifikasi template sesuai selera Anda.


Download plugin | Visit WP e-Commerce website


6. Quick Shop Plugin

Quick Shop menambahkan sidebar yang menampilkan keranjang belanja pengunjung di seluruh halaman. Quick Shop secara built-in mendukung Paypal dan layanan Email. Quick Shop juga menyediakan pilihan untuk menambahkan produk ke halaman Anda dengan tombol TinyMCE. Pengunjung juga dapat menghapus item tertentu dari keranjang belanja mereka.


Download plugin | Visit Quick Shop website

7. YAK Shopping Cart Plugin

YAK Shopping Cart Plugin meng-asosiasi produk Anda di bagian entri weblog dan karenanya pasca ID Anda menjadi kode produk Anda. Anda dapat dengan mudah meng-kategori-kan produk dan mengkonfigurasi pengaturan untuk keranjang belanja. YAK Shopping Cart menyediakan pilihan pembayaran kustom seperti cek, kartu kredit, paypal dan berbagai gateway pembayaran lainnya. Anda juga bisa menentukan tarif pengiriman berdasarkan berat atau flat rate.


Download plugin | Visit YAK website
