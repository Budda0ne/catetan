MOBA hingga saat ini terhitung genre game yang masih sangat diminati di Indonesia. Game MOBA yang dulu lebih dikenal sebagai game PC. Tetapi, dengan makin bertumbuhnya teknologi perangkat mobile, kini game MOBA versi mobile malah lebih digemari sehingga genre MOBA pun makin populer.

Sedangkan game MOBA paling terkenal di Indonesia untuk saat ini masih Mobile Legends. Terpengaruh saking menjamurnya pemain game bergenre MOBA, belum lama ini developer asal Tiongkok NetEase pun meluncurkan game MOBA terbaru bernama Onmyoji Arena. Game ini sendiri merupakan adaptasi game seri Onmyoji buatannya dan mengubahnya menjadi genre MOBA.

Game yang kini sudah bisa dipasang melalui Playstore di Indonesia ini, juga biasa disebut sebagai game MOBA Jejepangan atau game MOBA Setan Jepang. Tema permainan di game ini cukup unik, dimana tema budaya Jepang dengan dewa-dewa dan mitologi lainnya dijadikan sebagai sosok hero. ika umumnya sebutan untuk tokoh karakter yang ada di game MOBA adalah Hero, berbeda dengan Onmyoji mereka menyebutnya Shikigami.

Karena game ini bertemakan Jejepangan jadi Shikigami ini berbicara menggunakan bahasa Jepang, berlaku juga dengan suara announcer yang biasa terdengar saat membunuh hero lawan.

Dilihat dari segi grafis, yang ada di game ini lebih mantap jika dibandingkan dengan MOBALOG sebelah. Jika sudah memainkan Onmyouji Arena, kamu bisa lupa sedang bermain game mobile. Ini karena pergerakan di game ini cukup halus dan memberikan pengalaman bermain game MOBA (bisa dibilang) terbaik di platform mobile. Kamu bisa menemukan game Onmyouji Arena via Playstore

Game wibu ini sudah mengusung konsep permainan 5v5 di map yang terdiri dari 3 lane. Dengan tambahan 2 jenis waktu yang bisa berganti siang dan malam. Setiap karakter memiliki 4 kemampuan unik dan 1 passive. Untuk lebih jelasnya, berikut ini beberapa fitur yang bisa diakses dari menu yang ada di MOBA Onmyoji Arena:

1. Shikigami: Merupakan daftar lengkap karakter permainan atau hero yang disebut sebagai Shikigami pada game Onmyoji Arena kamu.

2. Prepare: Untuk menyiapkan apa saja yang akan dibawa selama melakukan match, seperti memilih Shikigami dan skill apa saja yang akan dipakai dan lainnya ada di sini.

3. Guild: Memiliki fungsi yang sama seperti game pada umumnya, tempat berkumpul, membentuk tim dan lain-lainnya.

4. Shop: Di tempat ini, kamu bisa melakukan transaksi in-game untuk mendapatkan barang tertentu mulai item biasa hingga item langka

5. Package: Di sini merupakan tempat penyimpanan semua item yang berhasil kalian kumpulkan atau dapatkan dari permainan.

6. Quest: Yakni misi harian yang bisa kamu penuhi agar mendapatkan hadiah maupun imbalan item tertentu.

7. Settings: Di sini terdapat beerbagai macam pengaturan yang lengkap termasuk pilihan grafis dengan Mode 60 FPS yang tersedia untuk semua jenis smartphone

Bagaimana sobat MOBALOG? Apa kamu tertarik mencoba atau malah sudah pernah coba? Game MOBA Onmyoji Arena ini bisa jadikan pengganti disaat kamu mulai bosan bermain Mobile Legends. Bisa jadi Kamu malah berhenti main Mobelejen karena alur permainannya lebih fair. Ganjalannya cuma satu: kalau game sebelah sudah lekat dengan istilah "moba kok analog", kalau kamu main moba jepangan ini bakalan sering dengar "moba kok wibu".
