Dan Karena sudah kadung jatuh cinta, kembali saya publish untuk versi downgrade-nya: MIUI 7 Rusia KitKat 6.3.17. Versi KitKat ini dirilis jauh sebelum munculnya versi terakhir MIUI 7 Rusia yang memakai Lollipop sebagai basis OS.

Kenapa kok seperti re-route/balik kucing dan malah posting versi downgrade-nya? Alasannya, beberapa user Lenovo A6000 Gagal move on lebih menyukai OS KitKat 4.4.4 sebagai base karena lebih light dan smooth daripada Lollipop. Jadi postingan ini sebagai alternatif saja, agar yang fanatik dengan KitKat juga bisa merasakan MIUI 7 Rusia Kitkat.

Lalu apakah fiturnya sama dengan dengan yang Lollipop based? Sepertinya tidak. Saat saya menulis belum install versi KitKat ini, jadi saya belum bisa memberi review dan pendapat untuk versi KitKat ini.

Namun, secara umum performance dari MIUI Russia KitKat version (kalau mengacu stock rom Lenovo A6000 KitKat) bisa jadi lebih ajib dari yang versi Lollipop. Oke, silahkan kasih pendapat anda di kolom komentar setelah memakai ROM ini. Nanti saya update review jika saya sempat untuk memasang di Lenovo A6000 saya.

Untuk step instalasinya masih sama saja dengan artikel sebelumnya. Rekomendasi pakai TWRP.

GAPPS memang sudah disertakan dalam ROM installer, namun jika anda mengalami masalah anda bisa menggunakan OpenGapps KitKat 4.4.x pico atau nano. Bisa juga memakai GAPPS dari basketbuild entah full version maupun versi minimal GAPPS.

Untuk penggantian GAPPS silahkan lakukan setelah Lenovo A6000 anda selesai boot ke system. Begitu juga jika ingin meng-install add-on lain semisal Dolby Atmos atau Custom Kernel. Ini untuk mencegah kesalahan instalasi dan tumpang tindih eksekusi script.
