Sebenarnya, pemasangan Framework pada Android 6.0 hampir sama dengan instalasi Xposed Framework Lollipop, caranya sedikit lebih ribet karena anda harus mengunduh framework dan aplikasi Xposed secara terpisah. Seperti yang sudah pernah saya bahas sebelumnya, anda sedikitnya mengerti SDK dan Architecture yang digunakan OS Android yang terpasang di device anda saat ini.

SDK atau API Level pengembangan Android Marshmallow (saat ini) Menggunakan API Level 23 (SDK 23). Sedangkan untuk Arsitektur Android yang dipakai adalah 32 bit untuk device dengan chipset ARMv7 dan 64 bit dengan chipset ARMv8 (Quallcomm maupun Mediatek). Meskipun beberapa chipset dengan ARMv7 juga menggunakan arsitektur 64 bit.

Sedangkan untuk chipset Intel biasanya disebut sebagai x86. Chipset Intel atau X86 yang digunakan di smartphone/tablet (misal: Asus Zenfone) umumnya menggunakan arsitektur 32 bit. Sedangkan arsitektur 64 (untuk saat ini) hanya dipakai di device seperti Android TV.
Jadi yang penting anda tahu SDK android anda dan tahu arsitektur yang dipakai ponsel anda saat ini. Coba baca : Cara Mudah Menenrukan SDK dan Arsitektur OS Android

Tambahan: untuk device baru buatan Samsung rata-rata mulai menggunakan cipset dengan arsitektur 64 bit ARMv8.

Perlu diperhatikan juga: ROM stock (Touchwiz UI) sepertinya tidak semua device didukung terutama untuk device baru tersebut, dan beresiko terjadi bootloop. Meskipun begitu, seperti halnya Xposed spesial untuk ROM MIUI, saat ini sudah tersedia Xposed Framework untuk Samsung dengan Stock ROM (Touchwiz UI) baik untuk yang odexed maupun deodexed. Meskipun tidak semua device. Lebih jelasnya akan saya bahas di artikel lain.

Cara Install Xposed Framework, Step by step.


Syarat agar bisa memasang Xposed tentu saja harus sudah root dan memiliki custom recovery. Jangan lupa melakukan nandroid backup dengan custom recovery ya. Anda sudah saya ingatkan.

Oke, anggap saja anda sudah memenuhi syarat diatas dan sudah paham tahu SDK dan Arsitektur yang dipakai oleh ponsel android anda. Sebagai contoh: Karena anda memakai Android Marshmallow 6.0 dengan arsitektur 64 bit berarti OS Android anda menggunakan SDK23 dan ARM64. Maka anda tinggal men-download file Xposed Framework SDK23 ARM64 di link yang diberikan dibawah ini:


Xposed Framework Marshmallow terbaru saat ini sudah versi 86.

Penting: Sebaiknya sebelum melakukan instalasi, anda melakukan Full Nandroid Back-up partisi system mengingat resiko bootloop.

Jika and a sudah mendownload Framework yang sesuai, letakkan file tersebut di sdcard. Matikan ponsel dan reboot ke menu recovery.

Cara masuk recovery mode itu variatif , tergantung model ponsel anda. Atau gunakan tool reboot recovery untuk mempermudah.

Dalam mode recovery, pilih install update dan pilih install zip from sdcard (CWM) atau Install dan pilih Install zip (TWRP) dan arahkan ke lokasi file *.zip

Jika sudah selesai, pilih menu reboot to system dari recovery mode.

Jika sudah masuk homescreen, install Aplikasi Xposed Framework 3.0 alpha 4 agar anda bisa mengatur modul-modul xposed yang akan anda install.
