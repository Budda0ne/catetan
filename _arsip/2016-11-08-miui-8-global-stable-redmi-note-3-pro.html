Kali ini untuk Redmi Note 3 SD (Snapdragon) dengan Nomor versi V.8.0.6.0. Meski tidak ada penambahan fitur yang berarti, tetap saja ini hal yang menggembirakan.


Dalam update versi terbaru MIUI 8 kali ini terdapat perbaikan bug yang ada pada fitur VoLTE (Voice over LTE). Sementara itu, fitur lainnya mash tetap seperti versi Global Stable untuk Redmi Note 3 sebelumnya. Dengan diberikannya Update ini merupakan saat yang tepat untuk meng-upgrade menjadi MIUI 8 bagi yang masih berkutat dengan OS MIUI 7.

Untuk Anda pengguna Redmi Note 3 SD yang tidak mendapat pemberitahuannya update otomatis, Anda bisa mengunduh file zip ROM Recovery MIUI V.8.0.6.0 untuk kemudian melakukan upgrade secara manual dari menu updater.

Selain menggunakan file zip, Anda juga bisa melakukan upgrade menggunakan file ROM Fastboot MIUI V.8.0.6.0 ber-ekstensi tgz dengan MiPhone/MiFlash dari PC/Laptop.

Anda bisa mengunduh kedua file tersebut dari link yang diberikan di bawah:
Recovery ROM V8.0.6.0 Download

Fastboot ROM V8.2.4.0 Download

Bagi yang ingin melakukan upgrade melalui menu Update yang tersedia, di sarankan untuk mengingat password Akun Mi. Hal ini karena saat Anda selesai melakukan upgrade akan dimintai password untuk mengaktifkan kembali ponsel Redmi Note 3 anda. Hal ini juga berlaku jika anda melakukan update melalui mode fastboot.
