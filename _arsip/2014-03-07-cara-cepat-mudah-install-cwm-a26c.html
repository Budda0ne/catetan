Evercoss A26C hampir sama dengan Evercoss A26B kalau dilihat dari segi bentuk. Tetapi banyak yang membandingkan A26C dengan Evercoss A7S karena dijual dengan kisaran harga yang sama.

Untuk mengikuti tutorial ini anda harus melakukan rooting evercross a26c anda terlebih dahulu.

Tujuan meng-install CWM Recovery ini adalah agar anda bisa memasang Custom ROM untuk A26C.

Sebelum anda mengikuti tutorial ini, anda HARUS setuju dengan Disclaimer dan juga mematuhi aturan DMCA & Copyright blog ini.

Cukup basa-basinya. Langsung saja ke pokok bahasan dan langsung dipraktek-kan, karena saya yakin anda tidak mungkin membuka artikel tutorial install cwm recovery evercoss a26c ini kalau belum memegang Evercross A26C. Tutorial ini merupakan cara cepat dan mudah-nya.

Siapkan dulu bahan dan aplikasi yang dibutuhkan untuk instalasi CWM Recovery A26C :

1. Download CWM Recovery (firmware lama ekstrak terlebih dahulu dan firmware baru cukup rename jadi recovery.img)
2. Download juga aplikasi Recovery Tools (bisa juga menggunakan mobile uncle apk)
Kedua bahan di atas, ada di bagian download di bawah artikel ini.

Setela anda mendownload keduanya, ikuti langkah-langkah berikut:

1. Letakkan File recovery.img di kartu memori anda di luar folder (/sdcard/--disini--)
2. Buka Pengaturan -> Pengaturan keamanan cari Sumber tidak dikenal dan centang
3. Tap (sentuh) recovery-tool.apk dan Install aplikasi Recovery Tools.
4. Setalah istalasi selesai, pilih opsi Buka.
5. Dari aplikasi ini, pilih Flash Recovery dan pilih other from sdcard.
6. Pilih file recovery.img tadi. Jika ada peringatan iyakan saja.
7. Pilih Reboot into recovery untuk masuk ke mode CWM Recovery.

Selamat, anda sudah berhasil memasang CWM Recovery untuk A26C dengan cara yang cepat dan mudah tapa PC.
Jangan lupa berkomentar, atau bertanya jika menemukan kesulitan. Selain berguna untuk mem-backup ROM, CWM Recovery ini juga bisa anda gunakan untuk memasang custom rom untuk A26C seperti Color OS R831T milik oppo.
