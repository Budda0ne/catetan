Bagaimana cara agar Game RR3 bisa dimainkan di Android dengan spek rendah dengan lancar? Tentu dengan menurunkan kualitas grafis-nya. Carannya bagaimana? Download dan Install RR3 Graphic. Download APK RR3 Graphic versi 2.6.2 (terbaru) di bawah dan install. Karena aplikasi RR3 Graphic sudah tidak bisa di-download lagi dari Playstore.

Aplikasi ini, sebenarnya berfungsi untuk mengatur kualitas grafis video dari game Real Racing 3 sesuai yang diinginkan. Tentu harus punya Game RR3 yang sudah ter-install sebelum menggunakan aplikasi ini.

RR3 Graphic sebenarnya di buat untuk memaksa agar kualitas grafis dalam game bisa lebih bagus. Karena semua HH dengan GPU Tegra 3 memiliki pengaturan grafis yang secara default diatur ke level rendah. Sehingga fitur-fitur visual pun tidak berfungsi, semisal spion belakang.
Beda RR3 High dan Low
Perhatikan: semakin tinggi kualitas grafis, semakin berkurang kinerja game karena makan resource yang tinggi.

Seperti yang sudah di jelaskan diatas, bukankah artinya malah akan membuat RR3 semakin nge-lag di HH spek rendah? Tunggu dulu, karena adanya pilihan untuk menurunkan kualitas grafis game RR3 dalam aplikasi ini, maka tool ini pun bisa digunakan untuk mengurangi lag di device dengan spek rendah (misal, ram 512mb) yang kebetulan setting default gpu-nya meminta grafis kualitas tinggi.

Begini Cara kerja Aplikasi RR3 Graphic


Aplikasi ini bekerja dengan cara memanipulasi file .plist dalam folder game yang ada di /sdcard/Android/data/com.ea.games.r3_[version]/.depot/eds/. Ketika mengaktifkan aplikasi ini, folder bernama 'eds' dikopi ke folder 'eds_backup'. Jika Anda memilih pengaturan kualitas tertentu (low, high, dll), semua file dalam folder 'eds' diganti dengan file yang sesuai dari 'eds_backup'. Kalau Anda menonaktifkan aplikasi, folder 'eds' dipulihkan lagi dari 'eds_backup' ke nilai default. Jika Anda tidak suka dengan aplikasi ini, Anda dapat mengubah pengaturan secara manual dengan meedit file .plist sendiri.
