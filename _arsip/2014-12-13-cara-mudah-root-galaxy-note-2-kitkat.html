Kali ini Samsung Galaxy Note 2 (N7100) dengan Android KiKat 4.4.2. Setelah Sekian lama, ternyata banyak sekali peningkatan keamanan yang dilakukan oleh samsung, yang membuat proses rooting jadi makin sulit. Setelah sebelumnya ada binary counter, sekarang ternyata ada KnoX yang membuat aplikasi superuser tidak berkutik.

Sebenarnya saya tidak sedang ingin melakukan rooting. Saya hanya ingin mencoba membackup system dari Samsung N7100 tanpa proses root. Tetapi, setelah browsing mencari referensi dan belum menemukannya, pada akhirnya saya memutuskan untuk melakuakn rooting pada Galaxy Note 2 N7100 terlebih dahulu, baru kemudian melakukan full back-up tidak perlu full back-up system karena di situs sammobile anda bisa mendownload Stock Firmware Samsung dari hampir semua model.

Untuk melakukan rooting pada Samsung Galaxy N7100 (dan ponsel Samsung lain yang sudah KitKat dan terisntall KnoX), ada langkah yang harus dilakukan adalah:
  1. Rooting dengan CF-root melalui odin downloader. Karena jika rooting kitkat tanpa PC akan sedikit sulit dan kadang tidak berhasil. kecuali root menggunakan aplikasi V-Root.
  2. Disable Aplikasi KnoX security. Ini karena jika KnoX tidak di disable, akan memblokir semua yang berbau superuser.

Untuk lebih jelasnya, ikuti panduan root Galaxy Note 2 berikut:

1. Rooting N7100 dengan CF-root melalui odin downloader

Untuk bisa melakukan rooting dengan CF auto root, anda harus sudah:
  1. Meng-install driver USB untuk Samsung Galaxy Note 2
  2. USB debugging pada Galnote 2 sudah di enable, caranya:
    • Masuk di settings -> more -> about device dan ketuk di Builod number beberapa kali hingga muncul notifikasi yang menyatakan anda sudah jadi developer.
    • Pilih menu back dan anda sudah bisa melihat opsi Developer Options pilih dan centang opsi USB Debugging.

Setelah kedua hal diatas sudah anda lakukan ikuti langkah berikut:
  1. Download 2 file yang saya sediakan di bagian download (baca lagi petunjuk di sana).
  2. Ekstrak kedua file tersebut di lokasi yang sama agar mudah.
  3. Jalankan Aplikasi Odin Downloader (Odin3.exe) dari file yang anda ekstrak tadi.
  4. Sekarang matikan Galaxy Note 2 anda dan hidupkan lagi dengan dalam mode download dengan cara menekan tombol [power] + [Volume Down] + [Menu] secara bersamaan, hingga muncul notifikasi download mode.
  5. Ketika sudah dalam keadaan download mode, colokkan Note 2 anda ke komputer dengan kabel data. Lihat dalam kotak ID:COM, jika sudah berubah warna menjadi biru atau kuning, artinya Note 2 anda sudah di deteksi oleh odin.
  6. Selanjutnya anda tinggal melakukan flashing file CF root ke ponsel dengan cara:
    • Klik tombol PDA di bagian kanan dan arahkan ke folder dimana anda mengekstrak file sebelumnya dan cari file dengan ekstensi .tar.md5
    • Pastikan opsi Re-Partiton tidak dicentang dan klik tombol Start dan tunggu hingga proses selesai.
  7. Proses tidak akan berjalan lama. Hanya sekitar 2 - 3 menit dan melakukan reboot otomatis. Tunggu hingga ODIN3 memunculkan notifikasi PASS!!

Lepaskan ponsel dari PC. Sekarang Galaxy Note 2 anda sudah root akan tetapi anda perlu untuk menonaktifkan KnoX agar aplikasi superuser tidak di blokir.

2. Menonaktifkan/men-disable KnoX

Sebenarnya ada beberapa cara untuk men-disable KnoX Security. Tapi untuk saat ini (karena anda sudah melakukan rooting) anda bisa menggunakan cara simple berikut ini:

1. Download Aplikasi KnoX disabler free di bagian download dan install di ponsel
2. Buka aplikasi tersebut dan pilih tombol Disable



Selain cara diatas, anda bisa juga men-disable KNOX dengan cara:
1. Unistall apklikasi superuser (chainDD)
2. Download aplikasi SuperSU (Chainfire) dari playstore atau dari sini
3. Jika muncul notifikasi disable KNOX pilih OK.

Jika aplikasi memberi notifikasi bahwa KnoX sudah di disable, maka selesailah sudah proses rooting Galaxy Note2.
