Langkah langkah root Xperia E3

Up# date: Untuk Sony Xperia E3/E3 Dual dengan OS Kitkat 4.4.4 jangan gunakan boot.img di download section, tetapi gunakan boot.img dari link yang saya berikan di kolom komentar
Untuk melakukan rooting, anda harus sudah melakukan unlock bootloader pada Sony Xperia E3 anda. Sebaiknya anda menyimpan semua data penting anda baik itu kontak, sms, atau lainnya karena proses unlock bootloader ini nantinya akan menghapus semua data di ponsel anda. Anda bisa mengikuti cara unlock bootloader di sini.

Pastikan bahwa baterai ponsel anda cukup untuk melakukan proses unlock bootloader dan rooting. Saya sarankan tidak kurang dari 75%.
  1. Download dan install USB Driver Sony Xperia E3 dan E3 Dual agar bisa dideteksi komputer/laptop.
  2. Download dan install Flashtool untuk melakukan flashing.
  3. Download file boot.img untuk tipe/varian ponsel anda, untuk install CWM Recovery.
Ketiga bahan diatas untuk memasang CWM Recovery, bisa anda dapatkan di bawah

Download juga file Update-SuperSU.zip di halaman ini, atau cek yang terbaru di sini. Simpan di sdcard ponsel anda.

Memasang CWM Recovery

Sebelum memulai, tutup software Sony PC companion dan non-aktifkan semua antivirus, karena akan menghalangi proses rooting.

Setelah anda yakin kedua aplikasi diatas sudah non-aktif, lanjutkan dengan langkah berikut:

  1. Buka software Flashtool yang sudah anda install sebelumnya dan pilih FASTBOOT mode.
  2. Sekarang matikan ponsel anda, kemudian sambil menekan tombol volume up konesikan Xperia E3 anda ke laptop/PC dengan kabel data.
  3. Langkah berikutnya adalah memilih file boot.img yang sudah anda download (sesuaikan dengan varian Xperia E3 anda) dari menu Flashtool dan klik start untuk mem-flash CWM Recovery ke ponsel.
Tunggu hingga selesai, ditandai dengan pesan notifikasi seperti ini:
sending ‘boot’ (15286 KB)…
OKAY [ 0.495s]
writing ‘boot’…
OKAY!
finished. total time: 0.497s
Artinya CWM recovery sydah terpasang.

Flashing SuperSU (Rooting)

Untuk melakukan rooting, anda tinggal melakukan flashing update-supersu.zip dengan cara:
  • Matikan ponsel anda (cabut baterai kalau perlu)
  • Nyalakan kembali dengan cara menekan tombol power, tombol volume up dan volume down secara bersamaan hingga masuk ke mode recovery.
  • Dalam mode recovery (CWM Recovery), pilih menu install zip from sdcard dan arahkan dimana anda menyimpan file update-supersu.zip, pilih file tersebut untuk melakukan rooting.
  • Jika proses flashing update-supersu.zip selesai, pilih reboot to system.

Selesai, kini Sony Xperia E3 anda, apapun tipe/varian-nya sudah root dan ter-install CWM Recovery. Anda bisa melakukan back-up dan restore system, menginstall dan menggunakan aplikasi yang memerlukan akses root dan sebagainya. Termasuk meng-install Custom ROM.
