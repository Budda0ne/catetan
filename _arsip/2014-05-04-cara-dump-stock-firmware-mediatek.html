Read Back/Dumping Stock Firmware Android Chipset Mediatek

Melakukan dumping stock rom adalah hal yang perlu anda lakukan jika anda berniat mengutak atik perangkat android anda. Karena sebagai pengembang, anda terkadang harus menanggung kerusakan akibat kesalahan kecil yang anda buat. Untuk mem-back-up stock "firmware yang belum tersentuh", anda harus melakukan readback melalui SP Flashtool.

Untuk melakuakan Dump original ROM (Firmware asli) dengan chipset MTK (Mediatek) anda membutuhkan file scatter sesuai dengan perangkat android anda sendiri. Jika anda belum punya sebaiknya anda membuat fie scatter sendiri terlebih dahulu.

Sebelum melakukan dumping rom melalui SP Flashtool, anda harus sudah:

1. Meng-instal driver Preloader Mediatek (USB VCOM)sesuai cengan tipe chipset.
2. Meng-install SP Flashtool (usakan versi yang baru).
3. Kabel data yang masih bekerja dengan baik.
4. File Scatter Sesuai dengan tipe dan seri perangkat anda.
5. Software text editor (notepad) lebih di sarankan menggunakan Notepad++

Setelah anda siap, anda bisa mengikuti langkah-langkah berikut:

1. Buka SP Flashtool dan klik Scatter loading. Pilih file scatter yang sudah anda siapkan. pastikan centang semua semua.
2. Klik pad tab/bilah Read back. klik Add jika pada jendela masih kosong.
3. Jika sudah ada 1 baris, dobel klik pada Start address dan beri nama file hasil dumping. Misalnya ROM_0 dan Save.

Menambahkan blok Dumping
4. Pada Jedela "Readback block start address" akan muncul. Pilih pilihan tombol di Hex. Dan fokus pada kolom Length. biarkan dulu, jangan di tutup.
5. Buka file Scatter yang anda siapkan tadi menggunakan Text Editor (Notepad++) scroll kebawah sampai anda menemukan partition_name: CACHE dan copy kode hex pada physical_start_addr

Nilai Hex dari file scatter
6. Beralihlah ke SP Flashtool dan tempel kode hex yang sudah anda kopi ke kolom Length. Tekan Ok.
7. Setelah itu anda tinggal menekan tombol Readback.

Readback
8. Tancapkan Perangkat Android anda dalam keadaan mati dan tanpa baterai ke komputer melalui kabel USB (gunakan panel belakang agar aliran data lancar)
9. Setelah perangkat di tancapkan, pasang baterai dan jangan di nyalakan. Komputer akan mendeteksi dan SP Flashtool akan memulai proses Dumping Stock ROM.

Proses Readback / Dumping ROM Selesai
10. Tunggu hingga muncul lingkaran hijau sebagai tanda proses readback sudah selesai. Anda bisa menutup SP Flashtool

Proses diatas akan memakan waktu sekitar 10-15 menit tergantung kecepatan komputer dan besar ROM yang di Dumping.

Lokasi Stock ROM mentah hasil Dumping

Jika anda melakukan langkah dengan benar, maka anda akan mendapatkan file Stock ROM mentah yang belum bisa anda gunakan untuk melakukan flashing dengan SP Flashtool. Agar bisa di gunakan, anda masih harus mengolahnya file ini dengan membuat ROM untuk SP Flashtool menggunakan MTK Droid tools
