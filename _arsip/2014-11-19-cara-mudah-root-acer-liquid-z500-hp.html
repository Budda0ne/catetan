Untuk melakukan root Acer Liquid Z500 dari ponsel bisa menggunakan Framaroot yang bisa anda download dari sini pilih versi 1.93. Akan tetapi jika kebetulan framaroot dari halaman tersebut tidak bisa digunakan atau tidak berhasil, anda bisa mendownload Framaroot khusus di bawah

Setelah anda mendownload Framaroot Apk, lanjutkan dengan mengikuti tutorial rooting berikut:

1. Buka menu Setting > Security > dan centang pilihan Unknown sources untuk meng-aktifkan USB Debugging.
2. Install faramaroot.apk yang sudah anda download tadi.
3. Setelah terpasang, bukia aplikasi framaroot dari launcher.
4. Setelah terbuak, pilih Install SuperSU dalam menu dropdown yang pertama.
5. Untuk pilihan exploit biarkan apa adanya. Lebih jelas soal exploit bisa di lihat di xda-developer.
6. Framaroot akan mulai melakukan rooting.
7. Reboot Acer Liquid Z500 jika sudah rooted.
8. Selesai, jika gagal, coba gunakan/pilih exploit yang lain.

Excellent! Untuk memastikan lihat di drawer (launcher). Jika SuperSU sudah ada maka Anda sudah berhasil melakukan rooting Z500 anda menggunakan Framaroot. Jika masih ragu, cek dengan root checker.
