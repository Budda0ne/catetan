Berbagai trik kostumasi OS Android berbasis Xposed Framework menjadi mandeg sejak OS Nougat dirilis. Pengembangan proyek Xposed untuk Nougat hilang timbul sejak saat itu. Beberapa kali pula beberapa developer mencoba membangkitkan kembali. Namun beberapa kali pula kandas. Pada akhirnya ErnyTech mampu meletakkan dasar untuk pengembangan Xposed Framework untuk Nougat. Namun, bukan itu yang ingin saya bagikan kali ini.

Berawal dari mulai populer-nya Magisk sebagai Systemless Interface agar bisa me-modifikasi Sistem Android tanpa menyentuh partisi sistem Android melalui Magisk Module yang tersedia. Apalagi setelah pengembang Magisk (topjohnwu) mem-port-ing Xposed Framework Installer menjadi salah satu Magisk Module mulai SDK21 hingga SDK23, saya lebih memilih untuk menunggu topjohnwu merilis Xposed Framework Installer SDK24 dan SDK25 a.k.a Nougat.

Dan akhirnya, setelah beberapa waktu bersabar, akhirnya terwujud. Xposed Framework Nougat pun muncul dan dapat dipasang dengan perantara Magisk Systemless. Yang istimewa, meski masih dalam tahap awal pengembangan, Xposed ala Magisk ini berjalan Dengan sangat baik. Dan saya lebih sreg menggunakan metode ini. Jika Anda setuju atau sependapat dengan saya, silahkan ikuti cara pasang Xposed Framework dibawah. Namun jika Anda lebih memilih untuk memasang Xposed Nougat tanpa Magisk, akan saya bahas lain waktu.

Memasang Xposed Framework Nougat dengan Magisk Systemless


Sebelum melakukan instalasi Xposed Nougat dengan cara ini, Magisk Systemless Interface sudah harus terpasang dan berjalan dengan baik pada devices Anda. Untuk lebih jelasnya silahkan baca Penjelasan Lengkap Tentang Magisk.

Download aplikasi Material Design Xposed Installer terlebih dahulu. (Penting!!) di sini atau di sini dan install.

Kemudian install Xposed Framework melalui aplikasi Magisk Manager dari menu Download dan pilih versi yang paling sesuai. Berbeda dengan instalasi langsung, metode ini lebih simpel karena Anda tidak perlu lagi pusing memikirkan apakah ponsel Anda menggunakan arsitektur arm, arm64, maupun x86. Yang perlu Anda ketahui hanyalah SDK Nougat yang Anda pakai saat ini. Jika versi Android Anda 7.0 berarti Anda perlu mengunduh Xposed Zip untuo SDK 24, sedangkan SDK 25 jika OS anda Android 7.1.

Atau jika tidak ketemu, silahkan pilih Xposed Installer yang sesuai dibawah ini dan flash langsung melalui recovery mode (TWRP).

Xposed SDK 24 atau Xposed SDK 25

Selanjutnya Anda bisa menambahkan Xposed Module baru melalui Material Xposed Installer dan mengatur kostumasi yang Anda inginkan seperti biasa.

Lalu bagaimana cara mencopot Xposed Magisk ini? Anda bisa langsung membuang Xposed Framework melalui Magisk Manager. Dan mencopot aplikasi Material Xposed Installer.

Ternyata cara install xposed framework untuk Nougat sangat mudah sekali bukan? Dengan demikian, Anda sudah bisa menikmati modul modul xposed mekipun OS Anda sudah menggunakan Android Nougat. Sayangnya, meskipun memasang Xposed secara Systemless, tidak menjadikan ponsel Anda lolos tes SafetyNet.
