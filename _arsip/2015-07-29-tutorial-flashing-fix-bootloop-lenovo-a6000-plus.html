Bagi user awam, bootloop merupakan kejadian yang paling menakutkan. Bootloop di Lenovo A6000 atau A6000 plus (dan ponsel android lainnya) bisa terjadi karena berbagai sebab yang kadang tidak disengaja. Tetapi, kejadian bootloop lebih sering terjadi pada Lenovo A6000 yang sudah root, meskipun ada juga kejadian bootloop meskipun ponsel belum di-root. Penyebab bootloop terjadi bukan karena ponsel rooted, tetapi karena kesalahan penggunaan dan keteledoran pengguna itu sendiri dalam menggunakan ponsel yang sudah di-root. Seperti kata Spiderman: "Kekuatan yang besar selalu di-ikuti dengan tanggung jawab yang besar."

Baca juga: Root Lenovo A6000 Without PC.

Selesai basa-basinya, jika kebetulan anda mengalami bootloop bahkan bricked, jangan panik. Ada solusi untuk mengatasi bootloop pada Lenovo A6000/plus anda. Namun sebelumnya, persiapkan dulu hal-hal berikut untuk Lenovo A6000:

  • Download dan Install USB Driver untuk Lenovo di PC/laptop.
  • Download dan Install Minimal ADB and Fastboot juga di drive C laptop/PC
  • Download Stock Rom sesuai tipe
  • Download Stock Recovery untuk Lenovo A6000.
  • Download CWM Recovery untuk Lenovo A6000plus.
  • Jangan lupa dengan kabel data dan sdcard :).

Panduan langkah demi langkah Fix Bootloop Lenovo A6000

Tutorial ini juga bisa anda pakai jika anda sudah bosan menggunakan Custom ROM dan ingin kembali menggunakan Stock ROM agar bisa mencicipi update firmware terbaru via OTA update. Pastikan anda sudah memenuhi 5 hal yang sudah disebutkan sebelumnya, dan ikuti langkah berikut:

  1. Copy file Stock Rom dan letakkan di sdcard anda, jangan di dalam folder agar mudah dicari
  2. Copy Stock Recovery ke lokasi anda meng-install ADB Fastboot (misal: C:\Program Files\Minimal ADB and Fastboot\) dan ganti nama filenya menjadi reco.img agar mudah.
  3. Matikan Lenovo A6000/plus, pastikan sudah mati dengan mencabut baterai dan memasang-nya kembali.
  4. Sekarang Nyalakan dengan menekan tombol Power bersamaan dengan tombol Volume down (-) agar masuk mode fastboot.
    Colokkan ponsel ke PC menggunakan kabel data. Lebih bagus dengan kabel data bawaan.
  5. Tunggu hingga PC/Laptop mendeteksi ponsel anda. Jika tidak terdeteksi anda harus meng-install ulang driver usb (bahan no.1) dan mengulangi dari langkah ke-4.
  6. Buka windows explorer dan arahkan C:\Program Files\Minimal ADB and Fastboot\ kemudian tekan dan tahan tombol [ctrl] dan [shift] kemudian klik kanan mouse, pilih Open Windows Command Here
  7. Pada Command prompt yang muncul, ketikkan perintah fastboot flash recovery reco.img
  8. Tunggu sampai muncul tulisan Finish dan diamkan sekitar 2-3 menit sebelum mencabut kabel data.
  9. Jika anda ingin cepat (tidak saya sarankan) tunggu sekitar 5 detik setelah finish.
  10. Untuk keluar dari mode fastboot, matikan ponsel dengan menekan tombol power kurang lebih sekitar 5-10 detik

Sampai tahap ini anda sudah mengembalikan stock recovery Lenovo A6000, tetapi bootloop belum sembuh. Selanjutnya tinggal melakukan Flashing Stock ROM Lenovo A6000. Berikut langkahnya:

  1. Matikan device dengan tekan dan tahan tombol power sampai layar mati, sekitar 5-10 detik.
  2. Nyalakan dalam mode Recovery dengan cara menekan tiga tombol bersamaan [power]+[vol up]+[vol down].
  3. Jika recovery mode sudah Stock Recovery Kraft_A6000_S032, pilih Apply update from sdcard
  4. Arahkan ke Stock Rom Kraft_A6000_S032
  5. Tunggu hingga proses flashing selesai sekitar 15 menit paling lama.
  6. Lenovo akan reboot otomatis setelah proses instalasi stok rom selesai.

Untuk booting pertama setalah flashing, biasanya memakan waktu sekitar 5 menit untuk masuk ke homescreen. Jika lebih dari 1 jam masih stuck di logo, sebaiknya masuk recovery mode lagi dan lakukan wipe data.

Panduan langkah demi langkah Fix Bootloop Lenovo A6000plus

Perhatikan, gunakan tutorial sesuai dengan jenis file stock rom yang anda gunakan.

Tutorial berikut untuk Flashable zip Stock ROM

  1. Pada ponsel anda sudah harus ter-install CWM Recovery jika belum gunakan tutorial ini dengan metode fastboot.
  2. Jika CWM sudah ter-install, copy file ke sdcard dan masuk-kan sdcard ke ponsel a6000plus anda.
  3. Nyalakan ke mode recovery dengan menekan tombol [power]+[vol up]+[vol down] bersamaan.
  4. Dalam mode recovery pilih menu install zip from sdcard dan pilih file Kraft_A6000-s_S0xx_xxxxxx.zip.
  5. Tunggu hingga proses flashing selesai.
  6. Pilih menu reboot to system jika sudah selesai.

Tutorial berikut untuk Backup Stock ROM

  1. Pada ponsel anda sudah harus ter-install CWM Recovery jika belum gunakan tutorial ini dengan metode fastboot.
  2. Jika CWM sudah ter-install, ekstrak file Backup_Kraft_A6000-S_S0xx_15xxxx dan copy folder 1970-xx-xx.xx.xx.xx ke sdcard di dalam folder /clockworkmod/backup/ di external sdcard. Jika dalam external sdcard tidak ada folder /clockworkmod buat sendiri folder tersebut dan buat folder backup didalamnya, baru copy folder 1970-xx-xx.xx.xx.xx di dalam folder backup tersebut.
  3. Masuk-kan sdcard ke ponsel a6000plus anda dan nyalakan ke mode recovery dengan menekan tombol [power]+[vol up]+[vol down] bersamaan.
  4. Dalam mode recovery pilih menu advance back-up restore dan pilih custom backup to sdcard1 dengan hanya mencentang boot.img. langkah ini agar proses restore tidak ada gangguan error.
  5. Kembali kemenu sebelumnya dan setelah proses backup tadi selesai, kemudian lakukan restore from sdcard1 dan pilih file 1970-xx-xx.xx.xx.xx yang tadi anda ekstrak (ingat, bukan file hasil back-up anda sendiri)
  6. Pilih menu reboot to system jika sudah selesai.

Booting pertama setelah selesai restore/install stock rom biasanya agak lama, sekitar 10 menit paling lama. Jika lebih dari itu silahkan boot ke recovery mode lagi dan lakukan wipe data, cache dan dalvik cache kemudian reboot to system lagi.

Nah, kini Ponsel Lenovo A6000 atau A6000plus anda sudah kembali seperti semula. Tinggalkan komentar jika sukses maupun ada yang perlu ditanyakan.
