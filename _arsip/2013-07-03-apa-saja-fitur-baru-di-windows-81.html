Update versi 8.1 ini sebenarnya hanyalah update minor. Microsoft sepertinya memperbaiki beberapa gangguan kecil yang dianggap dapat menghilangkan rasa nyaman pengguna OS Windows 8 di Windows 8.1.

Dari segi tampilan , Microsoft tidak melakukan perubahan yang berarti pada Windows 8.1. Pengguna tetap dapat menggunakan tampilan Metro di OS yang memiliki codename "Blue" tersebut.

Inilah 7 Fitur yang dirubah dan ditambahkan di Windows 8.1


Kembalinya tombol Start di Windows 8.1


Sebelum Windows 8 diluncurkan, para pengguna dapat dengan mudah membuka aplikasi, mencari fitur Control Panel, hingga mematikan PC. Hal tersebut disebabkan semua fitur dijejalkan oleh Microsoft ke dalam sebuah tombol, yaitu tombol Start.

Maka, tidak heran apabila pengguna lama Windows sebelum Windows 8 kesulitan setelah Windows 8 tidak lagi menghadirkan tombol Start. Untuk melihat aplikasi yang ada, pengguna harus masuk ke tampilan Metro, sedangkan tombol menu Control Panel dan Shutdown disatukan ke sebuah tempat yang dinamakan Charm Bar.

Microsoft menerim a banyak keluhan terkait tombol start yang hilang, Microsoft akhirnya mengembalikan tombol "Start" tersebut ke Windows 8.1, horeee... Namun, fungsinya sedikit berbeda dibandingkan dengan OS Windows sebelumnya. Tombol "baru" ini sebenarnya hanya menampilkan Start Screen ala Metro ketika diklik.

Jika Anda tidak ingin menampilkan tampilan Metro ketika tombol ini diklik, dengan sebuah pengaturan, anda dapat mengubah tampilan menjadi "All Apps" untuk mengembalikan fungsinya seperti tombol Start tradisional. Lihat gambar dibawah.



Pada gambar kedua diatas dapat dilihat, pengguna pun bisa mengakses beberapa menu lain, seperti Task Manager, Control Panel, Search, dan Run. Caranya adalah dengan mengklik kanan tombol Start, secara otomatis menu-menu tersebut akan ditampilkan.

Windows 8.1 mempunyai Lock Screen yang lebih keren


Lock Srcreen di Windows 8.1 tidak tampil monoton. Layar ini tampil lebih interaktif dari Windows 8. Pengguna bisa memasang slideshow foto, menerima panggilan Skype, dan mengakses kamera langsung dari layar tersebut. Lihat saja gambar berikut:


Kustomisasi secara bebas


Kali ini, Microsoft mengizinkan kustomisasi yang lebih leluasa untuk pengguna Windows 8.1. Sebagai contoh: pengguna kini dapat menyamakan wallpaper untuk tampilan desktop dan tampilan Metro. Fitur ini membuat pengguna tidak akan terlalu kebingungan apabila ingin berpindah dari model tampilan desktop ke Start Screen.


Pengguna tetap dapat mengubah ukuran kotak yang ada di tampilan Metro. Namun, caranya sedikit berbeda. Untuk mengubah dan memindahkan kotak-kotak aplikasi, pengguna harus menahannya terlebih dahulu. Hal tersebut terlihat kurang intuitif, tetapi ini dapat mencegah pergerakan atau perubahan ukuran kotak secara tidak sengaja.

Perubahan fitur Search


Fitur Search di Windows 8.1 kali ini lebih atraktif dan interaktif. Pencarian akan dilakukan untuk semua hal yang berkaitan dengan kata-kata yang ingin dicari oleh pengguna. Sistem melakukan pencarian dari hard drive yang ada di PC juga internet. Misalnya, jika pengguna mengetik kata Komputechno, maka pencarian akan mencari file yang berkaitan dengan kata tersebut di PC dan juga internet.

Melalui fitur Search di Windows 8.1, pengguna juga bisa langsung menjalankan aplikasi. Caranya, ketikkan beberapa huruf nama software, maka akan muncul daftar hasil pencarian. Klik saja nama aplikasi yang diinginkan, maka sistem secara otomatis akan menjalankan software ini.

Windows 8.1 bisa jalankan empat aplikasi bersamaan


Multitasking merupakan fitur yang sudah ada di Windows sejak dulu. Microsoft meningkatkan kemampuan Multitasking di Windows 8.1. User dapat menjalankan empat aplikasi sekaligus dalam beberapa layar terpisah.

Namun, fitur ini sangat bergantung pada ukuran layar. Contohnya, apabila pengguna menggunakan layar monitor 27 inci, sistem bisa menampilkan empat aplikasi tersebut dengan sangat baik. Apabila menggunakan tablet 8 inci, sistem hanya mampu menampilkan 2 window aplikasi saja.

Aplikasi di-"update" secara otomatis


Aplikasi yang di download dari Appstore Microsoft akan langsung diperbarui apabila pihak pengembang merilis versi barunya.

"Boot" langsung ke tampilan "desktop" klasik


Di Windows 8, saat PC dinyalakan, pengguna akan langsung menemui tampilan Metro. Di Windows 8.1, pengguna bisa mengatur sendiri pilihan saat boot-up, apakah sistem menggunakan metro atau langsung ke tampilan desktop.
