Jika Anda ingin membobol (bypass) blokade proxy server, entah itu di tempat kerja atau di laboratorium kampus/sekolah, ada beberapa pilihan yang tersedia untuk Anda. Yang pertama adalah cara yang cukup sederhana dan mudah dan hanya melibatkan Anda dengan Browser anda. Saya menyarankan anda menggunkana cara ini terlebih dulu untuk mem-bypass proxy kantor/kampus/sekolah anda. Jika ini solusi bypass pertama gagal, maka ada pilihan kedua, ketiga, dst. Alternatif kedua (dan selnjutnya) untuk membobol hadangan proxy ini mungkin sedikit lebih rumit.

Nah, jika Anda ingin menerobos server proxy kantor/kampus/sekolah, berikut ini adalah beberapa solusi terbaik yang bisa saya berikan. Tak perduli posisi anda dimana atau jenis server proxy ysng ingin Anda bobol baik itu kantor perusahaan anda atau kampus anda atau barangkali laboritorium internet sekolah anda.

Metode 1: Nonaktifkan Proxy Server di Browser

Jika Anda beruntung, melewati proxy di tempat kerja atau sekolah Anda mungkin hanya cukup dengan langkah yang sederhana nonaktifkan proxy di pengturan web browser. Banyak sekolah dan tempat kerja mengatur agar semua komputer dalam mengakses internet harus melalui proxy server pusat mereka. Ini adalah alasan mengapa Anda mungkin tidak dapat mengakses beberapa website dari tempat kerja atau sekolah. Ketika komputer Anda mengakses internet, anda meng-akses ke server proxy dan kemudian sercer proxy mencoba untuk mengakses website yang anda inginkan, yang kemudian terdeteksi dan ditolak oleh server proxy. Periksa pengaturan koneksi internet pada web browser (misalnya firefox) Anda dengan membuka Tools - Option - Advanced - Network - Settings.

Cobalah untuk menonaktifkan dengan memilih opsi (radio button) No Proxy dan hubungkan ke internet lagi. Jika Anda sekarang dapat mengakses web secara normal, maka cukup sampai di sini saja. Jika tisak, anda bisa melanjutkannya ke metode selanjutnya.

Metode 2:

Menggunakan Server Proxy Lain

Untuk byapass proxy server kali ini sedikit lebih kompleks, tapi lebih menjamin keberhasilan anda dalam menerobos blokade proxy anda. Banyak sekolah atau kantor memblokir Situs situs tertentu (seperti jaringan sosial atau situs "dengan klasifikasi tertentu") dan inilah sebabnya Anda mungkin mengalami kesulitan mengakses situs tersebut. Perusahaan/Kampus/Sekolah melakukan ini biasanya dengan tujuan untuk menghentikan mahasiswa atau karyawan dari membuang-buang waktu online yang berharga mengunjungi situs yang tidak produktif.

Namun, sekolah atau kantor Anda tidak akan mungkin memblokir akses keseluruh alamat di internet. Sebaliknya mereka memblokir situs-situs tertentu berdasarkan url website mereka atau alamat IP-nya. Oleh karena itu, Anda bebas untuk mengakses server proxy lain yang kemudian mengarahkan koneksi Anda ke situs web yang Anda inginkan sehingga Anda bisa mengakses situs yang diblokir bahkan dari tempat kerja atau sekolah.

Pada dasarnya, komputer Anda di sekolah atau tempat kerja yang menghubungkan ke situs yang tidak diblokir oleh sekolah atau tempat kerja. Kemudian, Anda menggunakan server proxy tersebut untuk menyambung kembali ke situs tujuan yang diblokir tadi. Mengapa dan bagaimana ini ini bisa dilakukan? Server proxy sekolah atau tempat kerja hanya dapat mendeteksi bahwa komputer anda terhubung ke situs web pertama yang tidak di blokir (dalam hal ini situs proxy server). Dia tidak tahu bahwa Anda "meminta tolong" pada situs pertama tadi untuk menghubungkan ke situs Web kedua yang diblokir. Anda bisa mencoba TurboHide atau proxMachineKlik link untuk melihat daftar laengkap server Proxy gratis.

Saya yakin cara ini efektif, tapi mungkin tidak efisien. Selanjutnya anda bisa mencoba beberapa alternatif berikutnya.

Metode 3: Bypass Proxy dengan cara HTTP Tunneling

Tunneling secara simple bisa dikatakan sebagai "memalsukan protokol". Metodenya adalah dengan menggunakan protokol SMTP (email) untuk membungkus protokol HTTP (Web), anda bisa membaca lebih lanjut di tentang tunneling di wikipedia. Metode/Cara ini biasanya digunkan jika perusahaan secara extrim memblokade seluruh akses internet dengn protokol HTTP, alias meblokir seluruh akses web tetapi tetap membuka akses untuk berkirim email. Untuk melakukan HTTP Tunneling anda membutuhkan software lain yang bisa membungkus protokol HTTP dan mengirimkannya melalui protokol lain semisal SMTP. Untuk metode ini akan ada artikel lanjutannya : Mem-bypass Blokade Proxy menngunakan Software HTTP Tunneling.

Metode 4: Bypass Proxy dengan menggunakan Virtual Private Network

Virtual Private Network bisa dikatakan sebagai "Jaringan Pribadi" atau jaringan lain yang terpisah dari sistem jaringan Kantor/kampus/sekolah anda sekarang. Ssehingga anda bisa mengaturnya sesuka hati anda tanpa harus menurut aturan dari jaringan kantor/kampus/sekolah anda. Anda bisa mengatur VPN di komputer secara langsung atau menggunakan software tertentu untuk mengkonfigurasi secara otomatis. mengenai VPN secara detail anda bisa membuka halaman berikut. Untuk menggunakan metode ini akan ada tutorial lanjutan: Mem-bypass Blokade Proxy Dengan Koneksi Virtual Private Network
