Jadi, simak langkah-langkah Cara Root Galaxy A3 pada Android 4.4.4 KitKat [Semua Tipe].

Sebelum melakukan proses root, cek dan lakukan hal berikut ini:
- Cas batera Galaxy A3 smartphone hingga minimal 65%.
- Aktifkan opsi "USB Debugging" pada smartphone Galaxy A3 Anda.
- Jangan lupa untuk melakukan backup data penting yang tersimpan di smartphone Anda.
- Instal "Samsung Galaxy A3 USB Driver" pada komputer Anda.

Langkah-langkah Root Galaxy A3 [SM-A300F, SM-A300H, SM-A3000]

  1. Download file rooting sesuai dengan nomor model perangkat Anda dan Tool Odin dari link di bawah untuk root Galaxy A3 xda.
  2. Ekstrak kedua file (odin dan file rooting) yang tadi anda download dalam satu folder.
  3. Jalankan Odin3 v3.09.exe dari folder tersebut.
  4. Sekarang, matikan ponsel Galaxy A3 sekitar 5 - 10 detik
  5. Boot Galaxy A3 ke recovery mode dengan menekan tombol power dan volume down dan home secara bersamaan hingga muncul peringatan lepas ketiga tombol dan tekan lagi tombol volume up untuk masuk mode download.
  6. Kemudian, hubungkan Galaxy A3 dengan komputer menggunakan kabel data USB
  7. Jika terhubung dengan sukses, maka salah satu kolom ID:COM pada Odin akan berubah warna menjadi biru.
  8. Pilih file rooting dengan mengklik tombol "PDA" dan arahkan ke lokasi file "CF_Auto_Root_xxxx.tar.md5" (dari folder tempat anda meng-ekstrak sebelumnya).
  9. Pastikan bahwa hanya pilihan "F. Reset Time" dan "Auto Reboot" yang tercentang di tool Odin.
  10. Setelah semua siap maka Anda bisa menekan tombol "Start" untuk memulai proses rooting, (jika anda menggunakan Odin yang anda download dari halaman ini, anda tinggal memilih file .tar.md5 dan klik start)
  11. Proses rooting ini tidak akan lama. Hanya memakan waktu sekitar 30 detik.
  12. Setelah selesai, Anda akan melihat kata "PASS" pada kolom ID:COM Odin dan ponsel akan reboot otomatis.
