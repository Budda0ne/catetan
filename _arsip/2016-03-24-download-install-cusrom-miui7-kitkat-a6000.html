Custom ROM MIUI 7 Kitkat untuk Lenovo A6000 ini sebenarnya sudah ada 2 versi yang di-porting oleh agan Risky Bagoes Alam sebelum ROM MIUI 7 Lollipop. Tapi sebelumnya (v2) saya tidak berminat untuk mempublikasikan dengan alasan masih agak buggy di fungsi Flashlight-nya. Nah, untuk saat ini boleh dibilang sudah sehat dan tidak batuk-batuk.

Fitur-fitur MIUI secara umum sudah bisa dinikmati tanpa kendala di ROM Global Stable ini. Hanya mungkin untuk performa game yang butuh koneksi lancar tanpa terputus agak meragukan (mis: Crisis Action). Tetapi saat digunakan untuk game seperti Clash of Clans atau Clash Royale sangat lancar menurut saya.

Dalam ROM MIUI 7 untuk Lenovo A6000 yang berbasis Android Kitkat ini sudah di tambahkan beberapa aplikasi yang cukup berguna. Terutama aplikasi blokir iklan paling efektif: Adaway (dan matilah blogger :'( seperti saya), juga ada Xperia Keyboard meskipun sebenarnya menurut saya lebih enak menggunakan Google Keyboard. Lagipula karena ada Xperia keyboard ini anda mesti melakukan pengaturan lagi setelah instalasi. Jangan kuatir, gampang kok.

Untuk fitur lainnya seperti Xposed Framework yang juga disertakan, bisa anda coba sendiri. Dan langsung sikat Custom ROM MIUI7 Kitkat dibawah beserta SuperSU-nya.

Syarat dan Cara Install MIUI7 Kitkat di Lenovo A6000

ROM terakhir yang dipakai Lenovo A6000 sebelum install MIUI Kitkat based ini sebaiknya masih Kitkat (stockrom maupun custom rom).

Selain itu sudah terpasang TWRP Kitkat. Kalau kebetulan sudah menggunakan TWRP Lollipop, silahkan ganti dengan TWRP Kitkat. Caranya ada di bagian akhir tutorial ini

Letakkan file yang sudah anda download sebelumnya, atau Download dulu diatas dan letakkan di sdcard anda (internal/external, bebas)

Masuk ke mode recovery (TWRP) dengan cara menekan tombol power dan kedua tombol volume bersamaan selama beberapa detik dari keadaan ponsel mati.

Dalam mode recovery lakukan Wipe (data, cache, dalvik).

Setelah selesai, kembali ke menu utama dan pilih menu install zip dengan urutan flashing ROM MIUI 7 kitkat dilanjtkan flashing SuperSU zip.

Lakukan reboot to system setelah proses instalasi selesai.

Setelah sukses instalasi ROM dan masuk setup wizard, sebaiknya pilih locale Rusia agar bisa melakukan beberapa penyesuaian dan module exposed berjalan lancar. Lakukan juga hal-hal berikut:

1. Xperia Keyboard

Masuk menu Settings kemudian additional setting pilih Xperia Keyboard. Matikan fitur (off) gesture input dan pilih i don't want any help.

2. Xposed Framework

Untuk mengaktifkan Xposed, buka aplikasi, pada menu framework dan pilih install/update dan berikan akses superuser kalau diminta (pilih grant) kemudian pilih lalu aktifkan mod MIUI Tweaks (agar tidak perlu login akun Mi), Mi Themes (download/instal theme), dan iFont untuk ganti font.

Oh iya, karena saya tidak begitu suka dengan Xposed Module, saya pilih Uninstall saat masuk menu Framework. Dengan konsekuensi: tidak bisa memakai theme dan fonts berbayar secara gratisan lagi. Ah, gak penting..

3. Adaway
Untuk adaway cukup buka aplikasi dan pilih grant untuk memberi akses superuser, nyalakan koneksi internet/data kemudian pilih opsi download files and apply ad blocking.
