Sony Flashtool 0.9.25 adalah versi terbaru update 2019 yang sekarang masih tetap bisa dipakai untuk FTF versi lama. Ini hanya karena bundle FTF versi lama memang tidak muncul. Untuk mengatasinya ada cara agar FTF yang lama bisa tetap dipakai. Yakni dengan cara meng-ekstrak isinya ke ke folder tertentu agar bisa terlihat.

Setelah di-ekstrak, jika ada folder "boot" rubah nama file sinb dan tab menjadi sin dan ta. Kemudian buat kembali file FTF dengan Flashtool versi terbaru ini.

Changelog 0.9.25.0

  • You will have to recreate the FTF with the release 0.9.25.0 of Flashtool if you already had a FTF

  • Updated Java to 10.0.1
