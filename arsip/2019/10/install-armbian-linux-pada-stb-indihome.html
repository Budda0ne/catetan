Instal Armbian di STB HG680P

Jika kamu pemakai OS Jendela veteran, mungkin akan merasa gamang jika mendengar kata Linux. Jangan kuatir, sebenarnya, tampilan beberapa distro Linux tak jauh beda dengan OS Jendela. Sulit memang menghilangkan prasangka bahwa memakai distro Linux itu sulit.

Balik lagi, STB HG680p bisa dijadikan layaknya SBC versi murah meriah dengan memasang Armbian. Dengan memasang Armbian S905x pada STB Indihome, bisa membuatmu lebih familiar dengan distro Linux dengan cara yang lebih hemat.

Bagaimana cara instal Armbian di STB HG680p? Hingga saat ini, sebenarnya sudah banyak tutorial yang dibagikan di internet. Tulisan ini lebih ditujukan sebagai catatan saja walaupun demikian bisa kamu jadikan referensi alternatif tutorial yang telah kamu baca sebelumnya.

Persiapan

Berikut adalah hal-hal saya gunakan dan sebaiknya kamu persiapkan jika ingin mencobanya:

  1. STB HG680 P. Jika kamu tidak pernahk langganan Indihome, kamu bisa beli di marketplace dengan kisaran harha 160 ribu-an hingga 200 ribu-an.
  2. Monitor. Sebaiknya yang punya port HDMI. Bisa juga menggunakan TV LED, yang penting ada port HDMI
  3. Keyboard dan Mouse tentu saja.
  4. Komputer (Desktop PC) atau laptop. Tak punya? Pinjam.
  5. Micro SD alias sdcard. Usahakan class 10. Dan jangan lupa adapternya
  6. Cardreader jika memakai Komputer.
  7. Koneksi Internet
  8. System Image Armbian. Misalnya saja Ubuntu Bionic Desktop. Pilih salah satu dari sini: Armbian Amlogic S905x.
  9. Alat untuk melakukan flashing (instal) Armbian ke microsd, bisa memakai etcher

Langkah-Langkahnya:

  • Gunakan laptop/komputer yang terhubung dengan internet untuk mengunduh Armbian yang diinginkan dan tool etcher.
  • Dalam hal tutorial ini coba unduh Armbian 5.91 Aml s905 Ubuntu Bionic Desktop. Sebaiknya jangan gunakan yang versi server agar ada tampilan layaknya layar desktop OS Jendela, kalau pakai Armbian server, nanti malah bingung.
  • Pasang tool etcher yang sudah diunduh, dan lakukan flashing Armbian ke microsd yang sudah dipersiapkan. Ingat, pindahkan semua file yangbada didalam microsd sebelum flashing.

  • Setelah proses validasi selesai, nyalakan STB seperti biasa jika sudah di-unlock. Kalau belum silahkan unlock dengan Pulpstone Android. Pasang aplikasi Terminal Emulator dari APKpure setelah STB menyala.

  • Masukkan microsd yang sebelumnya sudah di-flash Armbian ke slot microsd di STB Indihome.
  • Buka aplikasi Terminal Emulator dan ketik reboot update agar STB melakukan upgrade partisi system dengan file yang ada di microsd.

  • STB akan booting ke armbian, kemudian ikuti terus langkah-langkah seperti memasukkan user dan password root.

Tahap instalasi selesai.
