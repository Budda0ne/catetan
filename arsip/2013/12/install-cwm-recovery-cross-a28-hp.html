Ada banyak cara meng-install CWM recovery untuk Smartphone Android. Biasanya, tutorial-tutorial instal CWM recovery melalui PC lebih banyak. Untuk meng-install CWM Recovery melalui HP sendiri, juga sudah banyak. Cara install CWM Recovery dari hape lebih diminati karena lebih mudah dilakukan.

Untuk meng-install CWM Recovery (CWR), selain menggunakan aplikasi-aplikasi rooting dan installer CWR, bisa juga menggunakan tool/aplikasi ringan Terminal Emulator. Cara ini cukup simpel dan mudah.

Inilah beberapa foto CWM Recovery terbaru untuk Cross A28 yang sudah ter-install dan bekerja dengan baik.


Foto Tampilan CWM Recovery Cross A28 1


Foto Tampilan CWM Recovery Cross A28 2
Untuk meng-install CWM Recovery melalui HP Cross A28, anda membutuhkan:
  1. File image CWM Recovery untuk Cross A28 terbaru, bisa didonlot dari sini.
  2. Aplikasi Terminal Emulator, bisa di download dari playstore atau langsung dari sini.
  3. Kartu memori microSD (opsional)
  4. Dan tentu saja, Hape Cross A28 :)
Meng-install CWM Recovery melalui Terminal Emulator
Jika sudah anda siapkan hal-hal di atas, maka anda bisa mengikuti petunjuk simpel dari saya berikut ini:
  • Setelah anda mendownlad file image recovery (recovery.img), pindahkan ke sdcard. jika tidak punya, anda bisa meletakkan di dalam penyimpanan internal (emmc). Letakkan di luar, jangan di dalam folder (misal: /sdcard/recovery.img atau /emmc/recovery.img
  • Install terminal emulator dan buka aplikasi tersebut kemudian ketik :
    su
    untuk meminta akses superuser
  • Akan muncul peringatan, pilh saja allow (ijinkan) dan ketik-kan kode berikut:
    • jika file recovery.img anda letakkan di sdcard kodenya adalah:
      dd if=/sdcard/recovery.img of=/dev/block/mmcblk0p16
      kemudian tekan enter
    • jika anda meletakkan file recovery.img di dalam memori internal gunakan kode:
      dd if=/emmc/recovery.img of=/dev/block/mmcblk0p16
      kemudian tekan enter
  • Usahakan jangan dampai salah ketik ya? karena kalau dalah ketik bisa berabe. Proses-nya sangat singkat
  • Setelah complete, ketik-kan lagi perintah :
    reboot recovery
    untuk langsung masuk ke mode CWM Recovery. Tunggu kira-kira 5 detik, Cross A28 anda akan reboot sendiri
Jika anda melihat tampilan dengan menu-menu "mirip" gambar di bawah, maka saya ucapkan: "Selamat Cross A28 Anda sudah ter-install CWM Recovery terbaru".
Dengan memiliki Custom Recovery CWM ini, anda bisa melakukan back-up dan restore HH dengan mudah. Dan akan sangat berguna ketika anda ingin meng-install custom ROM untuk Android Cross A28 anda.
