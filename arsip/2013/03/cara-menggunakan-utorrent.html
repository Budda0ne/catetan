uTorrent adalah salah satu aplikasi BitTorrent Client yang sangat ringan dan memiliki fungsi luar biasa, dengan Aplikasi BitTorrent ini kamu dapat mendownload berbagai jenis musik dan film kesukaan kamu. lihat di wikipedia

Aplikasi uTorrent ini tersedia untuk berbagai sistem oprasi seperti Windows, Mac OS X, dan linux.

Bila kamu belum mengerti apa itu Torrent, sebaiknya kamu membaca terlebih dahulu Definisi Torrent

Tutorial ini akan memjelaskan kepada kamu cara menggunakan aplikasi uTorrent serta bagaimana cara download melalui file torrent dengan uTorrent. Simak cara menggunakan aplikasi utorrent berikut

  • Download terlebih dahulu aplikasi uTorrent di sini, kemudian install uTorrent dengan menjalankan uTorrent.exe yang sudah kamu unduh.
  • Pada saat pertama kali penggunaan kamu akan di hadapkan pada halaman konfigurasi untuk kecepatan download torrent, jadi setting lah sesuai dengan keinginan atau kecepatan internet yang kamu miliki.
  • Kemudian cari file torrent yang kamu inginkan, contohnya di http://isohunt.com atau http://kat.ph.
  • Ketikan nama file yang ingin kamu download di kotak pencarian
  • Jika kamu mengklik torrent yang ingin di download maka akan langsung muncul sebuah window baru.
  • Sebelum memilih file di situs koleksi torrent sebaiknya perhatikan komen-komen pengguna di bawah karena tidak semua file torrent yang ada disana sempurna. terkadang ada orang usil yang mengupload torrent yang berisi virus atau pun file yang korup.
  • Ketika file telah terbuka dengan aplikasi uTorrent. kamu dapat langsung mendownloadnya.
Jangan lupa, sebelum mendownload perhatikan juga di direktori/folder mana hasil download nantinya akan di tempatkan. Caranya dengan memilih option->directories pada utorrent
