Bagi XP lovers, tiada sistem operasi seindah Windows XP. Dari era kakek buyut hingga sekarang, masih saja menggunakan windows XP. Meskipun di ming-imingi Wndows Vista, kemudian Windows 7 dan terakhir Windows 8 dengan METRO UI-nya yang terkenal. XP Lovers masih beerkutat dengan sistem operasi tuwir ini.

Alasannya sih kalau masalah tampilan kenapa harus pusing? Banyak sekali developer yang mengembangkan penyempurna tampilan untuk Windows XP dari mulai yang sekedar Theme Pack hingga Tranformation Pack yang bisa merubah tampilan Windows XP secara Komplit.

Dengan hadirnya Windows 8, maka bermunculanlah tool-tool dan berbagai software untuk kustomisasi Windows XP menjadi Windows 8. Dari sekian banyak tool dan software tersebut, ada Windows 8 Transformation Pack. Nah yang terakhir inilah yang akan kita coba sama-sama.

Bagi yang sudah terbiasa, merubah tampilan Windows XP memang mudah, tapi bagi yang belum berpengalaman bisa menggunakan Windows 8 Transformation Pack untuk merubah tampilan Windows XP menjadi seperti Windows 8 dengan singkat dan mudah. Ikuti panduan berikut:

Pertama, download terlebih dahulu NetFramewok 2.0 (khusus 64bit) dan NetFramework 4.0 dari situs microsot atau klik link berikut ini NetFramework 2.0 | NetFramework 4.0. Juga downloadx20Windows 8 Transformation Pack dari tautan ini [mirror]. jika direct link tersebut sudah kadaluarsa buka halaman ini. Ketika mendownload adf.ly, tunggu 5 detik hingga muncul menu "Skip Ad" atau "Lewati" di pojok kanan atas, klik di situ.

Kedua, Install NetFramework 2.0. dengan menjalankan NetFx20SP2_x64.exe jika menggunakan Windows XP 64bit. Tunggu hingga selesai. Lanjutkan dengan meng-install NetFramework 4.0 dengan menjalankan dotNetFx40_Full_x86_x64.exe, jika menggunakan Windows XP 32bit cukup meng-install NetFramework 4.0 ini saja.

Ketiga, Cari dan buka file 8TP65.zip ekstrak ke lokasi yang mudah di cari misalnya; My Documents atau Dekstop. Kemudian cari file Windows 8 Transformation Pack 6.5.exe. Klik 2 kali untuk menjalankan.



Pada pilihan immerse ui, pilih Charm Bar atau Metro UI dengan klik dropdown menu. Centang seluruh opsi yang bisa di centang, Klik Configure. Kita hanya perlu menunggu hingga muncul pemberitahuan untuk merestart komputer. Restart.

Akhirnya, Windows XP pun rasa Windows 8 dari mulai Bootscreen, Logon UI, Desktop dan Windows Explore-pun serasa Wndows 8.
