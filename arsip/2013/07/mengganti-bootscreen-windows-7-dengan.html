Windows 7 Boot Updater sendiri merupakan sebuah program untuk mengupdate animasi dan teks pada saat boot Windows 7 (BootScreen). Program ini dibuat dalam bahasa pemograman C++ / CLI dan memerlukan Microsoft NET Framework. 2.0 dan Microsoft library WIM yang terinstall di Windows 7 anda.
Beberapa Fitur Windows 7 Boot Updater antara lain:
  • Mengubah animasi bootscreen
  • Mengubah teks boot “Starting Windows” dan “Microsoft Corporation”
  • Dapat mengkompilasi animasi dari file 105 BMP, PNG, GIF, atau gambar TIFF
  • Tidak memerlukan aplikasi lain (misalnya: ImageX)
  • Dirancang dengan desain interface yang simple agar mudah digunakan.
  • Otomatis membackup file winload.exe dan bootres.dll yang asli sebelum dimodifikasi
  • Ada pilihan antara menggunakan GUI dan CLUI alias Command Promt atau lebih dikenal dengan "perintah DOS"
  • Bisa digunakan untuk versi 32-bit atau 64-bit Windows 7
  • dan beberapa fitur minor yang lain.

Cara Mengganti Tampilan BootScreen di Windows 7


Anda bisa men-download dari coderforlife atau softonic untuk installer-nya.

Sebelum menjalankan Windows 7 Boot Updater, ketikkan dulu perintah berikut di command prompt:
bcdedit /set {bootmgr} testsigning on
  1. Fungsi perintah diatas adalah untuk mengaktifkan fitur test signing untuk menghindari masalah aatau resiko sistem crashx20ketika mengunakan perangkat lunak ini
  2. Unduh dan jalankan program untuk mengganti tampilan BootScreen Windows 7 anda.
  3. Jika anda mau menganti tulisan “Starting Windows” dan “© Microsoftx20Corporation” cukup centang kotak “Text” dan ketik-kan kata-kata di kotak “Message” dan “Copyright”, kemudian klik tombol “Go”. Prosesnya akan memakan waktu beberapa detik, jika sudah selesai anda dapat merestart untuk mencoba melihat hasilnya.
  4. Jika anda mau mengganti gambar animasi windowsnya, anda harus mempumnyai gambar animasi bootscreen sendiri, animasinya harus berupa sebuah file berukuran 200x1200px format file BMP yang berjumlah 105 file berformat BMP, PNG, GIF, atau TIFF, yang di simpan di dalam sebuah folder. Tentu saja anda bisa membuatnya sendiri. Atau anda bisa menggunakan BootScreen windows 7 Kustom yang sudah jadi
  5. Centang kotak “Animation” dan pilih lokasi folder dimana terdapat file-file gambar yang sudah anda siapkan sebelumnya kemudian klik tombol “Go”, prosesnya hanya memakan beberapa detik, kemudian kita dapat merestart sistem untuk melihat hasilnya
