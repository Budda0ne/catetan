Axioo PicoPad 5 anda belum rooted? Yuk kita root. Kali ini kebetulan ada teman yang tidak puas dengan PicoPad 5 miliknya, dan meminta bantuan kepada saya untuk me-modifikasi (mod) Axioo PicoPad 5 GEA-nya (yang ternyata sudah di-root). Untuk mod Axioo PicoPad 5 tentu harus di-root terlebih dahulu. Mari kita bahas proses rooting Axioo Picopad 5.

Pada dasarnya, cara melakukan rooting pada Axioo Picopad 5 ini sama dengan rooting pada perangkat lain yang mempunyai prosesor MT657. Namun, cara rooting kali ini lebih ringkas, simpel, mudah dipraktekkan pemula. Kenapa? Karena perintah/script ADB-nya dirangkum dalam file .bat sehingga berjalan secara otomatis.

Sudah siap? Ikuti langkah-langkah rooting berikut ini:
  1. Download file yang diperlukan di sini (2.6 MB).
  2. Extract file yang barusan anda download ke folder manapun di PC.
  3. Buka folder yang tadi Anda extract, perhatikan file berikut:
    • RootExplorer.apk
    • Superuser.apk
    • TPSparkyRoot.bat
    ketiga file tersebut adalah file yang diperlukan untuk proses root-ing
  4. Rooting Axioo Picopad 5
    • Install RootExplorer.apk dan Superuser.apk dalam ponsel Picopad 5
    • Kemudian jalankan mode “Debug USB” dengan cara:
    • Masuk “Pengaturan”
    • Tap pada “Pilihan pengembang”
    • Centang “Debug USB”
    • Ketika muncul cendela pop-up “izinkan debug USB”, tap Ok.
      • Gambar 1 Gambar 2 Gambar3
    • Hubungkan Picopad 5 ke PC melalui kabel USB.
    • Jika Anda baru pertama kali menyambungkan Picopad ke PC via kabel USB dalam mode debug USB, maka PC akan melakukan instalasi driver. Tunggu sampai driver selesai diinstall.
    • Setelah proses instalasi driver selesai dan Picopad 5 dapat dikenali PC, jalankan “TPSparkyRoot.bat “ dengan cara double clik pada file tersebut.
    • Maka cendela “Windows cmd” akan terbuka, di sini Picopad anda akan melakukan reboot
    • Jangan panik, perhatikan pada jendela cmd tertulis “Continue once device finishes rebooting. Press any key to continue”
    • Tunggu sampai Picopad Anda selesai booting dan meload Android dengan sempurna, baru tekan sembarang tombol pada keyboard PC/Komputer untuk melanjutkan proses.
    • Picopad 5 akan melakukan reboot untuk kedua kalinya. Tunggu sampai Android terbuka sempurna.
    • Klik sembarang tombol keyboard untuk melakukan root, atau tekan ctrl+c untuk menggagalkan.
    • Jika anda memilih melakukan root, maka ponsel anda akan melakukan proses reboot untuk kali ke 3.
    Selamat, Axioo Picopad 5 Anda sudah di-root.
Cara ini tidak hanya berjalan untuk Axioo Picopad 5, namun juga bisa dipraktekkan pada perangkat lain yang menggunakan prosesor MT6575 seperti Lenovo S880 dan Treq A10 Pocket II.
