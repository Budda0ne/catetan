Berbagi File Torrent


Anda mungkin sudah tak asing lagi dengan istilah Torrent, file yang digunakan sebagai media pertukaran file antar komputer ini. Pada dasarnya file torrent merupakan sekumpulan informasi yang memuat data file, pelacak, dan pc yang sudah dan sedang melakukan download. Dari file inilah, situs tracker seperti KAT, TPB, dan yang lain melakukan tugasnya, bertukar informasi seputar file yang file-sharing-kan.

Tutorial Membuat File Torrent

Berbeda dengan artikel sebelumnya, yang memuat cara download file Torrent, kali ini saya ingin menyampaikan kembali dasar-dasar langkah pembuatan file torrent dan bagaimana membagikannya di Internet. Ada 7 langkah dasar dalam membuat torrent kita siap dan bisa dibagi dengan rekan-rekan lain lewat dunia maya, yakni:
  1. Pilih program Torrent yang tepat
    2. Ada sekian banyak penyedia Torrent Client. Ada sebagian yang sudah disertai dengan fasilitas untuk membuat file torrent namun ada juga yang tidak. Program torrent gratis seperti μTorrent untuk Windows, Transmission untuk Mac dan Deluge untuk Linux sudah mendukung pembuatan file .torrent.
  2. Klik pilihan untuk membuat file torrent dari Menu.
    3. Biasanya pilihan ini ada di bawah menu File dan berjudul “Create New Torrent”. Jika Anda tidak menemukannya, silakan coba cari di bawah menu lain atau di bagian Toolbar. Ada juga yang menggunakan fasilitas drag-and-drop untuk mengaktifkan fasilitas untuk membuat file baru.x20
  3. Pilih file yang akan Anda share lewat torrent
    4. Jika Anda bermaksud membuat torrent dari beberapa file, alangkah lebih baik jika Anda mengelompokkannya dalam satu folder terlebih dahulu sehingga nanti Anda tinggal memilih folder itu saja. Biasanya nama file torrent Anda akan mengikuti nama folder yang Anda masukkan. Tapi tenang saja, nanti Anda bisa mengubahnya kembali.x20
  4. Masukkan situs tracker
    5. Situs tracker bertugas mengatur transfer data torrent Anda. Nama situs tracker mirip sekali dengan URL pada umumnya. Berikut sejumlah situs tracker yang banyak dikenal :x20 Jika Anda ingin torrent Anda terdaftar dalam suatu situs download torrent, Anda perlu menambahkan tracker dari situs tersebut. Jika Anda menggunakan tracker personal (meski kecil kemungkinan), pastikan Anda memilih check box mark it as private.
  5. Tambahkan informasi tambahan (additional information) jika perlu
    6. Anda bisa memasukkan informasi detail tentang file dan Anda sendiri. Sebaiknya Anda biarkan bagian “piece size” apa adanya. Setelah semua kolom terisi, Klik Create untuk membuat file .torrent anda. Selanjutnya mungkin Anda akan diminta mengisi lokasi penyimpanan file .torrent Anda tersebut.
  6. Jika .torrent sudah siap
    7. Tugas Anda berikutnya adalah seeding. Artinya Anda tidak boleh mengubah apalagi menghapus file yang Anda buat torrent-nya tadi. Jika Anda tidak melakukan seeding, file torrent Anda tidak akan ada artinya, karena tidak ada yang bisa mendownload file Anda.
  7. Bagi file Torrent Anda
    8. Jika Anda bermaksud membuat torrent agar rekan Anda bisa menerima file tertentu dari Anda, kirim saja file .torrent tersebut ke email rekan Anda. Namun jika Anda ingin membagikannya dalam sekup yang lebih luas, Anda bisa mendaftarkannya dalam situs download torrent. Mungkin Anda akan diminta untuk mendaftarkan diri terlebih dahulu sebelum bisa menambahkan/mengupload file torrent Anda. Tiap kali ada downloader baru, maka Anda akan mendapatkan seeder tambahan untuk file .torrent Anda.
