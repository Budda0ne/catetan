XBOOT adalah Tool untuk membuat Multiboot ISO USB dari Windows. Tool ini merupakan aplikasi berbasis Windows yang dapat digunakan untuk membuat Multiboot Live USB dan juga Multiboot ISO file yang selanjutnya dapat dibakar ke CD / DVD. XBOOT mendukung banyak Distribusi Linux, XBOOT juga memberikan keleluasaan dalam menentukan pilihan, apakah ingin menggunakan bootloader Grub atau bootloader syslinux. Hasil dari XBOOT juga cisa digunakan di emulator QEMU (Anda bisa mem-booting ISO dari Windows).

Maksud dari Live USB Multiboot adalah dalam satu USB drive terdapat beberapa OS yang bisa di booting melalui pilihan dalam bootloader yang terpasang.

Perhatikan, XBOOT adalah tool untuk OS windows bukan untuk digunakan di linux. Jika Anda menggunakan OS Linux, anda bisa menggunakan MultiSystem utnuk membuat Multiboot Live USB.

Membuat Multiboot ISO USB dari Windows dengan Tool XBOOT
  • Download, extract and run the latest version of XBOOT
  • (1) Drag and drop some ISOs into the XBOOT window (2) Select Create USB
  • (3) Select your USB Drive from the Droplist (4) Select a Bootloader to use (5) Click OK to begin creation
  • A copy notification window appears to keep you informed of the progress.
Ketika proses sudah komplit/selesai, Anda sudah bisa menggunakan Bootable USB Flash Drive yang berisikan beberapa distro Linuxx20sesuai dengan file-file ISO OS yang yang Anda pilih.
