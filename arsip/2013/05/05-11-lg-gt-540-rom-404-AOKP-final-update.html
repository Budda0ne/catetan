Kabar menyenangkan untuk anda pengguna Handheld LG Optimus GT-540: Update ICS AOKP ROM terbaru yakni versi 3. Yang sudah di publikasikan tanggal 10 Mei lalu dengan beberapa fitur ICS AOKP ROM yang sudah diperbaiki dan bisa anda download gratis. Kebetulan, update terbaru dari ICS AOKP ROM kali ini adalah update terakhir untuk Custom ROM yang berbasis AOKP [Android Open Kang] ini.

ICS AOKP ROM ini merupakan kelanjutan dari AOKP DevSwift"s Android 4.0.4 yang dikembangkan PolishVodka. Custom ROM ini sudah sangat sempurna meski perlu beberapa upgrade lagi. Tetapi secara keseluruhan ROM ini sudah berfungsi dengan baik dan tanpa masalah.

Perubahan dan Perbaikan ROM 4.0.4 AOKP Final Edition vol.3

Update Terakhir Android Open Kang Project untuk GT 540 Optimus

Dengan munculnya update ketiga (dan terakhir dari PolishVodka) untuk Android Open Kang Project LG GT540 kali ini berarti seluruh fitur-fiturnya sudah Fully working. Dan dinyatakan sudah stabil dan menjadi Custom ROM tercepat untuk Android 4.0.4 GT540!

Meskipun kelihatannya PolishVodka tidak menyertakan Swap, sistem-nya sudah mampu melakukan multitasking dengan free RAM yang lebih lega. Anda bisa menjalankan Music Player dengan EQ + Root Explorer + Gallery + Browser + Calculator + Calendarx20yang berjalan di background tanpa menutup salah satu aplikasi, meskipun tanpa swapping.

Untuk menggunakan AOKP Final Edition vol.3 di Hanheld GT-540 Optimus anda, anda harus memiliki setidaknya partisi sistem sebasar 165 MB
Berikut langkah untuk meng-install Cutom ROM AOKP ICS Final Edition ini
Masuk ke Recovery Mode CWM
Back-up ROM anda terlebih dahulu
Wipe data/factory reset
Mounts and storage --> format system , sdext , boot
Pilih file GT540_AOKP_FinalEdition_vol.3.zip dari sdcard
Install file tersebut
Reboot system setelah selesai
Reboot sekali lagi.
Selesai.

Fitur Ekstra lain yang bisa di tambahkan untuk LG GT-540

DevSwift's kernelx20- Anda bisa mencoba Original ROM kernel DevSwift_kernel.zip tetapi mungkin beberapa fitur tidak bekerja sebagaimana mestinya. Jika ingin kembali menggunkan kernel milik Miroslav anda tinggal Flashing dengan file Restore_Miroslav_kernel.zip.

BTDialingFix - Gunakan jika memang anda menggunakan headset bluetoth untuk melakukan panggilan.

12dalvikcache.txt - Rename dulu menjadi 12dalvikcache saja tanpa ekstensi txt, pindahkan ke system/etc/init.d dan rubah file permissionnya. Reboot. Script ini menciptakan folder dalvik-cache dalam cache partition dan memindahkan dalvik-cache milik aplikasi system. Dengan ini anda punya ruang tambahan dalam ROM untuk meng-install aplikasi. Dan anda perlu hati-hati saat menghapus dalvik-cache ketika dalam mode recovery.

Memory_optimizer - Download dari XDA Flash dengan CWM setelah Install cusrom.

DSPManager - Fitur ini dibuang dari ROM untuk menambah perfoerma, hanya jika anda membutuhkan saja. Bawaan dari ROM ICS: MusicFX from sudah bagus saat digunakan dengan Music Playr default.

wpa_supplicant_adhoc & wpa_supplicant_standard - untuk Ad-Hoc Wi-Fi. dan jika HH anda tidak bisa menemukan AP Wi-Fi

key_original - untuk mengembalikan fungsi tombol sesuai fungsi aslinya.
