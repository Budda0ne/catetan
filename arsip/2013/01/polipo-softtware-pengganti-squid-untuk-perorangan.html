Tapi masih ada lagi aplikasi yang cocok menjadi alternatif squid dan handy cache untuk perseorangan atau digunakan dalam kelompok kecil pengguna. Software ini adalah Polipo.

Polipo, adalah sebuah software proxy cache yang ringkas, cepat dan efektif untuk pengguna perorangan atau singgle user. Software ini cocok untuk pengguna modem Smartfren dan Flexy atau provider dial-up lainnya. Meski di rancang untuk pengguna tunggal, Polipo juga bisa digunakan bersama-sama dalam kelompok kecil, misalnya sebagai proxy cache dalam warnet kecil.

Polipo ini menggunakan metode yang unik, yakni Polipo akan menggunakan Pipelining HTTP/1.1 jika remote server (maksudnya web host dari suatu situs) mendukung pipelining. Jadi, otomatis kita mempunyai banyak saluran untuk arus data. Hal ini merupakan keunggulan tersendiri dibandingkan dengan Squid yang mejalankan koneksinya dengan koneksi persisten/konstan.

Polipo juga mampu menegoisasikan bentuk koneksi dari client (maksudnya komputer kita) ke remote server untuk menggunakan HTTP/1.1 meskipun web browser kita meminta koneksi HTTP/1.0. proses ini juga berjala otomatis. Jadi kita tidak perlu repot mengatur seperti saat menggunakan squid maupun handycache. Dan satu lagi, Polipo akan meneruskan meminta data meskipun secara tiba-tiba kita memutuskan koneksi kita. Sehingga ketika kita membuka kembali halaman itu sudah dalam kondisi ter-cache dan otomatis akan mempercepat waktu loading halaman (atau mungkin proses update software? misalnya, game online.)
