Similarity adalah program yang berguna untuk membantu Anda menemukan dan menghapus file yang serupa, file musik duplikat (MP3, WMA, OGG, ASF, WAV, FLAC, APE, WV) dengan konten suara yang sama atau serupa, tag musik (ID3, WMF, Vorbis ). Similarity adalah program terbaik di mana kesamaan bagian suara diperiksa, bukan tag musik saja atau isi file (perbandingan byte ke byte).

Selain itu, Similarity bisa menganalisis artis, judul, informasi album dari tag musik yang terdapat dalam sebuah file audio dan membandingkannya dengan entri yang sama pada file audio lainnya.
Anda dapat menyesuaikan sensitivitas kriteria pencarian Similarity untuk menemukan file yang duplikat atau sama. Dengan cara ini memungkinkan untuk mengidentifikasi judul yang sama dengan perbedaan yang lebih kecil. Duplikat file dapat dihapus atau diakses langsung.

Daftar file duplikat dapat diurutkan, diekspor ke playlist, dan anda tentukan untuk diapakan.

Fitur yang di miliki Similarity antara lain:
  • Tiga algoritma pembanding yang kuat (berbasis konten, berbasis tag dan experemental).
  • Mendukung format lossy: MP3, MP2, OGG, WMA, ASF.
  • Mendukung format lossless: FLAC, APE, WF, WMA.
  • Dukungan ID3v1, ID3v2, ASF, WMA, tag Vorbis.
  • Cari file dengan konten suara yang sama (bukan perbandingan byte ke byte , dengan menggunakan analisis suara).
  • Pemindaian konten suara yang sangat cepat .
  • User friendly.
  • Bekerja dengan perangkat removable media seperti USB, dll
  • Mencari di PC lokal dan Jaringan.
  • Mencari duplikat dari set-folder yang dipilih .
  • Menampilkan tag file dan properti.
  • Hasil dapat diekspor ke playlist.
