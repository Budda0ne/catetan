Weit.. kok tiba-tiba share aplikasi JAVA? Ceritanya, kebutulan ada teman yang masih setia dengan HP Nokia-nya yang hanya support aplikasi JAVA dan bertanya: "Apa file PDF bisa dibuka di HP JAVA?" saya katakan bisa kalau memang ada aplikasi untuk membaca PDF berbasis JAVA. Saya coba browsing dengan kata kunci "Mobile PDF Reader" dan "PDF Reader for JAVA" akhirnya saya menemukan beberapa Aplikasi untuk membuka file PDF.


aplikasi pembaca PDF ini berlisensi shareware dan freeware. Walaupun cukup lama untuk membuka satu file tapi cukup bermanfaat untuk membuka artikel-artikel ber-format PDF. Aplikasi ini saya anggap sebagai e-book reader untuk hape java :D . Adalah "Mobile PDF Reader" yang sebelumnya bernama "PDF Viewer" atau lazim disebut sebagai Adobe Reader Mobile.

Download Mobile PDF Reader (Java)

Diantara Aplikasi e-book reader yang ada, yang bisa membuka file PDF secara meyakinkan ada Mobile PDF Reader dan bisa didownload secara gratis melalui link download .jar dari 4shared atau dari link cadangan. Aplikasi ini mempunyai kelemahan dalam mebuka file pdf lumayan lama, tetapi cukup bagus.

Adobe Reader Mobile Java ini di buat oleh Zesium Software, aslinya bernama "PDF Viewer". Disebut Adobe Reader Mobile karena aplikasi ini memiliki tampilan icon dan startup seperti Adobe Reader versi komputer/PC. Tampilan-nya juga dinamis dan cukup menarik, tak ubahnya handphone java jadi seperti komputer.Anda bisa download secara gratis dari sini.

Jangan salah ya, aplikasi JAVA juga disupport oleh sistem operasi Android. Jadi bisa dicoba juga di Smartphone Android. Memang fungsi Adobe Reader Mobile tidak secanggih aplikasi PDF Smartphone, namun Adobe Reader Mobile sudah bisa membantu dalam membuka dokumen yang berektensi “PDF”.
