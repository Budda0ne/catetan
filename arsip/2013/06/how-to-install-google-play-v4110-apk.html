Cara meng-install Google Play v.4.1.10 APK



Langkah 1: Download file APK Google PLAY Store v4.1.10 dari link berikut ini dan simpan. Untuk lebih mudahnya, download langsung dari handphone atau tablet anda. dan jangan merubah nama file-nya.

Step 1: Download the Google PLAY Store v4.1.10 APK file from this link then save to your computer. To make things much easier, you can directly download the file on your Android tablet or smartphone. Do not rename the file and leave it as it is.

Langkah 2: Setelah Download file APK Google PLAY Store v4.1.10 selesai, colok-kan perangkat ke komputer dengan kabel usb. Biasanya termasuk dalam paket pembelian.

Step 2: After you have downloaded and saved the file on your computer, connect your device to your computer using a USB data cable. This is the cable that you got from the time that you’ve purchased your tablet or phone.

Langkah 3: Yang harus ada lakukan sekarang adalah meng-kopi file ke folder sdcard atau memori eksternal prangkat anda.

Step 3: Now, the next thing that you must do is copy the file from your computer then save to the root directory of your device. It would be best if you save the APK file on your phone’s SD card.

Langkah 4: Buka menu > Setting > Aplikasi. Centang opsi "Sumber tak dikenal" agar ponsel/tablet mengizinkan kita untuk meng-instal aplikasi dari sumber selain Play store-nya google.

Step 4: Go to Settings > Applications. Check the box for ‘Unknown Sources’ to allow your phone to install files that weren’t downloaded from the Google PLAY Store.

Langkah 5: Buka File Manager atau File Explorer, cari file ‘com.android.vending-4.1.10.apk’ yang sudah anda kopi tadi.

Step 5: Launch your device’s File Manager (also known as File Explorer) then locate the APK file that you have downloaded and copied to your tablet or smartphone. Better yet, search and select the file called ‘com.android.vending-4.1.10.apk’.

Langkah 6: Untuk meng-install, buka file dan pilih 'Terima'. Ikuti petunjuk yang di tampilkan. Jika sudah selesai, Google Play Store sudah ter-update

Step 6: To install the file, tap on the ‘Accept’ button. Follow the on-screen instructions for the Google PLAY Store app to be updated to its latest version.

Sekarang anda bisa men-cek apakah sudah ter-update atau belum dengan membuka aplikasi Google Play Store dan membuka bagian Setting. Scroll ke bawah sampai anda menemukan menu 'Tentang' lihat apakah sudah menjadi 4.1.10.

That’s it! If you do want to make sure that your app’s version has been successfully updated to v4.1.10, you can check by launching your app. Click on the Menu key to go to Settings then scroll until you see your app’s ‘About’ section. Check the Build Version and see if it’s on 4.1.10. You can also check the new features.
