Banyak dari kawan saya yang menjalankan usaha warnet mengeluhkan bahwa komputer di warnetnya ter-install begitu banyak software-software yang tidak berguna yang di-install oleh pelanggan mereka.Dan dari sekian banyak aplikasi itu membuat komputer semakin berat. Bahakan terkadang pelanggan yang nakal menggunakan aplikasi-aplikasi untuk mem-bobol billing yang mereka pasang. Solusinya adalah dengan memblokir akses file berekstensi .exe.

Ada banyak cara untuk memblokir akses file .exe. Salah satunya dengan software berikut, yaitu Trust-No-Exe. Sebenarnya software Trust-No-Exe bisa digunakan untuk banyak keperluan atau fungsi, antara lain bisa untuk:

- Memblokir akses file .exe
Bagi yang punya warnet, yang tidak ingin user semaunya sendiri menginstal atau pun menjalankan program di komputer warnetnya, bisa menggunakan software Trust-No-Exe. Misalnya: agar user tidak semaunya sendiri mengistal program download manager yang menyedot bandwith, atau program hack atau ilegal seperti netcut.
Atau bisa juga untuk komputer atau laptop pribadi, agar tidak sembarangan orang menginstal software di komputer atau laptop, termasuk software portable tidak bisa dijalankan.
- Mengamankan dari Virus
Virus tidak bisa masuk atau menginstal dirinya begitu saja ke komputer. Misalnya ketika flashdisk dicolok ke komputer, dan ternyata di flashdisk ada virus yang berjalan melalui autorun, secara otomatis software Trust-No-Exe akan memblokir akses virus tersebut.
- Mengamankan dari Worms/Trojan
Saat berinternet browsing, buka e-mail, chatting lebih aman. Karena terkadang ketika membuka website, e-mail, chatting, hacker ataupun cracker menyisipkan file berbahaya alias virus, yang terdownload secara otomatis ataupun tanpa disadari file-file berbahaya tersebut bersarang di Local Setting/Temp yang sewaktu-waktu bisa menginfeksi komputer.
