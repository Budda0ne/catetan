Microsoft masih merupakan yang terbesar dalam industri teknologi komputer. Mungkin ada beberapa kompetitor (meskipun jumlahnya sedikit) yang mampu mengangkat suara menantang raksasa komputer yang satu ini. Microsoft Windows masih tetap menjadi nama yang paling diminati di antara beberapa sistem operasi. Meskipun ada beberapa sistem operasi yang tersedia saat ini, Microsoft Windows masih memimpin di urutan teratas dengan basis pengguna yang memang luas.

Kesederhanaan dan kemudahan dalam penggunaannya adalah dua faktor penting yang membuat orang tertarik untuk menggunakan sistem operasi komputer dari Microsoft. Microsoft selalu berupaya untuk menambahkan fitur-fitur/kemampuan baru kedalam sistem operasi Microsoft Windows.

Seperti yang semua orang sudah ketahui, Microsoft Windows 7 adalah versi terbaru untuk keluarga Windows. Ini adalah sistem operasi Microsoft Windows yang menjanjikan ketangguhan dan kecepatan. Dengan tampilan yang menarik, sistem operasi ini dengan mudah menarik banyak mata untuk memperhatikan, atau paling tidak 'melirik'.

Namun, ada kecenderungan untuk komputer yang menjalankan Microsoft Windows semakin lambat secara bertahap. Komputer yang semakin lambat dalam periode tertentu permasalahan umum yang ada pada Microsoft Windows Vista, tetapi ketika muncul Microsoft Windows 7, sepertinya sudah tidak lagi. Berikut ini beberapa cara untuk mempercepat komputer yang bisa kamu coba sendiri.

Memeriksa Spesifikasi Hardware Sistem Anda:

Nah, setiap kali Anda menginstal sistem operasi baru dan komputer mulai melambat, hal pertama yang harus diketahui adalah: apakah komputer memenuhi semua persyaratan perangkat keras yang diajukan oleh Microsoft untuk sistem operasi tertentu. Dalam kebanyakan kasus, sebuah perangkat keras yang tidak memenuhi spesifikasi minimum dalam mendukung sebuah perangkat lunak adalah pokok permasalahannya.

Berikut ini adalah spesifikasi minimun perangkat keras untuk bisa menginstal dan menjalankan Microsoft Windows 7:
  • Prosesor:  1GHz
  • RAM 1GB
  • HDD: 16 GB
  • VGA Card: Support DX9
Jika komputer kamu memenuhi kriteria diatas, selanjutnya adalah memeriksa faktor-faktor lain dalam sistem operasi itu sendiri.

Membersihkan file 'sampah' yang ada di Sistem Operasi Windows:

Pertama-tama, cobalah untuk menghapus temporary files and folders dari komputer. File-file temporer dapat diakses dengan menggunakan perintah, 'Temp' dan '% temp%' di start > run. Sekalian, hapus juga cookie dan bersihkan file-file prefetch.Kedua folder dapat juga diakses dengan memasukkan perintah, 'cookie' dan 'prefetch' di start > run.

Menghapus semua file-file yang tidak berguna ini akan memberikan ruang bebas yang dibutuhkan oleh komputer dan sistem operasi. Kamu juga bisa menggunakan Disk Cleanup Utility untuk pembersihan hardisk untuk cara mudahnya.

Faktor lain yang dapat mempengaruhi kinerja komputer adalah adanya program yang tidak diinginkan. Uninstall semua program yang kurang berguna dari komputer. Dalam hal ini, mungkin ada beberapa program yang berjalan di background komputer kamu yang mungkin tidak langsung terlihat. Untuk menutup program ini, kamu bisa menggunakan Microsoft Windows Task Manager atau msconfig utility.

Menonaktifkan program-program ini menggunakan msconfig utility akan meningkatkan kinerja dan kecepatan komputer secara signifikan. Ini adalah beberapa langkah-langkah pemecahan masalah dasar dan lanjutan yang dapat Anda lakukan saat komputer Anda menjadi semakin lambat dan kurang reaktif atau biasa kita kenal dengan istilah 'lemot' atau 'lola'.
