Versi yang dibocorkan teridentifikasi sebagai Windows 8 Enterprise N - "N" adalah tanda bahwa versi tersebut ditujukan untuk pengguna Eropax20- di situs Web BitTorrent, yaitu sebuah situs file-sharing, tidak jelas apakah versi yang di-klaim sebagai Windows 8 Enterprise N yang dibocorkan itu benar-benar ASLI. Yang saya maksud asli adalah: Apakah akan sesuai(valid) dan sesuai dengan yang akan drilis pada tanggal 15 Agustus nanti.

Meskipun beberapa user yang men-download Windows 8 Enterprise N versi leaked menyatakan bahwa itu versi/build9200 yang tidak ASLI, namun pada Jumat pagi-nya dinyatakan bahwa versi bocoran terebut adalah "real deal".

Legitimate. It's Work, look real kata salah satu commenter di situs file-sharing populer itu. Banyak Downloader mem-posting screenshot untuk mendukung pendapat mereka bahwa versi leaked itu adalah Windows 8 RTM ASLI.

Yang lain-nya juga mendukung pendapat tersebut, termasuk "Canouna," user yang digunakan oleh administrator forum WinLeaked. Sedangkan di forum-forum diskusi lokal semacam kaskus juga sudah muncul tread khusus yang membahas tentang windows 8 yang membahasnya juga menyediakan link downloadnya.

Salinan yang dibocorkan itu berbahasa Inggris, tetapi menurut situs file-sharing tersebut, iso itu non-boot, yang berarti pembajak harus membuat media untuk instalasi. Petunjuk untuk membuat installer berbasis DVD atau USB flash drive juga diposting dix20situs file-sharing tersebut, dan di beberapa situs lain juga mulai menerbitkan tutorial instalasi Windows 8 Enterprise N Leaked.

Kebocoran dari rilis Windows mendatang ini sangat tidak biasa. Pada menjelang peluncuran Windows 7 tiga tahun lalu, versi bajakan muncul secara reguler, dari kebocoran Oktober 2008 sebuah "alpha" versi Microsoft hanya diperuntukkan bagi pengembang untuk Windows 7 Release Candidate, yang go public sekitar dua minggu sebelum Microsoft secara resmi meluncurkan preview.

Pihak Microsoft sendiri telah melakukan banyak tindakan pencegahan untuk Windows 8, meskipun demikian edisi Cina dari sistem operasi ini sudah mulai ada di Internet tiga hari sebelum Microsoft mengirimkan Release Candidate, yakni preview untuk publik versi terakhir, pada akhir Mei.

Sebelumnya, Microsoft telah memperingatkan pengguna untuk tidak menginstal Windows 8 Leaked build. Microsoft mengklaim versi ini sanagat mungkin terinfeksi dengan malware.

Perusahaan banyak melakukan pencegahan agar Windows 8 tidak bocor, meskipun edisi Cina dari sistem operasi ini dibuat untuk Internet tiga hari sebelum Microsoft mengirimkan Release Candidate, preview publik terakhir, pada akhir Mei.

Sebelumnya, Microsoft telah memperingatkan pengguna untuk tidak menginstal build leaked, mengklaim bahwa mereka dapat terinfeksi dengan malware.

Beberapa pelanggan dapat memperoleh salinan non-bajakan Windows 8 secepatnya 15 Agustus ketika RTM, atau "release to manufacturing" milestone (kode yang sama seperti yang akan diinstal pada PC baru) sampai pada MSDN (Microsoft Developer network) dan TechNet Services

Perusahaan yang mempunyai lisensi Software Assurance (Software Assurance pada dasarnya adalah anuitas yang memberikan perusahaan hak untuk menjalankan versi produk) akan bisa mendapatkan Windows 8 mulai 16 Agustus sebagai anggota yang akan dari Jaringan Mitra Microsoft.

Windows 8 akan diluncurkan 26 Oktober, dengan dua edisi ke pasaranx20- Windows 8 dan Windows 8 Pro - dan lainnya, edisi Enterprise yang bocor kemarin, menargetkan perusahaan dengan volume licensing agreements.
