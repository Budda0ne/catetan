Pengertian dan Fungsi HJSplit


HJSplit adalah sebuah program untuk memecah suatu file yang berukuran besar menjadi bagian-bagian kecil dalam waktu yang jauh lebih cepat dibandingkan melakukannya dengan Split Rar (menggunakan Program archiver WinRAR). HJSplit ini jg bisa digunakan untuk mengembalikannya seperti semula alias menggabungkan file yang di pecah dengan menggunakan HJ Split juga. Biasanya Hj Split digunakan oleh uploader yang mengupload file dengan kapasitas besar ke File hosting yang memberikan batasan file maksimum yang bisa di upload semacam Dropbox, Mediafire, Jumbo File, 4shared, Indowebster atau untuk mempermudah proses upload agar tidak perlu mengulang dari awal ketika gagal upload (bayangkan, Anda meng-upload file 1 Gb dan error ketika sudah mencapai 500Mb alias macet ditengah-tengah!).

Mendownload HJ Split

Untuk mendownload HJ Split bisa langsung ke homepagenya yakni http://www.hjsplit.org dengan beberapa pilihan software untuk sistem operasi Windows atau Linux. Tetapi Anda juga bisa mendownload dari file yang saya upload ke Mediafire di sini atau melalui File hosting lokal indowenster di sini

Cara penggunaan HJSplit

Untuk melakukan SPLIT (memisahkan file besar ke beberapa file kecil).
1. Double klik program HJSplit.exe (tidak perlu di-Install)
2. Klik Split
3. Klik input file dan browse file yang hendak dipecah
4. Klik output file bila ingin meletakkan file yang dipecah di folder lain (opsional)
5. Tentukan Split file size, misalnya 200Mb
6. Tekan start.
7. Setelah split complete, close. di folder yang sudah di tentukan akan muncul file dengan ekstensi atau akhiran ganda. dan ekstensinya dimulao dengan .001, 002 dan seterusnya sesuai dengan besarnya file yang dipecah
Untuk melakukan JOIN (menggabungkan file hasil dari pecahan HJ Split).
1. Double klik program HJSplit.exe (tanpa di-install)
2. Klik Join
3. Klik input file dan browse file yang ingin digabung (cukup file yg berekstensi .001 dan sisanya akan langsung dicari oleh HJ Split secara otomatis). Tapi Anda harus meletakkan semua file .001 sampai .00X pada folder yang sama.
4. Klik output file bila ingin meletakkan hasil gabungannya di folder lain (opsional)
5. Tekan start.
6. Setelah join complete, close.

Untuk memberi gambaran yang lebih luas, berikut ini adalah contoh nama file HJSplit.
Jika Anda memecah sesuatubanget.avi menjadi 3 bagian, maka file yang terbentuk adalah:
sesuatubanget.avi.001
sesuatubanget.avi.002
sesuatubanget.avi.003
atau secara global bisa di jabarkan seperti ini:
NamaFile.TipeFile.001
NamaFile.TipeFile.002
NamaFile.TipeFile.003

jadi kalau kalian mau menggabungkan 3 file dengan nama (misalnya):
sesuatubanget.exe.001
sesuatubanget.exe.002
sesuatubanget.exe.003
maka file yang terbentuk adalah: sesuatubanget.exe

Lalu, jika Anda masih gagal menggabungkan file-nya dengan HJ Split, kemungkinannya:
1. Urutan filenya salah. Benar2 harus di rename sesuai urutan asli (ini untuk file yang extensinya sudah dirubah, misalnya: sesuatubanget.exe.001 menjadi sesuatubanget-part1.exe. biasanya uploader akan memberikan petunjuk)
2. Pada folder option windows explorer --> view. Jangan centang pilihan "hide extension for known files"
3. Ada file yg corrupt saat di download
4. Anda menggabungkan file yang bukan hasil dari HJ Split (contoh : menggabungkan file hasil download youtube)

Apa itu tombol CHECKSUM?

Checksum ialah fitur untuk me-ngecek atau membandingkan antara file original dengan file hasil gabungan dari pecahan file original, apaka mempunyai nilai Checksum yg sama atau tidak. Jika hasil Checksum sama, berarti "kemungkinan besar" join berhasil dan file tidak corrupt.
