PCMan adalah program Billing Warnet Gratis yang diperuntukan untuk Warnet reguler maupun Warnet Game Online. Ini adalah Billing Warnet Alternatif yang juga GRATIS

Bagaimana cara mendapatkan program PCMan?
PCMan dapat langsung didownload di blog ini.

Apakah setiap menambah atau mengurangi PC Client harus melakukan instalasi ulang?
Tidak, PCMan dapat diatur menurut kebutuhan. PCMan dapat diatur dengan jumlah maksimal 300 client dan minimal 10 Client.

Apa yang membedakan PCMan dengan program billing yang lain?
PCMan memiliki tampilan grafis yang memudahkan pengunaan program billing ini, serta dilengkapi dengan system barcode yang berguna untuk kartu identitas warnet maupun untuk system penjualan.

Apakah kita harus membeli program PCMan?
Tidak, PCMan didistribusikan secara gratis untuk seluruh warnet

Apakah operator dapat merubah settingan PCMan?
Tidak, karena PCMan memiliki system pembagian level untuk mengunakan fitur-fitur di PCMan

Apakah PCMan memiliki system blok site dan blok program?
Ya, PCMan memiliki system untuk blok site dan blok program yang dapat diatur sesuai kebutuhan.

OS apa saja yang bias menggunakan PCMan?
Untuk saat ini hanya OS windows yang dapat dipasang PCMan

Apakan PCMan memiliki system back-up database ?
Ya, PCman memiliki system Back-up otomatis yang dapat diatur sesuai kebutuhan.

Jika PCMan memiliki fitur baru apakah harus mendownload ulang?
Tidak, karena system dari PCMan bersifat online jika PCMan terhubung dengan internet maka akan secara
otomatis mendownload dan memasukan kedalam PCMan anda.
Catatan:Sekarang program ini telah habis masa kontraknya dengan PT.Kreon Indonesia sehingga situs web maupun programnya tidak dapat di aktivasi secara OL lagi. Tapi jangan kuatir, ada patchingnya setelah instalasi sehingga program PCMan tidak akan meminta untuk aktivasi secara OL.
