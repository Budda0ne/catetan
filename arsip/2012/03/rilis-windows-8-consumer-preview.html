Rilis Windows 8 CP
Windows 8 Consumer Preview telah diperkenalkan diajang Mobile World Congres dan sudah dapat didownload oleh publik.

Windows 8 Consumer Preview adalah versi beta yang ditujukan untuk publik sebelum versi final dari Windows 8 dirilis (sebelumnya Microsoft juga sudah merilis Windows 8 Developer Preview), versi ini akan sama atau setidaknya akan mewakili mayoritas fitur yang akan hadir di Windows 8.

Beberapa fitur yang hadir di Windows 8 Consumer Preview adalah Metro UI Style, Windows Store, Internet Explorer 10, integrasi dengan Xbox 360, penyimpanan cloud SkyDrive, dukungan multi-touch, Secure Boot, Native USB 3.0 Support, dll.

Kalau kamu memiliki Laptop/PC yang sudah menggunakan Windows 7 tidak akan mengalami masalah yang berarti saat akan meng-install Windows 8 Consumer Preview. Ini karena spesifikasi minimal Windows 8 Consumer Preview cukup rendah, yaitu:
prosesor 1 GHz, RAM 1GB (32-bit) / 2 GB RAM (64-bit), ruang hardisk 16 GB (32-bit) / 20 GB (64-bit), VGA yg kompatibel dengan DirectX 9 dan WDDM 1.0.

Selain spesifikasi minimum diatas, untuk menjalankan beberapa fitur yg ada di Windows 8 Consumer Preview, ada beberapa persyaratan yang perlu dipenuhi.

Metro UI Style memerlukan minimum resolusi layar 1024×768 dan supaya fitur “snap” di metro bisa berfungsi monitor minimal harus beresolusi 1366×768
Secured Boot memerlukan UEFI BIOS yang tidak dimiliki oleh kebanyakan PC. (UEFI ada di setting BIOS)
Keberadaan Trusted Platform Module (TPM) akan membuat fitur BitLocker berjalan lebih mulus
Hyper-V memerlukan sistem 64 bit yg kompatibel dengan second level address translation (SLAT) dan tambahan 2GB RAM. SLAT bisa diaktifkan melalui setting BIOS.
Game dan software yg memakai resource VGA memerlukan kartu VGA yang kompatibel dengan DirectX 10.
Jika melakukan clean install, sebaiknya download dulu driver PC / Laptop dari website pembuatnya.

Tapi perlu diingat, karena Windows 8 ini masih Consumer Preview (versi beta) maka besar kemungkinan terdapat bug dan juga tidak kompatibel dengan hardware PC / Laptop yang kamu miliki. Tapi jika kamu adalah orang yang ingin tahu dan tidak sabar untuk mencobanya silahkan “Download Windows 8 Consumer Preview” dari situs Microsoft, atau link yang saya sediakan.
