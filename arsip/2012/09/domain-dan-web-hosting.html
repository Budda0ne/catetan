Jika anda mempunyai keinginan untuk mempublikasikan sesuatu melalui internet, anda membutuhkan website sebagai media publikasi yang efektif. Tentu saja dengan beberapa optimasi search engine agar website kita bisa terindeks Google atau Yahoo sebagai mesin pencari yang banyak digunakan (jangan lupakan Bing). Namun sebelum meng-online-kan website kita membutuhkan 2 layanan utama. Yakni Domain Name dan Web Hosting. Kedua jenis layanan ini bisa disewa dari penyedia layanan web domain/hosting.

Di Artikel ini akan saya jabarkan secara sederhana apa itu Domain dan Web Hosting.

Domain Name

Domain Name atau biasa disebut Domain saja, adalah alamat yang kita ketikkan untuk mengakses halaman website seperti google.com atau google.co.id. Harga sewa domain .COM rata-rata Rp 100.000 per tahun. Seperti domain milik saya ini: idlohulhaq.com, kita harus memesannya di registrar semacam godaddy atau hostgator. Khusus untuk domain Indonesia bisa didaftarkan melalui PANDI sebagai registrar domain .ID. Harga sewa yang paling murah adalah domain .WEB.ID dengan harga Rp 25.000 yang ditujukan untuk blog personal dengan syarat pendaftaran cukup KTP/SIM, sedangkan yang paling mahal .CO.ID dan .NET.ID dengan harga Rp 100.000 yang ditujukan untuk perusahaan dan penyedia jasa telekomunikasi dengan syarat pendaftaran Surat Ijin Usaha. Untuk mengetahui persyaratan lebih lengkapnya dan biaya domain Indonesia lihat di sini.

Sebelum melakukan pendaftaran domain yang akan kita pesan akan dicek terlebih dahulu, jika domain tersebut belum terdaftar maka domain tersebut bisa didaftarkan atas nama kita.

Web Hosting

Hosting adalah web server yang kita sewa untuk menyimpan file-file dan database website. Harga sewa web hosting mulai dari Rp 200.000 tergantung dari spesifikasi yang kita inginkan, biasanya pihak penyedia layanan Web Hosting memberikan pilihan dalam bentuk paket berdasarkan kapasitas space harddisk mulai dari 100MB untuk blog sampai unlimited untuk website dengan perkembangan data yang sangat besar.

Spesifikasi web hosting

  • Kapsitas Harddisk : Kapasitas ruang harddisk untuk menyimpan file dan database. Cara memilihnya total file website yang harus diupload ditambah kira-kira berapa pertambahan data selama 1 tahun.
  • Bandwidth : Kuota data transfer per bulan. Misalnya kita mengupload file sebesar 15MB dan pengunjung mengakses website kita dengan total file yang harus didownload sebesar 1MB jadi bandwidth yang sudah terpakai sebesar 16MB. Bandwidth yang besar sangat dibutuhkan bagi website yang memiliki traffic yang tinggi seperti website berita.
  • FTP : FTP (File Transfer Protocol), fitur untuk mengupload file menggunakan FTP Client. Dengan menggunakan FTP upload lebih cepat.
  • Addon Domain : Selain domain utama kita bisa menambahkan domain lain sehingga dalam 1 hosting bisa lebih dari 1 domain
  • Subdomain : bisa menambah subdomain baru contoh saya membuat subdomain j25 dan j15, j25.musaamin.web.id, j15.musaamin.web.id
  • Email : alamat email dengan nama domain sendiri, me@musaamin.web.id
  • Webmail : aplikasi untuk mengakses email dari web
  • POP : fitur untuk mengakses email dari email client seperti Mozilla Thunderbird
  • MySQL : menggunakan database MySQL. Diberikan informasi MySQL versi berapa dan berapa banyak database yang bisa kita buat agar disesuaikan dengan website kita
  • PHP : bahasa pemrograman web seperti yang digunakan oleh cms WordPress. Sama dengan MySQL diberikan pula PHP versi berapa yang dipakai jangan sampai website yang kita membutuhkan versi PHP yang lebih baru.
  • Lokasi Server : umumnya lokasi server ada 2 yaitu Indonesia (IIX) dan luar negeri (USA). Jika target pengunjung dari luar negeri dan bahasa yang digunakan pada website menggunakan bahasa Inggris sebaiknya pilih server USA, jika yang terjadi sebaliknya gunakan saja server Indonesia agar akses website lebih cepat.
Untuk memilih layanan web hosting sebaiknya Anda bertanya dulu dengan kawan-kawan Anda yang sudah pengalaman atau Anda bisa mencari review dari internet tentang layanan hosting yang akan kita sewa apakah layanannya ok, tidak sering down, dan memiliki technical support yang siap membantu kapan saja.
