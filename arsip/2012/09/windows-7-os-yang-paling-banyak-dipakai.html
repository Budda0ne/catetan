Windows 7 menggeser posisi Windows XP sebagai OS yang paling banyak dipakai didunia.

Windows XP yang berumur 11 tahun sudah cukup lama menjadi OS yang paling banyak dipakai di dunia, namun kini menurut data yang diperoleh dari Netmarketshare hal tersebut telah menjadi masa lalu karena posisinya saat ini telah diambil oleh Windows 7 dengan margin yg cukup tipis.

Dari data tersebut diketahui pemakaian Windows XP di PC telah menurun menjadi 42,52% dari sebelumnya mencapai 61,91% pada bulan September 2010. Sedangkan di sisi lain Windows 7 meningkat tajam dari 17,64% menjadi 42,76% dalam kurun waktu yang sama.
Dimana pertumbuhan Windows 7 diperkirakan akan terus meningkat karena semakin banyak user dan perusahaan korporat beralih dari XP ke Windows 7.

Kehadiran Windows 8 pun diperkirakan tidak akan menghentikan pertumbuhan Windows 7, alasannya user yang sudah menginstall Windows 7 kemungkinan tidak langsung mengupgrade ke Windows 8. Ini terjadi karena impresi awal dimana banyak user tidak menyukai absennya fitur Start menu, fitur start up to to desktop, serta kehadiran Modern UI (Metro) yang agak sulit (asing) digunakan bagi masyrakat awam.

Masih dari data Netmarketshare, Windows secara keseluruhan (semua versi) masih menjadi OS paling banyak dipakai dengan 91,77%; diikuti berturut-turut dengan Mac OS X 7,13% dan Linux sebanyak 1,10%.
