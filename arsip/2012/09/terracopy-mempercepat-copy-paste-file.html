Microsoft Windows memang mempunyai fitur untuk menduplikasi dan pemindahan file yang terintegrasi dalam windows explorernya, namun seringkali kita kecewa dengan performanya. Kenapa tidak mencoba tool untuk mempercepat copy paste file yang kecil tapi bermanfaat; Terracopy.

Untuk versi terbaru dari terracopy saat ini adalah Tera Copy Pro 2.27 Free & Full Version.
Saya menggunakan TerraCopy Pro 2.27, dan hasilnya lebih cepat daripada menggunakan Windows Explorer. Ada penambahan fitur yang tidak ada di versi versi TerraCopy sebelumnya yang tidak mampu dalam menangani proses copy-paste dalam jumlah banyak, tapi pada versiTerraCopy Pro 2.27 ini sudah ditambahkan fungsi tersebut. Untuk Anda yang terbiasa menggunakan TerraCopy mungkin sudah tahu kelebihannya. Namun jika Anda belum pernah mencobanya, Anda bisa melihat beberapa fitur yang ditawarkan oleh TeraCopy ini.

Fitur Baru yang ditawarkan TeraCopy versi terbaru ini:
• Added: ‘Remove other’ to remove non-selected files from the list [Pro].
• Added: Better handling multiple copy processes.
• Added: Multiple retries to delete source files and folders after moving.
• Added: Tray notification when adding files to the existing process.
• Fixed: Problem with some non-Latin filenames.
• Fixed: Copied files now have a correct creation time.
• Added: WaitBeforeMinimize option to ini file.

Uniknya lagi TerraCopy juga mendukung Language Interface Bahasa Indonesia. Dan Terracopy 2.27 ini juga bisa di jalankan di Windows 7 maupun Windows 8

TerraCopy Free & Pro
Untuk mencobanya Anda bisa mencoba versi free TerraCopy yang bisa Anda download disini, atau jika merasa kurang puas dengan TerraCopy gratisan masukkan saja serial number Terracopy ini agar bisa menggunakan TerraCopy Pro



Cara memasukkan Key TeraCopy 2.27 :
  1. Install TeraCopy seperti biasa seperti menginstall program lain.
  2. Jika sudah selesai, jalankan Tera Copy.
  3. Copy Serial dari file .txt(ctrl+a kemudian ctrl+c) yang sudah anda download.
  4. Klik More/Lebih lalu klik Get Pro Version/Dapatkan versi pro.
  5. Kemudian klik Enter Registration Key/Masukkan kunci registrasi dan klik OK.
  6. Dan restart Tera Copy.
  7. Selesai.
Catatan: meskipun anda sudah merigistrasikan TerraCopy, anda masih tetap melihat Dapatkan versi pro atau Get Pro Version
