Windows 8 sudah Release, itu artinya Windows 7 pun menjadi relatif jadul, apalagi Windows Vista dan Windows XP, namun karena masih banyaknya pengguna yang masih menggunakan Windows XP (terbukti dari masih banyaknya request/permintaan/pertanyaan tentang lokasi download file iso Windows XP), Meskipun saat ini pengguna Windows XP sudah terlampaui pengguna Windows 7. dan mengenai Windows Vista, saya tidak pernah menggunakannya. itulah kenapa sedikit sekali artikel yang membahas Windows Vista.

Kembali ke topik yang akan kita bahas, Windows Mana yang Terbaik? Windows XP, Vista atau Windows 7. Meskipun Windows 8 sudah release dan sudah bisa di download secara gratis, namun tidak ada salahnya jika kita kembali membahas ketiga sistem operasi yang masih banyak digunakan ini. Maka, kita bahas saja dari...

Windows XP

Windown XP ini merupakan versi Windows berbasis NT (apa itu NT? langsung ke mbak wiki aja dah)
Versi Windows berbasis arsitektur NT itu dikenal dan terkenal dengan stabilitas dan efisiensi yang lebih baik ketimbang versi Windows 9x. Windows XP menyajikan antarmuka grafis yang dirancang ulang secara signifikan, hingga perubahan tersebut dipromosikan oleh Microsoft sebagai tampilan yang lebih user-friendly dari versi Windows sebelumnya.
Sebuah fasilitas manajemen software baru yang disebut Side-by-Side Assembly diperkenalkan untuk memperbaiki masalah "DLL Hell" yang sering muncul pada Windows 9x. Windows XP juga merupakan versi pertama Windows untuk menggunakan aktivasi produk "Windows Product Activation" untuk memberantas pembajakan peranti lunak, meski hal ini menjadi kontroversi. Windows XP juga telah dikritik oleh beberapa pengguna untuk kelemahan keamanan komputer, integrasi beberapa aplikasi seperti Internet Explorer 6 dan Windows Media Player yang sangat ketat, dan untuk aspek-aspek dari standar antarmuka pengguna.

Windows Vista

Windows Vista memiliki daftar panjang untuk ratusan fitur-fitur baru, fitur-fitur yang diubahkan, dan fitur-fitur yang ditingkatkan kemampuannya. Salah satunya adalah Windows Vista memiliki fitur Aero Glass yang mana ini adalah sebuah themes dengan tampilan kualitas tampilan yang sangat bagus. Aero Glass akan menampilkan efek transparan yang sangat halus pada windows, selain itu Aero Glass tsb akan membentuk lengkungan di setiap ujungnya. dan ditambah pula dengan efek bayangan. OS ini juga memiliki tingkat KEAMANAN tertinggi karena dilengkapi fitur "TWO WAY" yang berarti OSnya bisa memonitor NETWORK COMMUNICATION secara inbound ataupun outbond. tetapi OS ini memiliki kekurangan yaitu kurang stabilnya OS yang telah terinstal dengan Hardware, makanya OS ini sering kali HANG jika spedifikasi Hardwarenya hanya standar-standar saja

Windows 7

Windows 7 adalah rilis terkini Microsoft Windows (sebelum Windows 8) yang menggantikan Windows Vista. Windows 7 dirilis untuk pabrikan komputer pada 22 Juli 2009 dan dirilis untuk publik pada 22 Oktober 2009, kurang dari tiga tahun setelah rilis pendahulunya, Windows Vista.
Tidak seperti pendahulunya yang memperkenalkan banyak fitur baru, Windows 7 lebih fokus pada pengembangan dasar Windows, dengan tujuan agar lebih kompatibel dengan aplikasi-aplikasi dan perangkat keras komputer yang kompatibel dengan Windows Vista, dan mungkin juga sebagai fondasi dari Windows 8.

Perbadaan Windows XP, Windows Vista dan Windows 7

Beberapa perbedaan mendasar pada sistem operasi Windows XP, Windows Vista dan Windows 7 di lihat dari:

1. Tampilan
Jelas saja beda, tampilan antara Windows XP, Vista dan 7

Windows 7 dan Vista memiliki Windows Aero yang sangat cantik dan sangat impresif pada awal peluncurannya (dan kini tampaknya mulai tersaingi Metro style dari windows 8). Sedangkan XP tidak memiliki itu. Ini mengingatkan saya betapa dulu saya sangat menyukai tampilan Windows XP daripada Tampilan Windows 98. Start menu pada Windows 7 dan Vista lebih keren tapi sederhana, Untuk mencari program di Windows 7 kita tidak perlu susah payah buka All Program layaknya Windows XP atau kotak Search yang pencariannya lola. Kita cukup mencarinya dari kotak search di Start Menu

2. Kestabilan
Untuk Kestabilan, Windows XP mungkin yang paling unggul, bahkan Microsoft pun menyadari hal itu dan akan terus mendukung Windows XP sampai tahun 2014 nanti, (saya anggap seperti itu karena sampai tahun tersebut XP masih mendapatkan support)

Dan yang paling parah adalah Windows Vista, ada spekulasi yang berkembang bahwa karena alasan itulah Microsoft mengeluarkan Windows 7 yang memiliki tingkat ke stabilan tinggi untuk menutupi kekecewaan pelanggan yang sudah menggunakan Windows Vista. Juga ada rumor, bahwa seharusnya Windows Vista masih belum cocok untuk di komersilkan mengingat daya tahan dan daya kestabilannya yang sangat kurang dibandingkan Windows 7 yang belakangan sudah diakui kestabilannya menyusul service pack yang pertama.

3. Keamanan
Dari rangkuman di atas juga, Windows Vista memiliki tingkat KEAMANAN tertinggi karena dilengkapi fitur "TWO WAY" yang berarti OSnya bisa memonitor NETWORK COMMUNICATION secara inbound ataupun outbond. Dan keamanan pada Windows XP kalah di bandingkan dua OS ini,

Di Windows Vista dan Seven memiliki Sistem UAC (User Account Control), yang akan memeringatkan anda jika sebuah program akan dijalankan. Fitur ini dapat anda atur di Control Panel sesuai keinginan. Jadi, jika ada virus yang masuk ke Windows 7 dan virus tersebut menjalankan sebuah program, maka secara otomatis UAC akan muncul dan memblokir program dan virus tersebut karena dijalankan tanpa seijin pengguna komputer. Sistem UAC (User Account Control) pada Windows Vista tidak bisa dimatikan dan terus berjalan. Ini sangat mengganggu, karena setiap menjalankan program, anda harus klik OK pada setiap peringatan. Tidak dengan Windows 7, anda bisa mengaturnya agar UAC dimatikan saja, atau dihidupkan sesuai dengan level keamanan. Mau pilih aman, atau pilih sangat aman, atau pilih sangat-sangat aman (saya menganggapnya sebagai tindakan paranoid :D) yang bisa anda atur sendiri.

Windows XP dalam keamanannya kurang unggul dikarenakan sistem firewallnya yang belum di tingkatkan level keamanannya atau di mutakhirkan. Dan juga Windows XP belum memiliki Windowsx20Defender yang siap memproteksi anda kapan saja. Makannya aktifkan Windows Automatic Update, agar Windows kita selalu terupdate dengan hotfix-hotfix dan patch-patch system baru Microsoft. Dan inilah point terpentinnya: windows XP begitu tergantung terhadap Patch-patch ini!! inilah kenapa saya lebih suka menyarankan anda meninggalkan Windows XP

4. Kecepatan
Kecepatan, Ini juga merupakan faktor penting dalam memilih OS. Selain di pengaruhi oleh spek Hardware kita, si OS itu sendiri yang juga mempengaruhi kecepatan komputer dalam mengakses data atau program.
Soal kecepatan berdasarkan hardware standar untuk Windows XP, dan hardware standar untuk Windows Vista dan Seven hasilnya adalah yang tercepat Seven, kedua Windows XP, dan ketiga Windows Vista.. (kata sebuah forum xD)
Ini juga di dukung dengan kemudahan akses Seven dengan viturnya, seperti yang saya sebutkan di atas.

Windows 7 dapat digunakan dengan mudah karena Start menu sudah didesain ulang agar terlihat lebih cantik namun lebih powerful (maksudnya lebih ber-daya-guna). Sekarang kita tidak perlu membuka windows explorer untuk mencari sesuau, cukup dengan buka Start menu pada Windows 7 dan anda dapat melakukan pencarian. Dan ada juga fitur Jump List pada taskbar, sehingga untuk mengakses program yang penting dapat diakses dengan mudah.
Sama dong seperti Start menunya Vista? Yep, anda benar, tapi tunggu dulu, Windows Search pada Windows 7 lebih cepat di bandingkan pada Windows Vista yang leletnya sangat mengganggu (bahkan membuat jengkel).

5. Penggunaan Hardware

Spesifikasi minimum Windows XP, Vista, dan 7

Spesifikasi Windows XP:
-Prosesor 233 MHz
-RAM 64 MB
-VGA 24 MB
-3GB Free Harddisk Space

Spesifikasi Windows Vista:
-Prosesor 800 MHz
-RAM 512 MB
-VGA 64 MB
-15GB Free Harddisk Space

Jika ingin menggunakan tampilan Aero, maka:
-Prosesor 1GHz
-RAM 1 GB
-VGA 128 MB dengan DirectX 9.0 Support

Spesifikasi Windows 7
-Prosesor 1GHz
-RAM 1 GB
-VGA 128 MB dengan DirectX 9.0 Support
-16GB Free Harddisk Space

Spesifikasi di atas adalah Spesifikasi Minimum, Kalau anda ingin lebih lancar ketika menggunakan Sistem operasi tersebut silahkan lampaui spesifikasi minimum di atas. dan kabar terbaru yang lumayan menyenangkan adalah: Windows 8 spesifikasinya hampir sama dengan Windows 7 bahkan ada spekulasi bahwa nantinya spesifikasi Windows 8 tidak setinggi Windows 7.

Kesimpulan: Secara umum Windows 7 memiliki nilai plus terbesar, disusul Windows XP, dan Windows Vista diposisi terakhir. namun dari segi keamanan Windows XP adalah yang paling buruk dalam segi keamanan ketika belum di-update (fresh install, dan belum di-install pacth dan hotfix-nya) dan Windows 7 tetap bearda di tempat pertama dengan keluwesan pemakaian UAC-nya.
