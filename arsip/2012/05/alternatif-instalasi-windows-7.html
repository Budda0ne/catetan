Tentang cara meng-install Windows dengan USB Pendrive atau USB Flashdik yang biasa kita sebut Flashdisk saja.


Cara lain, khusus untuk Windows 7 dapat menggunakan tools dari microsoft, yaitu Windows7 USB/DVD Download tools. Sesudah Download [mirror] dan diinstall lalu jalankan programnya,
  • langkah ke-1 : pilih lokasi file .iso Windows 7 kamu
  • langkah ke-2 : pilih, apakah akan di install ke USB flashdisk atau dibakar ke DVD.

Bonus : 3 (tiga) Tools untuk membuat USB Flashdisk Bootable

Baru-baru ini (sekitar seminggu yang lalu) saya "menemukan" lagi beberapa software untuk membuat flashdisk kita menjadi Bootable Installer --inilah alasan yang membuat saya ingin membuat tutorial install alternatif windows 7

Satu : Setup From USB
Setup From USB
Tools ini bisa di download di sini. Utility ini mengubah flashdisk kita menjadi CD/DVD installer untuk Windows Vista dan Windows 7. Cara pemakaiannya juga cukup mudah, tinggal pilih lokasi drive flashdisk kita dan sumber CD/DVD Windows Vista atau Windows 7 kita.

Dua : EasyBCD
EasyBCD
Ada cara lain lagi untuk membuat flasdisk kita agar bisa dipakai untuk meng-install Windows 7 ialah dengan memakai tool yang bernama EasyBCD
  • Bagi yang belum punya program EasyBCD, silahkan didownload dulu [download link]
  • Kemudian colokkan flaskdisk nya ke port USB komputer kita, lalu format lah dengan memakai partisi FAT32
  • Jalankan program EasyBCD dan klik tombol [Bootloader Setup] di panel sebelah kiri.
  • Dibawah kotak [Create Bootable External Media], pilihlah flashdisk yang akan kita buat untuk menginstall Windows 7.
  • Kemudian klik tombol [Install BCD] dan tunggu sebentar sampai muncul kotak dialog berikut:
  • Klik tombol [Yes] lalu tutuplah program EasyBCD, kemudian masukkan DVD Windows 7 ke komputer/laptop.
  • Setelah itu, copy-lah semua isi DVD Windows 7 tadi ke flashdisk yang telah kita siapkan tadi, jika anda punya file .iso Windows 7, ekstrak dahulu ke folder yang telah kita tentukan, kemudian baru di copy ke flashdisk tadi.
Selesai, sekarang flashdisk-nya sudah bisa dipakai untuk meng-install Windows ke laptop / komputer kita.
