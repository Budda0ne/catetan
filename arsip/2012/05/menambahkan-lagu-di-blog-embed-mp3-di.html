Menambahkan lagu di postingan Blog sederhananya meng-embed-kan File Mp3 di halaman Blog atau di halaman web (internet).

Di beberapa kesempatan, mungkin kita pernah [atau bahkan sering :)] menemui sebuah blog atau website, ketika di buka secara otomatis akan memutar lagu atau musik Mp3. Saya sendiri juga menggunakan trik itu dengan cara meng-embed-kan file Mp3 di dalam posting blog.

Apa fungsinya? Tentu saja untuk menarik minat pengunjung agar betah membuka - buka halaman blog tersebut. Tapi tentu ada resikonya, bagaimana jika pengunjung kita tidak suka lagu yang kita pasang? Tentu ada triknya agar lagu yang kita pasang tersebut bisa dihentikan oleh pengunjung. Sekarang ayo kita coba cara memasang file Mp3 di halaman blog kita

Step by step pemasangan Mp3-nya adalah sebagai berikut:
Langkah Pertama
Siapkan satu link file Mp3 yang bisa di streaming dengan kata lain bisat di akses langsung dari mana saja asal mengetahui link tersebut [biasanya adalah link statis / permalink yang merujuk ke file Mp3 tersebut]
kita ambil contoh saja seperti link yang saya dapatkan dari indowebster ini :
http://cdn.x.indowebster.com/download/90/p16togert31qc3kdvcforrap73.mp3/%5Bwww.indowebster.com%5D-Anda_-_Tentang_Seseorang.mp3
Pada teks yang berwarna ungu, nanti di hilangkan saja karena itu adalah request server untuk menamakan file yang di download menjadi [www.indowebster.com]-Anda_-_Tentang_Seseorang.mp3

Langkah Kedua
Buat postingan baru dan masuk ke mode HTML, jika ingin menempatkan di satu halaman
Buat widget baru dengan tipe HTML/Javascript, jika ingin menampilkan di widget atau sidebar

Langkah ketiga
Tuliskan kode berikut
<object data="http://flash-mp3-player.net/medias/player_mp3.swf" type="application/x-shockwave-flash" height="20" width="150"><param value="#ffffff" name="bgcolor"> <param value="mp3=http://link-file-sreaming.mp3&amp;loop=1&amp;autoplay=1&amp;volume=125" name="FlashVars">
</object>
Kode di atas saya di atas bisa di kembangkan dan di tambahi dengan beberapa variabel lain untuk mengatur cara memainkan file .mp3 tersebut, tetapi untuk sekarang cukup itu saja dulu.
Penjelasan Kode:
Coba perhatikan teks yang berwarna ungu di atas. nilai-nilai tersebut adalah parameter untuk mengatur bagamana nantinya file mp3 akan di mainkan di blog.
  • nilai mp3, mengatur file apa yang dimainkan [dalam hal ini adalh link streaming]
  • nilai loop, mengatur mp3 akan diulang atau tidak, nilai 1 (satu) berarti akan diulang-ulang jika sudah selesai atau habis. Sedangkan nilai 0 (nol) berarti file tersebut tidak akan di mainkan ulang jika sudah selesai.
  • nilai volume, mengatur kerasnya suara yang dihasilkan, nilainya dalam persen. jika tertulis 125, artinya adalah 125 persen dari volume yang diset di komputer.
Sekarang perhatikan teks yang berwarna hijau.
parameter : http://flash-mp3-player.net/medias/player_mp3.swf adalah adalah file untuk memainkan file mp3 dan bagaimana akan di tampilkan tinggi (height) dan lebarnya (width).
Langkah keempat
Sekarang kita tinggal memasukkan nilai-nila yang kita inginkan sehingga akan menjadi seperti ini: 
<object data="http://flash-mp3-player.net/medias/player_mp3.swf" type="application/x-shockwave-flash" height="20" width="150"><param value="#ffffff" name="bgcolor"><param value="mp3=http://cdn.x.indowebster.com/download/90/p16togert31qc3kdvcforrap73.mp3&amp;loop=1&amp;autoplay=1&amp;
volume="125" name="FlashVars"> </object>

Setelah selesai, publish-kan
Penjelasan Kode: Teks warna merah di atas adalah nilai yang sudah kita rubah atau kita masukkan. Kelebihan jika kita menampilkan player [http://flash-mp3-player.net/medias/player_mp3.swf] adalah pengunjung blog bisa mematikan lagu yang dimainkan jika tidak suka :)>
Penempatan Penempatannya bisa di mana saja bisa. Tidak terbatas pada satu halaman saja. Bisa saja di tempatkan di widget dengan menambahkan widget HTML/Javascript di blogspot.
