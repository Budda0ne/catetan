Selain virus, trojan, spyware, dan worm, gangguan yang tak kalah hebatnya adalah hacker yang dapat mengambil data-datakita secara ilegal. Salah satu metode yang populer untuk menjaga data-data dari ancaman hacker adalah memanipulasi data menggunakan algoritma enkripsi.

Untuk bahasan lebih lanjut mengenai enkripsi silahkan baca artikel Enkripsi di wikipedia.org. Secara singkat, Enkripsi adalah proses mengacak data sehingga tidak dapat dibaca oleh pihak lain. Pada kebanyakan proses enkripsi, Enkriptor harus menyertakan kunci(public key) sehingga data yang dienkripsi dapat didekripsikan kembali.

Berikut ini ada beberapa aplikasi enkripsi (Encryption Tool).

Deltacrypt Encryptor

Pada aplikasi DeltaCrypt Encryption, kita akan diminta untuk membuat public key terlebih dahulu. Enkripsi yang digunakan adalah RSA Encryption. Agar lebih cepat melakukan enkripsi, Kamu dapat menggunakan fitur Expres Commands.

Sofware ini d hargai oleh developernya US$ 30.00 untuk yang versi AutomaticPro dan AutomaticPro Limited tetapi ada juga versi rumahan (single user/home edition) yang ccukup murah, seharga US$ 1.99.

untuk mencoba aplikasi ini, silahkan kunjungi www.deltacrypt.com atau langsung ke halaman download

Dan jika menginginkan versi gratis dan bukan trial, ikuti tautan berikut.

Advanace File Encryptor

Aplikasi Dekart Private Disk dapat melakukan enkripsi terhadap semua data dengan cara membuat virtual disk padamedia penyimpanan. Kita bisa menentukan sendiri besarnya kapasitas virtual disk yangkita buat. Teknik algoritma enkripsi yang digunakan oleh aplikasi ini adalah algoritma AEX dan SHA-1.

Program trial bisa di dapatkan di www.dekart.com dengan langsung ke halaman download installer trial. Harganya sebesar US $ 65.00, namun kamu bisa mendapatkan versi gratisnya di sini

AEP 2012 Pro

Atau Advanced Encryption Package 2012 Professional yang dulunya di kenal sebagai Cryptocrat atau Cryptoexpert, memberikan key enkripsinya sebesar 448 bit dengan 20 macam algoritma yang dapat kamu pergunakan, diantaranya : DESX, BLOWFISH, RIJNDAEL/AES, CAST, 3DES, RC2, DIAMOND2, TEA, SAFER, 3WAY, GOST, SHARK, SQUARE, dll. Aplikasi ini dapat melakukan teknik enkripsi dengan mengkompresi beberapa file ke dalam arsip secara otomatis.

AEP 2012 Pro di hargai US$ 49.95 dan bisa didapatkan versi trialnya di www.aeppro.com di halaman download atau melalui tautan berikut.

Banyak program-program sejenis yang menawarkan kecanggihan teknologi enkripsi mereka seperti Handycrypto milik YOYS Software, Cryptoforge buatan Ranguel Technologies dan sebagainya.

TrueCrypt

Namun dari kesekian aplikasi tersebut adalah aplikasi berbayar (walaupun saya mencantumkan link gratisannya). Akan lebih menyenangkan jika kita menggunakan software enkripsi yang benar-benar gratis. Pilihannya adalah TrueCrypt yang merupakan program Open Source atau FOSS. Selain gratis, TrueCrypt juga mempunyai algoritma enkripsi yang sudah teruji dan banyak dipakai juga oleh pengembang software enkripsi komersial. Menggunakan AES, Serpent, Twofish, dan kombinasi dari ketiga macam algoritma enkripsi tersebut. Homepage komunitas pengguna TrueCrypt bisa dikunjungi di www.truecrypt.org. ada versi untuk Windows, Linux, dan Mac

Tetapi pilihan tentu di tangan kamu sendiri sebagai pemakai, apakh akan memakai program berbayar atau yang gratis dan open source.
