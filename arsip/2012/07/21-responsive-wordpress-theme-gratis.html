Akhir-akhir ini banyak sekali yang menanyakan pada saya tentang tempat untuk mendownload Tema untuk wordpress. Itulah alasan saya untuk memposting 21 Tema Wordpress Responsive Gratis, padahal saya sendiri tidak memakai Wordpress tetapi Blogger.

Wordpress Theme disingkat WP Themeatau jika di blog [dot]blogspot di sebut sebagai template ini saya dapatkan dari beberapa situs yang memberikan Theme Wordpres secara gratis kemudian saya rangkum setelah saya pilih hanya yang responsive saja.
