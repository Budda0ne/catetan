Facebook Messenger
Alasan mengganggu produktivitas kerja sudah tidak ada lagi.

Facebook Messenger untuk Windows 7 secara resmi keluar. Yah, secara teknis, itu sudah keluar setelah bocor beberapa bulan yang lalu dan memaksa Facebook untuk merilisnya ke pasaran awal Maret. Saat itu, aplikasi ini masih memiliki beberapa masalah, dan beberapa pengguna, termasuk saya sendiri, tidak bisa sama sekali digunakan, waktu itu saya mencoba masuk (login) akun facebook saya, di smessenger tertulis bahwa saya memang sudah masuk. tapi saya tidak melihat tampilan apapun pada saat itu.

Untuk yang versi baru ini sudah di perbaiki beberapa bug dan masalah yang pengguna temukan/dapatkan ketika aplikasi gratis pertama kali diluncurkan ke pasaran pada akhir Desember.
Fitur-fitur aplikasi tetap tidak berubah. Anda masih dapat menggunakannya untuk chatting dengan teman Anda di Facebook, menerima pemberitahuan dan mendapatkan berita ticker pada 7 desktop Windows Anda.


Jika Anda adalah pecandu Facebook yang fanatik (apa?), maka anda harus memiliki aplikasi ini. Dengan menginstall aplikasi ini memungkinkan bagi Anda untuk tetap up to date pada kejadian/status terbaru kawan-kawan anda tanpa harus membuka halaman web Facebook. Anda dapat meningkatkan produktivitas tanpa gangguan jika terjadi sesuatu yang penting di Facebook.
