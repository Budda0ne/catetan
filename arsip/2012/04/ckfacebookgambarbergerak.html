Kawan, dari dulu di facebook memang tidak ada yang namanya gambar bergerak (animasi). Tapi apa memang benar-benar tidak bisa? Bisa...

Iseng : Boleh, tetapi yang kreatif dan mudah.
Pernah juga saya merasa bosan ketika membuka akun facebook saya, yang saya lakukan hanya update status, komen, seskali mengunjungi beranda teman-teman facebook, upload foto, video dan chatting.
Seandainya ada gambar bergerak pasti agak hebh juga facebook ini. Ada beberapa trick untuk menampilkan gambar bergerak di facebook, tapi rata-rata agak sulit dan membutuhkan pengetahuan tentang kode-kode tertentu atau kalau tidak, menggunakan Aplikasi tertentu di facebook.
Yang saya inginkan adalah : saya ingin membuat sendiri tanpa harus menggunakan aplikasi di facebook ataupun harus belajar bahasa kode-kode yang rumit. Benar kan?
Menampilkan gambar animasi di Chat Facebook
Yang paling mudah menurut saya adalah menampilkan gambar bergerak di chat facebook. Langsung ke proses pengerjaannya:

Langkah Pertama : Mencari gambar

- Cari gambar animasi (file berakhiran ".gif") di google, ketikkan saja kata kunci semacam "gambar animasi", "gambar bergerak", atau langsung dengan jenis file yang kamu cari ".GIF"

- Setelah ketemu gambar yang diinginkan, klik kanan di gambar, 'copy image location' kalau menggunakan firefox atau 'copy image url' jika mamakai google chrome.

- Paste di notepad, untuk menyimpan sementara link gambar yang sudah diperoleh. misalnya : http://featherapocalypse.files.wordpress.com/2011/09/gambar-gif-2.gif


Write Notes / Tulis Catatan
Langkah Kedua : Membuat catatan

- Buatlah catatan di facebook, jika kesulitan mencari tautan Catatan/Note di facebook bisa klik link ini.

- Tuliskan judul di kolom judul (terserah mau diberi judaul apa)

- di kolom Isi, tuliskan tag HTML berikut : <img src="

- paste/tempelkan link gambar yang tadi sudah di copy (kalau tidak muncul, copy lagi link yangada di notepad tadi)

- kemudian ketikkan lagi kode HTML berikut : "></img> 
- sehingga menjadi : <img src="http://featherapocalypse.files.wordpress.com/2011/09/gambar-gif-2.gif"></img>
- set privasi menjadi Publik, Terbitkan.
Langkah Ketiga : Mengirim gambar ke teman chatting
Pop up Share

- setelah catatan kamu jadi, buka catatan kamu, gambarnya tidak bergerak? sebentar...

- cari link Bagikan/Share, di pop-up yang muncul klik di tombol [Di Kronologi Anda Sendiri] maka akan muncul beberapa pilihan, pilih [Dalam Pesan Pribadi]

- di kolom Masukkan nama teman, ketikkan nama teman chatting anda sa'at ini.

- bagikan catatan

Nah... Selesai.
