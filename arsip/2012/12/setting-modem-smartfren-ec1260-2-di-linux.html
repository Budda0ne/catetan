Bagi pengguna Linux, sabar ya. Karena setting Modem Smartfren EC 12060-2 di Linux memang agak ribet. Tidak seperti kala menggunakan modem EC1260-2 di windows, yang tiggal colok dan otomatis ter-install. meski agak ribet, tetapi masih bisa di atasi.

Bagian 1: Relatif Mudah

Berikut ini cara setting modem smartfren EC1260-2 pada linux:
1. Colokkan modem smartfren EC1260-2 pada pc, laptop, atau netbook yang terpasang Linux.
2. Buka terminal Linux yang anda gunakan, login sebagai root.
3. Lakukan cek modem anda pada daftar USB yang dikenali oleh Linux yang secara default dikenali sebagai media penyimpanan dengan cara mengetikkan perintah berikut untuk mengetahui serial produk dari modem Huawei EC1260-2 anda.
lsusb
Hasil list usb dari modem Huawei EC1260-2 yang saya dapatkan:
Bus 003 Device 003: ID 12d1:1505 Huawei Technologies Co., Ltd.
Perhatikan yang saya beri warna kuning. Dari hasil lsusb (list usb) di atas terdapat 12d1:1505 maka didapatkan:
vendor = 12d1
product = 1505
Setiap modem mempunyai id Product berbeda-beda. Maka sesuaikan dengan hasil dari pengecekan dengan lsusb di Linux anda.
4. Lanjutkan dengan perintah sebagai berikut untuk mengaktifkan modem smartfren EC1260-2 di Linux
modprobe usbserial vendor=12d1 product=1505
5. Setelah selesai, kita bisa menggunakan modem smartfren EC1260-2 untuk menghubungkan komputer, laptop, atau netbook ke internet. Cara paling gampang adalah menggunakan network connection.

Cara setting dial-up modem smartfren EC1260-2 pada Linux:
a. Klik pada icon network connection, kemudian pilih new cdma connection.
b. Klik next pada notifikasi Setup New Mobile Broadband Connection.
c. Pilih negara yang sesuai (Indonesia, misalnya)
d. Pilih koneksi SMART, kemudian klik Next
e. Simpan hasil konfigurasi koneksi smartfren
f. Klik network connection, kemudian pilih koneksi SMART (atau biasanya terdapat kata default pada belakang nama koneksi) untuk menghubungkan ke internet.

Bagian 2: Opsional dan Kondisonal

Bagaimana jika usb modem tidak dikenali oleh Linux (khusus Ubuntu)? Anda bisa melakukan langkah-langkah berikut (hanya untuk jaga-jaga saja :D):
1. Download usb-modeswitch-data dan install dengan cara klik 2x atau ketik
sudo dpkg -i usb-modeswitch-data_20100826-1+squeeze0_all.deb
2. Download usb-modeswitch dan install dengan cara klik 2x atau ketik
sudo dpkg -i usb-modeswitch_1.1.4-2_i386.deb
ingat: Sesuaikan dengan prosessor anda, karena saya memakai intel maka yang saya download i386
Cari file 12d1:1505 didalam folder/directory /etc/usb_modeswitch.d/, dan kalau tidak ada buat file di directory tsb dengan nama 12d1:1505 dengan cara 
sudo gedit /etc/usb_modeswitch.d/12d1:1505
3. Kemudian ketikkan script berikut di file tersebut
# Huawei, newer modems
DefaultVendor= 0x12d1
DefaultProduct=0x1505
TargetVendor=x200x12d1
TargetProductList="1001,1406,140b,140c,1412,141b,14ac"
CheckSuccess=20
MessageContent="55534243123456780000000000000011062000000100000000000000000000"
4. Tambahkan script berikut kedalam file /lib/udev/rules.d/40-usb_modeswitch.rules
ATTRS{idVendor}=="12d1", ATTRS{idProduct}=="1505", RUN+="usb_modeswitch '%b/%kx22
taruh script tadi setelah tulisan LABEL=modeswitch_rules_begin
Tambahkan script berikut kedalam file /lib/udev/rules.d/40-usb_modeswitch.rules
5. Dan yang terakhir adalah lakukan
sudo usb_modeswitch -c /etc/usb_modeswitch.d/12d1:1505

Setelah modem terdeteksi, sebnarnya ada dua cara untuk dial ke internet yaitu menggunakan Network Manager (caranya sama dengan langkah nomer 5 di bagian 1) atau wvdial atau gnome-ppp (GUI wvdial). Sekarang kita coba menggunakan wvdial caranya sebagai berikut:
1. Download wvdial dan install, caranya
$ sudo apt-get install wvdial
2. Ubah file wvdial.conf
$ sudo gedit /etc/wvdial.conf
dan isikan file tersebut dengan script berikut
[Dialer smartpret]
Init1 = ATZ
Init2 = ATQ0 V1 E1 S0=0 &C1 &D2 +FCLASS=0
Stupid Mode = 1
Modem Type = Analog Modem
Command Line = ATDT
ISDN = 0
New PPPD = yes
Phone = #777
Modem = /dev/ttyUSB0
Username = smart
Password = smart
Baud = 460800
3. Dan untuk dial lakukan
sudo wvdial smartpret

Setelah semua Setting Modem Smartfren EC1260-2 di Linux anda berskan, maka anda bisa terhubung ke internet menggunakan koneksi dari smartfren.
