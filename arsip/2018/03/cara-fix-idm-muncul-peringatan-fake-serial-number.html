Yap, IDM adalah singkatan dari Internet Download Manager. Yakni perangkat lunak (software) yang mampu mengunduh data-data yang ada di internet. Dan jika koneksi/jaringan terganggu atau putus, IDM mampu meneruskan kembali saat jaringan internet terhubung lagi.

Program ringan ini merupakan alat bantu download paling populer dengan menempati posisi teratas dalam memaksimalkan kecepatan mengunduh data. Tampilan dan grafis yang sederhana juga menjadi nila tambah tersendiri.

Meskipun begitu, IDM sebenarnya merupakan produk berbayar yang mengharuskan penggunanya membeli lisensi setelah menikmati masa percobaan (trial) selama 30 hari. Jadi jika kamu Sudah menggunakan IDM dan lewat 30 hari setelah tanggal pemasangan akan muncul pemberitahuan menyuruh registrasi atau membeli lisensi.

Biasanya, pengguna akan menghapus software IDM dan seluruh data-nya kemudian maemasang lagi untuk bisa menikmati masa trial. Wah, curang donk, melanggar hak cipta ini? Trik ini bisa dibilang abu-abu dan masih sah-sah saja dilakukan. Yang curang itu saat kamu memakai patch untuk menikmati gratis IDM selamanya.

Jendela Pop-Up Fake Serial Number

Tetapi, tahukah kamu bahwa Internet Download Manager sudah melakukan pencegahan atas tindakan pengguna yang menggunakan patch? Ya, program IDM mempunyai trigger agar saat terhubung dengan internet melakukan pengecekan lisensi yang ada ke server milik developer IDM. Jika cocok dan lisensi sudah diverifikasi memang dibeli secara resmi, IDM bisa digunakan dengan nyaman. Tetapi jika lisensi ternyata tidak ada dalam database mereka (fake serial number), IDM akan memunculkan pop-up secara terus menerus secara berkala.

Lalu, adakah cara untuk mencegah jendela pop-up fake serial number ini? Jika kamu tahu bagaimana cara kerja komputer terhubung ke internet. Kamu pasti tahu bagaimana mengatasinya tanpa saya beritahu.

Memblokir Alamat Server IDM

Setidaknya ada dua pilihan untuk menghilangkan jendela notifikasi ini. Kedua cara ini prinsipnya punya tujuan yang sama, yakni menghalangi software menghubungi alamat server IDM. Triknya bisa kamu baca berikut ini:
warning Kedua trik di bawah sebenarnya bukan diperuntukkan untuk mmendukung tindakan Piracy. Saya anggap ini sebagai bentuk pembelajaran tentang bagaimana melakukan penyaringan situs-situs jahat dengan memanfaatkan firewall dengan media yang akrab dan bisa diuji dan praktekkan secara langsung mengingat banyaknya pemakai IDM pak tani di Indonesia

Trik #1. Block alamat server IDM dengan edit file host

Cara untuk trik pertama adalah dengan mengarahkan alamat (domain) yang dihhubunngi oleh IDM ke alamat IP 127.0.0.1 alias localhost. Sehingga IDM tidak bisa memeriksa validitas serial number karena tidak mendapatkan jawaban dari server. Bagaimana caranya?
  1. Silahkan buka Windows Explorer dan arahkan ke lokasi C:/Windows/drivers/etc/ dan cari file bernama hosts.
  2. Klik kanan pada file, pilih Properties dan hilangkan centrang pada pilihan read only pada jendela Properties. Klik Apply lalu OK.
  3. Buka dengan notepad dengan cara klik dobel dan pilih program Notepad.
  4. Salin kode dibawah ini ke Notepad yang terbuka.
    # File Host Lokal Windows
    127.0.0.1 localhost

    ::1x20localhost



    # [IDM Host]

    127.0.0.1 tonec.com

    127.0.0.1 www.tonec.com

    127.0.0.1 registeridm.com

    127.0.0.1 www.registeridm.com

    127.0.0.1 secure.registeridm.com

    127.0.0.1 internetdownloadmanager.com

    127.0.0.1 www.internetdownloadmanager.com

    127.0.0.1 secure.internetdownloadmanager.com

    127.0.0.1 mirror.internetdownloadmanager.com

    127.0.0.1 mirror2.internetdownloadmanager.com

    127.0.0.1 mirror3.internetdownloadmanager.com

    127.0.0.1 star.tonec.com



    #
  5. Simpan file hosts di lokasi yang sama dengan nama yang sama tanpa ekstensi.
  6. Klik kanan pada file hosts" lagi, pilih Properties dan centrang kembali pilihan read only,
  7. Reboot Komputer/PC atau Laptop kamu
Seharusnya sekarang pop-up fake serial number IDM tidak bisa muncul lagi. Tetapi akan muncul permintaan untuk memberikan akses Administrator. Untuk mencegah IDM merubah file host kamu, tolak permintaan dari IDM tersebut. Selesai.

Trik #2. Block alamat server IDM dengan Firewall

Sedangkan trik yang kedua dengan memblokir alamat server software IDM melalui Firewall. Untuk memblokirnya bisa memakai firewall bawaan windows maupun software firewall lain. Cara untuk melakukan trik ini dengan firewall bawaan windows adalah sebagai berikut:
  1. Buka Windows Firewall dari Start menu dengan cara mengetik firewall di kolom pencarian. Pilih Windows Firewall dan pilih menu Advanced Settings
  2. Klik kanan pada Outbond Rules dan pilih New Rule akan muncul jendela Rules Type.
  3. Pada jendela ini pilih Custom dan klik Next. Lanjut Klik Next lagi untuk melompati Programs dan Protocols and Ports.

  4. Pada jendela Jendela Scope pilih These IP Addresses pada bagian Wich remote does this rule aplies to? kemudian tambahkan satu persatu 3 (tiga) alamat IP berikut dengan menekan Add dan klik OKt hingga selesai.
    • 169.55.0.224
    • 169.55.40.5
    • 50.22.78.28
    Ketiga alamat IP tersebut masing masing merumakan alamat server untuk domain internetdownloadmanager.com, registeridm.com dan tonec.com.

  5. Hasilnya akan terlihat seperti gambar berikut pada poin di atas Kemudian klik Next lagi
  6. Pastikan yang terpilih adalah Block the connection dan kilk Next
  7. Pastikan semua pilihan (Domain, Private dan Public) di centrang dan kilk Next lagi
  8. Pada kolom nama isikan saja Block Server IDM atau lainnya, baru klik Finish

Rules yang ada pada Windows Firewall akan tampak seperti gambar diatas. Setelah ini kamu lakukan, maka kamu akan terbebas dari pop-up fake serial number milik IDM.
Ulangi lagi semua langkah diatas pada Inbound Rules.

  1. Dobel klik pada rules, beralih ke tab Program and Services
  2. Pilih This program dan klik Browse (lihat gambar dibawah) arahkan ke lokasi program IDM
  3. Pilih IDMan.exe dan klik Open kemudian klik OK
