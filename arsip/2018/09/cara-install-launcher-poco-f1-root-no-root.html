Baru-baru ini Xiaomi mengeluarkan seri terbaru yang sudah rilis di negara india yaitu smartphone xiaomi Poco F1, POCO F1 dikabarkan akan hadir di Indonesia secara resmi melalui TAM sebagai distributor resmi.

Bukan, artikel ini tidak akan membahas kelebihan, kecanggihan, speksifikasi atau harga hape monster tersebut. Hanya ingin berbagi tips tentang Launcher yang disematkan pada Poco F1 agar bisa kita cicipi terlebih dahulu.

Poco Launcher tak berbeda jauh dengan launcher-launcher lain. Perbedaannya dengan launcher bawaan xiaomi (baca: MIUI) dimana tidak ada app drawer, semua aplikasi yang terpasang langsung ada di layar depan.

Sebelumnya, Poco Launcher ini sudah di gunakan pada Xiaomi Mi A1, karena Poco Launcher mengembalikan App Drawer yang sudah lama di tinggalkan oleh xiaomi sekarang di kembalikan lagi. Saat pertama menggunakan Poco Launcher di smartphone, ada kemiripan konsep dengan ZenUi Launcher milik ASUS, namun sedikit lebih baik.

Poco Launcher yang memiliki fitur app drawer (laci aplikasi) yang digolongkan sesuai kategori, tidak hanya menampilkan semua aplikasi yang terpasang dalam satu tempat.

Xiaomi Non-Root

Poco Launcher, layaknya aplikasi lain, karena itu bisa dipasang di smartphone xiaomi dan ponsel lain. Bahkan meskipun ponsel xiaomi kami belum ROOT pun bisa langsung instal dengan cara berikut:

  1. Download dulu aplikasinya, nih: Poco Launcher Beta.apk
  2. Instal seperti memasang aplikasi lain.
  3. Ketika menekan tombol home, pilih Poco Launcher.
  4. Selesai.

Root Mode

Jika Smartphone Xiaomi kalian sudah Rooted dan terpasang TWRP bisa ikuti Langkah-langkah sebagai berikut :

  1. Download file zip-nya terlebih dahulu Poco Launcher.zip
  2. Matikan Xiaomi kalian,lalu boot ke Twrp.
  3. Instal Paket Poco Launcher .
  4. Tunggu hinggal selesai penginstalan.
  5. Lalu Reboot.

Paket Poco Launcher sudah saya coba pada M IUI 9 dengan cara pasang via twrp dan berhasil 100%. Tidak mengalami error sedikit pun.

Ketika aplikasi ini dicoba di MIUI 10 mengalami error (force closed), Launcher system tidak bisa dibuka.

Bagi kalian yang instal aplikasi seperti cara pertama bisa uninstal Poco Launcher bila ingin kembali ke Launcher System MIUI. Sekarang sudah bisa dipasang melalui playstore.

Namun Untuk kalian yang Instal Poco Launcher melalui TWRP tidak bisa un-install, namun masih bisa ganti Launcher di Pengaturan > Layar Utama.
