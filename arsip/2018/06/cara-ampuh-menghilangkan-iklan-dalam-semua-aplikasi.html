Memblokir iklan yang tampil di aplikasi android bisa dibilang cukup tricky. Ada beberapa aplikasi yang menawarkan untuk menghilangkan iklan, sebagian membutuhkan akses root, sebagian lainnya tidak. Ada banyak pilihan jika punya akses root atau super user, akan tetapi perlu trik-trik khusus untuk yang tidak berhasil melakukan rooting.

Mungkin kamu pernah dengar aplikasi AdAway, yang pernah muncul di Playstore namun kemudian di banned. Aplikasi pemblokir iklan satu ini cukup terkenal dan masih bisa ditemukan di internet. Bisa jadi kamu termasuk salah satu penggunanya. Namun bukan aplikasi ini yang akan kita pakai untuk menghilangkan iklan karena masih banyak yang tidak begitu paham bagaimana cara melakukan rooting atau tahu cara root namun tidak berhasil.

Note: Sebelum kamu menyebut UC Browser, pahami dulu bahwa artikel ini tidak membahas tentang blokir iklan yang tampil di browser. Ini adalah artikel tentang bagaimana menghilangkan semua iklan yang tampil pada semua aplikasi android.

Untuk ponsel Android non-root, aplikasi sejenis ABP Plus atau AdFree, mungkin pernah berhasil dipakai, utamanya untuk ponsel OS Android yang lebih lawas. Untuk saat ini, aplikasi semacam ini banyak yang gagal (atau tidak optimal) mem-filter iklan. Metode baru untuk memblokir iklan dalam aplikasi pun bermunculan, terutama untuk ponsel non-rooted. Salah satunya dengan cara menggunakan aplikasi VPN semisal AdGuard yang mempunyai daftar alamat penyedia iklan di server, dan menghalangi request ke alamat penyedia iklan tersebut. Metode ini cukup berhasil, iklan memang tidak tampil, namun dari sisi performa tidak begitu baik jika dibandingkan dengan AdAway. Apalagi jika koneksi internet sedang buruk.

AdAway, tetap menjadi pemblokir iklan paling efisien karena metode pemblokiran yang dipakai adalah dengan menggunakan file hosts yang berisi daftar penyedia iklan secara lokal. Sehingga permintaan alamat IP bisa langsung diproses tanpa menunggu jawaban dari sever secara online. Namun, apa gunanya jika tidak bisa dipakai pada ponsel non-root?

Paham dengan masalah ini, ada solusi yang kemudian dicoba dikembangkan. Bagaimana jika menggabungkan kedua metode tersebut? Yoi, menggabungkan pemakaian VPN dengan pemakaian file hosts lokal. Adalah aplikasi Virtual Hosts yang menawarkan solusi baru pemblokiran iklan secepat AdAway namun bisa dipakai pada ponsel Android non-rooted.

Baca juga: 3 Aplikasi Virtual VPN berbasis Hosts Lokal.

Cara kerjanya adalah dengan membuat server VPN secara lokal, yang didalam server tersebut kamu bisa menambahkan alamat domain pebyedia iklan yang ingin diblokir melalui file hosts layaknya AdAway. Sehingga ponsel uang tidak bisa di-root masih bisa memblokir iklan secara komplit bahkan iklan yang tampil dalam aplikasi pun hilang tak berbekas.

Tertarik? Harus dong... Silahkan unduh dulu (gratis) aplikasi Virtual Hosts terbaru dan pasang di ponsel Android kamu. Karena membutuhkan file hosts secara lokal, kamu bisa membuat file hosts terlebih dahulu dengan aplikasi text editor kesukaan kamu, rekomendasi saya adalah Quoda. Untuk saat ini kamu tidak perlu membuat file hosts sendiri. Karena akan saya bahas di lain kesempatan kamu bisa menggunakan file host ini yang juga dioakai pada aplikasi AdAway.

Untuk cara memakainya sangat mudah, kamu bisa ikuti langkah berikut:

  • Buka aplikasi Virtual Hosts yang sudah kamu pasang.
  • Tekan tombol yang tampil dalam aplikasi tersebut. Kamu akan diminta untuk memilih file host.
  • Arahkan ke lokasi file host yang kamu buat sendiri maupun yang sudah kamu unduh sebelumnya.
  • Otomatis, akan muncul permintaan untuk menyetujui koneksi VPN. Cebtrang saja dan pilih Setuju (Accept).
  • Lihat tanda anak kunci yang muncul di status bar. Itu tandanya koneksi VPN sudah aktif
  • Koneksi kamu akan dilewatkan ke VPN (lokal) terlebih dahulu sehungga iklan tidak tampil.
Penggunaan aplikasi Virtual Hosts ini sebenarnya tidak perlu dibuatkan step by step-nya karena tampilan aplikasi vpn lokal ini cukup simpel. Sekali lihat pasti langsung paham, yang terpenting adalah kamu sudah punya daftar alamat (domain) penyedia iklan berupa file hosts.

Dengan menggunakan aplikasi virtual hosts atau virtual vpn ini, semua aplikasi gratis beriklan yang sebelumnya kamu pasang dari Playstore tidak akan bisa mebampilkan iklan lagi. Sekali lagi, semua aplikasi! bahkan ketika kamu membuka situs tertentu melaui browser chrome yang biasanya menampilkan iklan intrusif yang sangat mengganggu, hilang tak berbekas.

Aplikasi sejenis yang lebih dulu muncul dan popoler adalah DNS66 yang juga memanfaatkan repositori yang dipakai aplikasi AdAway untuk mengambil file hosts yang sudah jadi.
