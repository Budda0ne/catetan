Masalah baru kali ini ialah password admin modem indihome seringkali berubah sendiri tanpa diketahui sebabnya.
Dengan terganggunya akses admin merupakan kasus yang sangat-sangat mengganggu kenyamanan pelanggan indihome, dan lebih menjengkelkan daripada munculnya iklan sisipan. Seharusnya kasus ini mendapat perhatian dari pihak Telkom. Namun sepertinya tidak ada satupun pembahasan dari pihak Telkom mengenai kasus ini. Anehnya lagi beberapa password yang dirubah selalu ada kaitannya dengan nama Telkom dan Jatinegara.

Jadi bagi kalian yang juga menjadi korban kejahatan berubahnya password admin tanpa izin mungkin bisa mencoba salah satu cara yang akan saya bahas berikut ini, dalam kasus ini perangkat (milik kawan) bertipe ZTE F660. Caranya cukup mudah yaitu dengan mengakses sistem modem melalui port 23 lalu kita intip langsung database user yang menampilkan nama pengguna dan passwordnya tanpa ada enkripsi. Berikut langkah-langkahnya :
Cara Melihat Password Admin Modem Indihome
Pastikan anda sudah terhubung dengan internet dari modem indihome, baik via kabel maupun tanpa kabel (wireless)
Gunakan aplikasi Putty yang bisa di-download di sini ringan sekali) atau dapat menggunakan command prompt jika layanan telnet-nya sudah diaktifkan di OS windows. Namun di sini saya bahas yang pakai putty.

Nah sekarang buka putty-nya
Untuk host, yaitu alamat admin web page, secara default : 192.168.1.1, atau 192.168.1.245 dan kadang ada yang udah merubahnya dengan alamat lain.
Connection Type : Telnet
Login : root
Password : Zte521 (Z huruf besar) lalu tekan enter.

Ketikkan perintah “sendcmd 1 DB p UserInfo” lalu enter lagi.

Selesai, akan tampak passwordnya.



Langkah di atas bisa dilakukan jika port telnet (23) masih dibuka. Jika tertutup saya juga tidak tahu, mungkin solusi yang terpikir sekarang hanya reset modem ulang dari awal.

Perlu diketahui, meskipun mengaktifkan fitur “Enable Anti-Hacking Protection” yang ada di dalam modem ZTE tersebut tidak berpengaruh. Password tetap saja berubah dengan sendirinya.
