Fitur TWRP sendiri dikenal paling lengkap dibandingkan Custom Recovery yang lain. Sekedar informasi untuk pengguna yang baru kenal Android, dengan menggunakan Recovery Custom (seperti TWRP yang sedang kita bahas), anda bisa melakukan berbagai tindakan yang tidak bisa dilakukan dengan stock recovery. Bahkan menu back-up di custom recovery lebih menyeluruh sehingga anda lebih tenang saat melakukan customasi di ponsel anda nantinya. misalkan saja melakukan rooting, merubah dan mengganti ROM dan lain-lain.


Sebelum melakukan pemasangan TWRP 3.0.2-2 untuk Lenovo A7000 plus, silahkan download file yang dibutuhkan

TWRP 3.0.2-2

Aplikasi Rashr

Catatan: Lenovo A7000 plus dan K-50 menggunakan tipe cipset yang sama

Peringatan: Dengan mengikuti Tutorial Install TWRP 2.8.6.1 dan Restore Stock Recovery Redmi 2, berarti anda sudah setuju dengan disclaimer KnoAcc Blog
Instalasi TWRP bisa dilakukan dengan beragam cara. Jika belum root, bisa memasang dengan cara flashing memakai SP Flashtool karena Lenovo A7000 plus menggunakan cipset Mediatek. Tetapi kalau sudah root bisa langsung memasang menggunakan aplikasi Rashr dengan cara:

Install aplikasi Rashr.
Letakkan file TWRP yang sudah anda download (recovery_K50_3.0.2-2.img) di sdcard/internal sdcrad. Letakkan saja di luar agar mudah ditemukan.
Buka aplikasi Rashr dan pilih menu Flash custom recovery
Kemudian pilih file TWRP tadi. Tunggu hingga proses selesai.

Catatan: root bisa menggunakan Kingroot atau Kingoroot, baik tanpa PC maupun dengan PC

TWRP versi baru ini juga bisa langsung diupgrade jika sudah memasang TWRP versi sebelumya dengan cara:

Letakkan file TWRP di sdcard/internal sdcrad. Letakkan di luar saja.
Matikan A7000 plus anda dan masuk recovery mode dengan menekan tombol [power] + [vol up] + [vol down] bersamaan hingga muncul logo lenovo (bootlogo)
Setelah masuk recovery TWRP, pilih menu Install dan pilih install image kemudian cari dan pilih file recovery_K50_3.0.2-2.img di sdcard atau internal.
Swipe ke kanan untuk mengkonfirmasikan pemasangan TWRP baru. Tunggu hingga proses selesai.
Setelah selesai, kembali ke menu utama dan pilih Reboot lanjut Recovery untuk melihat apakah proses instalasi sudah berhasil.
Jika versi TWRP sudah berubah menjadi 3.0.2-2 berarti pemasangan TWRP sukses. Silahkan pilih Reboot dan pilih System untuk menyalakan ponsel.

Kini Lenovo A7000 plus anda sudah terpasang TWRP Recovery sebagai alternatif CWM Recovery.
