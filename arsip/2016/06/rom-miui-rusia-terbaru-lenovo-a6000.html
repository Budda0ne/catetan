Meskipun tidak berbeda jauh dengan MIUI lain, Varian MIUI developer Rusia mempunyai fitur lebih lengkap. Apalagi untuk Lenovo A6000 (baik biasa, SE maupun Plus) didukung secara official oleh developer di forum MIUI Rusia yang di maintain oleh GoodDeath. ROM ini dirilis tanggal 29 Mei lalu.


Versi update yang paling baru, MIUI v.6.5.26 bahkan sudah fitur yang harusnya baru diberikan jika menggunakan MIUI 8. Fitur tersebut semacam touch helper yang ada di device apple. Meskipun basis UI yang dipakai masih MIUI 7. Fitur khas MIUI 7 yang tidak muncul di MIUI Multirom dan Global Dev adalah Lockscreen Carousel yang bisa mengganti Wallpaper Lockscreen MIUI secara otomatis.

Meskipun sudah menggunakan OS Android Lollipop 5.1.1 sebagai base, saya lebih menyukai MIUI Rusia (miuisu) Karena terasa lebih smooth daripada ROM MIUI lain yang pernah saya coba di Lenovo.

Sempat kecewa dengan ROM Flyme OS Lenovo K30-T Karena boros baterai, saya merasa puas dengan manajemen daya ROM ini. Masalah baterai di Flyme OS bisa jadi disebabkan Karena system yang selalu memaksa pengguna untuk menggunakan koneksi internet. Hampir semua aplikasi bawaan (baik MIUI maupun Flyme OS) membutuhkan adanya akses data, tetapi kontrol di MIUI lebih baik dan tidak memaksa.

Untuk fitur lainnya hampir sama dengan MIUI lain, silahkan dieksplorasi sendiri.

Langkah Instalasi ROM MIUI di Lenovo A6000.


Perhatikan: MIUI Russia 6.5.26 (dan versi sebelumnya) sudah menyediakan custom recovery TWRP dalam paket zip. Jadi setelah Instalasi, custom recovery akan berganti menjadi TWRP 3.0.0.

Untuk memasang ROM, pastikan sudah mempunyai Custom Recovery, rekomendasi saya, paling baik menggunakan TWRP meskipun menggunakan CWM Recovery maupun PhilzTouch Recovery juga bisa.

OS sebelum instalasi bebas, baik Stock ROM atau Custom ROM, kitkat maupun lollipop bisa langsung melakukan instalasi.

Download ROM MIUI Russia 6.5.26 di bawah dan letakkan di external sdcard atau di Internal juga bisa.

Download juga SuperSU.zip jika ingin melakukan rooting dengan versi SuperSU terbaru. Sebenarnya tidak perlu, Karena setelah Instalasi berhasil anda akan ditawari oleh TWRP untuk melakukan root dengan SU bawaan TWRP (SuperSU stable).

Download GAPPS, juga tidak perlu. Tetapi jika anda ingin merampingkan paket aplikasi Google yang terpasang bisa mengunduh OpenGapps 5.1.1 ARM pico. Dan jika anda maniak Google Apps, anda bisa menggunakan paket aplikasi OpenGapps 5.1.1 ARM Full yang membuat ponsel anda seolah menggunakan Ponsel Xiaomi rasa Nexus.

Setelah semua hal diatas terpenuhi, boot ke mode Recovery dengan menekan tombol [power] + [vol up] + [vol down]. Jika belum bisa masuk TWRP Recovery secara langsung, pilih menu Recovery di layar sela yang muncul.

Jika sudah masuk TWRP, anda bisa melakukan wipe Cache dan Dalvik-cache. Tidak perlu wipe System Karena nantinya ROM installer akan memformat partisi System. Sementara untuk Data, sebaiknya anda wipe juga jika anda menggunakan ROM selain MIUI Lollipop. Karena sesuai pengalaman, tidak bisa masuk System setelah instalasi selesai. Tetapi jika menggunakan ROM MIUI lain yang sudah menggunakan base Lollipop 5.1.1 anda boleh tidak melakukan wipe Data agar data dan aplikasi yang sudah anda install tidak hilang.

Setelah proses wipe selesai, kembali ke layar utama TWRP dan pilih menu Reboot Recovery. Alasannya untuk member kesempatan TWRP membaca ulang tabel partisi System, Cache dan Data agar tidak muncul pesan kesalahan saat proses instalasi atau flashing.

Setelah masuk mode recovery lagi, pilih menu Install dari layar utama TWRP. Arahkan ke lokasi dimana anda meletakkan file ROM MIUI Rusia. Pilih dan swipe (geser bar indikator) untuk mengkonfirmasi pemasangan ROM. Proses install ini mungkin terlihat stag (mandeg) tetapi santai saja. Flashing ROM kan memakan waktu sekitar 10 hingga 15 menitan.

Setelah sukses (done), wipe cache dan Dalvik-cache lagi. Kemudian pilih menu Reboot pilih lagi menu System. Ingat, sebaiknya biarkan boot system selesai sebelum melakukan flashing file zip lain (mis: GAPPS/SuperSU).

Sampai disini proses instal ROM MIUI Rusia Lenovo A6000 sudah selesai.

Tips untuk advance user: Anda juga bisa memilih Reboot Recovery dan langsung melakukan install GAPPS maupun SuperSU terbaru. Saya yakin anda bisa mengatasi jika ada masalah yang muncul.
