Seri ponsel cerdas bundling dari operator ponsel Smartfreen 4G LTE series yakni: Andromax R, Andromax EC/ES dan Andromax Q/Qi, tak dipungkiri mempunyai daya tarik tersendiri bagi pengguna kelas menengah. Selain tergolong murah untuk kelas smartphone Android yang mendukung jaringan data LTE, minimal ponsel Andromax tersebut sudah menggunakan OS Lollipop. Tinggal menunggu waktu saja semua jajaran ponsel Andromax mendapatkan update Marshmallow.

Sayangnya, dari beberapa kelebihan diatas ADA satu kekurangan ponsel seri Andromax (termasuk seri lawas seperti Andromax G/G2 atau Andromax C/C2/C3) yakni TIDAK BISA menggunakan kartu sim slot 2 untuk meng-akses internet alias terkunci di kartu sim slot 1. Dimana slot 1 hanya bisa di isi dengan kartu dari provider bundling yakni Smartfren. Artinya, ponsel jajaran Andromax cuma bisa menggunakan Smartfren sebagai penyedia layanan data.

Eit, jangan kecewa dulu. Tak butuh waktu lama bagi komunitas pengguna Android di Indonesia untuk membobol proteksi (Unlock) yang di lakukan oleh operator Smartfren. Alhasil, Ponsel Andromax EC/ES, Andromax R dan juga Andromax Q/Qi pun bisa dipakai untuk ber-internet-an dengan kartu GSM dari operator lain. Lalu, Bagaimana caranya agar bisa internetan dengan kartu GSM? Simak langkahnya di bagian tutorial unlock. Sebelum itu, silahkan persiapkan file yang dibutuhkan dan penuhi beberapa syarat dibawah:

1. Download file untuk Unlock GSM 4G LTE dibawah. Catatan: Update file untuk unlock gsm 4g lte smartfren tanggal 29 Juni 2016 dan memindahkan file di Mega dan Mediafire.

Apakah melakukan unlock gsm andromax perlu rooting terlebih dulu? Tidak, tetapi untuk yang tidak punya PC tentu harus root agar bisa memasang TWRP recovery.

Tutorial Unlock Data GSM di Andromax EC/ES, Q/Qi, dan Andromax R

Sebelum melakukan unlock gsm lte, lakukan back-up EFS (imei dan modem) asli. Ini untuk keperluan restore jika anda ingin re-locking Andromax. Caranya:

1. Masuk Recovery Mode (TWRP) dengan cara menekan tombol [power] + [volume up] bersamaan selama beberapa detik.

2. Setelah masuk TWRP, lakukan Back Up EFS MODEM dan BOOT dengan men-centang opsi partisi: DIAG, MODEM, MODEM1, MODEM2.

3. Swipe (geser) ke kanan indikator di bawah untuk melakukan patch modem

4. Namai back-up anda agar tidak lupa.

Jika anda ingin langsung melakukan unlock GSM, sebaiknya lakukan reboot terlebih dahulu. Kalau saya lebih memilih Reboot into recovery dengan memilih menu Reboot kemudian pilih menu Recovery agar lebih cepat.

Okay, sekarang saatnya melakukan unlock. Yang perlu adna lakukan hanyalah melakukan flashing file flashable yang sudah anda download sebelumnya.

  1. Dalam mode recovery, Pilih menu Install
  2. Jika file zip anda letakkan di Internal, tekan tombol Choose storage dan pilih Internal storage. Begitu juga jika letak file zip di sdcard, pilih External Storage.
  3. Cari file zip yang anda diwnload, pilih, dan swipe untuk konfirmasi Instalasi. Tunggu proses flashing selesai.

Setelah langkah ini, sangat di anjurkan untuk wipe Cache dan Dalvik-cache. Tidak juga terserah, Karena setelah wiping, proses boot akan berjalan agak lama (android is upgrading bla bla bla..) seperti ponsel baru di factory reset.

Anda boleh meletak-kan kartu GSM yang akan dipakai di slot sim 1 atau 2 agar bisa digunakan untuk internet-an.
