Dan cara transfer file dari iOS ke Android dan sebaliknya boleh dubilang yang paling banyak ditanyakan.
Berbeda dengan Android yang memberikan kebebasan lebih baik dalam hal transfer data, iOS sendiri tidak begitu baik menyediakan tool untuk mentransfer data tanpa harus memakai software tertentu via PC atau laptop dan harus mencolokannya ke perangkat mobile. Lebih susah lagi jika transfer file tanpa menggunakan kabel data.

Sebenarnya, masalah dan keterbatasan transfer data antar platform bisa mudah diatasi menggunakan aplikasi SHAREit, yang populer di OS Android terlebih dahulu. Aplikasi ini juga mampu melayani platform Windows Mobile, tidak hanya bisa transfer berbagai macam data ke sesama platform saja tetapi juga lintas platform.

Bagaimana cara menggunakan aplikasi ini untuk berbagi data dan file antar platform? Simak cara bagaimana cara menggunakan SHAREit berikut:

Dalam tutorial ini iPhone sebagai pengirim file dan data, sedangkan Snartphone Android sebagai penerima data.

1. Unduh dan install aplikasi SHAREit terlebih dahulu di smartphone Anda. Kemudian buka aplikasi di perangkat Anda (iOS atau Android). Anda akan menemukan pilihan Send (untuk mengirim data) atau Receive (menerima data).

2. Setelah aplikasi terpasang di iPhone, buka dan pilih menu Kirim (Send). Sedangkan aplikasi shareIt yang terpasang di Android pilih menu Terima (Receive).



3. Di ponsel pengirim (iOS/Apple) akan muncul pilihan kategori file apa saja yang ingin Anda kirim. Setelah dipilih, Anda bisa melanjutkan pengiriman file dengan menekan tombol OK.

Sedangkan di ponsel penerima (Android) karena platform yang berbeda, pili tombol "Connect ti iOS/WP" yang ada di bagian menu ponsel penerima.



4. Kini di ponsel penerima file akan muncul pemberitaguan yang menampilkan sebuah kode jaringan (hostspot) dengan nama random. Di ponsel pengirim (iOS/Apple) pastikan untuk memilih tombol "Connect to hotspot now".

Setelah dipilih, akan terbuka halaman pengaturan jaringan. Pilih jaringan (hotspot) yang sama dengan kode yang ada di ponsel penerima (Android).



5. Anda tinggal menunggu kedua smartphone beda platform ini terhubung. Anda tinggal memilih tombol Send untuk memulai proses transfer file yang diinginkan.



Proses transfer dengan menggunakan aplikasi ini sangat cepat karena menggunakan koneksi wireless atau tanpa kabel.
