Misalnya saja eggster dari Google Chrome berupa game Dino, OS Android Lollipop dengan game "contekan" dari game Flappy Bird. Eggster bisa diakses menggunakan "pemicu" tertentu (kadang) tidak sengaja ditemukan pemicunya. Nah kali ini, ada fitur tersembunyi dari fitur perpesanan Facebook, yaitu Game Catur.

Mungkin jika anda bukan penggemar game olah otak ini, anda tidak tertarik untuk memainkannya. Tetapi jika Anda ingin mencobanya, ada cara agar anda bisa memainkan catur rahasia yang berada di fitur chatting Facebook maupun aplikasi Messenger. Penasaran? Berikut adalah cara untuk bermain catur rahasia Facebook Messenger ini.

Seperti yang sudah saya sebutkan sebelumnya, ada pemicu untuk membuka fitur rahasia ini. Caranya mulai percakapan baru dengan teman anda dan ketik "@fbchess play" untuk mengaktifkan game catur. Nah, dalam permainan ini, setidaknya anda paham cara memainkan catur. Karena cara memainkannya menggunakan kode-kode lokasi bidak Catur milik anda dengan mengetikkan "@fbchess" sebagai awalan. Sebagai contoh: "@fbchess PE4" akan memindahkan pion Anda (P) ke lokasi E4. Huruf tidak harus besar karena tidak case-sensitive.

Bingung? Ketik saja "@fbchess help"

Anda juga bisa mengetik "@fbchess help" untuk meminta bantuan, tapi hanya bantuan dasar saja: K adalah untuk Raja (King), Q untuk Ratu/Ster (Queen), B untuk Luncum (Bishop), N untuk Kuda/Ksatria (Knight), R untuk Benteng (Rook), dan P untuk Pion (Pawn). Sedangkan untuk memindahkan lokasi bidak Catur, cukup menambahkan lokasi tujuan ke perintah. Misalnya "NBD2" akan memindahkan Kuda dalam lajur B (kolom vertikal) ke kotak D2. Perintah "x" untuk memakan bidak musuh (mis B2XC5 berarti Luncum di kotak 2 (baris horisontal) memakan bidak di kotak C5); Perintah "=" untuk mempromosikan Pion menjadi Ratu (mis E8 = Q); dan perintah "0-0-0" untuk melakukan Lukir.

Jika Anda tidak menentukan warna Anda akan ditugaskan secara otomatis. Seperti biasa di permainan catur, putih jalan dulu.
Fitur rahasia game catur Facebook ini terakhir kali disebutkan oleh pengguna reddit di akhir bulan Desember tahun lalu. Entah berapa lama fitur rahasia ini bisa diakses, bisa jadi ini fitur permanen.
