Custom ROM MIUI 7 ini merupakan hasil dari porting agan Risky Bagoes Alam member dari group OFFICIAL INA Lenovo A6000 di Facebook.

Basis ROM MIUI 7 untuk Lenovo A6000 ini diambil dari Stock ROM Redmi 2 MIUI Lollipop Beta 6.2.25 Global. Jadi sudah terpasang Playstore tanpa harus meng-install GAPPS secara manual, baik flashable zip maupun dengan GAPPS installer.

Meskipun sebenarnya MIUI Lollipop ini sudah tersedia akses root dari sono-nya (native) tapi bagi sebagian user masih lebih suka menggunakan SuperSU sebagai aplikasi manajemen akses root. Terutama bagi pengguna yang tidak familiar dengan UI MIUI, atau baru saja menggunakan ROM MIUI.

Porter CustomROM ini juga memberikan tips/ menganjurkan agar menurunkan frekuensi minimal CPU agar A6000 lebih adem. Alasannya, frekuensi CPU di ROM ini di setting terlalu tinggi baik untuk min_freq maupun max_freq. Sehingga saat CPU selalu kerja keras meskipun tidak begitu banyak aplikasi yang dibuka. Ini juga mengakibatkan baterai boros. Untuk user awam, bisa menggunakan aplikasi NoFrill CPU Control atau Kernel Adiutor untuk mempermudah menurunkan settingan Frekuensi CPU bawaan ini.

Step dan Syarat Instalasi Custom ROM MIUI 7

ROM terakhir yang dipakai Lenovo A6000 sebelum install MIUI Lollipop ini harus Lollipop (stockrom maupun custom rom).

Selain itu sudah terpasang TWRP Lollipop
Letakkan file yang sudah anda download sebelumnya, atau Download dulu diatas dan letakkan di sdcard anda (internal/external, bebas)

Masuk ke mode recovery (TWRP) dengan cara menekan tombol power dan kedua tombol volume bersamaan Beberapa detik dari keadaan ponsel mati.

Dalam mode recovery lakukan Wipe (data, cache, dalvik)

Setelah selesai, kembali ke menu utama dan pilih menu install zip dengan urutan flashing ROM MIUI 7 kemudian SuperSU zip.

Lakukan reboot to system setelah proses instalasi selesai.

ROM sudah menggunakan Bahasa Indonesia

GAPSS sudah di minimalisir, tinggal Playstore dan Google keyboard (agar ukuran ROM tidak terlalu besar).
