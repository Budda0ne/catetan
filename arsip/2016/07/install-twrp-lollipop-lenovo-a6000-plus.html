Setelah melakukan update Lenovo A6000 maupun Lenovo A6000+ menjadi Android lollipop, kini custom recovery yang saya bagikan tidak berfungsi dengan baik karena memang sudah tidak compatible lagi dengan OS Lollipop. Sehingga meskipun ada sisa custom recovery di partisi recovery, mode recovery hanya stuck di logo Lenovo saja.

Untuk mengatasi hal ini, sangat mudah. Tinggal flash lagi saja custom recovery lagi. Eit... tetapi custom recovery yang sudah compatible dengan lollipop ya... Baik versi Sevenmaxs maupun versi MIUI Rusia compatible dengan versi Lollipop dan yang sudah bisa digunakan saat ini baru TWRP, custom recovery di kitkat hampir semuanya tidak menyertakan dtb. Ini yang jadi penyebab utama gagalnya fungsi recovery kitkat di lollipop. Selain itu ada beberapa modifikasi di recovery.fstab nya dan ada tambahan di twrpp.fstab agar bisa memanggil semua partisi yang dibutuhkan secara sempurna.

Bagi yang sudah tidak sabar bisa langsung mengikuti tutorial yang diberikan di bawah. Tutorial berikut menggunakan PC, sedangakan untuk opsi instalasi TWRP tanpa menggunakan PC masih tetap sama dengan tutorial install recovery tanpa pc sebelumnya. Hanya file recovery yang digunakan tentu harus di download di bagian download di artikel ini.

Pastikan bahwa anda sudah meng-install Minimal ADB Fastboot di komputer/laptop anda. saya sarankan menggunakan windows 7 karena di Windows 8/8.1 keatas agak ribet instalasi drivernya.

Download file TWRP Lollipop di bagian download dan letakkan dalam folder dimana anda meng-install adb fastboot.

Rename menjadi twrp.img atau nama yang mudah di-ingat.

Matikan Lenovo A6000 atau A6000plus

Nyalakan dalam mode fastboot dengan cara menekan tombol volume down (-) bersamaan dengan tombol power.

Tancapkan ponsel ke komputer atau laptop dengan kabel data usb.

Buka folder Minimal ADB Fastboot dan tekan tombol CTRL + SHIFT jangan lepaskan dan klik kanan di ruang koson dalam folder tersebut.

Pilih Open command window here.

Dalam Command Promt yang muncul ketik 

fastboot devices
Jika muncul list device maka ponsel sudah terhubung.

Selanjutnya tinggal mengetik perintah 

fastboot boot twrp.img
untuk meng-akses TWRP recovery sementara (hot boot).

Sedangkan perintah: 

fastboot flash recovery twrp.img
untuk melakukan instalasi permanen.

Jika melakukan flashing permanen maka anda perlu mematikan ponsel terlebih dahulu sebelum masuk ke recovery mode dengan cara menekan tiga tombol secara bersamaan selama beberapa detik.

Tetapi jika melakukan boot sementara dengan TWRP Recovery maka setelah "sending boot" OKAY, ponsel anda akan boot otomatis ke dalam mode recovery.

Rekomendasi Custom ROM yang bisa di coba setelah install TWRP : MIUI7 for Lenovo A6000/plus Lollipop based.

Troubleshoot instalasi TWRP

Untuk yang masih belum bisa melakukan flashing TWRP permanen, lakukan langkah berikut:

  1. Letakkan file recovery.img di sdcard dan juga dalam folder minimal adb.
  2. akukan hot boot TWRP dari kondisi fastboot perintahnya sama.
  3. Setelah masuk mode recovery (TWRP), pilih menu install, pilih image, dan arahkan ke file recovery.img yang ada di sdcard.
  4. Setelah sukses, pilih menu reboot recovery.
Note: Cara diatas juga bisa dilakukan saat ingin mengganti recovery saat ini dipakai dengan custom recovery lain, misal: TWRP kitkat.

Troubleshoot ganti bahasa Rusia ke bahasa Inggris

  1. Dari menu awal, pilih tombol dibagian kanan ke tiga dari atas.
  2. Pilih tombol logo dunia (paling kanan) dan pilih English (scroll menu) dan tekan tombol di kanan bawah.
