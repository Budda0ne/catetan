Melakukan Unsecure Boot dan Memasang TWRP


Extract file ROM Andromax R2 & file boot unsecure & TWRP
Kopi file SuperSU.zip ke eksternal sd, jangan di dalam Internal memori karena akan terhapus saat proses unsecure boot.

Ganti nama file boot max R2 05.00.img menjadi boot.img
Ganti juga nama file TWRP 2.8.7.0 Blade S6 - Max R2 05.00.img menjadi recovery.img

Kopi kedua file (boot.img dan recovery.img) ke dalam folder i56d2g.v1, jika ada peringatan, pilih replace untuk menimpa file yang ada didalam folder tersebut.

Kini, Install QPST di komputer/laptop, yang anda butuhkan adalah tool QFil yang ada dalam paket software QPST.

Aktifkan Auto load usb device di Andromax R2, dengan cara masuk ke Telepon kemudian tekan *1973461# di dial pad. Pada menu Debug setting, centang Auto load usb device.

Jika diminta untuk memasukkan password, isi dengan SmartfrenI56D2G14082015

Kini hubungkan Andromax R2 ke PC/laptop dengan kabel usb.

Buka tool QFil dari start menu > QPST dan pastikan HP dideteksi sebagai device HS-USB Diagnostic 9006

Dalam QFil, Set bagian Select Build Type menjadi Flat Build

Pilih tombol Browse dan masuk ke folder i56d2g.v1 kemudian pilih file prog_emmc_firehose_8929.mbn.

Pilih lagi tombol Load XML kemudian pilih rawprogram_unsparse.xml dan jika muncul pop-up lagi, pilih patch0.xml.

adri sini langsung tekan tombol Download untuk memulai proses flashing. Proses berlangsung selama lebih kurang 20 menitan. Santai saja.

Setelah selesai, cabut ponsel anda dari Kabel data USB kemudian tekan tombol [power] sekitar 10 detik sampai keluar logo Andromax 4G LTE.

Tunggu hingga Andromax R2 booting sempurna dan masuk Homescreen, kemudian matikan ponsel.

Kini Andromax R2 sudah unsecure, dan siap untuk di root.



Rooting Andromax R2 dengan TWRP (Recovery Mode)

Sebenarnya tidak harus dari TWRP. Dengan sudah boot sudah unsecure, artinya anda bisa melakukan rooting dari manapun baik dengan aplikasi rooting biasa (semisal kingroot) maupun dengan mengunakan tool rooting pc. Akan tetapi karena sudah terpasang TWRP bukankah lebih baik langsung root dari TWRP saja?

Kini, masuk ke mode recovery dengan menekan tombol [Power] + [Volume Down] hingga muncul teks berwarna hijau, lanjutkan menahan tombol Power sampai bergetar.

Jika berhasil masuk recovery mode (dalam hal ini TWRP), pilih menu Install.

Pilih menu Change directory, pilih External Storage dan pilih file Supersu.zip Swipe bar dibawah ke kanan untuk install.

Tunggu hingga proses flashing selesai dan pilih reboot to system. Tunggu hingga ponsel booting.

Setelah masuk home screen silahkan cek apakah sudah ada aplikasi SuperSU? Jika ya, selamat Andromax R2 anda sudah berhasil root. Kalau anda tidak percaya, silahkan pasang aplikasi root checker manapun untuk membuat anda yakin.
