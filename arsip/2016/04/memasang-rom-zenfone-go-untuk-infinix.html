--- categories: [arsip] title: "Memasang ROM Zenfone GO untuk Infinix Hot 2 (X510)" date: "2016-04-22T16:00:00.000+07:00" # author: Phate Holloway tags: - CustomROM - Infinix - Android last_modified_at: "2016-04-25T16:22:10.297+07:00" header: image: https://2.bp.blogspot.com/-sEYe1tM9WyI/Vt1wHp9JgOI/AAAAAAAACfI/-o5GdvoU29o/w480/Zenfone-GO-Infinix.jpg # blogger_id: tag:blogger.com,1999:blog-8168996750712797250.post-4047874549508860253 redirect_from: - /2016/04/memasang-rom-zenfone-go-untuk-infinix.html --- Saat ini sudah banyak sekali developer Indonesia yang melakukan developing dan porting ROM untuk Android Infinix Hot 2. Salah satu yang terbaru adalah Custom ROM Zenfone GO.

Beberapa fitur Custom ROM Zenfone GO ini sudah sangat mirip dengan ROM aslinya termasuk aplikasi-aplikasi khas Asus. Hanay saja ada sedikit ganjalan yakni hasik foti kamera tidak tersimpan di External Sdcard tetapi di Internal Sdcard.

Perlu diperhatikan, sebelum anda melakukan instalasi Custom ROM Zenfone GO ini, Posisi ROM saat ini harus menggunakan Infinix ROM (Stock ROM Infinix, bukan ROM Android One) agar tidak ada masalah yang serius (brick). Jadi silahkan download dulu Infinix ROM (sudah disediakan di bawah) dan Flash menggunakan SP Flashtool.

Selanjutnya anda perlu meng-instal Custom Recovery untuk melakukan flashing. Anda bisa menggunakan Tutorial Instal TWRPx20Infinix Hot 2.

Jika Syarat atas sudah terpenuhi, silahkan Download Custom ROM Zenfone GO dan file untuk Fix Bug.

Letakkan 2 file zip yang sudah anda download di
eboot sdcard dan Reboot ke mode Recovery.

Setelah berada TWRP recovery lakukan fullflash wipe dengan memilih menu Wipe dan Advance Wipe

Pilih (centang) pilihan Dalvik Cache, Cache, Data dan System dan Swipe ke kanan tombol di bawah untuk mengkonfirmasi.

Tunggu hingga proses Wiping selesai baru kembali ke menu utama TWRP Recovery.

Sekarang pilih INSTALL, dan cari file CustomROM Zenfone GO yang ada di scard, pilih file tersebut dan Swipe ke kanan untuk konfirmasi instalasi.

Masih dalam mode Recovery, flash juga file Fix Bug caranya sama seperti langkah diatas.

Sekarang reboot Infinix Hot 2 anda dengan memih menu Reboot dan pilih System. Ingat jangan pilih install jika TWRP menawarkan untuk melakukan install SuperSU. Rooting bisa dilakukan kapan saja setelah proses Instalasi selesai.

Proses booting pertama akan memakan waktu yang lumayan lama sekitar 10 menitan, santai saja asal tidak sampai 1 jam jangan panik.kalau saya sekitar sebatang rokok belum habis sudah masuk system)

Perlu diketahui juga, jika pada saat masuk system berenti di Checking info, reboot saja agar tidak terlalu lama menunggu.
