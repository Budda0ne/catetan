Cara Flashing AOFP via SP Flashtool hampir sama dengan langkah-langkah flashing ponsel Android mediatek lainnya. Hanya saja, jika anda mengikuti cara ini akan menjadikan IMEI terdeteksi "00000000000" karena partisi nvram ikut kena format. Tetapi jangan kuatir, masih bisa di perbaiki IMEI nya dari engineering mode maupun SN Write tools.

Peringatan dokter: Dengan mengikuti tutorial di halaman ini, berarti anda sudah setuju dengan disclaimer dan tos. Jika tidak, silahkan cari dokter lain.

Cara ini sebenarnya merupakan gabungan dari flashing via SP Flashtool dengan flashing via Recovery. SP Flashtool digunakan untuk membangkitkan AOFP dari kondisi hardbrick dengan cara memasang boot dan recovery. Kemudian recovery yang terpasang digunakan untuk meng-install stock ROM AOFP yang berupa flashable.zip. Sebelum mengikuti tutorial ini sebaiknya persiapkan dulu bahan dan alat yang dibutuhkan (baca: Download) ada di bawah postingan.

Memperbaiki Alcatel Onetouch Flas Plus Harbrick

Langkah ini bertujuan agar ponsel anda bisa di boot ke dalam mode recovery sehinga bisa melakukan langkag lanjutan. Caranya:

1. Install Driver USB VCOM terlebih dahulu. Jika menggunakan Windows 8 keatas dan driver tidak bisa di-instal, disable force signature terlebih dahulu.
2. Ekstrak File For_HARDBRICK FIX (bukan kulit manggis).
3. Buka SP Flashtool jalankan file Flash_Tool.exe dan pilih menu load scatter dan cari di lokasi anda meng-ekstrak file Harbrick fix tadi.
4. Pastikan SEMUA file yg di folder for hardbrick sudah ter load di SP Flashtool.
5. Lanjutkan dengan memilih format whole flash di Tab Format dan tekan start sebelum ponsel di hubungkan.
6. Matikan ponsel dan colok via kabel data dalam keadaan mati dan tunggu hingga proses format selesai.
7. Jika sudah selasai (OK) cabut ponsel. Tetapi jangan dinyalakan. Biarkan saja dulu.
8. Perhatikan SP Flashtool dan pilih lagi tab Download dan langsung tekan downloa tanpa merubah apapun, kemudian tancapkan lagi ponsel Alcatel Onetouch (masih dalam keadaan mati) agar proses berjalan, tunggu proses hingga selesai
9. Jika proses selesai, cabut ponsel dan biarkan tetap mati dulu.

Tahap pertama selesai.

Install stock ROM Alcatel Onetouch Menggunakan TWRP

Lanjutan ke tahap kedua agar AOFP bisa sehat kembali dengan meng-install ulang dengan stockrom via TWRP. Caranya:

1. Copy ROM 1 yang tadi anda dowload ke sdcard. Dan pasang ke ponsel.
2. Boot langsung ke mode recovery dengan tombol power dan volume up bersamaan hingga ponsel masuk TWRP Recovery.
3. Setelah berhasil masuk Recovery mode, install zip ROM (Rom by iwanw) dari external SD card tunggu sampai ada pemberitahuan selesai (Done),
4. Pilih wipe cache setelah selesai, pilih menu reboot system.

Tahap kedua selesai.
