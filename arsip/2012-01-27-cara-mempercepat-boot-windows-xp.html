Pernah anda merasa jengkel menunggu booting windows begitu lambat?

Ada tips yang perlu untuk dicoba :

Mempercepat Booting dengan Otomatis Hapus File Prefetch (cara pertama)

Setiap kali kita booting windows, OS windows akan membuat file simpanan dari kegiatan sebelumnya. File itu ada di folder Prefetch di direktori windows (%SystemRoot%Prefetch)

Kalau di folder tersebut banyak tersimpan file-file yang ter save, bisa membuat windows kita menjadi lelet pada saat booting/start-up. Dengan cara ini akan secara otomatis menghapus file-file yang tersimpan di %SystemRoot%Prefetch setiap kali kita shut down atau restart komputer.

Step 1 :
Buka aplikasi notepad.

Ketikkan “del c:\windows\prefetch\ntosboot-*.*/q” (tanpa tanda kutip )
dan save as dengan nama ntosboot.bat dalam drive c:\.

Step 2 :
Masuk ke menu [Start] [Run] (Anda juga dapat membuka jendela Run dengan menekan kombinasi tombol Windows + R) dan ketikkan gpedit.msc.

Step 3 :
Klik [Computer Configuration] [Windows Settings] [Script]–lalu klik 2 kali pada Shutdown.

Step 4 :
Di dalam Windows Shutdown Properties klik [add] lalu [browse], kemudian cari letak file ntosboot.bat yang anda simpan tadi dan klik open.

Step 5 :
Klik OK, Apply dan OK sekali lagi untuk menutupnya.

Step 6 :
Klik menu [Start] [Run] dan [devmgmt.msc]

Step 7 :
Klik IDE ATA/ATAPI controllers–Primary IDE Channel ( Klik 2X untuk membuka properties )

Step 8 :
Pilih Advanced Settings
Pada Device 0 atau Device 1
Ganti [Device Type] menjadi None ( Ganti saja opsi Device Type yang tidak terkunci ).

Step 9 :
Klik IDE ATA/ATAPI controllers–Secondary IDE Channel ( Klik 2X untuk membuka properties )
Ulangi step 8.

Step 10 :
Restart Komputer anda dan lihat hasilnya.

Memperercepat booting dengan defrag boot (cara kedua)

Untuk cara yang kedua ini adalah dengan melakukan boot defrag. Boot defrag dapat mempercepat proses start-up windows dengan menempatkan boot file pada posisi yang berdekatan antara satu dan lainnya.

Fitur ini sebenanya sudah secara default berjalan pada windows xp. Tapi jika anda pernah meng-upgrade windows xp anda, maka fitur ini tidak akan berfungsi. Untuk itu anda harus mengaktifkannya secara manual.

Untuk dapat mengaktifkan fitur tersebut, Anda dapat mengikuti langkah berikut:

Step 1 :
Tekan tombol [Start] [Run].

ketik "regedit" tanpa tanda petik, untuk membuka jendela Registry Editor.
Setelah itu, tekan [Enter].

Step 2 :
Setelah jendela Registry Editor terbuka, arahkan pada

"HKEY_LOCAL_MACHINESOFTWAREMicrosoftDfrgBootOptimizeFunction".

Step 3 :
Pilih [Enable] dari daftar yang terdapat pada bagian kanan.

Step 4 :
Klik kanan, kemudian pilih [Modify]

Step 5 :
Ubah nilai ke [Y] untuk mengaktifkan dan [N] untuk menon-aktifkannya.

Restart komputer.
