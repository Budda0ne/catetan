Dengan bekal CPU dual core frekuensi 1 Ghz dan RAM "hanya" 256 MB ponsel ini saya anggap lemah jika digunakan untuk memainkan game atau aplikasi besar. Karena itulah banyak yang berusaha untuk melakukan rooting agar bisa melakukan modifikasi system (tweaking). Akan tetapi, dengan memodifikasi system sembarangan terkadang Advan Vandroid S3A ini akan mengalami bootloop bahkan hardbrick.

Untuk mengatasi masalah bootloop Advan Vandroid S3A, salah satu caranya adalah dengan melakukan restore dari file backup CWM Recovery. Namun jika anda lupa atu tidak mempunyai back-up ROM dari CWM Recovery, satu-satunya cara adalah dengan melakukan flashing stok firmware Advan Vandroid S3A. Jika kebetulan OS anda masih JellyBean 4.2.2 dan ingin upgrade ke kitkat 4.4.2 atau OS Kitkatt downgrade Jellybean anda juga bisa menggunakan tutor ini tergantung Firmware yang anda pilih.

Tutorial Install Stock Firmware S3A Kitkat 4.4.2


Langkah-langkah flashing (baik upgrade maupun downgrade) bisa mengikuti tutorial berikut:
  1. Pastikan anda sudah meng-install driver VCOM (MTK USB Preloader). Untuk meng-install driver, ekstrak file yang sudah anda download dan buka folder hasil ekstrak. Jika anda menggunakan Windows 64 bit jalankan installdrv64.exe dan jika Windows 32bit jalankan InstallDriver.exe.
  2. Ekstrak SP Flashtool yang anda download di drive C:\ kemudian buka folder tersebut dan jalakan Flash_tool.exe
  3. Klik menu Options dan pastikan USB MODE dan DA Download All di centang. Jika tidak, klik di USB MODE.
  4. Ektrak Firmware Kitkat atau Jellybean yang sudah anda download.
  5. Klik di tombol Scatter-Loading dan pilih file MT6572_Android_scatter.txt dari hasil ekstrak firmware, klik tombol open.
  6. Pastikan file file untuk flashing sudah tercentang semua sesuai dengan versi android yang anda pilih.
    List File ROM Kitkat Advan S3A
    List File ROM Jellybean Advan S3A
  7. Jika sudah, Pilih Tombol Download atau tekan F9, jika muncul peringatan klik Yes.
  8. Matikan ponsel dengan melepas baterai selama 5 detik kemudian psang lagi baterai tetapi jangan di nyalakan.
  9. Tancapkan Advan Vandroid S3A ke komputer/laptop dengan kabel USB.
  10. Proses akan berjalan, tunggu hinga proses selesai yang di tandai Download OK dalam lingkaran hijau.

Jika tidak jalan atau gagal, ulangi langkah no. 1 hingga no. 8 tetapi pada langkah no. 9 jangan masukkan baterai, langsung tancapkan ke kabel data usb bersamaan dengan menekan tombol volume down (-) sampai proses berjalan.

Jika anda benar-benar mengikuti tutorial dengan teliti, saya yakin proses flashing akan berjalan lancar. Jika proses diatas selesai, cabut ponsel anda dari kabel data dan nyalakan secara normal. Advan Vandroid S3A anda akan kembali normal. namun jika masih bootloop atau stuck di logo, lakukan factory reset dari recovery mode dengan cara:

1. Matikan ponsel.
2. Nyalakan dengan menekan volume up bersamaan dengan tombol power selama 5 detik untuk masuk recovery mode
3. Dalam mode recovery, pilih menu wipe data, wipe cache dan wipe dalvik cache.
4. setelah seledsai, pilih menu reboot system.

Nah, selesai sudah. Silagkan tinggalkan komentar anda jika sukses. Atau ada yang gagal? jangan malu bertanya. Jangan lupa, sertakan juga pesan kesalahan yang muncul.
