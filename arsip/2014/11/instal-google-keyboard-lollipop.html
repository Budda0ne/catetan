Seperti yang sudah banyak kita tahu, Android Lollipop (5.0) terkenal dengan design material UI. Material UI ini juga ada/muncul dalam tampilan Google Keyboard 4.0 di L Develepoer Preview. Aplikasi ini, sama seperti Google Search 4.0, memerlukan file library (*.so), yang tidak disertakan dalam APK keyboard itu sendiri. Anda mungkin bisa meng-installnya, tetapi ketika dijalankan akan mengalami force close. Untuk meng-instal Keyboard Material UI ini, anda memerlukan file library tersebut.


Google Keyboard 4.0, mempunyai tambahan tema Material Light dan juga Material Dark. Selain itu, update keyboard yang baru mendukung lebih banyak layout, termasuk QWERTY, QWERTZ, AZERTY, Dvorak, Colemak, dan Keyboard PC.

Tutorial Install Keyboard Lollipop (Material Design)


Anda bisa meng-install Google Keyboard 4.0 ini di Android versi 4+ (ICS keatas). Tetapi ada syaratnya. Syarat untuk meng-install Google Keyboard 4.0 (Material UI) HH anda harus dalam keadaan root, karena anda perlu mengganti beberapa file dalam folder system.

Jika Anda ingin mencoba meng-install-nya, ikuti tutorial berikut:

  • Download Keyboard APK dan libjni_latinimegoogle.so di bawah.
  • Instal APK Keyboard, seperti biasa. Keyboard kemungkinan akan force close saat Anda mencoba untuk menggunakannya.
  • Dengan root explorer, cek apakah ada file libjni_latinimegoogle.so dalam folder /system/lib, jika ada
    Back up file tersebut di sdcard dan ganti dengan file libjni_latinimegoogle.so yang tadi Anda download.
  • Rubah juga permisson-nya, set ke rw-r--r--.
  • Reboot dulu Android anda.
  • Setelah menyala, anda bisa mengatur tema dari Pengaturan Bahasa dan Masukan.
  • Pilih Google Keyboard, dan atur tema sesuai kesukaan anda.

Ada 4 pilihan tema: Material Light, Material Dark, Holo White dan Holo Blue.
