Kali ini benar-benar dalm bentuk ROM bukan sekedar asus launcher-nya saja. ROM ini merupakan ROM porting handheld china kloningan Zenfone 5. Meskipun belum bisa dikatakan sama sepenuhnya, setidaknya secara keseluruhan sudah mirip dengan aslinya.

Untuk performa, rasanya tidak ada masalah kalau untuk penggunaan harian. Jika ingin meningkatkan performa, anda bisa menambahkan beberapa tweak semisal TurboBoost untuk RAM atau PureGraphic untuk display.

Custom ROM LEXUS Zenfone 5 ini sudah dual 3G (3G hot swap) selain itu juga disertakan patch Dolby Sorround (untuk meningkatkan kualitas suara) oleh agan Hamzah Boyolali Art Dev sebagai porter ROM. Patch Dolby Sound ini juga sekaligus untuk patch bug hilangnya suara speaker/output sound (bukan mic).

Sedangkan tampilan untuk Evercoss A26C, perbedaan yang paling terlihat dengan ROM Zen-UI asli berada pada statusbar dan quick setting-nya yang masih AOSP banget. Sementara komponen interface yang lain sudah menyerupai Zen-UI. Lebih lengkapnya lihat screenshoot berikut:

Install ROM LEXUS Zenfone 5 untuk Evercoss A26C

Dengan mengikuti tutorial instalasi ini, anda berarti sudah setuju dengan Disclaimer dan mengerti serta patuh dengan DMCA Copyright di blog ini. Dan anda juga paham bahwa dengan menginstall Custom ROM di handheld anda bisa menyebabkan garansi anda gugur resiko kehilangan data, backup data penting anda.

Syarat: harus sudah ter-install CWM Recovery, jika belum, baca di tutorial install CWM Recovery.

Untuk memasang ROM ini, anda harus melakukan Backup IMEI melalui aplikasi Mobile Uncle atau aplikasi backup IMEI lainnya, sebagai antisipasi jika terjadi INVALID IMEI. Buat juga Nandroid Backup melalui CWM Recovery sebelum melakukan flashing.

Langkah-langkah instalasi:
  1. Download ROM di download section dan letakkan di sdcard tapi jangan di dalam folder. Letakkan dix20/sdcard/Lexus-zen5Fix by BylArtDev.zip
  2. Matikan Evercoss A26C anda, dan hidupkan lagi dalam mode CWM Recovery dengan menekan tombol [Power] dan [Volume Up]
  3. Ketika sudah masuk mode CWM Recovery, gunakan tombol [Volume Up] dan [Volume Down] untuk navigasi dan tombol [Power] untuk memilih menu:
    • pilih menu Wipe Data/Factory Reset pilih yes
    • pilih lagi menu Wipe Cache Partition pilih yes lagi
    • pilih menu Advanced, pilih Wipe Dalvik Cache
    • Pilih Install Zip From SdCard kemudian pilih file Lexus-zen5Fix by BylArtDev.zip
  4. CWM Recovery akan melakukan proses flashing, tunggu hingga selesai.
  5. Setelah Selesai, pilih GO BACK (menu paling bawah) dan pilih Reboot System

Setelah ROM ter-install dan bisa booting dengan normal, maka akan di temukan beberapa bug seperti
  • Suara tidak keluar
  • System UI dan Kamera force close
  • Animasi charging saat mati aneh
Jika anda mengalami hal-hal diatas, maka anda bisa melakukan fix-ing bug tersebut dengan cara:

Download fix (Dolby+Sound, UI, Baterai dan Kamera) dari download section dan letakkan di akar sdcard (/sdcard/fix_xxx.zip) dan boot ke recovery langkahnya sama seperti di atas tetapi hanya lakukan wipe dalvik-cache saja. Flash satu persatu.
