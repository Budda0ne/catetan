Tutorial Install ROM ColorOS R831K A.K.A NEO3

  1. Download ROM dari link di atas
  2. Matikan HH
  3. Masuk ke Recovery Mode dengan cara menekan tombol Volume Up & Power secara bersaman
  4. Pastikan Wipe Pre Flash / Wipe Data / Cache / Dalvik Cache / sebelum install zip Jangan Wipe System..
  5. Pilih Install Zip From SD-CARD dan cari ROM yg telah di download.
  6. Tunggu proses instalasi sampai selesai
  7. Setelah selesai Reboot System
  8. Booting pertama cukup lama (3-5menit)
  9. Flash Bug_Fix.zip
