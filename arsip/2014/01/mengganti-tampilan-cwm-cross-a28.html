User Cross A28 mempunyai CWM Recovery terbaru versi 6.0.1.2 yang ternyata bisa dirubah-rubah tampilannya. Caranya dengan menggunakan template modding (semacam themer) khusus unruk cwm recovery yang dibuat agan Jhonatan Agsunod dari Filipina.

Theming TeamBURST CWM Recovery v6.0.1.2

Anda bisa menggunakan file ini sebagai template untuk mempercantik tampilan CWM Recovery. Tetapi, Developer-nya mengatakan bahwa theming ini khusus di buat untuk TeamBURST CWM Recovery v6.0.1.2 saja meskipun mungkin bisa digunakan untuk CWM Recovery versi yang lain. Jika anda ingin mencobanya di CWM Recovery lain, Anda sendiri yang menanggung resiko-nya.

Cara menggunakan Theming TeamBURST CWM Recovery ini:
  • Download file ini ke komputer (PC/Laptop) kemudian ekstrak. (hanya untuk mengecek apakah file yang anda download tadi korup atau tidak)
  • Pindahkan file zip ke SDcard
  • Matikan Handheld dan nyalakan lagi ke Mode Recovery.
  • Pilih menu Install zip from sdcard
  • Cari file TeamBURST_CWM_Patcher.zip dan install
  • Tunggu hingga selesai. HH akan reboot sendiri dan otomatis masuk ke mode recovery dengan tampilan yang baru.

Selamat, CWM Recovery anda kini menjadi lebih cantik tampilannya.

Karena file zip ini bisa digunakan sebagai template juga, maka anda pun bisa mengganti-ganti tema CWM sesuai selera anda.
Ikuti panduan membuat tema sendiri dengan menggunakan Theming TeamBURST CWM Recovery sebagai template berikut ini:
  • Pertama, ekstrak file TeamBURST_CWM_Patcher.zip
  • Buka folder hasil ekstrak file tadi dan masuk ke /res/images/
  • Disana ada beberapa gambar yang bisa anda edit sesuka hati, tetapi jangan sampai merubah ukuran dan format gambar tersebut.
  • Kompres ulang foldeer META-INF dan folder res menjadi satu file zip

Selanjutnya anda bisa meng-install via CWM seperti sebelumnya.
