Rilis IGOS Nusantara 9

Tutorial Install IGN9 di Lenovo C440 menggunakan boot mode UEFI secara ringkas tapi pas.

Persiapan Instalasi:

Download iso image dari sini [1]

Buat liveUSB dengan sebagaimana ada di sini [2]. Saya menggunakan (USB Flash disk ada di /dev/sdb1):

dd if=IGN-D9.0-live64-sembilan-desktop.iso of=/dev/sdb1 oflag=direct bs=8M

Set bios di Lenovo C440

Ubah [Boot Mode] ke UEFI Only atau jika [Auto], ubah Boot Priority ke UEFI
Simpan dan reboot

Proses Instalasi:

Pada saat booting dari liveUSB, grub tidak memunculkan splash ataupun menu pilihan, tapi langsung masuk ke mode command : grub>

ketik perintah:

search --set -f /isolinux/vmlinuz0
linuxefi /isolinux/vmlinuz0 root=live:LABEL=IGN9 selinux=0 rootfstype=auto ro rd.live.image acpi=force
initrdefi /isolinux/inited0.img
boot

Komputer akan booting dan masuk ke mode LiveUSB.
Jalankan gParted (Aplikasi > Perkakas Sistem > GParted), hapus semua partisi yang ada.
Install ke Harddisk.
Pilih harddisk yang akan diinstalasi dan pilih mode otomatis untuk pembuatan partisi.
Lanjutkan instalasi sampai selesai.

Post Instalasi

Meregistrasikan boot loader dengan EFI (langkah ini entah diperlukan atau tidak) [3]

efibootmgr -c -p 1 -l \\EFI\\fedora\\bootx64.efi -L IGN9

Ubah Seting BIOS lagi

Startup > CSM --> Disabled
