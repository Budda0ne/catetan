Galaxy Ace GT-S5830 on Hand

Samsung Galaxy Ace atau seri GT-S5830 seperti yang kita tahu hanya mendapatkan dukungan Update dari Samsung sampai Android Froyo 2.2. Namun, developer dari Cyanogen Mod memberikan dukungan hingga Jelly Bean versi 4.3 [CM 10.2] meskipun masih Nightly build. Biasanya, Nightly build masih terdapat beberapa bug di dalamnya. Tetapi, bug-bug-nya sudah di berikan fixer.

Untuk anda pengguna Galaxy Ace GT-S5830 yang ingin meng-upgrade versi OS Androidnya ke Jelly Bean 4.3 bisa mengikuti panduan instalasi yang saya berikan. Tapi ingat, Semua resiko anda sendiri yang menanggung. Tutorial yang saya berikan di tutorial ini sudah saya coba di GT-S5830 teman saya dan bisa berjalan normal meski tidak semua fiturnya berjalan.

Sebelum anda melakukan instalasi Custom ROM JB 4.3 untuk GT-S5830, ada beberapa bahan yang perlu anda download di bagian bawah postingan (bagian download). Selain itu, Galaxy Ace anda juga harus sudah dalam kondisi ROOT dan ter-install CWM Recovery.
Untuk rooting, anda bisa membaca Cara Efektif Rooting GT-S5830 [Galaxy Ace]
Untuk instalasi CWM Recovery anda bisa membaca Cara Mudah Memasang CWM Recovery untuk Galaxy Ace [GT-S5830].

Tutorial Upgrade GT-S5830 menjadi Jelly Bean 4.3 [CM 10.2]
Lakukan step by step, jangan terburu-buru. Jika ada tanda (opsional), anda bisa mengabaikan-nya.
  1. Download file cm-10.2-20131008-NIGHTLY-JB.4.3.zip dari link download [di bawah]
  2. Download file GPS-V5-cm10-10.220131014.zip dari link download [di bawah]
  3. Download GAPPS JB 4.3 (opsional)
  4. Letak-kan ketiga file zip yang sudah anda download ke SDCard (/sdcard/disini), letakkan di luar folder agar mudah mencarinya.
  5. Matikan Galaxy Ace dan masuk ke mode recovery dengan menekan tombol [power] dan [volume down]. jika kerepotan atau tidak bisa masuk recovery mode, gunekan cara ini:
    • install terminal emulator, download dari playstore
    • buka terminal emulator dan ketik "su" tanpa tanda kutip, allow jika muncul peringatan superuser.
    • - setelah muncul tanda pagar (#) ketikkan "reboot recovery"
    Tunggu hingga masuk mode Recovery
  6. Setelah anda masuk recovery mode, gunakan tombol volume untuk memilih menu yang ada.
  7. Pilih menu Wipe System, Wipe Data dan Wipe Cache.
  8. Pilih menu "install zip from sdcard" tekan tombol [home]
  9. Pilih menu "choose zip from sdcard" tekan tombol [home]
  10. Pilih "cm-10.2-20131008-NIGHTLY-JB.4.3.zip" tekan tombol [home]
  11. Pilih "GPS-V5-cm10-10.220131014.zip" tekan tombol [home]
  12. Pilih "gapps-jb-20130813-signed.zip" tekan tombol [home] (opsional)
  13. Kembali ke menu awal dengan menekan tombol [back]
  14. Pilih menu "Reboot system now" tekan tombol [home]
Selesai. Galaxy Ace GT-S5830 anda sekarang sudah Upgrade ke Android Jelly Bean 4.3. Booting awal biasanya agak lama, santai saja.

Jika masih ada yang masih kurang jelas, bisa tanya-tanya di kolom komentar. Atau ninggalin jejak setelah sukses upgrade juga boleh :)
