Menu Pilihan Dump Utility ASUS Memopad ME172V

Sebagai pemilik Tablet ber OS Android, tentu belum puas kalau belum meng-utak-atik jeroan Tablet anda. Tetapi alangkah baiknya jika anda mempunyai back-up ROM Original-nya sebelum melakukan rooting ASUS Memopad ME172V.

Untuk kali ini, saya akan memberikan cara untuk mem-backup ROM Original ASUS Memopad ME172V terlebih dahulu sebelum anda melakukan hal-hal lain ke Tablet Memopad ME172V anda.

Dengan Mengikuti Tutorial ini anda setuju dengan Disclaimer

Untuk melakukan back-up ASUS Memopad ME172V, setidaknya anda harus menyediakan microSD untuk menyimpan hasil back-up ROM nanti. Lakukan langkah-langkah berikut jika anda mempunyai PC/Laptop. Jika anda tidak punya, sebaiknya anda mengikuti cara kedua meskipun pada dasarnya sama saja cara kerjanya.

Back-up Melalui PC/Laptop
  • Download toolsnya di bagian download (di bawah)
  • Extract file Me172vDumpUtil_v0.2-backup.zip ke sdcard (gunakan card reader)
  • Pastikan ASUS Memopad sudah di-charge sampai full 100% [untuk jaga-jaga]
  • Lepaskan sdcard dari card reader dan masukkan sdcard ke tablet dalam keadaan mati
  • Nyalakan tablet dan tunggu beberapa saat sampai muncul menu berlayar biru dengan 3 pilihan
  • Gunakan tombol "Volume Up" dan pilih "Dump me172v flash and some info" lalu tekan tombol "Power" dan biarkan proses berjalan
  • Setelah proses selesai (sekitar 5 menit) maka tablet akan mati dengan sendirinya
  • Lepaskan sdcard dari tablet dan nyalakan kembali Memopad ME172V
  • Copy dan simpan semua file di folder /me172v/dump ke tempat yang aman (hard disk PC/laptop)
  • Selesai
Back-up Langsung dari Tablet (tanpa PC)
  • Download toolsnya di bagian download (di bawah)
  • Extract file Me172vDumpUtil_v0.2-backup.zip ke /sdcard
  • Pastikan ASUS Memopad sudah di-charge sampai full 100% [untuk jaga-jaga]
  • Matikan dan nyalakan lagi Memopad ME172V
  • Tunggu beberapa saat sampai muncul menu berlayar biru dengan 3 pilihan
  • Gunakan tombol "Volume Up" dan pilih "Dump me172v flash and some info" lalu tekan tombol "Power" dan biarkan proses berjalan
  • Setelah proses selesai (sekitar 5 menit) maka tablet akan mati dengan sendirinya
  • Lepaskan sdcard dari tablet dan nyalakan kembali Memopad ME172V
  • Masukkan sdcard setelah Memopad ME172V menyala, jika anda memasukk-kan sdcard dalam sebelum menyala, anda akan mengulang lagi proses back-up tersebut.
  • Buka sdcard dan hapus file dengan nama scriptcmd
  • Selesai

Nah, Setelah anda mempunyai back-up ROM ASUS Memopad ME172V, anda pun bisa melanjutkan proses lanjutan semisal Rooting atau install CWM Recovery.
