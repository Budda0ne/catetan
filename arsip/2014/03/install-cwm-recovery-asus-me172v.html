CWM Recovery ASUS Memopad ME172V

Seperti biasa, Tutorial instal CWM Recovery selalu mempunyai banyak cara dan alternatif yang bisa digunakan. Dalam tutorial ini anda akan saya berikan 2 pilihan cara meng-install CWM Recovery ASUS Memopad ME172V.

Cara pertama menggunakan terminal emulator dan cara kedua menggunakan Dumping tool seperti di tutorial Back-up Stock ROM Memopad ME172V dengan sedikit manipulasi. Saya lebih menyukai cara yang menggunakan Terminal Emulator. Hanya saja ASUS Memopad ME172V harus dalam kondisi ROOT.

Dengan Mengikuti Tutorial ini anda setuju dengan Disclaimer

Install CWM Recovery dengan Terminal Emulator

Cara ini adalah cara paling mudah menrut saya anda tingggal mengetikkan beberapa baris kode melalui terminal emulator. Ikuti instruksi berikut.
  • Pastikan ASUS Memopad ME172V anda sudan ter-root dan mempunyai aplikasi terminal emulator
  • Download file mmcblk0p3.img di bagian download (link merah)
  • Tempatkan filemmcblk0p3.img di memori internal atau eksternal jika punya. Letak-kan di luar jangan didalam folder (misal: /sdcard/mmcblk1p3.img atau /extSdcard/mmcblk1p3.img)
  • Buka terminal dan ketik-kan SU, jika muncul peringatan pilih OK saja
  • Jika anda meletak-kan di memori internal, lanjutkan dengan mengetik perintah:
    dd if=/sdcard/mmcblk1p3.img of=/dev/block/mmcblk1p3
  • Jika anda meletak-kan di memori eksternal (microsd), lanjutkan dengan mengetik perintah:
    dd if=/extsdcard/mmcblk1p3.img of=/dev/block/mmcblk1p3
  • Selesai.

Penting, ketika mengetik perintah di atas cek sebelum menekan enter. perhatikan spasi dan huruf-hurufnya. harus sama agar tidak terjadi kesalahan. mmcblk1p1 jika dengan huruf besar adalah MMCBLK1P1

Install CWM Recovery dengan tool Dumping Utility

Ikuti instruksi berikut:
  • Pastikan Memopad ME172V sudah di-charge sampai full 100% [untuk jaga-jaga]
  • Download toolsnya di bagian download (di bawah)
  • Download file mmcblk1p3.img
  • Extract file Me172vRootUtil_v0.1-backup.zip ke /sdcard/disini
  • Copy-kan file mmcblk1p3.img ke dalam folder /sdcard/me172v/dump/ pilih overwrite (timpa)
  • Matikan dan nyalakan lagi Memopad ME172V
  • Tunggu beberapa saat sampai muncul menu berlayar biru dengan 3 pilihan
  • Gunakan tombol [Volume Down] dan pilih "Restore me172v flash (previously saved)" lalu tekan tombol [Power] dan biarkan proses berjalan
  • Setelah proses selesai akan muncul permintaan untuk mengeluarkan sdcard
  • Keluarkan sdcard dan Memopad ME172V akan otomatis reboot
  • Masukkan sdcard setelah Memopad ME172V menyala, jika anda memasukk-kan sdcard sebelum menyala, anda akan mengulang lagi proses rooting tersebut.
  • Buka sdcard dan hapus file dengan nama scriptcmd, agar anda bisa menggunakan sdcard anda seperti biasa
  • Selesai

Jika sudah selesai, anda bisa langsung mengecek apakah CWM Recovery sudah ter-install di tabelt ASUS Memopad ME172V anda atau belum dengan mengetik perintah melalui terminal emulator:
reboot recovery
Jika anda melihat menu seperti gambar di atas, maka anda sudah berhasil melakukan instalasi cwm recovery ke tablet ASUS Memopad ME172V anda. Selamat.
