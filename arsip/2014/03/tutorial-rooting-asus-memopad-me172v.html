ASUS Memopad ME172V Sudah terpasang Aplikasi Superuser (ROOT)

Pastikan anda mem-Back-up ROM ASUS Memopad ME172V anda terlebih dahulu, sebelum melakukan rooting. Ini penting untuk dilakukan agar nantinya anda bisa mengembalikan ASUS Memopad ME172V anda seperti semula jika terjadi hal-hal yg tidak diinginkan.

Seperti tutorial sebelumnya (back-up ROM), anda juga membutuhkan memori ecternal (microSD) untuk melakukan Rooting dengan cara ini. Jika anda tidak punya memori eksternal anda bisa menggunakan tool rooting VROOT sebagai alternatif.

Dengan Mengikuti Tutorial ini anda setuju dengan Disclaimer

Tutorial root ASUS Memopad ME172V ini bisa anda lakukan baik langsung melalui tablet maupun dengan bantuan PC/Laptop. Jika anda tidak mempunyai PC/Laptop anda bisa mengikuti panduan kedua.


Menu Rooting ASUS Memopad ME172V

Rooting Melalui PC/Laptop

Untuk melakukan rooting menggunakan PC/Laptop, ikuti instruksi berikut:
  • Download toolsnya di bagian download (di bawah)
  • Extract file Me172vRootUtil_v0.1-root.zip ke sdcard (gunakan card reader)
  • Pastikan ASUS Memopad sudah di-charge sampai full 100% [untuk jaga-jaga]
  • Lepaskan sdcard dari card reader dan masukkan sdcard ke tablet dalam keadaan mati
  • Nyalakan tablet dan tunggu beberapa saat sampai muncul menu berlayar biru dengan 3 pilihan
  • Gunakan tombol [Volume Up] dan pilih "Root me172v" lalu tekan tombol [Power] dan biarkan proses berjalan
  • Setelah proses selesai akan muncul permintaan untuk mengeluarkan sdcard
  • Keluarkan sdcard dan Memopad ME172V akan otomatis reboot
  • Selesai
Rooting Langsung dari Tablet
Untuk melakukan dari Tablet, tidak jauh berbeda, hanya ada beberapa perubahan langkah-langkahnya. Ikuti instruksi berikut:
  • Download toolsnya di bagian download (di bawah)
  • Extract file Me172vRootUtil_v0.1-root.zip ke /sdcard/disini
  • Pastikan Memopad ME172V sudah di-charge sampai full 100% [untuk jaga-jaga]
  • Matikan dan nyalakan lagi Memopad ME172V
  • NTunggu beberapa saat sampai muncul menu berlayar biru dengan 3 pilihan
  • Gunakan tombol [Volume Up] dan pilih "Root me172v" lalu tekan tombol [Power] dan biarkan proses berjalan
  • Setelah proses selesai akan muncul permintaan untuk mengeluarkan sdcard
  • Keluarkan sdcard dan Memopad ME172V akan otomatis reboot
  • Masukkan sdcard setelah Memopad ME172V menyala, jika anda memasukk-kan sdcard dalam sebelum menyala, anda akan mengulang lagi proses rooting tersebut.
  • Buka sdcard dan hapus file dengan nama scriptcmd, agar anda bisa menggunakan sdcard anda seperti biasa
  • Selesai

Untuk mengecek apakah ASUS Memopad ME172V sudah di root, anda bisa membuka drawer aplikasi dan lihat apakah sudah ada aplikasi Superuser di situ. Jika belum yakin, anda bisa menggunakan aplikasi Root Checker.

Selamat, anda sudah berhasil melakukan rooting tablet ASUS Memopad ME172V anda. Jika anda mau, anda bisa melanjutkannya dengan meng-instal CWM Recovery ASUS Memopad ME172V anda untuk mempermudah dalam melakukan back-up dan restore tablet anda termasuk meng-install Custom ROM.
