CWM (ClocWorkMod) dan TWRP (TeamWin Recovery Project)merupakan recovery kustom yang paling banyak digunakan pengguna ponsel android. Sebenarnya Xiaomi Redmi 1S (WCDMA) memiliki Mi recovery secara default, tetapi Recovery kustom mempunyai fitur lebih banyak daripada recovery bawaan.

Dalam tutorial ini, saya akan meberikan cara menginstal CWM pada Xiaomi Redmi 1s dan cara menginstal TWRP pada Xiaomi Redmi 1s (pilih salah satu), sehingga nantinya anda bisa menginstal custom ROM pada Redmi 1S Anda dengan lebih mudah.

Baca: Menginstal CyanogenMod 11 pada Redmi 1s untuk contoh penggunanaan CWM Recovery.

Sebelum memasang CWMR, pastikan baterai ponsel Anda dalam keadaan terisi, usahakan berada dikisaran 60 persen. CWM Recovery ini hanya untuk ponsel Xiaomi tipe Redmi 1S, jadi jangan gunakan pada ponsel Xiaomi lainnya (mis: Redmi Note).

Panduan Install CWM Recovery

Berikut langkah-langkah untuk install CWM recovery di Xiaomi Redmi 1S anda:

  1. Aktifkan opsi Developer options, caranya buka Settings pilih About phone dan pencet di Android version 5 kali, hingga muncul notifikasi Developer options sudah aktif.
  2. Persiapkan dulu dengan mendownload file Zip CWM Recovery (signed_cwm_update.zip) di sini dan rubah namanya menjadi update.zip saja.
  3. Letakkan file update.zip di akar internal storage (memori internal)
  4. Masuk ke Recovery Mode (bawaan), dengan cara:
    • Matikan ponsel kemudian dalam kondisi mati tekan tombol volume up + tombol power, secara bersamaan.
    • Bisa juga masuk ke Recovery Mode dari keaadaan ponsel nyala, dengan cara masuk ke Settings pilih Updater dan pencet tombol menu (sebelah kiri tombol home) dan pilih Reboot to Recovery mode
    • Dalam Recovery Mode, gunakan tombol volume up dan volume down untuk navigasi menu dan gunakan tombol power untuk memilih.
    • Pilih menu English pilih Install update.zip to system konfirmasikan YES.
  5. Proses flashing akan berlangsung cepat.
  6. Pilih menu Back pilih Reboot dan Reboot to system dan tunggu hingga ponsel menyala.
Selesai, CWM Recovery sudah ter-install.

Panduan Install TWRP

Langkah-langkah untuk menginstal TWRP di Xiaomi Redmi 1S anda sama persis dengan cara install CWM recovery di atas, hanya saja perbedaannya hanya file yang digunakan.Caranya:

  1. Aktifkan opsi Developer options, caranya sama.
  2. Download file Zip TWRP di sini dan kali ini anda tidak perlu merubah nama karena memang sudah bernama update.zip :D
  3. Letakkan file update.zip di akar internal storage (memori internal)
  4. Masuk ke Recovery Mode (bawaan) dan ikuti lakukan panduan sebelumnya
  5. Setelah complete, reboot to system dan tunggu hingga ponsel menyala.

Selesai, TWRP pun ter-install.

Tidak masalah apakah anda menggunakan CWM atau lebih memilih TWRP, fungsinya tidak berbeda jauh tetapi saya lebih suka menggunakan TWRP Recovery.
Dan sekarang anda dapat dengan mudah menginstal ROM lain di ponsel Xiaomi Redmi 1s anda.
