Punya ponsel Xiaomi Redmi 1S dan belum melakukan rooting? Bagi anda yang suka mengoprek sistem Android, pasti anda tidak mau ketinggalan untuk me-root ponsel Xiaomi Redmi 1S anda. Bagi yang belum tahu apa maksud dari root, anda bisa membaca artikel tentang root dan superuser. Singkatnya, ketika anda melakukan root, maka anda mempunyai kebebasan dalam mengatur ponsel anda. Dalam tutorial ini, selain cara root juga desertakan cara untuk Unroot juga.

Meskipun beberapa vendor ponsel meng-haram-kan rooting (garansi hilang), Tetapi Xiaomi mengatakan bahwa root pada Redmi 1S ini tidak akan menghilangkan garansi ponsel, jadi aman. Hal ini dibuktikan dengan adanya fitur untuk mengaktifkan dan menonaktifkan akses root (Root Permission) dalam ponsel ini. Untuk menge-cek apakah sudah root, coba masuk ke menu Security -> Permission -> Root permission. Jika ponsel belum root maka menu ini tidak bisa diklik.

Cara root Xiaomi Redmi 1S ini mudah dilakukan dan tanpa harus menggunakan PC/komputer. Rooting Xiaomi Redmi 1S bisa dilakukan langsung dari ponsel dan sangat mudah dilakukan. Namun ada beberapa syarat sebelum root Xiaomi Redmi 1S yakni: Xiaomi Redmi 1S menggunakan ROM versi stable.

Rooting Xiaomi Redmi 1S
Untuk melakukan rooting, ikuti panduan berikut:
  1. Aktifkan Developer options, caranya buka Settings pilih About phone dan pencet di Android version 5 kali, hingga muncul notifikasi Developer options sudah aktif.
  2. Download file update.zip (dibawah)
  3. Pindahkan file download update.zip tadi ke akar internal storage (memori internal)
  4. Masuk ke Recovery Mode (bawaan), dengan cara:
    • Matikan ponsel kemudian dalam kondisi mati tekan tombol volume up + tombol power, secara bersamaan.
    • Bisa juga masuk ke Recovery Mode dari keaadaan ponsel nyala, dengan cara masuk ke Settings pilih Updater dan pencet tombol menu (sebelah kiri tombol home) dan pilih Reboot to Recovery mode
  5. Dalam Recovery Mode, gunakan tombol volume up dan volume down untuk navigasi menu dan gunakan tombol power untuk memilih.
  6. Pilih menu English pilih Install update.zip to system konfirmasikan YES.
  7. Proses rooting akan berlangsung sangat cepat.
  8. Pilih menu Back pilih Reboot dan Reboot to system dan tunggu hingga ponsel menyala.

Proses root Xiaomi Redmi 1S selesai dan anda bisa meng-cek apakah rooting sudah berhasil dengan cara masuk ke menu Security pilih Permission dan klik Root permission untuk mengaktifkan akses root. Jika pilihan tersebut bisa di klik berarti ponsel Xiaomi Redmi 1S sudah berhasil di-root. Jangan lupa untuk menghapus/memindahkan file update.zip ke dalam folder.

Un-root Xiaomi Redmi 1S

Sedangkan untuk proses unroot Xiaomi Redmi 1S ini langkahnya sama dengan proses rooting. Langkah un-root juga sama-sama mudah. Caranya:

  1. Hapus file update.zip (jika belum) dari internal storage (memori internal).
  2. Download file UnRoot.zip ada di bawah dan letakkan di akar internal storage.
  3. Rename file UnRoot.zip menjadi update.zip
  4. Masuk ke Recovery Mode, caranya sama.
  5. Lanjutkan seperti saat melakukan rooting dari no. 5 s/d no.8
