Saya termasuk yang sudah mengganti nomor id profil FB saya menjadi username singkat. Tetapi dalam berbagai kesempatan (terutama developer) masih sering diminta memasukkan Nomor ID dan bukan username. Lalu bagaimana cara mengetahui lagi nomor id profil saya? Ada 2 Alternatif yang bisa anda ikuti:

Sponsor blog ini
1. Dengan melihat link Foto yang Anda upload.
2. Dengan menggunakan Open Graph milik Facebook.

Dua cara mudah untuk mengetahui nomor id FB Facebook di atas bisa juga untuk melihat Nomor ID Facebook orang lain. Anda cukup memakai salah satu dari dua cara berikut:

1. Mengetahui Nomor ID Facebook dengan melihat link Foto yang Anda upload

Cara pertama utnuk mencari nomor id Facebook adalah dengan cara membuka link foto kita di Facebook. Nah dari link tersebut akan ditemukan beberapa nomor. Salah satunya adalah nomor id profile fb kita. Ikuti langkah berikut:

1. Masuk ke facebook.com dengan username dan password anda.
2. Kemudian klik di salah satu Foto yang sudah anda upload dari beranda.
3. Pada Foto tersebut, klik kanan dan pilih Open link in new tab
4. Perhatikan addres bar browser, anda akan melihat link yang mirip seperti ini:
https://www.facebook.com/photo.php?fbid=287250837970968&set=t.1514647497&type=1
5. Letak nomor FB ID anda ada setelah &set=t. dan sebelum &type=1
Dalam contoh di atas, nomor ID Facebook profile-nya adalah 1514647497.

2. Mengetahui Nomor ID Facebook dengan menggunakan Open Graph milik Facebook.

Trik ini adalah cara mengetahui FB ID yang paling mudah. Dan sampai saat ini masih bisa anda gunakan.

1. Copy link berikut: https://graph.facebook.com/username
2. Ganti username dengan username facebook anda. ( misal: facebook.com/nasyrih )
3. Copy/paste ke browser (IE/Firefox/Chrome) dan klik enter.
Nomor yang paling di baris paling atas iulah nomor id fb anda.
