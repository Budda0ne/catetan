Hal ini seperti yang di janjikan oleh pabrikan Mito sendiri. Tetapi, bagi yang sudah terlanjur melakukan rooting apalagi meng-install custom ROM, tentu tidak bisa menikmati Update OTA yang diberikan.

Jika Anda ingin meng-upgrade Mito A77 Fantasy Selfie menjadi Android KitKat, mau tidak mau anda harus melakukan flashing Firmware KitKat untuk Mito A77 Fantasy Selfie melalu SP Flashtool. Firmware KitKat ini adalah hasil dumping stockrom yang sudah di-upgrade via OTA update. Jadi ketika anda sduah melakukan flashing, otomatis OS Android Mito A77 Fantasy Selfie anda akan berubah menjadi versi Kitkat 4.4.2.

Tutorial Flashing Firmware Mito A77 Fantasy Selfie


Sebelum memulai, persiapkan dulu bahan dan alatnya:
1. PC/Laptop dengan OS Windows (XP/7/8)
1. Handheld (Mito A77 Fantasy Selfie)
2. Kabel data USB (micro)
3. Firmware Kitkat, download di bawah
4. Driver USB VCOM, download di bawah (link updated)
5. SP Flashtool, download SP Flashtool terbaru

Setelah lengkap alat dan bahannya, ikuti langkah-langkah berikut:

  1. Install Driver VCOM, Jika muncul peringatan abaikan dan pilih Install anyway
  2. Ekstrak file firmware Mito A77 Kitkat di desktop agar lebih mudah mencari.
  3. Ekstrak juga software SP Flashtool di desktop.
  4. Matikan Mito A77 dengan cara mencabut baterai, tunggu sekitar 10 detik.
  5. Lepas sdcard, simcard (boleh tidak, tapi sebaiknya dilepas)
  6. Pasang kembali baterai, jangan dinyalakan.
  7. Buka Software SP Flashtool (Flash_tool.exe)
  8. Pilih menu Scatter Loading, dan cari file bernama MT6572_Android_scatter.txt dari folder Firmware di desktop
  9. Otomatis akan terload semua file yang dibutuhkan. Klik Download.
  10. Sambungkan Mito A77 ke PC/Laptop dengan kabel USB, otomatis akan terdeteksi.
  11. Biarkan proses flashing berjalan, waktunya berkisar 10 menit.
  12. Jika sudah muncul lingkaran hijau, itu artinya proses selesai dan sukses.
  13. Cabut Mito A77 dari PC/Lapto dan lepaskan baterai selama 10 detik.
  14. Pasang kembali baterai dan nyalakan.

Upgrade Mito A77 Fantasy Selfie ke Android 4.4.2 KitKat sudah selesai. Proses Boot up akan memakan waktu sekitar 5 menit dan paling lama 10 menit.
