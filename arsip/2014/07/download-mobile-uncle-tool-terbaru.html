Aplikasi ini hanya untuk perangkat dengan chipset Mediatek saja.

Aplikasi ini sebenarnya bukan hanya berfungsi untuk membantu melakukan install CWM (atau custom recovery lainnya) seperti yang anda tahu. App Mobile Uncle ini lebih seperti All In One app for rooted devices.

Beberapa fungsi lain yang dimiliki oleh Mobile Uncle ialah disa digunakan untuk back-up dan restore IMEI jika terjadi kasus Invalid IMEI untuk chipset MTK.

Untuk versi terbaru Mobile Uncle (versi 2.9.9) sudah ada beberapa perbaikan dengan dukungan MTK quad-core dengan skema MT6589 aircraft Recovery module. Update dukunagn ini sebenarnya sudah ada sejak Mobile Uncle v.2.8.0. Sehingga jika sebelumnya anda mengalami gagal install recovery menggunakan Mobile Uncle versi lama, anda bisa mencoba aplikasi versi terbaru ini untuk meng-install custom recovery.
