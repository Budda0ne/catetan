Evercross A7T menggunakan Board MT6572, sama dengan Evercoss A26C bedanya, layar lebih luas A26C dan ram A7T hanya 256 mb saja. Fitur lainnya hampir sama.

File Original Firmware Evercoss A7T JellyBean 4.2.2 ini menggunakan MTK Droid Tools 2.5.3 untuk proses backup dan SP Flashtool untuk Readback ROM.

Sementara saya sediakan file stock firmware-nya dulu. Untuk Tutorial Flashing A7T dengan SP Flashtool, akan saya berikan menyusul seperti biasanya (sedang saya siapkan). Kalau anda butuh segera, silahkan anda cari tutorial di google terlebih dulu.
