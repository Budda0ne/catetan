Pernah melihat kawan anda memamerkan handphone buatan Oppo? Cantik kan? Anda tetu tertarik dengan ColorOS. ColorOS ini juga sama dengan MIUI OS yang juga dikembangkan dari OS Android oleh pabrikan Xiaomi. ROM ColorOS R831T ini berbasis Android 4.2.2 (Jelly Bean)

Nah, Berterimakasihlah kepada developer lokal kita yang berhasil mem-porting Color OS untuk Evercoss A26C (lihat bagian kredit). Karena kini anda juga bisa merasakan ColorOS di Handheld kesayangan anda: Evercoss A26C.

Fitur yang ditambahkan kedalam ROM Color OS R831T ini selain aplikasi aplikasi bawaan Color OS antara lain:
  • Super user
  • Walkman Audio Player
  • Noozxoiddde EIZO Rewire FX
  • Kamera Google
  • Galery Google
OTA Update masih berfungsi normal, anda juga bisa mendowload-nya. Tapi ingat: "JANGAN LAKUKAN UPDATE, KARENA A26C Anda AKAN MENGALAMI BOOTLOOP"
Tidak ada custom ROM yang sempurna, aplagi ROM portingan. Bugs yang paling terasa :
  • Reboot (fast reboot) saat mengakses file-file di root folder dan aplikasi-aplikasi yang membutuhkan hak Super user.
  • Saat anda mengakses menu Wallpaper, dari Homescreen (Swipe Up) Domescreen akan force Close (tidak terlalu mengganggu)
  • Microphone tidak berfungsi, gunakan headset
  • GPS tidak berfungsi
Selain bugs diatas, init.d support ROM Color OS ini belum aktif.

Istimewanya adalah, ROM Oppo R831T ini sangat smooth. Benchmark scoring antutu juga menunjukkan performanya diatas Galaxy S2

Untuk menunjang kinerja Evercoss A26C anda, anda bisa menambahkan Turbo Boost Swap jika anda sudah mengaktifkan init.d support

Cara Pasang/Instalasi Custom ROM Oppo R831T

Sebelum meneruskan tutorial Instalasi ini, anda berarti sudah setuju dengan artikel Disclaimer dan mengerti serta patuh dengan artikel DMCA Copyright di blog ini. Dan anda juga paham bahwa dengan menginstall Custom ROM di handheld anda bisa menyebabkan garansi anda gugur resiko kehilangan data, backup data penting anda.

Syarat: harus sudah ter-install CWM Recovery, jika belum, baca di tutorial install CWM Recovery.

Untuk memasang ROM ini, anda harus melakukan Backup IMEI melalui aplikasi Mobile Uncle atau aplikasi backup IMEI lainnya, sebagai antisipasi jika terjadi INVALID IMEI. Buat juga Nandroid Backup melalui CWM Recovery sebelum melakukan flashing.

Langkah-langkah instalasi:

1. Download ColorOS.zip dan letakkan di sdcard tapi jangan di dalam folder. Letakkan di /sdcard/--disini- - (menjadi /sdcard/ColorOS.zip)
2. Matikan Evercoss A26C anda, dan hidupkan lagi dalam mode CWM Recovery dengan menekan tombol [Power] dan [Volume Up]
3. Ketika sudah masuk mode CWM Recovery, gunakan tombol [Volume Up] dan [Volume Down] untuk navigasi dan tombol [Power] untuk memilih menu:
  • pilih menu Wipe Data/Factory Reset pilih yes
  • pilih lagi menu Wipe Cache Partition pilih yes lagi
  • pilih menu Advaced, pilih Wipe Dalvik Cache
  • Pilih Install Zip From SdCard kemudian pilih file ColorOS.zip
5. CWM Recovery akan melakukan proses flashing, tunggu hingga selesai.
6. Setelah Selesai, pilih GO BACKx20(menu paling bawah) dan pilih Reboot System Now
Jika Proses Instalasi berjalan mulus, anda akan mendapati Evercross A26C anda sudah terpasang/terinstall ColorOS R831T.
