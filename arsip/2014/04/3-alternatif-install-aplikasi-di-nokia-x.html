Nokia X telah resmi diperkenalkan. Handphone ini merupakan smartphone pertama dari Nokia yang menjalankan sistem operasi Android, yang masih versi 4.1.2 Jelly Bean (AOSP).

Meski berbasis Android, Nokia tidak menyertakan layanan-layanan Google (G-apps) untuk Nokia X, bahkan toko aplikasi Google Play pun di buang. Lalu bagaimana pemilik Nokia X memperoleh aplikasi Android untuk dijalankan di ponselnya? Bagaimana cara install aplikasi di Nokia X?

Jawabannya ternyata sederhana dan mudah dilakukan. Ada tiga cara memperoleh dan menjalankan aplikasi Android di Nokia X. Berikut ini cara install aplikasi android di Nokia X secara singkat.

1. Melalui Nokia Store


Ini adalah toko resmi Nokia yang menyediakan aplikasi-aplikasi Android. Nokia menjelaskan bahwa developer bisa langsung menyetorkan file installer (APK) dari aplikasi mereka ke Nokia Store. Pemilik Nokia X pun bisa mengunduh aplikasi dari toko aplikasi ini.

2. Melalui toko aplikasi pihak ketiga


Di samping toko aplikasi resmi seperti Google Play dan Nokia Store, aplikasi Android untuk Nokia X juga dapat diperoleh dari toko aplikasi pihak ketiga semisal Yandex.

Tautan-tautan menuju toko-toko aplikasi ini akan disediakan di Nokia Store untuk mempermudah pengguna dalam mencari aplikasi yang diinginkan.

3. Dengan sideloading aplikasi


Perangkat Android biasa bisa memasang aplikasi dari file mentah (file APK) yang diperoleh dari sumber manapun. Demikian juga dengan Nokia X.
Yang diperlukan hanyalah mengaktifkan pilihan "allow installation of apps from unknown sources" di menu Setting dan mendownload file APK ke memori ponsel. Tapi ingat, anda harus mengunduh file APK dari sumber terpercaya untuk menghindari kemungkinan infeksi malware.

Nokia sendiri menjanjikan bahwa 75 persen aplikasi Android yang beredar saat ini bisa dijalankan di Nokia X tanpa modifikasi apapun. Sebanyak 25 persen sisanya -terutama yang memakai layanan Google, seperti Maps atau mekanisme pembayaran dalam aplikasi - perlu di-porting terlebih dahulu menggunakan SDK Nokia X dan di-submit ke toko aplikasi Nokia Store.
