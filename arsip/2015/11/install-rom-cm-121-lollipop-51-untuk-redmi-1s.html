Sebenarnya agak eman untuk mengganti Official ROM MIUI7 dengan CyanogenMod 12.1 ROM ini. Tapi mau bagaimana lagi, dengan aplikasi bawaan yang berat dan membuat kepala pening karena system kehausan, mau tidak mau harus diganti dengan ROM yang lebih ringan. Meskipun harus mengorbankan beberapa fitur bawaan Xiaomi yang lumayan berguna.

Sebagai informasi, CM 12.1 release R29 Unofficial ini sudah lebih stabil dan minim bugs, karena merupakan rilis stable. Dibangun berdasarkan OS Lollipop 5.1 (MIUI versi 7 untuk RedMi 1S masih KitKat), ROM ini tentu lebih baik dalam optimalisasi penggunaan RAM yang hanya 1 GB saja. Ditambah lagi dengan peng-aktif-an ZRAM untuk swapping bisa membantu menambah kapasitas RAM yang cekak, terutama jika anda meng-install banyak aplikasi sosial media.

Selesai sudah sedikit review dari saya, dan jika anda tertarik untuk meng-install ROM CM 12.1 Lollipop ini, anda bisa mempersiapkan hal-hal berikut:

1. Install TWRP Recovery dengan mengikuti panduan berikut ini.
2. Download dan letakkan di sdcard file ROM CM 12.1 Unofficial di bawah.
3. Download dan letakkan di sdcard juga file bootloader/modem dari OS Kitkat.
4. Download juga gapps jika anda membutuhkan aplikasi dari google (recomended).

Instalasi CyanogenMod 12.1 Lollipop untuk RedMi 1S

Karena instalasi ini akan menghapus semua data anda yang berada di ponsel, silahkan back-up data yang anda anggap penting.
Dengan mengikuti tutorial flashing yang saya berikan berarti anda sudah setuju dengan Disclaimer dan harus memahami copyright notice di KnoAcc Blog jika ingin membagikan tutorial ini. Jika tidak setuju silahkan minggat dari sini.

Jika 4 hal diatas sudah dipenuhi, anda bisa mengikuti panduan instalasi berikut untuk memasang CM 12.1 di ponsel RedMi 1S anda:
  1. Pastikan bahwa 2 file zip: firmware-kk-7.0.5.0.zip dan cm-12.1-20151117-UNOFFICIAL-armani.zip berada di sdcard. Sedangkan file gapps-511-base-20151114-1-signed.zip hanya jika anda membutuhkan aplikasi google.
  2. Matikan ponsel Redmi 1S anda dan boot ke mode recovery (TWRP) dengan menekan tombol volume up (+) dan power secara bersamaan.
  3. Dalam mode recovery TWRP, lakukan Wipe Data, System dan Cache untuk melakukan fresh install.
  4. Setelah selesai, lakukan flashing ber-urut (queue) dengan urutan ROM CM 12.1, GAPPS 5.1.1 dan Firmware KK
  5. Tunggu hingga proses instalasi ketiganya selesai kemudian pilih menu reboot to system.

Proses boot pertama mengkin akan memakan waktu agak lama, sekitar 5 menit atau paling lama 15 menit. Dan RedMi 1S anda kini sudah menggunakan OS Lollipop 5.1.1 yang bebas bugs, ringan dan menggemaskan. ROM ini sudah saya ujicobakan di 3 (tiga) ponsel RedMi 1S yang berbeda. Yang rata-rata sudah mengunakan MIUI7 dan pemiliknya mengeluh masalah lemot dan bikin darah naik.

Troubleshoot

1. Untuk meng-aktifkan Developer options, buka di Settings > About phone dan tekan 5 kali pada Build number.
2. Untuk mengaktifkan akses root (native root, tidak perlu melakukan rooting lagi), buka Settings > Developer options > Root access, dan aktifkan untuk aplikasi, adb atau keduanya juga bisa.
3. Untuk meng-aktifkan mode 3G only, buka dialer dan ketik *#*#4636#*#*, buka pilihan "Phone Information", disana anda bisa memilih mode jaringan yang anda inginkan.
4. Jika ingin melakukan update GAPPS lakukan reboot dan masuk ke recovery mode lakukan wipe System, Cache dan Dalvik cache tetapi jangan wipe Data untuk mempertahankan data sensitif anda. Ini juga untuk fix jika ada masalah dengan aplikasi google (mis: com.google.android has stopped)
