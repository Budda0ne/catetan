Tetapi fitur cantik ini tidak ada lagi di Windows 7 dan setelahnya, Tetapi, jangan kuatir, anda masih bisa menikmati fitur Dreamscene ini. Ikuti cara menjadikan video sebagai wallpaper di Windows 7. Trik ini menggunakan tool DreamScenes Activator.

Siapkan DreamScenes atau Video yang akan dijadikan Wallpaper


Pertama, Anda harus mempunyai DreamScenes (video wallpaper). Bagaimana jika belum punya Dreamscenes? Saya menemukan beberapa video yang keren di dreamscene.org dan deviantart.com. Anda bisa mendownload-nya di bawah. File berupa file rar/zip, setelah anda download, ekstrak terlebih dahulu sebelum memakainya sebagai Video Wallpaper.

Menggunakan Windows 7 DreamScene Activator


Jika Anda menjalankan Windows 7 Anda dapat menggunakan Windows 7 DreamScene Activator. Tool portable gratis yang bisa dijalankan di Windows 7 versi 32bit maupun versi 64bit. Dengan DreamScene Activator ini anda dapat menjadikan video dengan ekstensi MPG atau WMV sebagai wallpaper di desktop Windows 7 anda.

Download dan ekstrak Windows 7 DreamScene Activator (link juga ada di bawah). Setelah diekstrak, jalankan aplikasi sebagai administrator. Klik kanan pada file Dreamscene.exe dan pilih Run as administrator.

Klik pada tombol Enable DreamScene. Windows Explorer akan menutup sendiri, kemudian terbuka kembali.

Fitur DreamScene kini sudah aktif. Untuk memainkan (menjadikan) video DreamScene, cari file yang sudah anda download sebelumnya, jika sudah ketemu, klik-kanan file tersebut dan pilih menu Set as Desktop Background.

Kini Video anda bisa anda jadikan desktop. Tapi ingat, tidak semua video bisa dijadikan sebagai Video Wallpaper. Untuk bisa dijadikan sebagai Video Wallpaper, format video harus MPG atau WMV. Jika anda ingin memainkan semua jenis viseo sebagai wallpaper adna bisa menggunakan VLC Media Player

Cara Alternatif: Menggunakan Video sebagai Wallpaper dengan VLC.

Meskipun menurut pengembangnya untuk dijalankan di Windows 7 saja, ternyata tool ini bisa dijalankan untuk Vista Home Premium 32bit juga.

Anda bisa menghentikan sementara DreamScene kapansaja dengan cara klik-kanan di desktop dan memilih menu Pause DreamScene.

Dan jika anda ingin mengembalikan desktop anda seperti semula, anda bisa men-disable fitur Dreamscenes dengan membuka kembali too ini kemudian pilih Disable DreamScene.

Desktop anda pun kembali seperti sebelum menggunakan wallpaper video.
