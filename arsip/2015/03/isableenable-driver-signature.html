Masalah yang kemudian muncul adalah masih banyaknya perangkat yang masih menggunakan unsigned drivers. Misalnya saja beberapa driver Android debugging atau COM Port untuk MTK device, Beberapa printer Epson dan HP tipe lama. Jika di Windows 7 akan muncul pesan kesalahan yang bisa di abaikan berbeda dengan Windows 8 dan Windows 8.1

Update : Cara ini juga bisa dipakai di Windows 10.

Karena beberapa masalah itulah kali ini akan saya beritahukan cara untuk meng-install nya tanpa pesan error.

Digitally signed drivers, merupakan driver yang sudah mempunyai electronic fingerprint yang menandakan bahwa driver tersebut asli buatan perusahaan pembuat devices. Ini untuk balasan keamanan, Karena jika sudah dimodifikasi oleh pihak lain, maka digital signatures ini akan berubah tidak valid. Driver driver ini di tandai dengan code signing certificates untuk Benjamin keaslian driver.

Cara Non-aktif-kan Driver Signature Verification di Windows 8.1 atau 10 64bit

Untuk menonaktifkan (disable) driver signature enforcement, kita hanya perlu masuk ke menu Troubleshooting options di bagian boot manager Windows 8.1 atau Windows 10. Cara termudah untuk meng-akses menu tersebut adalah dengan trik rahasia yang jarang terpakai.

Langsung saja pilih menu Restart dari pilihan menu Power. Untuk Windows 8 berada di bawah Charms bar atau di layar login, sedangkan untuk Windows 10 berada di Start Menu.

Tetapi berbeda dengan biasanya, kali ini tekan tombol [shift] di keyboard saat anda memilih menu Restart.
Sekali lagi, trik ini bisa dipakai untuk menu Power di Windows 8 atau 10, baik itu yang berada di layar login, Charms bar, Start Menu, ataupun Start Screen

Setelah Windows restart, pilih menu Troubleshoot option.

Kemudian lanjutkan dengan memilih Advanced options.

Di bagian ini keta bisa memilih menu Startups settings

Karena saat ini kita sedang merubah pengaturan booting Windows, maka kita harus me-restart PC/komputer sekali lagi.

Nah setelah restart, anda akan disuguhi beberapa pilihan startup dengan no urut masing-masing. Untuk men-disable signature verification (signature enforcement) kita bisa memilih no. 7 dengan menekan tombol F7.

Setelah memilih menu disable signature, Windows akan reboot sendiri. Dan kini kita bisa melakukan instalasi driver tanpa pesan kesalahan meskipun driver tersebut unsigned.
