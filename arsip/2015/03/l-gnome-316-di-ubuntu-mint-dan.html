GNOME 3.16 sudah dirilis dengan berbagai fitur yang menawan. Fitur baru meliputi perbaikan di bidang antar muka, pengelola file Nautilus, Overlay Scrollbar, Mutter, Control Center, Gaming dan penampil dokumen. Tidak hanya perbaikan fitur yang diberikan, tapi banyak hal baru juga ditambahkan dalam versi 3.16 ini. Kita sudah membahas fitur-fitur yang paling menonjol dalam artikel sebelumnya. Sekarang, anda bisa mencoba menginstal di distro Linux yang berbeda.

Note: Perlu diketahui bahwa meskipun GNOME 3.16 sudah resmi dirilis tetapi saat ini distribusi Linux tidak memiliki paket resmi yang direkomendasikan untuk di install. Hampir semua distro Linux masih dalam tahap bersiap-siap untuk melengkapi rilis baru mereka dengan versi ini. Tapi anda masih bisa mencobanya pada sistem Linux yang anda pakai, saat ini paket GNOME 3.16 masih dalam pengujian, dan ada di repositori ujicoba (staging-repositories), yang berarti masih memiliki bug dan dalam tahap fixing oleh pengembang. Menginstal paket pengujian tersebut pada sistem tidak dianjurkan karena dapat merusak sistem anda. Sebelum melanjutkan untuk menginstal GNOME 3.16 dari repositori ujicoba, sangat disarankan agar Anda membuat cadangan data penting Anda.

Memasang GNOME 3.16 di Ubuntu dan Linux Mint


GNOME 3.16 saat ini masih ada di repositori ujicoba untuk sistem operasi berbasis ubuntu. Jika Anda menjalankan ubuntu atau sistem operasi yang berbasis ubuntu seperti Linux Mint, Anda harus terlebih dahulu menambahkan staging repositori GNOME ke manajer paket Anda. Buka terminal dan ketik perintah di bawah untuk meambahkan repositori ujicoba:

sudo add-apt-repository ppa:gnome3-team/gnome3-staging
sudo add-apt-repository ppa:gnome3-team/gnome3


Karena ini adalah repositori ujicoba, manajer paket akan meminta Anda untuk mengkonfirmasi penambahan ini untuk memahami konsekuensinya, tekan "Enter" untuk melanjutkan.

Setelah selesai, kita perlu memperbarui repositori paket manajer kita dengan menjalankan perintah berikut:

sudo apt-get update

Sudah, dan silahklan lakukan instalasi GNOME3.16 dengan mengetik perintah berikut:

sudo apt-get install gnome


Memasang GNOME 3.16 di Arch Linux


Distr0 Arch Linux yang terkenal selalu paling update jika ada rilis terbaru dari paket perangkat lunak. Tapi paket GNOME 3.16 untuk Arch Linux belum ada pada saat artikel ini ditulis. Meskipun demikian, telah ditambahkan ke repositori pengujian. Untuk menguji versi GNOME baru, Anda harus mengaktifkan repositori pengujian Arch Linux terlebih dahulu. Setelah selesai, Anda akan dapat menginstal versi baru dari GNOME.

Pertama-tama aktifkan repositori "Testing" Arch Linux GNOME. Seperti yang mungkin sudah Anda tahu, sistem operasi berbasis Arch Linux menggunakan "pacman" sebagai manajer paket. Sekarang edit file konfigurasi Arch Linux Package Manager di editor teks "gedit" dengan menjalankan perintah berikut di terminal.

sudo gedit /etc/pacman.conf


Sekarang tambahkan baris berikut dalam file ini untuk mengaktifkan repo pengujian GNOME.
[testing]

Include = /etc/pacman.d/mirrorlist

[gnome-unstable]

Include = /etc/pacman.d/mirrorlist

Sekarang tinggal memperbarui manajer paket Anda dan arch Linux Anda dengan perintah berikut, segera setelah selesai diproses, Anda akan mendapati GNOME 3.16 yang berjalan pada sistem operasi Anda.

sudo pacman -Syyu
