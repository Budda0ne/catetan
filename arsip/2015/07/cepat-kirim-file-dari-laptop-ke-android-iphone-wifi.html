Tetapi, ada kalanya anda lupa dimana menaruh kabel data anda. Mungkin anda akan berpikir untuk menggunakan konektifitas bluetooth jika mengalami hal ini. Tetapi tahukah anda bahwa kita bisa mengirim file secara langsung menggunakan wi-fi theter (portabel hotspot)? Kelebihan menggunakan Wi-fi tentu berada di perbedaan kecepatan berkirim file dibbandingkan menggunakan bluetooth. Apalagi jika file yang anda kirimkan berukuran besar, misalnya file terkompresi seperti file Custom ROM atau file video.

Cara Kirim File dengan CepatJika anda sudah familiar dengan wi-fi tether-ing, mungkin anda akan menyebutkan aplikasi populer Superbeam di android. Anda tidak salah, tetapi akan saya beritahu anda aplikasi lain yang gratis dan tidak perlu membeli lisensi pro untuk membuka fitur-fitur lengkapnya. Aplikasi ini bernama ShareIt yang dibuat oleh Lenovo untuk jajaran ponsel cerdas buatannya. Jangan kuatir, Selain pemilik ponsel Lenovo juga bisa memasang dan menggunakannya secara bebas. Bahkan aplikasi ini juga tersedia untuk platform selain, seperti ponsel buatan Apple (iOS) dan Nokia (Windows Phone)

Cara Mengirim dan Menerima File


Untuk mengirim dan menerima file, anda harus meng-install Aplikasi ShareIt di PC dan ponsel anda (Android, iPhone dan Windows Phone). Aplikasi ini bisa anda download di bawah. Jika sudah anda install anda bisa menggunakan aplikasi tersebut untuk mengirim file dengan membuka aplikasi tersebut di masing masing perangkat.

Caranya juga sangat mudah karena interfacenya sangat sederhana. Hanya ada tombol Kirim (Send) dan Terima (Receive). Untuk mengirim file baik dari komputer atau ponsel cukup dengan menekan tombol Send, kemudian pada perangkat yang akan dikirimi file tinggal menekan Receive. Proses pengiriman file ini bahkan berlangsung lebih cepat daripada menggunakan kabel data.

Anda juga bisa mengatur apakah akan menggunakan Jaringan Wi-Fi (Wireless LAN) dimana anda terhubung sekarang atau menggunakan koneksi hostspot privat (wi-fi theter) ponsel anda.
