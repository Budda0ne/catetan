Anda mungkin termasuk yang ingin segera melakukan flashing ROM racikan Microsoft ini. Dan mungkin anda juga bertanya-tanya bagaimana cara meng-install Windows Mobile 10 ke dalam Xiaomi Mi 4 anda.

Agar mempermudah anda dalam melakukan flashing, Microsoft sendiri telah menyediakan tool MiFlash khusus (versi 20150601) untuk melalukan pemasangan ROM Windows Mobile 10. Cara instalasinya sendiri sangat mudah jika menggunakan MiFlash.

Windows Mobile 10 ini HANYA untuk Xiaomi MI4 LTE (4G). Jangan gunakan untuk tipe lain. Resiko hardbrick menanti, meskipun (mungkin, saya belum coba) di flash ulang dengan metode flash via fastboot

Sebelum itu, pastikan juga bahwa anda sudah men-download Full ROM Windows Mobile 10 (952 MB) di bawah bagi anda yang tidak menjadi anggota Windows Insider.

Mempersiapkan MiFlash Tool for Windows Mobile 10


MiFlash khusus ini untuk mempermudah instalasi Windows Mobile 10 pada ponsel Mi 4 LTE. Ingat! Jangan gunakan MiFlash lama anda, karena bisa saja terjadi ketidak-akuratan saat proses flashing. Karena driver Qualcomm juga sudah ter-integrasi dalam MiFlash ini.

Berikut panduan singkat instalasi MiFlash Tool:

  1. Download MiFlash Tool di atas dan ekstrak.
  2. Install MiFlash dengan double klik MiFlash20150601.exe, klik next untuk melanjutkan instalasi.
  3. Jika pada saat instalasi muncul jendela dengan peringatan Windows cant verify the Driver, klik saja Install this driver software anyway.
  4. Jika instalasi sudah selesai, klik tombol Finish

Selanjutnya anda bisa membuka aplikasi MiFlash untuk melakukan flashing.

Melakukan Flashing Windows Mobile 10 dengan MiFlash Tool


Pastikan bahwa driver sudah benar-benar terpasang. Terutama jika anda mempunyai/memasang Nokia recovery tool di PC/Laptop anda. Sebaiknya anda meng-uninstall semua driver yang berhubungan dengan ponsel Lumia anda.

Pada saat anda melakukan flashing, sangat dianjurkan terkoneksi dengan koneksi wi-fi agar tidak memakan paket data anda. Karena kemungkinan akan ada update aplikasi dan sinkronisasi akun Microsoft.
Selanjutnya, ikuti langkah-langkah berikut dengan cermat dan hati-hati.

  1. Ekstrak file 10586.1102.3063.Retail.zip
  2. Pastikan setting tanggal sudah diatur dengan benar, agar aplikasi bawaan bisa ter-install semua.
  3. Aktifkan Developer Options dengan cara masuk menu Settings > About phone > ketuk 5 kali pada MIUI version agar menu USB Debugging bisa diakses.
  4. Aktifkan USB Debugging dengan masuk menu Settings > Other Settings > Developer Options > USB debugging dan aktifkan pilihan USB debugging.
  5. Tancapkan Mi4 ke PC/Laptop dengan kabel data.
  6. Buka folder C:\Program Files (x86)\Xiaomi\MiPhone\Google\Android kemudian tekan [ctrl] + [shift] + klik kanan mouse secara bersamaan pilih Open command promt here.
  7. Saat jendela CMD muncul, ketik perintah adb reboot edl dan tekan enter.
  8. Buka aplikasi MiFlash, klik menu refresh agar Mi4 terdeteksi (port COM xx)
  9. Klik di tombol browse dan arahkan ke folder tempat anda meng-ekstrak ROM (langkah no. 1)
  10. Jika sudah, lakukan flashing dengan menekan tombol flash.
  11. Proses akan berjalan. Tunggu hingga muncul The operation completed successfully.

Kini Xiaomi Mi4 LTE anda sudah terpasang Windows Mobile 10. Setelah selesai melakukan flashing, ponsel tidak otomatis menyala sendiri, tetapi ponsel dalam keadaan mati. Jadi nyalakan secara manual dengan menekan tombol power sampai muncul logo MIUI yang kemudian disusul munculnya logo Windows.

Review Fitur Penting Windows Mobile 10 untuk Mi 4 LTE


Windows Mobile 10 tersedia dalam bahasa Inggris dan bahasa China saja. Melakukan pengaturan umum Windows Mobile 10 tergolong mudah dan berjalan sangat lancar. Pada saat Account Login ke Microsoft account dan pada saat sinkronisasi kontak, email, kalender, one drive, app store dan fitur microsoft lainnya lumayan cepat dan tidak ada masalah. Dapat dikatakan Windows Mobile 10 dapat berjalan dengan baik di Xiaomi MI 4 LTE tanpa ada masalah.

Kamera
Secara umum tidak masalah. Entah jika membandingkan dengan performa kamera bawaan saat menggunakan ROM MIUI. Tetapi menurut saya, hasil kamera sangat bagus. Personally, I use the camera from Windows market - Camera 360 sight, matter of habit

Baterai
Untuk penggunaan baterai hampir sama dengan ROM MIUI. Meskipun aktif menggunakan koneksi 4G LTE, masih bisa bertahan 1 harian. Karena di Windows Mobile tidak ada aplikasi khusus yang membantu menghemat baterai (mis: untuk membatasi penggunaan data secara otomatis, optimasi koneksi wi-fi. dll), cara yang bisa dilakukan hanya dengan melakukan pembatasan proses yang berjalan di latar-belakang melalui pengaturan sistem.

Fitur-fitur lain
Berkenaan dengan kualitas audio, saat menggunakan headphone/headset tergolong istimewa. Namun sedikit lebih pelan jika menggunakan speaker. Mungkin yang sedikit mengganjala adalah tidak ditemukannya fitur equalizer atau mode tertentu untuk headphones (seperti yang ada di ponsel Lumia.
Ada juga fitur-fitur yang tidak bekerja dengan baik seperti:
  • Notifikasi LED - tidak berfungsi.
  • Ir module - tidak berfungsi
  • Glance screen - tidak ada
  • Double touch to wake - tidak ada
Mungkin akan ada penambahan dan pengembangan dari developer MIUI untuk hal-hal tersebut, tetapi entahlah.
Sedangkan fitur yang berfungsi dengan sangat baik diantaranya:
  • GPS sangat cepat dan akurat, kinerjanya lebih baik daripada android.
  • Jaringan WiFi juga memuaskan, tidak tiba-tiba putus atau ngadat tanpa sebab.

Nah, untuk perkara update, Windows Mobile 10 akan diberikan melalui program windows insider atau secara langsung dari pengaturan di pengaturan update (settings > update settings).

Secara umum, ROM ini sudah sangat bagus sebagai permulaan untuk Windows Mobile. Semoga saja, kedepannya fitur-fitur yang tidak berfungsi atau tidak diberikan akan dilengkapi.
