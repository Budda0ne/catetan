Kedua ponsel cerdas ini sudah mendukung teknologi 4G LTE yang booming di Indonesia. Hingga memaksa Operator seluler berlomba-lomba menyediakan layanan data super cepat ini.

Seperti biasa, tak sedap rasanya jika memasak tanpa garam (ups..), maksudnya mempunyai hape android tapi masih perawan (ups lagi). Maksud saya adalah rooting Andromax EC dan ES yang bisa dilakukan tanpa PC. Silahkan root dengan menggunakan Kingroot, pilih versi 4.0 atau diatasnya. Tujuannya tentu agar bisa leluasa meng-obrak-abrik system operasi Android sesuai keinginan. Meskipun toh secara umum performance dari dua ponsel ini cukup memuaskan.

Baru update: Unlock Dual GSM LTE Andromax

Tetapi, dengan melakukan rooting, garansi akan hangus. Selain itu ada resiko rusaknya System yang bisa mengakibatkan bootloop (softbrick) dan permasalahan lainnya. Sedangkan pihak Smartfren juga smart (dengan caranya sendiri) untuk tidak menerima klaim garansi ponsel Andromax yang sudah rooted.

Untuk itu perlu agar anda mempunyai full-backup agar sewaktu waktu bisa melakukan restore Andromax Es dan Es ke kondisi semula. Untuk itulah penting untuk mempunyai Custom Recovery, dalam hal ini adalah TWRP Recovery.

Cara Install TWRP Recovery Andromax EC dan Andromax ES

Silahkan root Andromax EC atau Andromax ES terlebih dulu dan download image TWRP yang dibutuhkan di bawah ini:

Andromax Es

Andromax Ec

Update TWRP 3.0.2.0

Andromax Ec


Jika sudah memiliki hak akses superuser (root) dan file TWRP, anda bisa mengikuti langkah-langkah berikut:

1. Install aplikasi Rashr di Playstore.
2. Letakkan file TWRP di sdcard, ekstrak dan ambil file recovery.img jika file yang anda download berupa file zip.
3. Buka aplikasi Rashr. Saat pertamakali membuka anda akan di minta untuk melakukan back-up kernel (boot.img) dan partisi recovery (recovery.img). Beri tambahan nama Stock di file hasil back-up agar tidak tertukar dengan TWRP.
4. Pilih menu install recovery from sdcard dan pilih file recovery.img (TWRP)
5. Tunggu hingga proses install TWRP selesai.

Jika anda diminta untuk melakukan reboot setelah TWRP Recovery terpasang, sebaiknya pilih reboot to recovery untuk melihat apakah sudah berhasil atau belum.

Jika anda bisa masuk TWRP, saya ucapkan selamat. Segera lakukan back-up full dengan men-centang semua opsi yang tersedia di menu Back-up dan jika punya, segera amankan di PC atau Laptop.
