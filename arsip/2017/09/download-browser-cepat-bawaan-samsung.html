Samsung bagi sebagian orang adalah jaminan mutu. Tak hanya produk hardware-nya, software dari Samsung juga. Salah satu yang saat ini diminati adalah aplikasi browser Samsung Internet yang ternyata juga bisa memblokir konten yang tidak perlu alias memblokir iklan.

Nah, jika Anda ingin mencoba Browser samsunh yang cepat ini, Samsung memperkenalkan Samsung Internet Beta yang bisa di unduh di Playstore yang memang disediakan agar bisa dipasang di semua smartphone Android. Browser ini memberikan akses awal ke fitur terbaru dari mobile browser yang aman, prifat dan dioptimalkan untuk ponsel Samsung. Selain sudah mendukung Android 5.0 keatas juga dilengkapi dengan fitur Keamanan & Privasi sehingga membantu Anda melindungi keamanan dan privasi saat browsing Internet. Dan tentu saja fitur Konten Blocker menjadi andalan Samsung Internet untuk Android sehingga memungkinkan aplikasi pihak ke-3 (bisa di integrasikan dengan baik dengan aplikasi adblock plus dan lainnya) untuk menyediakan filter untuk pemblokiran konten tertentu. Sehingga Anda dapat menelusuri web tanpa konten gangguan iklan yang menyulitkan Anda saat sedang browsing. Untuk fitur blokir iklan ini bisa diaktifkan pada menu ekstensi dan melihat berapa banyak konten yang telah diblokir, dengan interface Content Blocker Status baru.

Sementara di bagian navigasi, Anda bisa menggeser ke kiri atau kanan di bar URL atau toolbar bawah, untuk berpindah ke tab berikutnya atau tab sebelumnya. Anda bisa menambahkan Quick Menu baru dengan mengaktifkan dalam menu Extensions, untukx20akses cepat ke fitur tertentu.

Fitur lain Browser bawaan Samsung ini adalah mode high contrast. Jika membutuhkan peningkatan kontras untuk melihat halaman web lebih nyaman, Modus Kontras Tinggi dapat diaktifkan dalam pengaturan aksesibilitas.

Belanja Online pun lebih aman dan lebih mudah dari sebelumnya dengan dukungan untuk Pembayaran Web API baru, memberikan Anda pilihan untuk memeriksa dan membayar aman dengan hanya beberapa sentuhan. Fitur lainnya: Badge Progressive Web Apps juga Progresif Web Apps adalah aplikasi yang dapat diinstal seperti aplikasi atau digunakan dalam browser. Ketika halaman yang Anda lihat mendukung fitur ini, lencana indikasi akan muncul di bar URL.

Asisten Belanja Amazon dapat diaktifkan untuk memungkinkan Anda untuk mendapatkan penawaran terbaik saat berbelanja di web dengan membandingkan produk dengan Amazon.

Video Asisten memungkinkan Anda beralih di antara berbagai mode tampilan saat menonton video. Pop-up pemain memungkinkan Anda menelusuri web saat menonton video.

Tempatkan telepon ke Gear VR atau Daydream headset ketika Anda melihat halaman web dan akan terbuka di immersive Virtual Reality.

Bisa dikatakan, Samsung Internet adalah versi upgrade dari Browser Chrome karena memang dikembangkan dengan inti Web Browser buatan Google yakni Chrome Browser.

Jika Anda tertarik silahkan unduh browser bawaan Samsung Galaxy S8 ini di sini atau silahkan langsung mengunduhnya dari Playstore untuk versi terbaru.
