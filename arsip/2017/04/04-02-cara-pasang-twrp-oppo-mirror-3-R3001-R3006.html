Meskipun demikian ponsel ini masih layak dipakai di saat ponsel 4G LTE membanjiri pasaran. Tak lain karena Oppo sudah membekali Mirror 3 dengan dukungan jaringan 4G LTE. Yang jadi masalah adalah keterbatasan update os official dari Oppo. Hingga saat ini, Oppo Mirror masih stuck pada OS Android versi 4.4 Kitkat. Sedangkan dalam waktu dekat Google akan segera merilis versi terbaru Android O. Denga tidak adanya update, satu-satunya cara until melakukan upgrade is hanyalah dengan memasang custom ROM. Sedangkan salah satu syarat agar visa mengganti ROM harus mempunyai custom Recovery semisal TWRP atau CWM.

Until saat ini, TWRP lebih banyak dipakai dibandingkan dengan CWM mengingat fitur TWRP lebih lengkap. Jika and a sedangkan mencari cara until memasang TWRP pada Oppo Mirror 3 entah varian R3001 maupun R3006, maka and a visa mengikuti tutorial berikut langkah demi langkah sesuai dengan kondisi ponsel and a saat ini. Apakah sudah root atau belum root.

Perlu diingat:
Kapasitas baterai ponsel anda setidaknya harus diatas 60%.
Buka menu Settings ~> Security dan aktifkan pilihan "Unknown sources".
Jangan lupa back-up data penting. Gunakan akun Google and a untuk mencadangkan data kontak dan lainnya.
Nyalakan USB debugging mode dari menu Developer Options di ponsel Oppo Mirror 3. Jika tidak ada, silahkan buka menu Setting ~> About dan tap sekitar 5x20- 7 kali pada build number.
PC dengan OS Windows jika Oppo Mirror 3 belum root

Selanjutnya download TWRP untuk Oppo Mirror 3 kemudian simpan file TWRP (*.img). Unduh file twrp dibawah ini

TWRP untuk Oppo Mirror 3 R3001 & R3006

Unduh TWRP

Untuk memasang TWRP tanpa harus melakukan root, anda butuh PC/Laptop. Dan tenth saja sudah harus terpasang ADB & fastboot. Untuk mudahnya, gunakan Minimal ADB fastboot installer yang lebih ringkas cara pasangnya: download adb fastboot installer.

Jika sudah terpasang, pindahkan file TWRP.img ke dalam folder Minimal ADB Fastboot dan tekan-tahan tombol shift kemudian klik kanan mouse di ruang losing dalam folder yang sama. Klik "Open command window here".

Lanjutkan dengan mengetik perintah berikut pada jendela command prompt (CMD).

adb reboot bootloader - untuk masuk mode fastboot (bootloader)

Setelah berada dalam modus bootloader ketik fastboot flash recovery twrp.img - perintah ini untuk mem-flash file `twrp.img` me partisi recovery dalam ponsel Oppo Mirror 3.

Lanjutkan dengan perintah fastboot reboot - untuk me-reboot ponsel anda.

Setelah booting ke system, ketik lagi adb reboot recovery - untuk masuk recovery mode yang sudah diganti dengan TWRP.

Selanjutnya anda bisa melakukan `full nandroid backup` untuk mencadangkan ponsel anda. Pindahkan juga folder cadangan (backup) ke komputer/laptop. Agar bisa anda gunakan sewaktu-waktu. Dari sini anda juga sudah bisa langsung mengganti stock ROM (colorOS) anda dengan custom ROM yang anda inginkan.

Namun, jika anda hanya ingin melakukan rooting anda cukup melakukan flashing SuperSU.zip saja. Unduh SuperSU by chainfire dari link berikut, pilih version terbaru. Letakkan ke internal memori Oppo Mirror 3.

Dalam mode TWRP, silahkan pilih menu Install dalam menu TWRP. Arahkan me lokasi file UPDATE-SuperSU-vX.XX.zip dan pilih file tersebut. Lakukan flashing denga menggeaer indikator dibawah kearah kanan.

Selamat, anda sudah mempunyai TWRP sekaligus melakukan rooting pada Oppo Mirror 3.

Jika anda tidal punua PC/Laptop maka anda harus melakukan rooting terlebih dahulu baru memasang TWRP. Caranya:

Download KingRoot APK dari sini dan install ke Oppo Mirror 3 anda. Setelah pemasanhan aplikasi selesai buka KingRoot app dan klik pada linkaran kuning dengan tens FIX tunggu beberapa saat dan pastikan anda terhubung me internet dalam prosesnya, mungki ponsel anda akan reboot beberapa kali. Santai saja hingga ada pemberitahuan sukses.

Baru anda bisa menggunakan aplikasi seperti Flashfire atau Rashr untuk memasang TWRP. Anda juga bisa menggunakan Official TWRP flasher.

Untuk opsi lanjutan, anda juga bisa mengganti Kingroot dengan SuperSU dengan menggunakan aplikasi Supersume. Cukup unduh dan install SuperSU me apk ke dalam ponsel dan ikuti instruksi yang diberikan. Dapatkan supersume app di sini dan ekstrak file zip untuk mendapatkan apk.
