Termasuk ponsel nostalgia yang termasuk "murah" di awal-awal Oppo masuk Indonesia. Seperti banyak ponsel bercipset Mediatek, R1001 yang menggunakan MTK 6572 juga mentok tidak mendapatkan update versi Android nya. Setidaknya, pembuatnya masih memberikan beberapa update Fix bug untuk pembaruan sistem operasi ColorOS.

Mungkin karena ini pula Oppo Joy serinfkali me ngalami mentok di logo saat dinyalakan atau biasa disebut dengan bootloop. Penyebabnya bervariasi. Salah satunya bisa jadi karena penyimpanan internal yang terlalu penuh dan menyebabkan system tidak punya ruang lagi untuk berjalan. Dan akhirnya ponsel pun reboot/restart sendiri terus menerus.

Cara-cara Memperbaiki Bootloop


Untuk memperbaikinya ada dua jalan yang biasa di pakai. Yakni dengan menggunakan PC jika sudah parah dan melaui recovery mode jika bootloop ringan.

Memperbaiki tanpa bantuan PC Komputer


Dengan memanfaatkan recovery mode Oppo bisa diperbaiki dengan mudah. Pertama perlu dicoba untuk melakukan hard reset atau factory reset dengan cara mematikan ponsel terlebih dahulu.

Cara Hard Reset / Factory Reset


Nyalakan kembali Oppo Joy dengan menekan tombol power dan volume min secara bersamaan selama beberapa saat. Tunggu hingga beberapa muncul menu pilihan bahasa. Pilih English dan pilih menu Wipe dan pilih lagi menu Wipe Data. Konfirmasi OK saja dan tunggu beberapa saat. Apabila sudah selesai, pilih saja Reboot System.

Namun adakalanya cara Hard Reset / Factory Reset tidak juga berhasil. Jangan panik masih ada jalan kedua. Jika cara pertama masih belum berhasil berarti ada yang salah dengan partisi system-nya.

Cara Update/Upgrade


Cara kedua menggunakan file Stock rom untuk memperbaiki bootloop. Stock ROM Oppo Joy bisa sekaligus meng-upgrade versi ColorOS sesuai update terbaru yang disediakan Oppo Mobile di situs resminya di sini atau langsung saja unduh file zip ini kemudian simpan ke sdcard.

Selanjutnya masuk ke recovery mode seperti sebelumnya. Pilih saja menu Install dan pilih Install dari Sdcard dan pilih file zip yang sudah di unduh sebelumnya.

Tunggu hingga proses selesai dan reboot ke system jika sudah selesai.

Sudah mencoba dua cara tanpa PC diatas namun masih belum mampu mengatasi bootloop? Atau jangan jangan anda sudah tidak bisa masuk Recovery Mode lagi? Saatnya Anda menggunakan bantuan PC Komputer Untu memperbaiki yang Oppo Joy mentok di logo.

Memperbaiki dengan bantuan SP Flashtool menggunakan PC Komputer


Perlu diperahtikan bahwa cara ini cenderung lebih beresiko daripada cara-cara sebelumnya. Namun patut dicoba. Namun saya peringatkan bahwa: anda tanggung resikonya sendiri jika terjadi apa-apa setelah melakukan flashing.

Persiapan dan Bahan

Pertama, siapkan dulu bahan flashing berikut:

1. Software SP Flashtool, ada beberapa versi termasuk yang terbaru. Namun untuk saat ini saya memakai SP_Flash_Tool_v5.1408.00. Silahkan unduh filenya di sini. Ekstrak di lokasi yang mudah di ingat.

2. Firmware Oppo Joy R1001 yang hanya bisa dipakai menggunakan SP Flashtool. Silahkan unduh di sini. Ekstrak (password: oppo3) juga di lokasi yang sama dengan SP Flashtool.

3. Driver VCOM. Umumnya driver akan otomatis terpasang. Namun jika ponsel tidak terdeteksi, silahkan cari dan unduh dari sini kemudian Install driver.

Jangan lupa untuk charge baterai setidaknya 40% agar ponsel tidak mati di tengah jalan.

Langkah - langkah Flashing Firmware Oppo Joy R1001




1. Matikan ponsel dan cabut baterai terlebih dahulu dan pasang kembali. Hubungkan kabel usb ke komputer dan Ponsel dalam keadaan mati.
2. Sekarang buka folder SP Flashtool dan jalankan spflashtool.exe.
3. Klik tombol Scatter dan buka folder Firmware cari file scatter (.txt), pilih dan klik OK.
4. Sekarang tekan tombol Download. Jika ada error, silahkan ganti dari Download Mode ke Firmware Upgrade.
5. Proses akan berjalan selama beberapa menit. Tunggu hingga penanda proses berubah hijau dan muncul lingkaran hijau (100%) yang menandakan flashing sukses.
6. Cabut Ponsel dari kabel data dan lepas baterai.

Proses flashing Oppo Joy sudah selesai. Pasang lagi Baterai dan nyalakan ponsel. Seharusnya ponsel Anda sudah normal kembali. Namun, jika ada pesan Error saat menggunakan SP Flashtool, silahkan kopi dan paste di kolom komentar.
