Dimana letak test point Xiaomi Redmi 3? Lokasi test point berada di bagian bawah samping lempengan penutup simcard/sdcard.

Untuk lebih jelasnya bisa di lihat pada gambar di bawah

Keterangan: Lakukan short pada 2 titik kuningan yang ditandai menggunakan logam. Bisa dengan pinset lancip atau clipper kertas.

Fungsi dari test point untuk alternatif masuk mode fastboot alias download mode jika redmi 3 tidak bisa masuk mode download/fastboot mode menggunakan tombol Power + Vol -. Bisa juga dipakai saat ingin mengganti rom dari global ke versi china dan sebaliknya atau saat ingin melakukan unbrick Xiaomi Redmi 3
