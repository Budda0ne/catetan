Sehingga muncul peringatan "Memori Penuh" atau ... insufficient space ... padahal Memori Internal (Data ) maupun External (SDcard) masih banyak ruang yang kosong-melompong.

Catatan: Memori yang di maksud di sini adalah kapasitas storage, bukan RAM

Banyak sekali pengguna Smartphone Android yang menanyakan hal tersebut di sosmed (facebook dan twitter bahkan di forum). Anda beruntung karena ada disini, karena dibawah ini akan Saya jelaskan secara detail penyebab dan solusi untuk mengatasi problem memori penuh pada android.

1. Perhatikan Cache Anda dan Bersihkan Secara Berkala.

Cache selalu ada di Android, karena dengan adanya Cache mempercepat Android lsaat membuka aplikasi yang sama yang pernah dibuka sebelumnya. Masalahnya, hal positif ini menjadi sesuatu yang negatif jika file cache terlalu menumpuk. Akibatnya akan menyita kapasitas memori Android anda. Solusinya, anda mesti secara berkala membersihkan file-file-cache ersecbut. Untuk itu anda perlu aplikasi untuk membersihkan file cache tersebut. Anda bisa menggunakan:
App 2 SD atau Link2SD. Dengan dua aplikasi ini Anda dapat menghapus Cache yang ada pada waktu pertama kali membuka aplikasi ini secara manual. Beda dari App 2 SD dan Link2SD adalah Link2SD memerlukan izin super user sehingga bisa digunakan di Android yang sudah di Root-ing. Apa itu Rooting dan fungsinya bisa anda baca di artikel Perbedaan dari Rooting dan Unlock Bootloader
Cache Cleaner. Keunggulan aplikasi ini dibanding dengan aplikasi sebelumnya adalah Anda dapat menyetel jarak waktu secara otomatis untuk menghapus file-file Cache yang telah menumpuk di memori internal.

Cari File Selain Aplikasi dan Game Anda, Pindahkan ke External Memory / SDcard

Untuk mengetahui seberapa banyak file diluar aplikasi maupun game, maka bukalah Setting lalu cari Storage. Jika smartphone Android Anda Jelly Bean dan memiliki sdcard, maka usahakan untuk memindah file-file tersebut dari internal memori ke sdcard dengan jalan pilih transfer data to sdcard. Jika masih Ice Cream Sandwich maka pindahkan file-file tersebut secara manual lewat File Explorer/File Manager.

Selalu Cek File Aplikasi dan Game Anda, Pindah ke SDcard / External Memory

Pakailah aplikasi Link2SD atau App 2 SD, yang dapat Anda download di Playsore, untuk memindahkan file-file tersebut secara aman. Untuk Ice Cream Sandwich dapat dipindahkan ke sd card, jika sudah Jelly Bean dapat dipindahkan antar internal memori yang telah dipartisi. Untuk mempartisi SDcard, anda bisa menggunakan Tutorial Meningkatkan Kapasitas Memori Internal Android

Kontrol Browser Google Chrome Atau Browser yang Lain!
Bila Anda memakai chrome atau browser lain dalam jangka waktu yang lama, maka semakin banyak sekali File Cache dan Data yang mengendap. Agar lebih efektif dan efisien, lakukan penghapusan dengan cara masuk ke opsi setting di chrome, lanjutkan ke Privacy, kemudian Clear browsing data, dalam menu ini ada beberapa opsi untuk memilih apa saja yang ingin Anda hapus.

Bersihkan Folder dan File Sampah Anda!

Setiap Anda meng-uninstall sebuah aplikasi maupun game, biasanya selalu meninggalkan file data ataupun folder yang tidak berguna, yang beberapa diantaranya berisi file yang tidak berguna lagi. Jadi tidak masalah untuk membersihkannya. Anda tidak perlu kuatir, karena ada banyak aplikasi yang dapat mencarinya secara otomatis, supaya anda tidak salah hapus. Aplikasi yang saya rekomendasikan adalah Clean Master. Dengan aplikasi ini Anda dapat memilih clear junk files untuk membersihkan file-file sampah tersebut.

Cek Folder "lost.dir" di Memory Internal dan External Anda

Pada bagian terakhir ini sering luput dari para user android untuk dibersihkan dari memori. Menurut pengalaman Saya sendiri, juga folder lost.dir ini sering kali terlupakan. Kadang, 5 Bagian sebelumnya sudah dilakukan tetapi tetap saja memori penuh. Setelah dan mengecek ukuran folder-folder di memory kapasitas folder lost.dir ini besar sekali, saya hapus saja. Untuk melakukan ini, masuk ke Setting lalu pilih Storage, cari Misc, lalu disitu ada folder lost.dir, pilih lalu delete. Dan memory yang full sebelumnya menjadi lebih lega.

Jika Anda telah mengecek dan melaksanakan semua instruksi untuk mengatasi masalah kapasitas memori penuh di atas, Saya yakin ruang kosong di memori Anda akan lebih lega dari sebelumnya. Berikan komentar anda, dan jika masih tidak menyelesaikan masalah Anda bisa mengajukan pertanyaan anda.
