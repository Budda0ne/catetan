Layanan hosting file handal milik Google ini punya nilai lebih bagi banyak orang, terutama para penghobi download file besar karena kecepatan downloadnya yang luarbiasa cepat dibandingkan hosting lain. Maklumlah, yang punya raksasa teknologi Google.

Meskipun Google Drive merupakan server yang mendukung kecepatan penuh, namun sayangnya, kuota download perharinya dibatasi oleh Google. Kamu pasti pernah mengalami Limit Kuota Download pada saat ingin mengunduh file dari Google Drive bukan? Untuk mengatasinya, ada tips jitu untuk menghapus batasan kuota pada Google Drive. Namun sebelumnya akan dijelaskan dulu kenapa hal ini bisa terjadi.

Tampilan seperti di atas tentu membuat kesal dan pusing sendiri, ya, apalagi kalau kamu sedang buru-buru download game, atau file "tertentu". Munculnya pemberitahuan Sorry, You can"t view or download this file at this time. sering terjadi pada file yang dibagikan oleh situs download. Entah itu situs film, software, atau arsip besar lain yang kebetulan diunggah pada Google Drive. Sebabnya adalah dalam waktu 24 jam banyak sekali yang mengakses file tersebut dan batasan ini pun muncul.

Mengatasi Limit Download Google Drive

Setelah tahu penyebabnya, mari kita lakukan cara mengatasinya bersama-sama.

CATATAN: Penting! Syarat pertama, kamu harus memiliki akun Gmail dan sudah terdaftar memiliki akun Google Drive (akun gratis disediakan ruang 15 GB)

Langkah 1: Memisah dan mengganti URL

Pada halaman peringatan limit kuota google drive kerika kamu akan mengunduh file, perhatikan URL atau link pada address bar. Sebagai contohnya adalah file unduhan yang dibagikan oleh situs Bagas31 yang akan terlihat seperti ini:

https://drive.google.com/uc?id=1jv3nL5ZjUf7sUJFtlHJhsvA0DZkbZgmp&export=download

Letakkan/klik di kolom url browser yang kamu pakai dan lerakkan pada bagian &export=download hapus.Kemudian letakkan pada uc dan ganti dengan open sehingga link-nya akan menjadi:

https://drive.google.com/open?id=1jv3nL5ZjUf7sUJFtlHJhsvA0DZkbZgmp

Perhatikan gambar dibawah ini:

Langkah 2: Kloning ke akun pribadi

Saat ini akan muncul halaman baru yang berbeda dari sebelumnya. Perhatikan baik-baik pada bagian pojok kanan atas layar, Klik Logo Google Drive dengan tanda +.

Langkah 3: Copy File

Setelah itu, logo nya akan berubah menjadi kotak seperti folder, lalu klik My Drive.

Langkah 4: Membagikan File

Setelah itu, kamu akan diarahkan ke akun Google Drive kamu. Di sana akan terdapat file yang ingin kamu download. Klik kanan pada file tersebut, kemudian klik “Make a Copy” / “Buat Salinan”.

Langkah 5: Download salinan file

Akan muncul file copy-an (salinannya), klik kanan pada file tersebut, lalu klik “Download”

Akan muncul peringatan, klik “Download Anyway” untuk melanjutkan.

Selamat, kamu berhasil mendownload file nya tanpa limit kuota lagi.

PENTING: Setelah proses download selesai, jangan lupa menghapus file salinan dari Google Drive kamu dan pastikan terhapus juga di folder Trash agar kamu bisa mengunduh file lain yang kebetulan kena limit kuota. Semoga Bermanfaat
