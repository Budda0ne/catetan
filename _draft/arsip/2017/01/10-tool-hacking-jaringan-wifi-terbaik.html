Postingan ini hanya untuk merangkum beberapa (10 saja) tool yang paling banyak dipakai untuk WiFi hacking dan juga paling sering dicari di Internet.


Kami tidak bertanggung jawab atas segala kerusakan yang anda (sebagai pengguna) alami dan atau anda sebabkan dengan menggunakan WiFi hacking tools dalam daftar ini. Pastikan anda menggunakan tool ini untuk pembelajaran atau untu ujicoba keamanan di jaringan anda sendiri.
10 tool hacking jaringan WiFi terbaik dan populer dipakai oleh praktisi bidang keamanan tersebut antara lain:

1. Aircrack-ng


Aircrack-ng merupakan generasi lanjutan darix20Aircrack dengan banyak tambahan fitur baru yang banyak dipakai oleh para hacker/cracker untuk membobol jaringan WiFi. Aircrack-ng sendir adalah sebuah program cracking password koneksi WiFi dengan melakukan recovery kunci AP yang menggunakan sistem keamanan 802.11 WPA-PSK and WEP.

Cara kerja Aircrack-ng adalah dengan melakukan crack-ing kunci WEP dengan FMS attack, PTW attack, dan dictionary attacks. Sedangkan untuk enkripsi WPA, dengan menggunakan dictionary attack saja.

2. AirSnort


AirSnort mendukung OS Windows dan Linux layak jadi tool cracking WiFi terpopuler. Tool AirSnort menjadi paling populer digunakan untuk men-decrypting enkripsi WEP pada jaringan Wi-Fi 802.11b.

Sayangnya tool ini sudah tidak tidak di-update oleh pengembangnya. Meskipun tool ini tidak update lagi, tetap saja alat ini masih layak pakai mengingat tidak semua jaringan WiFi saat ini bebas celah. Anda bisa masih bisa mendownload gratis AirSnort dari situs sourceforge.

3.Kismet


Kismet juga salah satu dari software yang bagus untuk pendeteksi jaringan, dan bisa melakukan packet sniffing pada jaringan 802.11 semua lapisan (layer a/b/g/n). Lebih canggih dari AirSnort, software Kismet mendukung platform Linux, OSX, Windows juga platform BSD.

Kismet melakukan identifikasi dengan mengumpulkan paket data yang diterima dan mendeteksi nama jaringan standar juga jaringan yang tersembunyi (SSID-nya tidak di broadcast).

4. Cain and Able


Cain and Able adalah yang terbaik, sangat direkomendasikan dan juga tak kalah populer dibanding 3 tool ha*king wifi sebelumnya. Umumnya digunakan untuk melakukan sniffing jaringan, cracking sandi ter-enkripsi.

Anda bisa menggunakan Cain and Cable dengan kamus kata sandi (bisa anda cari secara online dengan kata sandi "encrypted passwords dictionary"), brute-force dan cryptanalysis attacks dan banyak fitur lain.

Tool ini juga mampu menemukan snadi jaringan wireless dengan meng-analisa protokol jalur yang digunakan dalam AP WiFi.

5.WireShark


Wireshark merupakan packet analyzer gratis dan dikembangkan secara open-source. Mampu merekam secara live data dari ethernet, IEEE 802.11, ATM, Bluetooth, USB dan jenis koneksi lainnya.

Alat ini termasuk salah tool-tool hacking favorit untuk 2017. Karena kelengkapan fiturnya. Selain iti, Wireshark juga mendukung platform Linux, Windows, OSX, Solaries, FreeBSD dan lainnya yang mejadikannya menjadi tool hacking paling lengkap.

6.CommView for Wi-Fi


CommView for WiFi juga merupakan wireless analyzer dan network monitor paling bertenaga . bisa digunakan di berbagai lapisan freq 802.11 (lapisan jaringan a/b/g/n/ac).

Software ini melakukan scanning pada terminal WiFi dan access points jaringan wireless. CommView bisa melihat berbagai data statistik koneksi IP: alamat IP, ports, sessions, dan lainnya.

Kekurangannya hanya mendukung platform Windows. Menjadikannya tool hacking wifi khusus OS Windows versi 7/8/8.1/10 baik 64 maupun 32 bit.

7. Airjack


AirJack sebenarnya adalah device driver (atau kumpulan device driver) untuk 802.11(a/b/g) injeksi dan pengambilan data frame mentah (raw frame). Tujuan aslinya adalah digunakan dalam melakukan pengembangan aplikasi yang yanh membutuhkan akses komunikasi data mentah (raw protokol) jaringan 802.11.

8. inSSIDer


inSSIDer, juga termasuk tool paling banyak dipakai. Berguna untuk menampilkan alamat fisik (MAC Address) seluruhx20wireless hotspot’s di sekitar, menampilkan enkripsi yang dipakai, kekuatan sinyal dari AP dan channel yang dipakai oleh.

inSSIDer adalah software berbayar dengan harganya sekitar $19.99. Sebelumnya Software ini mendapatkan gelar “Best Opensource Software in Networking” saat masih dikembangkan secara opensource.

9. WepAttack


Wepattack is an open source Linux tool for 802.11WEP keys. This tool is based on dictionary attack on WEP keys in WLAN networks.

10.NetStumbler


Terkhir dalam daftar ini, NetStumbler juga biasa disebut sebagai Network Stumbler sigunakan dalam platform Windows untuk mendeteksi Wireless LAN yang memakai jarinhan 802.11 b/a/g.

Versi ringan dari software ini juga tersedia dengan nama MiniStumbler.

Selain tool-tool hacking/cracking WiFi yang sudah diaebutkan di atas masih banyak lagi tool lainnya. Namun untuk saat ini cukup 10 saja sesuai judul.
