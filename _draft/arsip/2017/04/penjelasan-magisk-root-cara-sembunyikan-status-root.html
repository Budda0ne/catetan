Magisk Root merupakan alternatif terbaik dari SuperSu yang dikembangkan oleh Chainfire. Framework Magisk sendiri dikembangkan oleh developer yang juga anggota forum XDA-developers, topjohnwu. Metode pengelolaan Root pada Magisk bisa dikatakan lebih baik dari SuperSu dalam berbagai aspek. Bahkan banyak pengembang ROM sekarang memasukkan Magisk di dalam ROM, sebut saja Ressurrection Remix. Anda dapat menyembunyikan status "root" dari aplikasi yang tidak mau berjalan ketika ponsel sudah dalam keadaan root. Misalkan saja pada aplikasi mobile banking. Jika ponsel Anda yang sudah root menggunakan SuperSu tanpa Magisk, ketika Anda memasang aplikasi perbankan biasanya, tiba-tiba aplikasi perbankan tersebut menutup sendiri secara otomatis disertai munculnya peringatan bahwa "Aplikasi ini tidak dapat berjalan di perangkat berakar".

Sekarang apa yang akan anda lakukan? Mau tidak mau, tentu melakukan unroot ponsel Anda agar bisa menjalankan aplikasi tersebut. Apalagi jika Anda sangat membutuhkan aplikasi ini.

Contoh lainnya adalah AndroidPay. AndroidPay tidak akan pernah bisa dipakai pada perangkat yang sudah root. Selain itu, ada juga game populer seperti Pokemon Go yang juga menolak untuk dijalankan pada rooted device, untuk menghindari pemain curang.

Apakah ada solusi lain agar tetap bisa menggunakan aplikasi perbankan, AndroidPay, atau game Pokemon Go dalam keadaan root? Jawabannya, ada. Yakni dengan menggunakan Magisk sebagai wadah untuk menampung root dan berbagai modifikasi system lainnya. Dalam artikel kali ini saya akan memberitahukan bagaimana cara kerja magisk dan juga cara menggunakannya secara dengan se jelas-jelasnya.

Membandingkan Root MagiskSu dengan SuperSu

Magisk, tidak merubah file sistem anda. Maksudnya tidak merubah file apapun yang ada di dalam partisi System sehingga tetap perawan. Metode root MagiskSu tidak menambahkan file su binari di partisi sistem. Hal ini biasanya disebut sebagai "systemless". Sedangkan SuperSu (reguler) mengubah partisi sistem Anda dan menambahkan beberapa file dalam partisi sistem untuk mendapatkan akses root, meskipun pada beberapa device sudah bisa menggunakan SuperSu systemless, terutama untuk device lollipop keatas.

Kelebihan dan Keuntungan menggunakan Root Magisk

Seperti yang sudah saya jabarkan diatas, jika ponsel android Anda root dengan Magisk, Anda tetap dapat menjalankan hampir semua aplikasi yang tidak dapat berjalan jika ponsel sudah rooted.

  • Anda masih bisa menjalankan aplikasi perbankan.
  • Bisa memainkan Pokemon Go (jika menyembunyikan status root)
  • Anda bisa menggunakan Snapchat.
  • Anda dapat menginstal pembaruan OTA jika dalam stock ROM.
  • Instal systemless Xposed Framework dari Module yang disediakan dalam menu Magisk.
  • Bypass Safetynet agar bisa menggunakan Android Pay.

Lalu, bagaimana cara kerja Magisk hingga bisa menghindari deteksi root?

Magisk sendiri bekerja pada perangkat yang menjalankan Android Lollipop keatas. Seperti apa cara kerjanya? Ada dua komponen yang dibutuhkan agar Magisk agar bisa berjalan. Pertama adalah Magisk Installer berupa file zip dan kedua adalah Magisk App atau Magisk Manager. 

Pertama-tama kita perlu melakukan flashinh file zip Magisk Installer melalui TWRP yang memodifikasi boot image ponsel dan membuat file magisk.img di partisi /data yang di mount ke direktori root smartphone Anda. Magisk juga membuat file yang diperlukan pada partisi /cache dan /data selama proses pemasangan. Jadi tidak ada perubahan pada partisi /system. 

Tapi, perlu anda perhatikan bahwa jika sudah melakukan root dengan SuperSu by Chainfire maka Magisk Installer akan mencoba untuk menghapus semua file yang berkaitan dengan SuperSu dari partisi /system. Magisk juga akan menghapus file binari SU dan beberapa file dependensi SuperSU.

Selanjutnya yang kita butuhkan adalah aplikasi Magisk Manager. Berfungsi seperti aplikasi SuperSU yang digunakan untuk mengelola perizinan root user. Magisk Manager memiliki lebih banyak fitur. Dalam aplikasi Magisk Anda akan menemukan beberapa pengaturan tambahan yang bisa Anda jelajahi. Lebih jelasnya, sebagian fitur bisa anda lihat beberapa tangkapan layar di bawah ini.

Bagaimana Cara Memasang Magisk di Android?

Ada dua cara untuk memasang Magisk, yakni:

1. Langsung menggunakan TWRP untuk flashing zip Magisk Installer, baru kemudian memasang aplikasi Magisk.

2. Menggunakan bantuan Magisk Manager yang bisa Anda unduh dari Google Play. Prosesnya lebih mudah tapi tetap butuh TWRP. Karena aplikasi Magisk akan reboot ke recovery secara otomatis dan mem-flash file zip dengan TWRP.

Langkahnya cukup sederhana, jika Anda sudah tahu tentang cara pakai TWRP dan terbiasa dengan flashing ROM.

Langkah Flashing & Install Magisk

Berikut ini ada dua cara yang bisa dipilih dalam memasang Magisk Systemless Interface.

Cara 1: memasang Magisk secara langsung

  1. Reboot ke recovery mode (TWRP), pilih menu Install
  2. Cari file zip Magisk Installer yang sudah anda unduh sebelumnya.
  3. Konfirmasi proses flashing dengan menggeser indikator ke arah kanan.
  4. Tunggu hingga proses selesai
  5. Sebaiknya pilih wipe cache dan dalvik sebelum reboot to system.
  6. Sekarang instal aplikasi magisk dari Play Store dan buka Magis Manager untuk melihat apakah root sudah berhasil. Anda akan melihat tanda centang hijau di aplikasi jika Anda berhasil root.

Cara 2: Memasang Magisk dengan bantuan Magisk Manager.

Cara ini kebalikan dari cara 1.

  1. Download Magisk Manager dari Google Play.
  2. Buka Magisk Manager dan gesek dari kiri untuk membuka menu dan klik menu "Install".
  3. Aplikasi Magisk secara otomatis akan mendeteksi lokasi partisi /boot perangkat Anda, lanjut dengan mengklik "Download dan Install" .
  4. Sebuah pop-up akan muncul, lanjut dengan klik "Install". File zip akan diunduh dari repo magisk dan kemudian aplikasi secara otomatis membuat script recovery.
  5. Magisk akan me-reboot otomatis dan proses instalasi zip berjalan. Jika pemasangan sukses, reboot untuk mengaktifkan root magisk.

Jika perangkat Anda stuck di logo karena memasang magisk, silahkan rebot kembali ke mode TWRP dan flash fila "Magisk-Unistaller" yang terletak pada folder "MagiskManager" di penyimpanan internal.

Meskipun cara kedua lebih mudah, saya lebih merekomendasikan cara pertama.

Cara Menyembunyikan Status Root dari Deteksi Aplikasi

Selanjutnya, untuk menyembunyikan status root, silahkan ikuti langkah di bawah ini:

  1. Buka Aplikasi Magisk dan Klik menu Pengaturan (Setting).
  2. Aktifkan pilihan Magisk Hide dan tekan tombol back
  3. Anda sekarang akan melihat menu Magisk Hide di bilah navigasi samping.
  4. Klik menu Hide dan Pilih aplikasi dari daftar aplikasi yang ingin Anda sembunyikan.

Ada fitur lain yang bisa didapat menggunakan Magisk semisal sytemless file hosts yang bisa dipakai untuk memblokir iklan, Xposed Framework systemless, dan juga mod audio seoerti ViperAudio atau Dolby Atmos tanpa merubah partisi system.

Fitur tambahan disertakan sebagai modul tambahan yang bisa di unduh dari menu Module. Untuk fitur tambahan akan dibahas pada kesempatan lain.
