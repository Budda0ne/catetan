Cara Cepat Mudah Rooting
Sekarang akan saya tunjuk-kan cara paling cepat dan paling mudah untuk melakukan rooting pada smartphone Evercoss A7V anda tanpa PC. Tapi saya ingatkan: dengan mengikuti tutorial yag saya berikan di blog ini, saya anggap anda setuju dengan disclaimer tutorial Android ini dan setuju untuk mematuhi dan memahami ketentuan hak cipta blog ini.

Cara root untuk Evercoss A7V sangatlah mudah. Prses rooting tanpa menggunakan PC, jadi langsung dari handheld Anda.

Sebelum melakukann rooting, download dulu aplikasi Root Master di bagian bawah postingan. (Jika anda mau, anda bisa menggunakan Framaroot 1.9.3, tetapi saya tidak menjamin berhasil)

Step by Step untuk melakukan rooting Evercoss A7V:

  • Buka menu Pengaturan -> Pengaturan Keamanan dan scroll kebawah sampai anda melihat Sumber tak dikenal dan Centang/Contreng.
  • Sesudah mendownload Root Master segera install Root Master, jika muncul peringatan membahayakan abaikan dan Install saja.
  • Setelah selesai, buka Aplikasi Root Master.
  • Untuk melakukan rooting, pilih menu Root
  • Lanjutkan dengan memilih warna merah dengan tulisan China.
  • Centang/contreng/cek di pilihan HS, lihat SuperSu dan pilih.
  • Pilih Continue dan lanjutkan dengan memilih Normal

Papparappa... Evercross A7V anda sudah root. untuk mengecek apakah Evercocc A7V sudah benar-benar root atau belum, gunakan aplikasi Root Checker yang bisa anda download di playstore.

Untuk berjaga-jaga jika anda melakukan kesalahan dan mengalami bootloop atau kerusakan system yang lain, anda sebaiknya mempunyai CWM Recovery yang bisa anda gunakan untuk membackup ROM Evercross A7V anda. Baca tutorial install CWM Recovery untuk A7V.
