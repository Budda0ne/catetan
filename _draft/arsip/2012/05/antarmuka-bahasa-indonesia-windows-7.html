Paket Antarmuka Bahasa Bahasa Indonesia Windows 7 ini menyediakan versi terjemahan parsial untuk bagian Windows yang paling umum digunakan. Setelah memasang LIP (Paket Antarmuka Bahasa), maka teks di dalam Wisaya (Jendala Munculan, pen.), Kotak Dialog, Menu, dan Topik Bantuan dan Dukungan akan ditampilkan dalam Bahasa Indonesia oleh Antarmuka Bahasa Indonesia Windows 7 (Indonesian LIP).
Jika ada kata/kalimat yang belum diterjemahkan, akan di tampilkan dalam bahasa dasar Windows 7. Contohnya, jika kamu membeli Windows 7 versi bahasa Spanyol :) , dan memasang LIP Bahasa Indonesia, beberapa teks akan tetap dalam bahasa Spanyol. Kamu juga bisa memasang lebih dari satu LIP pada bahasa dasar tunggal. Paket Antar Muka Bahasa Windows ini dapat dipasangkan ke semua edisi Windows 7.

Sedikit ngelantur sebelum men-download LIP

Ketika saya menuliskan artikel ini, dan menyelesaikan paragraf di atas, sebenarnya saya belum mencoba memasangnya di Windows 7 saya. Ada bayangan yang muncul di benak saya "Jangan-jangan saya malah jadi pusing ketika berhadapan dengan istilah-istilah windows/teknologi dalam Bahasa Indonesia?" Hahahaha...

Maklum, sejak pertamakali menggunakan Windows (98) dulu, saya memang tidak pernah menggunakan Paket Antarmuka Bahasa Indonesia, kecuali sekali sa'at mencoba menginstall Paket Antarmuka Bahasa Indonesia di Windows XP. Itupun hanya bertahan setengah hari. :)

Download dan Petunjuk Antarmuka Bahasa Indonesia Windows 7 (LIP)
Persyaratan Sistem (System Requirement)
  • Sistem Operasi yang Didukung: Windows 7
  • Pemasangan bahasa dasar yang diperlukan oleh Windows 7: Inggris
  • 4.63 Mb ruang bebas untuk Download
  • 15 Mb ruang bebas untuk konfigurasi (penyetelan)

Petunjuk Peng-Install-an

PERINGATAN:
Jika Kamu menggunakan enkripsi BitLocker yang sudah diaktifkan, silahkan mematikannya sebelum memulai pemasangan LIP. 

Caranya:
Buka Control Panel, pilih System and Security, kemudian BitLocker Drive Encryption. Klik Suspend Protection.

Karena ada pengunduhan (download, pen) yang terpisah untuk Windows 7 versi 32-bit danx2064-bit, sebelum kamu mulai men-download, kamu harus mengecek versix20Windows 7 yang telah terpasang.
Cara menentukan versix20Windows 7 yang telah terpasang:

Klik tombol Start kemudian klik-kanan pada komputer dan pilih Properties. Ini akan menampilkan informasi dasar dari komputer Anda.
Lihat di seksi Sistem untuk tipe Sistem. Ini akan mengindikasi jika Pengoperasian Sistem Windows 7 Anda adalah Sistem Pengoperasian 32-bit atau Sistem Pengoperasian 64-bit.

Untuk memasang versi 32-bit, lakukan dengan cara:
  1. Klik teks Download, lalu klik Open untuk memasang LIP
    2. atau
  2. Klik teks tombol Download
    • Klik Save untuk menyimpan file di komputer,
    • Cari lokasi filenya kemudian klik dua kali pada file unduhan untuk memasang LIP
Untuk memasang versi 64-bit, cara yang dilakukan juga sama.

Download LIP_id-ID-32bit.mlc [Indonesian LIP u/ Sistem 32-bit]
Download LIP_id-ID-64bit.mlc [Indonesian LIP u/ Sistem 64-bit]
